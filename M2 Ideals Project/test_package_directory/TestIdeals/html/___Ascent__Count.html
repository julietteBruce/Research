TestIdeals :: AscentCount

AscentCount -- an option for ascendIdeal

Description

AscentCount is an option for ascendIdeal, that tells the function to return the number of steps it took before the ascent of the ideal stabilized.

Functions with optional argument named AscentCount :

  • "ascendIdeal(...,AscentCount=>...)" -- see ascendIdeal -- find the smallest ideal containing a given ideal which is compatible with a given Cartier linear map

For the programmer

The object AscentCount is a symbol.
