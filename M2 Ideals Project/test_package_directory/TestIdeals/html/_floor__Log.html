floorLog -- floor of a logarithm

Synopsis

  • Usage:
    floorLog(b, x)
  • Inputs:
    • b, a number, greater than 1; the base of the logarithm
    • x, a number, positive
  • Outputs:
    • an integer, the floor of log_b(x)

Description

floorLog(b, x) computes floor(log_b x), correcting occasional errors due to rounding.

i1 : floor(log_3 3^5)

o1 = 4
 
i2 : floorLog(3, 3^5)

o2 = 5

Ways to use floorLog :

  • "floorLog(Number,Number)"

For the programmer

The object floorLog is a method function.
