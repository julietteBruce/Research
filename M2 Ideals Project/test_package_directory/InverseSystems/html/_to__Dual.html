InverseSystems :: toDual

toDual -- finds the inverse system to an ideal up to a given degree

Synopsis

  • Usage:
    M = toDual (ZZ, I)
  • Inputs:
    • I, an ideal, in a standard graded polynomial ring
  • Optional inputs:
    • DividedPowers => ..., default value true
  • Outputs:
    • M, a matrix, interpreted as list of generators of inverse system to I

Description

If I is an ideal, then toDual(d,I) = inverseSystem(d, I). See the documentation of InverseSystems for the basic theory and of inverseSystem for the function.

If I is a matrix, then the code first replaces it with the ideal generated by the entries.

Caveat

The Base ring of the input must be a standard graded polynomial ring over a field.

See also

  • InverseSystems -- Macaulay's Inverse Systems
  • inverseSystem -- Inverse systems with equivariance
  • fromDual -- Ideal from inverse system

Ways to use toDual :

  • "toDual(ZZ,Ideal)"
  • "toDual(ZZ,Matrix)"

For the programmer

The object toDual is a method function with options.
