Graphs :: deleteEdges

deleteEdges -- Deletes a list of edges from a graph

Synopsis

  • Usage:
    G' = deleteEdges(G,E)
  • Inputs:
    • G, a graph,
    • E, a list,
  • Outputs:
    • G', a graph, the graph resulting from deleting edges in list E from graph G

Description

This method deletes the specified edges from the graph, but preserves the original vertex set of the graph, and outputs the adjusted graph.

i1 : G = cycleGraph 10;
 
i2 : deleteEdges(G,{{1,2},{3,4},{7,8}})

o2 = Graph{0 => {1, 9}}
           1 => {0}
           2 => {3}
           3 => {2}
           4 => {5}
           5 => {4, 6}
           6 => {5, 7}
           7 => {6}
           8 => {9}
           9 => {0, 8}

o2 : Graph
