Matroids :: isNonCrossing

isNonCrossing -- whether 2 subsets are non-crossing

Synopsis

  • Usage:
    isNonCrossing(A, B)
  • Inputs:
    • A, a list,
    • B, a list,
  • Outputs:
    • a Boolean value, whether A and B are non-crossing subsets of [n]

Description

This function determines whether two disjoint subsets of [n] are non-crossing.

i1 : isNonCrossing({0,1,7},{2,4,5})

o1 = true
 
i2 : isNonCrossing({2,3,6},{1,4})

o2 = false

See also

  • isPositivelyOriented -- whether a matroid is positively oriented

Ways to use isNonCrossing :

  • "isNonCrossing(List,List)"
  • "isNonCrossing(Set,Set)"

For the programmer
