Hadamard :: Point

Point -- a new type for points in projective space

Description

A point in projective space is represented as an object in the class Point. An element of this class is a BasicList.

Methods that use an object of class Point :

  • Point * Point -- entrywise product of two projective points
  • Point == Point -- check equality of two projective points

For the programmer

The object Point is a type, with ancestor classes BasicList < Thing.
