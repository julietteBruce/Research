Hadamard :: point

point -- constructs a projective point from the list (or array) of coordinates.

Synopsis

  • Usage:
    point(L)
  • Inputs:
    • L, a list, or array or visible list
  • Outputs:
    • an instance of the type Point,

Description

i1 : point {1,2,3}

o1 = Point{1, 2, 3}

o1 : Point
 
i2 : point [1,4,6]

o2 = Point{1, 4, 6}

o2 : Point

Ways to use point :

  • "point(VisibleList)"

For the programmer

The object point is a method function.
