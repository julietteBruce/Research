Hadamard -- a package to study Hadamard products of varieties.

Author

Version

This documentation describes version 0.1 of Hadamard.

Exports

    • Point -- a new type for points in projective space
  • Functions and commands
    • hadamardPower -- computes the Hadamard powers of varieties
    • hadamardProduct -- computes the Hadamard product of varieties
    • idealOfProjectivePoints -- computes the ideal of set of points
