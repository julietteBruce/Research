EllipticCurves :: toShortWForm

toShortWForm -- A method to transform an elliptic curve from Weierstrass form to short Weierstrass form.

Synopsis

  • Usage:
    toShortWForm(E)
  • Inputs:
    • E

Description

i1 : E=ellCurve({1,2,3,4,6},GF(29))

o1 = Elliptic Curve defined by Y^2 + 1 XY + 3 Y = X^3 + 2 X^2 + 4 X + 6( over(GF 29 .))

o1 : EllipticCurveW
 
i2 : toShortWForm(E)

o2 = Elliptic Curve defined by Y^2 = X^3 + 11 X + (-4)( over(GF 29 .))

o2 : EllipticCurve

Ways to use toShortWForm :

  • "toShortWForm(EllipticCurveW)"

For the programmer

The object toShortWForm is a method function.
