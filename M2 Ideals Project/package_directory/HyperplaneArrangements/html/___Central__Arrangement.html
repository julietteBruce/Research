HyperplaneArrangements :: CentralArrangement

CentralArrangement -- the class of all central hyperplane arrangements

Description

A central arrangement is a finite set of linear hyperplanes. In other words, each hyperplane passes through the origin.

Functions and methods returning a central hyperplane arrangement :

  • cone(Arrangement,RingElement) -- creates an associated central hyperplane arrangement
