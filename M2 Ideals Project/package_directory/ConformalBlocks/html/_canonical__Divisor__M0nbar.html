ConformalBlocks :: canonicalDivisorM0nbar

canonicalDivisorM0nbar -- returns the class of the canonical divisor on the moduli space of stable n-pointed genus 0 curves

Synopsis

  • Usage:
    canonicalDivisorM0nbar(14)
  • Inputs:
    • n, an integer,
  • Outputs:
    • K, an instance of the type SymmetricDivisorM0nbar,

Description

This function returns the class of the canonical divisor $K$ on the moduli space $\bar{M}_{0,n}$. See e.g. [KM] for a formula for $K$ in the standard basis.
