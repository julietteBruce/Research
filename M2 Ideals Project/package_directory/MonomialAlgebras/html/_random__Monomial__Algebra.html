MonomialAlgebras :: randomMonomialAlgebra

randomMonomialAlgebra -- Generate random monomial algebra.

Synopsis

  • Usage:
    randomMonomialAlgebra(a,d,c)
  • Inputs:
    • a, an integer, positive
    • d, an integer, positive
    • c, an integer, positive
  • Optional inputs:
    • Num => ..., default value 1, Option of randomSemigroup and randomMonomialAlgebra to return a list of several semigroups.
    • SetSeed => ..., default value false, Option to set the random seed for randomSemigroup and randomMonomialAlgebra.
    • Simplicial => ..., default value false, Option of randomSemigroup and randomMonomialAlgebra to return a simplicial semigroup.
  • Outputs:
    • a MonomialAlgebra, or a list of those if the option Num is bigger or equal to 2.

Description

Generate a random monomial algebra such that the semigroup generated by the degrees is in \mathbb{N}^d of full dimension with coordinate sum a and codimension c.

Note that the random number generator can be controlled by the command setRandomSeed. Alternatively you can also use the option SetSeed.

The option Num can be used to return a list of semigroups.

The option Simplicial can be used to return a simplicial semigroup such that the standard vectors a*e_i are among the Hilbert basis.
