MonomialAlgebras :: net(MonomialAlgebra)

net(MonomialAlgebra) -- Pretty print for monomial algebras

Synopsis

  • Function: net
  • Usage:
    net M
  • Inputs:
    • M, a MonomialAlgebra,
  • Outputs:
    • a net,

Description

Pretty print for monomial algebras. Prints the associated polynomial ring of M.

i1 : M = monomialAlgebra {1,4,8,9,11}

      ZZ
o1 = ---[x ..x ]
     101  0   5

o1 : MonomialAlgebra generated by {{11, 0}, {0, 11}, {1, 10}, {4, 7}, {8, 3}, {9, 2}}
 
i2 : net M

      ZZ
o2 = ---[x ..x ]
     101  0   5
