MonomialAlgebras :: isGorensteinMA

isGorensteinMA -- Test whether a simplicial monomial algebra is Gorenstein.

Synopsis

  • Usage:
    isGorensteinMA R
    isGorensteinMA B
    isGorensteinMA M
  • Inputs:
    • R, a polynomial ring, with B = degrees R and K = coefficientRing R, or
    • B, a list, with the generators of an affine semigroup in \mathbb{N}^d.
    • M, a MonomialAlgebra,
  • Outputs:
    • a Boolean value,

Description

Test whether the simplicial monomial algebra K[B] is Gorenstein.

Note that this condition does not depend on K.

i1 : R=QQ[x_0..x_2,Degrees=>{{2, 0}, {0, 2}, {1, 1}}]

o1 = R

o1 : PolynomialRing
 
i2 : isGorensteinMA R

o2 = true
