Divisor :: cleanSupport

cleanSupport -- removes primes with coefficient zero from a divisor

Synopsis

  • Usage:
    cleanSupport( D1 )
  • Inputs:
    • D1, an instance of the type BasicDivisor,
  • Outputs:
    • an instance of the type BasicDivisor,

Description

This function returns a divisor where all entries with coefficient zero are removed.

i1 : R = QQ[x,y,z];
 
i2 : D = divisor({1,0,-2}, {ideal(x), ideal(y), ideal(z)})

o2 = 0*Div(y) + -2*Div(z) + Div(x)

o2 : WeilDivisor on R
 
i3 : cleanSupport(D)

o3 = -2*Div(z) + Div(x)

o3 : WeilDivisor on R

Ways to use cleanSupport :

  • "cleanSupport(BasicDivisor)"

For the programmer

The object cleanSupport is a method function.
