Divisor :: toQWeilDivisor

toQWeilDivisor -- create a Q-Weil divisor from a Weil divisor

Synopsis

  • Usage:
    toQWeilDivisor( D )
  • Inputs:
    • D, an instance of the type QWeilDivisor,
  • Outputs:
    • an instance of the type QWeilDivisor,

Description

Turn a Weil divisor into a Q-divisor (or do nothing to a Q-divisor).

i1 : R = ZZ/5[x, y];
 
i2 : D = divisor(x);

o2 : WeilDivisor on R
 
i3 : E = toQWeilDivisor(D)

o3 = Div(x)

o3 : QWeilDivisor on R
 
i4 : toQWeilDivisor(E)

o4 = Div(x)

o4 : QWeilDivisor on R
 
i5 : F = divisor({3, 0, -2}, {ideal(x), ideal(y), ideal(x+y)})

o5 = -2*Div(x+y) + 3*Div(x) + 0*Div(y)

o5 : WeilDivisor on R

See also

  • toWeilDivisor -- create a Weil divisor from a Q or R-divisor
  • toRWeilDivisor -- create a R-divisor from a Q or Weil divisor

Ways to use toQWeilDivisor :

  • "toQWeilDivisor(QWeilDivisor)"
  • "toQWeilDivisor(WeilDivisor)"

For the programmer

The object toQWeilDivisor is a method function.
