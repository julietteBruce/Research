Divisor :: MTries

MTries -- an option used by embedAsIdeal how many times to try embedding the module as an ideal in a random way.

Description

After making some canonical attempts, embedAsIdeal tries to embed the module into a ring as an ideal in a random way. The value of this option is how many times that random embedding is attempted. The default value is 10.

Functions with optional argument named MTries :

  • "embedAsIdeal(...,MTries=>...)" -- see embedAsIdeal -- embed a module as an ideal of a ring

For the programmer

The object MTries is a symbol.
