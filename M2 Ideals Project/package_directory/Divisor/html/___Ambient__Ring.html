Divisor :: AmbientRing

AmbientRing -- an option used to tell divisor construction that a particular ambient ring is expected.

Description

If set to a ring, then when calling divisor, the primes are all checked whether they are ideals in that ring.

Functions with optional argument named AmbientRing :

  • "divisor(...,AmbientRing=>...)" -- see divisor -- constructor for (Weil/Q/R)-divisors

For the programmer

The object AmbientRing is a symbol.
