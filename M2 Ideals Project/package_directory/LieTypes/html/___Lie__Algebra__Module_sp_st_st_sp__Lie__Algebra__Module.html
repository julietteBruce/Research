LieTypes :: LieAlgebraModule ** LieAlgebraModule

LieAlgebraModule ** LieAlgebraModule -- tensor product of LieAlgebraModules

Synopsis

  • Operator: **
  • Usage:
    U ** V
  • Inputs:
    • U, an instance of the type LieAlgebraModule,
    • V, an instance of the type LieAlgebraModule,
  • Outputs:
    • W, an instance of the type LieAlgebraModule,

Description

Computes the tensor product of two Lie algebra modules.

i1 : g=simpleLieAlgebra("A",2)

o1 = g

o1 : LieAlgebra
 
i2 : U=irreducibleLieAlgebraModule({4,2},g)

o2 = U

o2 : g module
 
i3 : V=irreducibleLieAlgebraModule({3,1},g)

o3 = V

o3 : g module
 
i4 : U**V

o4 = {LieAlgebra => g                                        }
      isIrreducible => false
      DecompositionIntoIrreducibles => HashTable{{0, 5} => 1}
                                                 {1, 3} => 1
                                                 {1, 6} => 1
                                                 {2, 1} => 1
                                                 {2, 4} => 2
                                                 {3, 2} => 2
                                                 {3, 5} => 1
                                                 {4, 0} => 1
                                                 {4, 3} => 2
                                                 {5, 1} => 2
                                                 {5, 4} => 1
                                                 {6, 2} => 2
                                                 {7, 0} => 1
                                                 {7, 3} => 1
                                                 {8, 1} => 1

o4 : g module
