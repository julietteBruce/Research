ResidualIntersections :: isStronglyCM

isStronglyCM -- Checks if the given ideal is Strongly Cohen Macaulay

Synopsis

  • Usage:
    b = isStronglyCM I
  • Inputs:
    • I, an ideal,
  • Outputs:
    • b, a Boolean value, true if I is Strongly Cohen Macaulay

Description

I is Strongly Cohen Macaulay if each Koszul homology module is Cohen-Macaulay. The routine computes the Koszul homology with koszulDepth.

i1 : R = ZZ/101[x_1..x_5];
 
i2 : I = ideal{x_1*x_3,x_2*x_4,x_3*x_4,x_1*x_5,x_3*x_5};

o2 : Ideal of R
 
i3 : isStronglyCM I

o3 = true

See also

  • koszulDepth -- Computes the depths of the Koszul homology
  • hasSlidingDepth -- Checks if an ideal has the sliding depth property

Ways to use isStronglyCM :

  • "isStronglyCM(Ideal)"

For the programmer

The object isStronglyCM is a method function.
