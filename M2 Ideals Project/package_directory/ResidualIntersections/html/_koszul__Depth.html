ResidualIntersections :: koszulDepth

koszulDepth -- Computes the depths of the Koszul homology

Synopsis

  • Usage:
    L = koszulDepth I
    d = koszulDepth(k,I)
  • Inputs:
    • I, an ideal,
    • k, an integer, the homological index to compute
  • Outputs:
    • L, a list, a list of the depths of Kozul homology
    • d, an integer, the depth of the k-th Koszul homology

Description

The one parameter version computes the depths of the non-vanishing Koszul homology of I. The two parameter version computes only the depth of the k-th Koszul homology.

i1 : R = QQ[x_1..x_6];
 
i2 : I = ideal{x_1*x_2,x_1*x_3,x_2*x_4*x_5,x_1*x_6,x_4*x_6,x_5*x_6};

o2 : Ideal of R
 
i3 : koszulDepth I

o3 = {3, 0, 2}

o3 : List
 
i4 : koszulDepth(2,I)

o4 = 2
