SumsOfSquares :: solveSOS(...,Solver=>...)

solveSOS(...,Solver=>...) -- picking a semidefinite programming solver

Description

Macaulay2 has a powerful solver already configured. If necessary you can use this variable to change the semidefinite programming solver used in the background. See Solver for details.

Further information

  • Default value: null
  • Function: solveSOS -- solve a sum-of-squares problem
  • Option key: Solver -- picking a semidefinite programming solver

Functions with optional argument named Solver :

  • "optimize(...,Solver=>...)" -- see Solver -- picking a semidefinite programming solver
  • "lowerBound(...,Solver=>...)"
  • solveSOS(...,Solver=>...) -- picking a semidefinite programming solver
  • "sosdecTernary(...,Solver=>...)"
  • "sosInIdeal(...,Solver=>...)"
