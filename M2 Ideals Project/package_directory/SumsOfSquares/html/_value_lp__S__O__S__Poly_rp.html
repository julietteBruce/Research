SumsOfSquares :: value(SOSPoly)

value(SOSPoly) -- expansion of a weighted SOS decomposition

Synopsis

  • Function: value
  • Usage:
    value(s)
  • Inputs:
    • s, an instance of the type SOSPoly,
  • Outputs:
    • a ring element, a polynomial

Description

Given an object of type SOSPoly, this method expands the polynomial and computes $f = \sum_i d_i g_i^2$.

i1 : R = QQ[x,y];
 
i2 : s = sosPoly(R, {x+1,y}, {2,3} )

              2        2
o2 = 2*(x + 1)  + 3*(y)

o2 : SOSPoly
 
i3 : value( s )

       2     2
o3 = 2x  + 3y  + 4x + 2

o3 : R
