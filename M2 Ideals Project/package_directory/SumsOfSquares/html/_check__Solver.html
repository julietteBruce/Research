SumsOfSquares :: checkSolver

checkSolver -- tests a semidefinite programming solver

Synopsis

  • Usage:
    checkSolver(solver)
    checkSolver(solver,fun)
  • Inputs:
    • solver, a string, either "M2" or "CSDP" or "SDPA" or "MOSEK"
    • fun, a function, optional, one of optimize, solveSOS, sosdecTernary, sosInIdeal, or lowerBound
  • Optional inputs:
    • Verbosity => ..., default value 0, control the level of information printed

Description

This method tests whether various functions work properly using a specified solver. The most basic invocation uses only the name of the solver. This runs all tests with that solver and prints a summary. All tests should pass for CSDP and MOSEK.
