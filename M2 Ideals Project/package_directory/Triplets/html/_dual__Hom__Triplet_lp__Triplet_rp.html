Triplets :: dualHomTriplet(Triplet)

dualHomTriplet(Triplet) -- the dual homology triplet

Synopsis

  • Function: dualHomTriplet
  • Usage:
    dualHomTriplet T
  • Inputs:
    • T, an instance of the type Triplet, a homology triplet (B,H,C)
  • Outputs:
    • the dual homology triplet (conj(B,n),C,H), an instance of the type Triplet

Description

i1 : T = triplet({1,2,3}, {1,3}, {0,2,3})

o1 = {{1, 2, 3}, {1, 3}, {0, 2, 3}}

o1 : Triplet
 
i2 : dualHomTriplet(T)

o2 = {{0, 1, 2}, {0, 2, 3}, {1, 3}}

o2 : Triplet
