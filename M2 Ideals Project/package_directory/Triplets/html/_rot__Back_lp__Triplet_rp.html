Triplets :: rotBack(Triplet)

rotBack(Triplet) -- backward cyclic permutation

Synopsis

  • Function: rotBack
  • Usage:
    rotBack T
  • Inputs:
    • T, an instance of the type Triplet, a triplet of degree sequences
  • Outputs:
    • an instance of the type Triplet, the cyclically shifted backward triplet.

Description

i1 : T = triplet({1,2,3}, {0,2}, {0,2,3})

o1 = {{1, 2, 3}, {0, 2}, {0, 2, 3}}

o1 : Triplet
 
i2 : rotBack(T)

o2 = {{0, 2, 3}, {1, 2, 3}, {0, 2}}

o2 : Triplet
