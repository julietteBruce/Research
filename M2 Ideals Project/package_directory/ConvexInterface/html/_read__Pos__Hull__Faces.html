ConvexInterface :: readPosHullFaces

readPosHullFaces -- Read the result of a previous mPosHullFaces or mPosHullFacesAndDuals computation.

Synopsis

  • Usage:
    readConvexHullFaces(L)
  • Inputs:
    • fn, a string,
  • Outputs:
    • a list,

Description

Read from the file fn the result of a previous mPosHullFacesAndDuals or mPosHullFaces computation stored via the Option toFile.

Ways to use readPosHullFaces :

  • "readPosHullFaces(String)"

For the programmer

The object readPosHullFaces is a method function.
