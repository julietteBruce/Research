ConvexInterface :: mIsSubcone

mIsSubcone -- Test whether a cone is a subcone of another cone.

Synopsis

  • Usage:
    mIsSubcone(L1,L2)
  • Inputs:
    • L1, a list, of vectors generating the cone C1
    • L2, a list, of vectors generating the cone C2
  • Optional inputs:
    • toFile => ..., default value null, Store result in a file.
  • Outputs:
    • a Boolean value,

Description

Test whether C1 is a subcone of C2.

i1 : L1={{1,0},{0,1}};
 
i2 : L1=apply(L1,vector)

o2 = {| 1 |, | 0 |}
      | 0 |  | 1 |

o2 : List
 
i3 : L2={{1,2},{2,1}};
 
i4 : L2=apply(L2,vector)

o4 = {| 1 |, | 2 |}
      | 2 |  | 1 |

o4 : List
 
i5 : mIsSubcone(L1,L2)

o5 = false
 
i6 : mIsSubcone(L2,L1)

o6 = true

Ways to use mIsSubcone :

  • "mIsSubcone(List,List)"

For the programmer

The object mIsSubcone is a method function with options.
