MultiprojectiveVarieties :: shape(MultiprojectiveVariety)

shape(MultiprojectiveVariety) -- shape of the ambient multi-projective space of a multi-projective variety

Synopsis

  • Function: shape
  • Usage:
    shape X
  • Inputs:
    • M, a multi-projective variety, a subvariety of $\mathbb{P}^{k_1}\times\cdots\times\mathbb{P}^{k_{n}}$
  • Outputs:
    • a list, the list of integers $\{k_1, \ldots, k_n\}$

Description

i1 : X = PP_(ZZ/65521)^{2,3,1};

o1 : ProjectiveVariety, PP^2 x PP^3 x PP^1
 
i2 : shape X

o2 : List
 
i3 : p = point X;

o3 : ProjectiveVariety, a point in PP^2 x PP^3 x PP^1
 
i4 : shape p
