MultiprojectiveVarieties :: ∏

∏ -- product of multi-projective varieties

Synopsis

  • Usage:
    ∏ {X,Y,Z,...}
  • Inputs:
    • a list {X,Y,Z,...} of multi-projective varieties
  • Outputs:
    • a multi-projective variety, the product $X\times Y \times Z\times \cdots$

Description

i1 : K = ZZ/33331;
 
i2 : X = PP_K^(2,2);

o2 : ProjectiveVariety, surface in PP^5
 
i3 : Y = PP_K^({1,1,1},{2,3,1});

o3 : ProjectiveVariety, threefold in PP^2 x PP^3 x PP^1
 
i4 : Z = PP_K^(1,4);

o4 : ProjectiveVariety, curve in PP^4
 
i5 : ∏ {X,Y,Z};

o5 : ProjectiveVariety, X x Y x Z
 
i6 : assert(oo == ∏ {X ** Y,Z} and ∏ {X ** Y,Z} == ∏ {X, Y ** Z})

For the programmer
