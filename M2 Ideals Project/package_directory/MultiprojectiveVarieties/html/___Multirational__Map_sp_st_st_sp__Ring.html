MultiprojectiveVarieties :: MultirationalMap ** Ring

MultirationalMap ** Ring -- change the coefficient ring of a multi-rational map

Synopsis

  • Operator: **
  • Usage:
    Phi ** K
  • Inputs:
    • Phi, a multi-rational map, defined over a coefficient ring F
    • K, a ring, the new coefficient ring (which must be a field)
  • Outputs:
    • a multi-rational map, a multi-rational map defined over K, obtained by coercing the coefficients of the multi-forms defining Phi into K

Description

It is necessary that all multi-forms in the old coefficient ring F can be automatically coerced into the new coefficient ring K.
