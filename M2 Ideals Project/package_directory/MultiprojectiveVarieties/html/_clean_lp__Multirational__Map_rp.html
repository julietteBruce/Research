MultiprojectiveVarieties :: clean(MultirationalMap)

clean(MultirationalMap) -- clean the internal information of a multi-rational map

Synopsis

  • Function: clean
  • Usage:
    clean Phi
  • Inputs:
    • Phi, a multi-rational map
  • Outputs:
    • a multi-rational map, which is identical to Phi, but new to the system

Description

This is only useful for comparing computation times for various algorithms.

i1 : Phi = 1_(PP_QQ^2);

o1 : MultirationalMap (automorphism of PP^2)
 
i2 : Psi = clean Phi;

o2 : MultirationalMap (rational map from PP^2 to PP^2)
 
i3 : Phi == Psi

o3 = true
 
i4 : Phi === Psi

o4 = false

Ways to use this method:

  • clean(MultirationalMap) -- clean the internal information of a multi-rational map
  • "clean(RationalMap)"
