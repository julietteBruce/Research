MultiprojectiveVarieties :: multirationalMap(MultiprojectiveVariety)

multirationalMap(MultiprojectiveVariety) -- identity map

Synopsis

  • Function: multirationalMap
  • Usage:
    1_X
    multirationalMap X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a multi-rational map, the identity map on X

Description

i1 : X = PP_QQ^{2,3,1};

o1 : ProjectiveVariety, PP^2 x PP^3 x PP^1
 
i2 : 1_X;

o2 : MultirationalMap (automorphism of X)

The command 0_X returns instead the empty subscheme of X.

i3 : 0_X;

o3 : ProjectiveVariety, empty subscheme of PP^2 x PP^3 x PP^1
