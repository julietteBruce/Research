MultiprojectiveVarieties :: Saturate

Saturate -- whether to compute the multi-saturation of the ideal (intended for internal use only)

Synopsis

  • Usage:
    projectiveVariety(I,Saturate=>false)

Description

Use this option only in the case you know that the ideal I is already multi-saturated, otherwise nonsensical answers may result.

See also

  • projectiveVariety -- the closed multi-projective subvariety defined by a multi-homogeneous ideal
  • projectiveVariety(...,MinimalGenerators=>...) -- whether to trim the ideal (intended for internal use only)

Functions with optional argument named Saturate :

  • "projectiveVariety(...,Saturate=>...)"

For the programmer

The object Saturate is a symbol.
