MultiprojectiveVarieties :: ring(MultiprojectiveVariety)

ring(MultiprojectiveVariety) -- the coordinate ring of the variety

Synopsis

  • Function: ring
  • Usage:
    ring X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a ring, the coordinate ring of $X$

Description

i1 : X = PP_QQ^({2,1},{1,3});

o1 : ProjectiveVariety, threefold in PP^2 x PP^3
 
i2 : ring X

                 QQ[x0 ..x0 , x1 ..x1 ]
                      0    2    0    3
o2 = ---------------------------------------------
        2                              2
     (x1  - x1 x1 , x1 x1  - x1 x1 , x1  - x1 x1 )
        2     1  3    1  2     0  3    1     0  2

o2 : QuotientRing
