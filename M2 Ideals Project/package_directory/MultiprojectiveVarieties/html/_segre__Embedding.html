MultiprojectiveVarieties :: segreEmbedding

segreEmbedding -- the Segre embedding of the variety

Synopsis

  • Usage:
    segreEmbedding X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a multi-rational map, the Segre embedding of X (calculated through segre ring $X$)

Description

i1 : X = PP_(ZZ/3331)^({2,1,1},{2,1,3});

o1 : ProjectiveVariety, 4-dimensional subvariety of PP^5 x PP^1 x PP^3
 
i2 : segreEmbedding X

o2 = multi-rational map consisting of one single rational map
     source variety: 4-dimensional subvariety of PP^5 x PP^1 x PP^3 cut out by 9 hypersurfaces of multi-degrees (0,0,2)^3 (2,0,0)^6 
     target variety: PP^47

o2 : MultirationalMap (rational map from X to PP^47)

See also

  • segre -- Segre embedding
  • segre(MultirationalMap) -- the composition of a multi-rational map with the Segre embedding of the target

Ways to use segreEmbedding :

  • "segreEmbedding(MultiprojectiveVariety)"

For the programmer

The object segreEmbedding is a method function.
