MultiprojectiveVarieties :: target(MultirationalMap)

target(MultirationalMap) -- the target for a multi-rational map

Synopsis

  • Function: target
  • Usage:
    target Phi
  • Inputs:
    • Phi, a multi-rational map
  • Outputs:
    • a multi-projective variety, the target of Phi

Description

Note that, instead, the target of a standard rational map is the coordinate ring of the target variety (this is done mainly for efficiency reasons).
