MultiprojectiveVarieties :: projections

projections -- projections of a multi-projective variety

Synopsis

  • Usage:
    projections X
  • Inputs:
    • X, a multi-projective variety, a subvariety of $\mathbb{P}^{k_1}\times\mathbb{P}^{k_2}\times\cdots\times\mathbb{P}^{k_n}$
  • Outputs:
    • the list of the projections $X\to \mathbb{P}^{k_i}$, for $i=1,\ldots,n$
