MultiprojectiveVarieties :: MultiprojectiveVariety % MultiprojectiveVariety

MultiprojectiveVariety % MultiprojectiveVariety -- subvariety of a projective variety

Synopsis

  • Operator: %
  • Usage:
    X % Y
  • Inputs:
    • X, a multi-projective variety
    • Y, a multi-projective variety, which contains $X$
  • Outputs:
    • a multi-projective variety, the same variety $X$ thought of as a subvariety of $Y$

Description

i1 : Y = GG(ZZ/33331,1,4);

o1 : ProjectiveVariety, GG(1,4)
 
i2 : p = point Y

o2 = p

o2 : ProjectiveVariety, a point in PP^9
 
i3 : p % Y

o3 = p

o3 : ProjectiveVariety, a point in PP^9 (subvariety of codimension 6 in Y)
 
i4 : Fano p

o4 = curve in PP^4 cut out by 3 hypersurfaces of degree 1

o4 : ProjectiveVariety, curve in PP^4
