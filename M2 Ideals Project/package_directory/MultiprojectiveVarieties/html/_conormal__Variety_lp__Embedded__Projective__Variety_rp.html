MultiprojectiveVarieties :: conormalVariety(EmbeddedProjectiveVariety)

conormalVariety(EmbeddedProjectiveVariety) -- the conormal variety of a projective variety

Synopsis

  • Function: conormalVariety
  • Usage:
    conormalVariety X
  • Inputs:
    • X, an embedded projective variety
  • Optional inputs:
    • SingularLocus => ..., default value null,
    • Strategy => ..., default value "Saturate",
    • Variable => ..., default value null,
  • Outputs:
    • a multi-projective variety, the conormal variety of $X$

Description

i1 : X = PP_QQ^(2,2);

o1 : ProjectiveVariety, surface in PP^5
 
i2 : C = conormalVariety X;

o2 : ProjectiveVariety, 4-dimensional subvariety of PP^5 x PP^5
 
i3 : p2 = multirationalMap last projections C;

o3 : MultirationalMap (rational map from C to PP^5)
 
i4 : image p2 == dual X

o4 = true
