MultiprojectiveVarieties :: segre(MultiprojectiveVariety)

segre(MultiprojectiveVariety) -- the Segre embedding of the variety

Synopsis

  • Function: segre
  • Usage:
    segre X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a rational map, the map returned by segre ring $X$

Description

This function is intended for internal use. Use the segreEmbedding function instead.
