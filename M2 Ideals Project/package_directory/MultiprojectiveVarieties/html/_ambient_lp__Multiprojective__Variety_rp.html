MultiprojectiveVarieties :: ambient(MultiprojectiveVariety)

ambient(MultiprojectiveVariety) -- the ambient multi-projective space of the variety

Synopsis

  • Function: ambient
  • Usage:
    ambient X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a multi-projective variety, the product of the projective spaces where $X$ is embedded

Description

i1 : X = PP_QQ^({1,1,1},{2,1,3});

o1 : ProjectiveVariety, threefold in PP^2 x PP^1 x PP^3
 
i2 : ambient X;

o2 : ProjectiveVariety, PP^2 x PP^1 x PP^3
