MultiprojectiveVarieties :: describe(MultiprojectiveVariety)

describe(MultiprojectiveVariety) -- describe a multi-projective variety

Synopsis

  • Function: describe
  • Usage:
    describe X
    ? X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a description of X

Description

? X is a lite version of describe X.

i1 : X = source graph rationalMap PP_(ZZ/65521)^(1,3);

o1 : ProjectiveVariety, threefold in PP^3 x PP^2
 
i2 : ? X

     (1,1)
