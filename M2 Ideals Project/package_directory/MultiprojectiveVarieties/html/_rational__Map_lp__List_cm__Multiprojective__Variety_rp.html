rationalMap(List,MultiprojectiveVariety) -- the multi-rational map defined by a list of rational maps

Synopsis

  • Function: rationalMap
  • Usage:
    rationalMap Phi
    rationalMap(Phi,Y)
  • Inputs:
    • Phi, a list, whose elements are either rational maps or representatives of them (e.g., row matrices or ring maps)
    • Y, a multi-projective variety, optional
  • Optional inputs:
    • Dominant => ..., default value null,
  • Outputs:
    • a multi-rational map, the same as multirationalMap(Phi,Y), or multirationalMap Phi (if Y is not specified)

  • rationalMap -- makes a rational map
