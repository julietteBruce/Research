MultiprojectiveVarieties :: multirationalMap(MultirationalMap,MultiprojectiveVariety)

multirationalMap(MultirationalMap,MultiprojectiveVariety) -- change the target of a multi-rational map

Synopsis

  • Function: multirationalMap
  • Usage:
    rationalMap(Phi,Y)
    multirationalMap(Phi,Y)
    check rationalMap(Phi,Y)
  • Inputs:
    • Phi, a multi-rational map
    • Y, a multi-projective variety, which must be compatible with Phi
  • Outputs:
    • a multi-rational map, defined in the same way as Phi but with Y as target

Description

i1 : Phi = rationalMap {super specialQuadraticTransformation 1}

o1 = Phi

o1 : MultirationalMap (rational map from PP^7 to PP^8)
 
i2 : Y = image Phi

o2 = Y

o2 : ProjectiveVariety, hypersurface in PP^8
 
i3 : Psi = rationalMap(Phi,Y)

o3 = Psi

o3 : MultirationalMap (dominant rational map from PP^7 to Y)
