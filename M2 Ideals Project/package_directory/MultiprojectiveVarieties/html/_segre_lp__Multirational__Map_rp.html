MultiprojectiveVarieties :: segre(MultirationalMap)

segre(MultirationalMap) -- the composition of a multi-rational map with the Segre embedding of the target

Synopsis

  • Function: segre
  • Usage:
    segre Phi
  • Inputs:
    • Phi, a multi-rational map
  • Outputs:
    • a rational map, the composition of the multi-rational map Phi with the Segre embedding of the target of Phi
