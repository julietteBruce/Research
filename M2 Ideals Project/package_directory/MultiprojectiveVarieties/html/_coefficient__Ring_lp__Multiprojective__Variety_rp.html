MultiprojectiveVarieties :: coefficientRing(MultiprojectiveVariety)

coefficientRing(MultiprojectiveVariety) -- the coefficient ring of the variety

Synopsis

  • Function: coefficientRing
  • Usage:
    coefficientRing X
  • Inputs:
    • X, a multi-projective variety
  • Outputs:
    • a ring, the coefficient ring of $X$

Description

i1 : X = PP_QQ^({2,1},{1,3});

o1 : ProjectiveVariety, threefold in PP^2 x PP^3
 
i2 : coefficientRing X

o2 = QQ

o2 : Ring
