RandomSpaceCurves :: hilbertNumerator

hilbertNumerator -- calculate Hilbert numerator from Hilbert function

Synopsis

  • Usage:
    p=hilbertNumerator(L,r,t)
  • Inputs:
    • L, a list, values of the Hilbert function
    • r, an integer, dimension of ambient projective space
    • t, a ring element, variable in which the hilbertNumerator is given

Description

i1 : T=QQ[t];
 
i2 : hilbertNumerator({0,0,4,6,3,0,0,0,0},3,t)

       8     7     6     5     4      3     2
o2 = 3t  - 6t  - 2t  + 8t  + 3t  - 10t  + 4t

o2 : T

Caveat

The Hilbert function has to be given at positions 0 to d+r+1 where d is the regularity of the considered variety

Ways to use hilbertNumerator :

  • "hilbertNumerator(List,ZZ,RingElement)"

For the programmer

The object hilbertNumerator is a method function.
