Visualize :: visualize(Graph,Warning=>...)

visualize(Graph,Warning=>...) -- an option to squelch warnings

Synopsis

  • Usage:
    H = visualize (G, VisPath => "/PATH/TO/DIRECTORY/", Warning => false)
  • Inputs:
    • G, a graph, a graph

Description

When using the option VisPath a warning is produced if files will be overwritten. Warning is used to squelch these warnings. The default value is true, meaning the warning will be displayed.

Further information

  • Default value: true
  • Method: visualize(Graph) -- visualizes a graph in a modern browser
  • Option key: Warning -- an option to squelch warnings
