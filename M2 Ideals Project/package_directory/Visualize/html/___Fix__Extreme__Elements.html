Visualize :: FixExtremeElements

FixExtremeElements -- an option to fix extreme elements of a poset

Description

When using this option, the visualized poset will be drawn such that the extreme elements are placed at the top and bottom of the drawing.

See also

  • Basic Workflow for Visualize
  • visualize(Poset) -- visualizes a poset in the browser

Functions with optional argument named FixExtremeElements :

  • "visualize(Poset,FixExtremeElements=>...)"

For the programmer

The object FixExtremeElements is a symbol.
