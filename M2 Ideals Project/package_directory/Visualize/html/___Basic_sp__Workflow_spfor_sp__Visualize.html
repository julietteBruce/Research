Basic Workflow for Visualize

The basic workflow this package requires the user to open a port for communication between Macaulay2 and the web browser. Once the port is open, the visualize method can be used freely. In particular, the workflow is as follows:

  • 1. Load or install the package.
  • 2. Open a port with openPort method for communication with the browser. It is up to the user to choose port and also to close the port when finished.
  • 3. Define an object you wish to visualize. For example, a graph, poset, digraph, etc.
  • 4. Run the visualize method. This will open the browser with an interactive interface. This session is in communication with Macaulay2 through the open port above. At this point you can edit and manipulate the created object.
  • 5. End the session and export work back to Macaulay2.
  • 6. Continue manipulating the object and repeat steps 3-5 as necessary.
  • 7. When finished, close the port with closePort or restart Macaulay2.
