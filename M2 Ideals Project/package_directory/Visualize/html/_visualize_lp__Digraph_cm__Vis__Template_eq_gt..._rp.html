Visualize :: visualize(Digraph,VisTemplate=>...)

visualize(Digraph,VisTemplate=>...) -- an option defining the template path

Synopsis

  • Usage:
    H = visualize (D, VisPath => "/PATH/TO/TEMPLATE/")
  • Inputs:
    • D, a digraph, a digraph

Description

This option is used internally to pass paths from one method to another. A savvy user would be able to use this option to create a personal template and pass the path for Visualize.m2 to use.

Further information

  • Default value: ../../../../Macaulay2/packages/Visualize/templates/visDigraph/visDigraph-template.html
  • Method: visualize(Digraph) -- visualizes a digraph in the browser
  • Option key: VisTemplate -- an option defining the template path
