Normaliz :: generators(MonomialSubalgebra)

generators(MonomialSubalgebra) -- generators of a monomial subalgebra

Synopsis

  • Function: generators
  • Usage:
    gens S
  • Inputs:
    • an instance of the type MonomialSubalgebra,
  • Optional inputs:
    • CoefficientRing => ..., default value null,
  • Outputs:
    • a list, whose entries are the generators of the monomial subalgebra S

Description

i1 : R=ZZ/37[x,y,z];
 
i2 : S=createMonomialSubalgebra {x^2*y, x*z, z^3}

     ZZ  2         3
o2 = --[x y, x*z, z ]
     37

o2 : monomial subalgebra of R
 
i3 : gens S

       2         3
o3 = {x y, x*z, z }

o3 : List
