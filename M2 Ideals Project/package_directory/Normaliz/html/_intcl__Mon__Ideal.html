Normaliz :: intclMonIdeal

intclMonIdeal -- normalization of Rees algebra

Description

This function computes the integral closure of a monomial ideal I=(f_1,\ldots,f_m)\subset R in the polynomial ring R[t] and the normalization of its Rees algebra. The normalization of the Rees algebra is the integral closure of R[f_1t,\ldots,f_mt] in R[t]. For a definition of the Rees algebra (or Rees ring) see Bruns and Herzog, Cohen-Macaulay rings, Cambridge University Press 1998, p. 182.

Ways to use intclMonIdeal :

  • intclMonIdeal(Ideal) -- normalization of Rees algebra
  • intclMonIdeal(Ideal,RingElement) -- normalization of Rees algebra

For the programmer

The object intclMonIdeal is a method function with options.
