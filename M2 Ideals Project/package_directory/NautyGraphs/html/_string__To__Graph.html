NautyGraphs :: stringToGraph

stringToGraph -- converts a Sparse6 or Graph6 String to a Graph

Synopsis

  • Usage:
    G = stringToEdgeIdeal S
  • Inputs:
    • S, a string, a string in nauty's Sparse6 or Graph6 format
  • Outputs:
    • G, a graph,

Description

This method converts a Sparse6 or Graph6 string $S$ to an instance of the class Graph, stored in $G$.

i1 : stringToGraph "Dhc"

o1 = Graph{0 => {1, 4}}
           1 => {0, 2}
           2 => {1, 3}
           3 => {2, 4}
           4 => {0, 3}

o1 : Graph

See also

  • graphToString -- converts a graph to a string in the Graph6 format
  • stringToEdgeIdeal -- converts a Sparse6 or Graph6 String to an edge ideal in the given polynomial ring

Ways to use stringToGraph :

  • "stringToGraph(String)"

For the programmer

The object stringToGraph is a method function.
