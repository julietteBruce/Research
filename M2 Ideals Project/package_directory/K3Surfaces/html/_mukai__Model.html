K3Surfaces :: mukaiModel

mukaiModel -- Mukai models

Synopsis

  • Usage:
    mukaiModel g
  • Inputs:
    • g, an integer, the genus
  • Optional inputs:
    • CoefficientRing => ..., default value ZZ/65521
  • Outputs:
    • an embedded projective variety, the Mukai model of genus $g$ and degree $2g-2$

Description

i1 : X = mukaiModel 9;

o1 : ProjectiveVariety, 6-dimensional subvariety of PP^13
 
i2 : (degree X, sectionalGenus X)

o2 = (16, 9)

o2 : Sequence
 
i3 : parametrize X

o3 = multi-rational map consisting of one single rational map
     source variety: PP^6
     target variety: 6-dimensional subvariety of PP^13 cut out by 21 hypersurfaces of degree 2
     dominance: true

o3 : MultirationalMap (dominant rational map from PP^6 to X)

Ways to use mukaiModel :

  • "mukaiModel(ZZ)"

For the programmer

The object mukaiModel is a method function with options.
