K3Surfaces :: K3(String)

K3(String) -- show available functions to construct K3 surfaces of given genus

Synopsis

  • Function: K3
  • Usage:
    K3 "G"
    K3("G",[Unique=>true])
  • Inputs:
    • G, a string, the string of an integer
  • Optional inputs:
    • CoefficientRing => ..., default value ZZ/65521, make a lattice-polarized K3 surface
    • Singular => ..., default value null, make a lattice-polarized K3 surface
    • Verbose => ..., default value false, make a lattice-polarized K3 surface
  • Outputs:
    • a list, a list of terns (g,d,n) such that ((K3(g,d,n))(a,b) is a K3 surface of genus G, for some integers a,b
