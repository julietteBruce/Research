Classic > poly(String)

poly(String) -- make a polynomial using classic Macaulay syntax

Synopsis

  • Function: poly
  • Usage:
    poly s
  • Inputs:
    • s, a string
  • Outputs:
    • a ring element, created from s using classic Macaulay syntax, as described in Classic

Description
