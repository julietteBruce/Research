Classic :: monomialIdeal(String)

monomialIdeal(String) -- make a monomial ideal using classic Macaulay syntax

Synopsis

  • Function: monomialIdeal
  • Usage:
    monomialIdeal s
  • Inputs:
    • s, a string, in the form: "f1,f2,...,fr"
  • Outputs:
    • a monomial ideal

Description

Creates a monomial ideal using an abbreviated format. Each polynomial has the form described in Classic. The polynomials are separated by commas. Spaces and newline characters are ignored.

i1 : R = ZZ[a..e];
 
i2 : I = monomialIdeal "abc,bcd,cde11"

                                       11
o2 = monomialIdeal (a*b*c, b*c*d, c*d*e  )

o2 : MonomialIdeal of R
