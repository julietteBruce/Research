Seminormalization :: isSeminormal

isSeminormal -- checks if a ring is seminormal

Synopsis

  • Usage:
    b = isSeminormal(R)
  • Inputs:
    • R, a ring,
  • Outputs:
    • b, a Boolean value, true if the ring is seminormal

Description

This returns true if the ring is seminormal, otherwise it returns false.

i1 : R = QQ[x,y]/ideal(y^2 - x^3);
 
i2 : isSeminormal(R)

o2 = false
 
i3 : S = ZZ/5[a,b]/ideal(a^2-b^2+a^3)

o3 = S

o3 : QuotientRing
 
i4 : isSeminormal(R)

o4 = false

Ways to use isSeminormal :

  • "isSeminormal(Ring)"

For the programmer

The object isSeminormal is a method function.
