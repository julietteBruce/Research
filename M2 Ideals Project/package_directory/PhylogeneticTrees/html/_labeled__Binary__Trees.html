PhylogeneticTrees :: labeledBinaryTrees

labeledBinaryTrees -- enumerate all binary labeled trees

Synopsis

  • Usage:
    labeledTrees n
  • Inputs:
    • n, an integer, the number of leaves
  • Outputs:
    • a list, of all binary trees with n leaves

Description

This function enumerates all possible binary trees with n leaves labeled by $0,\ldots, n-1$, including all possible labelings. The trees are represented as an object of class LeafTree.

i1 : L = labeledBinaryTrees 4

o1 = {{{0, 1, 2, 3}, {set {1, 3}, set {0}, set {1}, set {2}, set {3}}}, {{0,
     ------------------------------------------------------------------------
     1, 2, 3}, {set {1, 2}, set {0}, set {1}, set {2}, set {3}}}, {{0, 1, 2,
     ------------------------------------------------------------------------
     3}, {set {2, 3}, set {0}, set {1}, set {2}, set {3}}}}

o1 : List
