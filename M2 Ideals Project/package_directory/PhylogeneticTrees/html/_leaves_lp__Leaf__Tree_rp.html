PhylogeneticTrees :: leaves(LeafTree)

leaves(LeafTree) -- list the leaves of a tree

Synopsis

  • Function: leaves
  • Usage:
    leaves T
  • Inputs:
    • T, an instance of the type LeafTree,
  • Outputs:
    • a set, the leaves of T

Description

This function outputs the leaves of the tree as an object of class Set.

i1 : T = leafTree(4,{{0,1}});
 
i2 : vertices T

o2 = {set {set {0, 1}, set {2}, set {3}}, set {set {0, 1, 2}}, set {set {0,
     ------------------------------------------------------------------------
     1, 3}}, set {set {0}, set {1}, set {2, 3}}, set {set {0, 2, 3}}, set
     ------------------------------------------------------------------------
     {set {1, 2, 3}}}

o2 : List
 
i3 : #(vertices T)

o3 = 6

Caveat

The leaves have a different representation from the one in the output of vertices(LeafTree).
