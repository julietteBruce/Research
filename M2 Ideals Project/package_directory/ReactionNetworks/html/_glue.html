ReactionNetworks :: glue

glue -- combine two networks

Description

This function takes two reaction networks, or a reaction network and a list and creates a new, combined network.

i1 : N1 = reactionNetwork("A <-- 2B, A + C <-- D, B + E --> A + C")

o1 = 2B-->A
     D-->A+C
     B+E-->A+C

o1 : ReactionNetwork
 
i2 : glue(N1, {"A --> 2B", "A + C --> D", "D --> B+E"})

o2 = A-->2B
     2B-->A
     A+C-->D
     D-->A+C
     D-->B+E
     B+E-->A+C

o2 : ReactionNetwork

See also

Ways to use glue :

  • "glue(ReactionNetwork,List)"
  • "glue(ReactionNetwork,ReactionNetwork)"
