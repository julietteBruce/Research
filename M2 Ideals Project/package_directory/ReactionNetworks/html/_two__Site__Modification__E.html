ReactionNetworks :: twoSiteModificationE

twoSiteModificationE -- an example for chemical reaction motif

Description

A two-site phosphorylation system in which modifications are carried out by different kinases and phosphatases for each phosphoform. Simply, it is assumed that both phosphorylation and dephosphorylation occur sequentially.

i1 : twoSiteModificationE()

o1 = S_0+E_1-->X_1
     X_1-->S_0+E_1
     X_1-->E_1+S_1
     S_1+E_2-->X_2
     X_2-->S_1+E_2
     X_2-->E_2+S_2
     S_1+F_1-->Y_1
     Y_1-->S_1+F_1
     Y_1-->S_0+F_1
     S_2+F_2-->Y_2
     Y_2-->S_2+F_2
     Y_2-->S_1+F_2

o1 : ReactionNetwork

To rename species, use substitute(ReactionNetwork,List).

Ways to use twoSiteModificationE :
