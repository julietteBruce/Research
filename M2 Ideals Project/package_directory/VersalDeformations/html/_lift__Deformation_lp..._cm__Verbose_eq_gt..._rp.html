VersalDeformations :: liftDeformation(...,Verbose=>...)

liftDeformation(...,Verbose=>...) -- control the verbosity of output

Description

Verbose is the name of an optional argument. Its value is an integer, which specifies how verbose output of the above commands should be. Default value is 0 which gives the tersest possible output. Highest verbosity is attained with the value 4.

Further information

  • Default value: 0
  • Function: liftDeformation -- lift a solution of the deformation equation to the next order
  • Option key: Verbose -- request verbose feedback
