VersalDeformations :: correctionMatrix

correctionMatrix -- calculate how first order deformations perturb obstruction vector

Synopsis

  • Usage:
    (M,L) = correctionMatrix(F1,R1)
  • Inputs:
    • F1, a matrix
    • R1, a matrix
  • Outputs:
    • (M,L), a sequence

Description

F1 should be some first order perturbations of a matrix with R1 a lift of the corresponding relations, as in the output of firstOrderDeformations. M is a matrix representing the effect of these perturbations one order higher, and L gives a parameter-free version of the perturbations and lifted relations.

Ways to use correctionMatrix :

  • "correctionMatrix(Matrix,Matrix)"

For the programmer

The object correctionMatrix is a method function.
