Polyhedra :: Cone + Polyhedron

Cone + Polyhedron -- computes the Minkowski sum of a cone and a polyhedron

Synopsis

  • Operator: +
  • Usage:
    Q = C + P
  • Inputs:
    • C, a convex rational cone
    • P, a convex polyhedron
  • Outputs:
    • Q, a convex polyhedron

Description

Computes the Minkowski sum of C and P.This is the polyhedron C + P = {c + p | c in C, p in P}. Note that C and P have to lie in the same ambient space.

See also minkowskiSum.
