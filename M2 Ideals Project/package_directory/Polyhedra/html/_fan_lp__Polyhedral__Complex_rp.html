Polyhedra :: fan(PolyhedralComplex)

fan(PolyhedralComplex) -- Take the fan over a polyhedral complex

Synopsis

  • Function: fan
  • Usage:
    F = fan PC
  • Inputs:
    • PC, an instance of the type PolyhedralComplex,
  • Outputs:
    • F, an instance of the type Fan,

Description

The polyhedral complex is embedded at height one and the fan is taken over it.
