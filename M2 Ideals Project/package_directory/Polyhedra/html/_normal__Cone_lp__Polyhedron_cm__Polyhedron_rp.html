Polyhedra :: normalCone(Polyhedron,Polyhedron)

normalCone(Polyhedron,Polyhedron) -- computes the normal cone of a face of a polyhedron

Synopsis

  • Function: normalCone
  • Usage:
    C = normalCone(P,Q)
  • Inputs:
    • P, a convex polyhedron
    • Q, a convex polyhedron, which must be a face of P
  • Optional inputs:
    • BasisElementLimit => ..., default value infinity,
    • DegreeLimit => ..., default value {},
    • MinimalGenerators => ..., default value true,
    • PairLimit => ..., default value infinity,
    • Strategy => ..., default value null,
    • Variable => ..., default value "w",
  • Outputs:
    • C, a convex rational cone

Description

The normal cone of a face Q of a polyhedron P is the cone in the normal fan (see normalFan) that corresponds to this face. This is the cone of all vectors attaining their maximum on this face.
