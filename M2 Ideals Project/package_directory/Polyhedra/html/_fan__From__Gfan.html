Polyhedra :: fanFromGfan

fanFromGfan -- Construct a fan from output data of {\tt Gfan}

Synopsis

  • Usage:
    F = fanFromGfan L
  • Inputs:
    • L, a list,
  • Outputs:
    • F, an instance of the type Fan,

Description

Method used by the package Tropical.m2 to construct a fan from Gfan output. Gfan produces more data than just rays and maximal cones and this constructor will save that data to avoid recomputation.

Ways to use fanFromGfan :

  • "fanFromGfan(List)"

For the programmer

The object fanFromGfan is a method function.
