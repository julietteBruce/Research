Polyhedra :: isFullDimensional

isFullDimensional -- Determine whether a polyhedral object is full-dimensional

Synopsis

  • Usage:
    b = isFullDimensional PO
  • Inputs:
    • PO, an instance of the type PolyhedralObject,
  • Outputs:
    • b, a Boolean value,

Description

Checks whether the dimension of the polyhedral object is equal to its ambient dimension.

o1 = C
