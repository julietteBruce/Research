Polyhedra :: secondaryPolytope

secondaryPolytope -- computes the secondary polytope of a compact polyhedron

Synopsis

  • Usage:
    Q = secondaryPolytope P
  • Inputs:
    • P, a convex polyhedron
  • Outputs:
    • Q, a convex polyhedron

Description

The secondary polytope parametrises the regular subdivisions of a polytope. See ... to be added. 

i1 : P = crossPolytope 2

o1 = P

o1 : Polyhedron
 
i2 : Q = secondaryPolytope P

o2 = Q

o2 : Polyhedron
 
i3 : vertices Q
