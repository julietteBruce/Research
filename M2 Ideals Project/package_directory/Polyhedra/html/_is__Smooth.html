Polyhedra :: isSmooth

isSmooth -- checks if a Cone or Fan is smooth

Synopsis

  • Usage:
    b = isSmooth C
    b = isSmooth F
  • Inputs:
    • C, a convex rational cone
  • Outputs:
    • b, true if the Cone/Fan is smooth, false otherwise

Description

isSmooth checks for a Cone if the rays are a subset of a basis of the lattice. For a Fan it checks smoothness for every Cone. 

i1 : C = coneFromVData matrix {{1,2,3},{3,1,2},{2,3,1}}

o1 = C

o1 : Cone
 
i2 : isSmooth C

o2 = false
 
i3 : F = hirzebruch 3

o3 = F

o3 : Fan
 
i4 : isSmooth F

o4 = true

Ways to use isSmooth :

  • "isSmooth(Cone)"
  • "isSmooth(Fan)"

For the programmer

The object isSmooth is a method function.
