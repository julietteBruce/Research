SRdeformations :: net(Face)

net(Face) -- Printing faces.

Synopsis

  • Function: net
  • Usage:
    net(F)
  • Inputs:
    • F, a face,
  • Outputs:
    • a net,

Description

Prints a face. The vertices are printed without any brackets. The number of vertices is printed.

i1 : R=QQ[x_0..x_4];
 
i2 : C=simplex R;
 
i3 : F=C.fc_2_0

o3 = x x x
      0 1 2

o3 : face with 3 vertices
