SRdeformations :: target(FirstOrderDeformation)

target(FirstOrderDeformation) -- The target of a deformation.

Synopsis

  • Function: target
  • Usage:
    target(f)
  • Inputs:
    • f, a first order deformation,
  • Outputs:
    • a module,

Description

Returns the target of the homomorphisms represented by f.
