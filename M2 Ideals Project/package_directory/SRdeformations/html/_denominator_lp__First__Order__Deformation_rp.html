SRdeformations :: denominator(FirstOrderDeformation)

denominator(FirstOrderDeformation) -- The denominator of a deformation as a vector.

Synopsis

  • Function: denominator
  • Usage:
    denominator(f)
  • Inputs:
    • f, a first order deformation,
  • Outputs:
    • a vector,

Description

Returns the denominator of a deformation as a vector.

i2 : addCokerGrading(R);

             0 1   1 2   2 3   3 4   0 4
