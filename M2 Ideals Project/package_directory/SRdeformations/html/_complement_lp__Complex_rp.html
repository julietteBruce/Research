SRdeformations :: complement(Complex)

complement(Complex) -- Compute the complement CoComplex.

Synopsis

  • Function: complement
  • Usage:
    complement(C)
  • Inputs:
    • C, an embedded complex,
  • Outputs:
    • an embedded co-complex,

Description

Given a Complex it returns the CoComplex of complement Faces and vice versa.

The complement (co)complex cC is remembered in C.complementComplex=cC and cC.complementComplex=C.
