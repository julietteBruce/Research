SRdeformations :: ofComplex

ofComplex -- The complex of a face.

Synopsis

  • Usage:
    ofComplex(F)
  • Inputs:
    • F, a face,
  • Outputs:
    • an embedded complex,

Description

Returns the complex of F, i.e., the first entry in F.ofComplex.

i1 : R=QQ[x_0..x_4];
 
i2 : C=simplex R

o2 = 4: x x x x x  
         0 1 2 3 4

o2 : complex of dim 4 embedded in dim 4 (printing facets)
     equidimensional, simplicial, F-vector {1, 5, 10, 10, 5, 1}, Euler = 0
 
i3 : F=C.fc_2_0

o3 = x x x
      0 1 2

o3 : face with 3 vertices
 
i4 : ofComplex F

o4 = 4: x x x x x  
         0 1 2 3 4

o4 : complex of dim 4 embedded in dim 4 (printing facets)
     equidimensional, simplicial, F-vector {1, 5, 10, 10, 5, 1}, Euler = 0

For the programmer

The object ofComplex is a method function.
