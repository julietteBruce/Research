SRdeformations :: vert

vert -- The vertices of a face or complex.

Synopsis

  • Usage:
    vert(F)
    vert(C)
  • Inputs:
    • F, a face,
    • C, an embedded complex,
  • Outputs:
    • a list,

Description

The vertices of F or C.

i1 : R=QQ[x_0..x_4];
 
i2 : addCokerGrading R;

              5        4
o2 : Matrix ZZ  <--- ZZ
 
i3 : C=simplex R;
 
i4 : vert C

o4 = {x , x , x , x , x }
       4   3   2   1   0

o4 : List
 
i5 : F=C.fc_2_0

o5 = x x x
      0 1 2

o5 : face with 3 vertices
 
i6 : vert F

o6 = {x , x , x }
       0   1   2

o6 : List

  • "vert(Face)"

For the programmer
