SRdeformations :: FirstOrderDeformation == FirstOrderDeformation

FirstOrderDeformation == FirstOrderDeformation -- Compare two first order deformations.

Synopsis

  • Operator: ==
  • Usage:
    f1==f2
  • Inputs:
    • f1, a first order deformation,
    • f2, a first order deformation,
  • Outputs:
    • a Boolean value,

Description

Checks whether f1 and f2 are equal.
