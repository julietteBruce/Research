SRdeformations :: complexToIdeal

complexToIdeal -- The monomial ideal associated to a complex.

Synopsis

  • Usage:
    complexToIdeal(C)
  • Inputs:
    • C, an embedded complex,
  • Optional inputs:
    • embedded => ..., default value null, Use only faces of the embedding complex.
  • Outputs:
    • an ideal,

Description

The Stanley-Reisner ideal associated to a complex, i.e., the ideal of non-faces of C.

If the option embedded=>true is given, then only Faces of embeddingComplex C are used.
