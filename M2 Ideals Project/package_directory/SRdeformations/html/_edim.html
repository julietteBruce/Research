SRdeformations :: edim

edim -- The embedding dimension of a complex or co-complex.

Synopsis

  • Usage:
    edim(C)
  • Inputs:
    • C, an embedded complex,
  • Outputs:
    • an integer,

Description

The embedding dimension of a complex or co-complex.
