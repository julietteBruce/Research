SRdeformations :: fc

fc -- The faces of a complex.

Synopsis

  • Usage:
    fc(C)
    fc(C,d)
  • Inputs:
    • C, an embedded complex,
    • d, an integer, positive
  • Outputs:
    • a list,

Description

Returns a list with entries lists with the faces of C starting in dimension -1. If d is given then faces of dimension d are returned.
