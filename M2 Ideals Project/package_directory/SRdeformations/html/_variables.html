SRdeformations :: variables

variables -- The variables of a complex or co-complex.

Synopsis

  • Usage:
    variables(C)
  • Inputs:
    • C, an embedded complex,
  • Outputs:
    • an embedded complex,

Description

The variables of a complex of co-complex. Compare with vert.
