gfanInterface :: gfanTropicalWeilDivisor

gfanTropicalWeilDivisor -- the tropical Weil divisor of a piecewise linear function

Synopsis

  • Usage:
    W = gfanTropicalWeilDivisor(C,F)
  • Inputs:
    • C, an instance of the type Fan, a k-cycle
    • F, an instance of the type Fan, a piecewise linear function
  • Optional inputs:
    • i1
    • i2
  • Outputs:
    • W, an instance of the type Fan, the Weil divisor

Description

This method computes the tropical Weil divisor of a piecewise linear function.

The following example is taken from the gfan manual.

i1 : QQ[x_1..x_3];
 
i2 : C = gfanTropicalHyperSurface (x_1*x_2 + x_2*x_3 + x_1*x_3 + x_1*x_2*x_3)

o2 = (Fan{...1...}, {1, 1, 1, 1, 1, 1})

o2 : Sequence
 
i3 : F = "..."

o3 = ...

gfan Documentation This program computes the tropical Weil divisor of piecewise linear (or tropical rational) function on a tropical k-cycle. See the Gfan manual for more information.Options:-i1 value: Specify the name of the Polymake input file containing the k-cycle.-i2 value: Specify the name of the Polymake input file containing the piecewise linear function.

Ways to use gfanTropicalWeilDivisor :

  • "gfanTropicalWeilDivisor(Fan,Fan)"
