gfanInterface :: MPLConverter

MPLConverter -- Converts lists or ideals into markedPolynomialLists

Synopsis

  • Usage:
    G = MPLConverter{L}
    G = MPLConverter(I)
  • Inputs:
    • L, a list, of polynomials
    • I, an ideal, contained in a polynomial ring
  • Outputs:
    • G, a marked polynomial list, a marked polynomial of I

Description

This method converts ideals and lists into marked polynomial lists. The output is a MarkedPolynomialList. The input can be given as an Ideal or List of polynomials.

i1 : QQ[x,y];
 
i2 : I = ideal(x^2 + y^2, x*y + y^2 + y^3);

o2 : Ideal of QQ[x..y]
 
i3 : MPLConverter(I)

        2     2    3           2
o3 = {(x ) + y , (y ) + x*y + y }

o3 : MarkedPolynomialList
 
i4 : QQ[x,y];
 
i5 : MPLConverter{x^2 + y^2, x*y + y^2 + y^3}

        2     2    3           2
o5 = {(x ) + y , (y ) + x*y + y }

o5 : MarkedPolynomialList

Ways to use MPLConverter :

  • "MPLConverter(Ideal)"
  • "MPLConverter(List)"

For the programmer

The object MPLConverter is a method function with options.
