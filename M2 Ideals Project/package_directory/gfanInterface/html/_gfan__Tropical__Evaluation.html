gfanInterface :: gfanTropicalEvaluation

gfanTropicalEvaluation -- evaluates a tropical polynomial function

Synopsis

  • Usage:
    Y = gfanTropical(f, L)
  • Inputs:
    • f, a ring element, a polynomial
    • L, a list, of points
  • Outputs:
    • Y, a list, the values of the tropicalization of f evaluated at each of the points in L

Description

This method evaluates a tropical polynomial function at a list of points.

i1 : QQ[x,y,z]

o1 = QQ[x..z]

o1 : PolynomialRing
 
i2 : gfanTropicalEvaluation(x*y+z^2, {{1,1,0}, {0,0,3}, {1,1,3} })

o2 = {2, 6, 6}

o2 : List
