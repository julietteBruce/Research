gfanInterface :: gfanTropicalMultiplicity

gfanTropicalMultiplicity -- multiplicity of a tropical cone

Synopsis

  • Usage:
    gfanTropicalMultiplicity(L)
  • Inputs:
    • L, a marked polynomial list,
  • Outputs:
    • M, an integer, the multiplicity of the tropical cone of L

Description

This method compute the multiplicity of a tropical cone. The tropical cone is described by a marked reduced Groebner basis for its initial ideal.
