M0nbar :: Fcurves

Fcurves -- lists the F-curves for a given n

Synopsis

  • Usage:
    Fcurves n
  • Inputs:
    • n, an integer,
  • Outputs:
    • a list,

Description

Given an integer $n$, this function returns a list of the F-curves on $\bar{M}_{0,n}$.
