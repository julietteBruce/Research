M0nbar :: nonadjacentBasis

nonadjacentBasis -- computes the nonadjacent basis of divisors on M0nbar

Synopsis

  • Usage:
    nonadjacentBasis n
  • Inputs:
    • n, an integer,
  • Outputs:
    • a list,

Description

The nonadjacent basis is described in Carr, A polygonal presentation of $Pic(\bar{M}_{0,n})$, arXiv:0911.2649.
