M0nbar :: CurveClassRepresentativeM0nbar

CurveClassRepresentativeM0nbar -- class implementing curves on the moduli space of stable n-pointed genus zero curves

Description

This class represents curve classes in $ NS_1(\bar{M}_{0,n}) $. A curve class is just a linear combination of F-curves.

i1 : L= { {{{1,2},{3},{4},{5}},1}, {{{1,3},{2},{4},{5}},1} };
 
i2 : curveClassRepresentativeM0nbar(5,L)

o2 = CurveClassRepresentativeM0nbar{CurveExpression => HashTable{{{1, 2}, {3}, {4}, {5}} => 1}}
                                                                 {{1, 3}, {2}, {4}, {5}} => 1
                                    NumberOfMarkedPoints => 5
