SimpleDoc :: stacksProject

stacksProject -- link to a Stacks Project tag

Synopsis

  • Usage:
    stacksProject(tag, description)
  • Inputs:
    • tag, a string, an alphanumeric Stacks Project tag of the form "TTTT"
    • title, a string, the link title
  • Outputs:
    • a markup list, a link to the Stacks Project node with the given tag, usable inside a documentation node

Description

i1 : stacksProject("01HR", "Affine schemes")

o1 = Affine schemes (see https://stacks.math.columbia.edu/tag/01HR )

o1 : HREF
 
i2 : html oo

o2 = <a href="https://stacks.math.columbia.edu/tag/01HR">Affine schemes</a>
