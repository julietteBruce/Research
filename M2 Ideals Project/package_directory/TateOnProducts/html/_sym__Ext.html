TateOnProducts :: symExt

symExt -- from linear presentation matrices over S to linear presentation matrices over E and conversely

Synopsis

  • Usage:
    symExt(m,E)
  • Inputs:
    • m, a matrix, a linear presentation matrix over S
    • E, a ring, the Koszul dual ring of S
  • Outputs:
    • a matrix, the corresponding linear presentation matrix over E

Description

Same method as in the single factor case

o1 = {1, 2}

i2 : (S,E) = productOfProjectiveSpaces n
