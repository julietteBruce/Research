TateOnProducts :: coarseMultigradedRegularity

coarseMultigradedRegularity -- A truncation that has linear resolution

Synopsis

  • Usage:
    R = coarseMultigradedRegularity M
  • Inputs:
    • M, a module, multi-graded module over a multi-graded polynomial ring
    • M, a chain complex, multi-graded module over a multi-graded polynomial ring
  • Optional inputs:
    • Strategy => a string, default value "MinimalResolution"
  • Outputs:
    • R, a list, degree such that truncate(R,M) has linear resolution

Description

Uses a free resolution and takes the maximum degree of the terms.
