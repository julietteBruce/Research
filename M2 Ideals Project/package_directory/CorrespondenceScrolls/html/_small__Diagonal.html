smallDiagonal -- Ideal of the small diagonal in (P^1)^n

Synopsis

  • Usage:
    I = smallDiagonal n
    I = smallDiagonal S
  • Inputs:
    • n, an integer, number of factors of P^1
    • S, a ring, ring with 2n vars, corresponding to (P^1)^n for some n
  • Outputs:
    • I, an ideal, ideal of the small diagonal
