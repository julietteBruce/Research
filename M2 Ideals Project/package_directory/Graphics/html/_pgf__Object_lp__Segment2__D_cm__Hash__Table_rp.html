Graphics :: pgfObject(Segment2D,HashTable)

pgfObject(Segment2D,HashTable) -- create the string describing a line segment in pgf

Synopsis

  • Function: pgfObject
  • Usage:
    pgfObject(mysegment,myoptions)
  • Inputs:
    • mysegment, an instance of the type Segment2D,
    • myoptions, a hash table, giving values to some options
  • Outputs:
    • a string, containing the description of the segment in pgf

Description

The options that are not given in myoptions are set at their default values if they are contained in defaultPGFOptions. Options not listed in possiblePGFOptions are discarded.

i1 : myoptions = hashTable {"fill"=>"black"}

o1 = HashTable{fill => black}

o1 : HashTable
 
i2 : pgfObject(segment(point(20,30),point(40,50)),myoptions)

o2 = \draw[] (20,30) -- (40,50);
