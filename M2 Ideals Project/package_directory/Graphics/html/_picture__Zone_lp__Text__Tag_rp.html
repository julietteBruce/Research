Graphics :: pictureZone(TextTag)

pictureZone(TextTag) -- find the zone that contains the tag

Synopsis

  • Function: pictureZone
  • Usage:
    pictureZone(t)
  • Inputs:
    • t, an instance of the type TextTag,
  • Outputs:
    • a list, of the form {xmin,xmax,ymin,ymax} or {xmin,xmax,ymin,ymax,zmin,zmax}

Description

i1 : pictureZone(textTag(point(20,30),"my tag"))

o1 = {20, 20, 30, 30}

o1 : List
 
i2 : pictureZone(textTag(point(20,30,40),"my tag"))

o2 = {20, 20, 30, 30, 40, 40}

o2 : List

Ways to use this method:
