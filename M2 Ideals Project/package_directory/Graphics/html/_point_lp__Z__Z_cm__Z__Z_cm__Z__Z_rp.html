point(ZZ,ZZ,ZZ) -- a point from three coordinates

Synopsis

  • Function: point
  • Usage:

    • x, an integer,
    • y, an integer,
    • z, an integer,
  • Outputs:
    • an instance of the type Point3D, the point with coordinates x, y and z
