Graphics :: svgPicture(Picture,ZZ,ZZ,String)

svgPicture(Picture,ZZ,ZZ,String) -- create an SVG picture from a Picture object and store it in a file

Synopsis

  • Function: svgPicture
  • Usage:
    svgPicture(mypict,w,h,filename)
  • Inputs:
    • mypict, an instance of the type Picture,
    • w, an integer, the width of the SVG picture
    • h, an integer, the height of the SVG picture
    • filename, a string, the name of the storage file

Description

If the filename is without a .svg suffix, it will be added.

The heading used is stored in defaultSVGHeading and the SVG opening bracket (resp. closing) used is stored in defaultSVGOpening (resp. defaultSVGClosing).
