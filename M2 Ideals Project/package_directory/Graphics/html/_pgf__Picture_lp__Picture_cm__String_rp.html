Graphics :: pgfPicture(Picture,String)

pgfPicture(Picture,String) -- create a pgf picture from a Picture object and store it in a file

Synopsis

  • Function: pgfPicture
  • Usage:
    pgfPicture(mypict,filename)
  • Inputs:
    • mypict, an instance of the type Picture,
    • filename, a string, the name of the storage file

Description

If the filename is without a .tex suffix, it will be added.
