Graphics :: viewPicture(String)

viewPicture(String) -- view a Picture in a browser

Synopsis

  • Function: viewPicture
  • Usage:
    viewPicture(filename)
  • Inputs:
    • filename, a string, the name of the file (possibly with path) containing the picture

Caveat

For the moment, it only works with SVG pictures and browsers supporting them.

Ways to use this method:

  • viewPicture(String) -- view a Picture in a browser
