Graphics :: pictureZone(Polygon3D)

pictureZone(Polygon3D) -- find the zone that contains the polygon

Synopsis

  • Function: pictureZone
  • Usage:
    pictureZone(p)
  • Inputs:
    • p, an instance of the type Polygon3D,
  • Outputs:
    • a list, of the form {xmin,xmax,ymin,ymax,zmin,zmax}

Description

i1 : pictureZone(polygon({point(20,30,40),point(25,30,40),point(20,35,45)}))

o1 = {20, 25, 30, 35, 40, 45}

o1 : List
