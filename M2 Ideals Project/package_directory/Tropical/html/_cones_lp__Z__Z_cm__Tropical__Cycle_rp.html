Tropical :: cones(ZZ,TropicalCycle)

cones(ZZ,TropicalCycle) -- computes the cones of a tropical cycle

Synopsis

  • Function: cones
  • Usage:
    cones(k,T)
  • Inputs:
    • k, an integer,
    • T, a tropical cycle,
  • Outputs:
    • L, a list, the cones of codimension k in T

Description

This function computes the cone of codimension k of the fan associated to the tropical cycle T.

i1 : QQ[x,y,z,w,t]

o1 = QQ[x..z, w, t]

o1 : PolynomialRing
 
i2 : I=ideal(x^2-y*z+w^2,w^3-y^3*x+z^3,t-w+x);

o2 : Ideal of QQ[x..z, w, t]
 
i3 : T=tropicalVariety I;
 
i4 : cones(2,T)

o4 = {{0, 1}, {0, 2}, {0, 3}, {0, 6}, {1, 4}, {1, 5}, {2, 4}, {2, 5}, {3, 4},
     ------------------------------------------------------------------------
     {3, 5}, {4, 6}, {5, 6}}

o4 : List
