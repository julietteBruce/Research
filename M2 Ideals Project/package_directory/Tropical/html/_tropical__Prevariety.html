Tropical :: tropicalPrevariety

tropicalPrevariety -- the intersection of the tropical hypersurfaces

Synopsis

  • Usage:
    tropicalPrevariety(L)
    tropicalPrevariaty(L,LS)
    tropicalPrevariety(L,Strategy=>S)
    tropicalPrevariety(L,LS,Strategy=>S)
  • Inputs:
    • L, a list, of polynomials
    • LS, a list, of Symmetries (optional)
  • Optional inputs:
    • Strategy => a string, default value "gfan", Strategy (currently only "gfan")
  • Outputs:
    • F, an instance of the type Fan, the intersection of the tropical hypersurfaces of polynomials in L

Description

This method intersects the tropical hypersurfaces coming from the tropicalizations of the polynomials in the list L. If there are symmetries that leave the specified polynomials fixed, they can be specified by passing a list with the symmetries as second argument, with the same format as the option Symmetry.
