SLPexpressions :: undeclareVariable

undeclareVariable -- clears assignment of an input gate to a variable name

Synopsis

  • Usage:
    undeclareVariable x
  • Inputs:
    • x, a symbol, or an IndexedVariable
  • Outputs:
    • an instance of the type InputGate,

Description

This method undoes the action of declareVariable, by reverting the assignment of an InputGate to a symbol.

i1 : declareVariable x

o1 = x

o1 : InputGate
 
i2 : x + 1

o2 = (x + 1)

o2 : SumGate
 
i3 : undeclareVariable x

o3 = x

o3 : Symbol
 
i4 : x

o4 = x

o4 : Symbol

See also

  • declareVariable -- assigns an input gate to a given variable name

Ways to use undeclareVariable :

  • "undeclareVariable(InputGate)"

For the programmer

The object undeclareVariable is a method function.
