SpecialFanoFourfolds :: Singular

Singular -- whether to transfer computation to Singular

Synopsis

  • Usage:
    Singular => true/false

Description

This option allows you to transfer part of the computation to Singular.

See also

  • associatedK3surface -- associated K3 surface to a rational fourfold

Functions with optional argument named Singular :

  • "associatedK3surface(...,Singular=>...)"
  • "mirrorFourfold(...,Singular=>...)"

For the programmer

The object Singular is a symbol.
