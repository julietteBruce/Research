SpecialFanoFourfolds :: check(ZZ,CongruenceOfCurves)

check(ZZ,CongruenceOfCurves) -- check that a congruence of curves is well-defined

Synopsis

  • Function: check
  • Usage:
    check f
    check(n,f)
  • Inputs:
    • n, an integer, (optional with default value 5)
    • f, a congruence of curves
  • Optional inputs:
    • UserMode => ..., default value null,
    • Verbose => ..., default value false,
  • Outputs:
    • a congruence of curves, the same object passed as input, but an error is thrown if the congruence fails when evaluated on n random points.

Ways to use this method:

  • "check(CongruenceOfCurves)"
  • check(ZZ,CongruenceOfCurves) -- check that a congruence of curves is well-defined
