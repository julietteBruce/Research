SpecialFanoFourfolds :: map(SpecialGushelMukaiFourfold)

map(SpecialGushelMukaiFourfold) -- associated quadratic map

Synopsis

  • Function: map
  • Usage:
    map X
  • Inputs:
    • X, a special Gushel-Mukai fourfold, containing a surface $S\subset Y$, where $Y\subset\mathbb{P}^8$ is the unique del Pezzo fivefold containing $X$
  • Optional inputs:
    • Degree => ..., default value null,
    • DegreeLift => ..., default value null,
    • DegreeMap => ..., default value null,
  • Outputs:
    • a rational map, the rational map from $Y$ defined by the linear system of quadrics through $S$

Ways to use this method:

  • map(SpecialGushelMukaiFourfold) -- associated quadratic map
