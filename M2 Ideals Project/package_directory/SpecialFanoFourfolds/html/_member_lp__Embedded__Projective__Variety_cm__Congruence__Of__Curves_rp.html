SpecialFanoFourfolds :: member(EmbeddedProjectiveVariety,CongruenceOfCurves)

member(EmbeddedProjectiveVariety,CongruenceOfCurves) -- test membership in a congruence of curves

Synopsis

  • Function: member
  • Usage:
    member(C,f)
  • Inputs:
    • C, an embedded projective variety, a curve
    • f, a congruence of curves
  • Outputs:
    • a Boolean value, whether the curve $C$ belongs to the congruence $f$

See also

  • CongruenceOfCurves EmbeddedProjectiveVariety -- get the curve of a congruence passing through a point

Ways to use this method:

  • member(EmbeddedProjectiveVariety,CongruenceOfCurves) -- test membership in a congruence of curves
