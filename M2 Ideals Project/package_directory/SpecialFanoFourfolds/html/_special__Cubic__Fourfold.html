SpecialFanoFourfolds :: specialCubicFourfold

specialCubicFourfold -- make a special cubic fourfold

Synopsis

  • Usage:
    specialCubicFourfold(S,X)
    specialCubicFourfold(S,X,NumNodes=>n)
  • Inputs:
    • S, an embedded projective variety, an irreducible surface $S\subset\mathbb{P}^5$, which has as singularities only a finite number $n\geq 0$ of non-normal nodes (this number $n$ should be passed with the option NumNodes, otherwise it is obtained using a probabilistic method)
    • X, an embedded projective variety, a smooth cubic fourfold $X\subset \mathbb{P}^5$ containing the surface $S$
  • Optional inputs:
    • InputCheck => ..., default value 1
    • NumNodes => ..., default value null
    • Verbose => ..., default value false, request verbose feedback
  • Outputs:
    • a special cubic fourfold, the special cubic fourfold corresponding to the pair $(S,X)$
