SpecialFanoFourfolds :: specialFourfold(String,ZZ)

specialFourfold(String,ZZ) -- load a prebuilt example of fourfold

Synopsis

  • Function: specialFourfold
  • Usage:
    specialFourfold(str,i)
  • Inputs:
    • str, a string, such as "prebuilt-example-in-P5" or "prebuilt-example-in-P7".
    • i, an integer
  • Optional inputs:
    • InputCheck => ..., default value 1, make a Hodge-special fourfold
    • NumNodes => ..., default value null, make a Hodge-special fourfold
    • Verbose => ..., default value false, request verbose feedback
  • Outputs:
    • a Hodge-special fourfold, the i-th example of fourfold in accordance with some classification (e.g., specialFourfold("prebuilt-example-in-P5",i) is the same as source trisecantFlop(i).

Description

This function requires the package TrisecantFlops. If not present the user will be asked to automatically install the package.

See also

  • trisecantFlop -- examples of trisecant flops
