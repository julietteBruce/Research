SpecialFanoFourfolds :: SpecialFanoFourfolds

SpecialFanoFourfolds -- A package for working with Hodge-special fourfolds

Description

This package contains several tools related to the rationality problem for cubic fourfolds, Gushel-Mukai fourfolds, and some other special Fano fourfolds. See arXiv:2204.11518 for some applications.

The following are some references that have benefited from this package.

References

  • M. Hoff and G. S., Explicit constructions of K3 surfaces and unirational Noether-Lefschetz divisors, available at arXiv:2110.15819 (2021).
  • G. S., Some new rational Gushel fourfolds, available at arXiv:2003.07809 (2020).
  • G. S., On some families of Gushel-Mukai fourfolds, available at arXiv:2002.07026 (2020).
  • M. Hoff and G. S., New examples of rational Gushel-Mukai fourfolds, available at arXiv:1910.12838 (2020).
  • F. Russo and G. S., Trisecant Flops, their associated K3 surfaces and the rationality of some Fano fourfolds, available at arXiv:1909.01263 (2020).
  • F. Russo and G. S., Explicit rationality of some cubic fourfolds, available at arXiv:1811.03502 (2019).
  • F. Russo and G. S., Congruences of 5-secant conics and the rationality of some admissible cubic fourfolds, available at arXiv:1707.00999 (2018).
