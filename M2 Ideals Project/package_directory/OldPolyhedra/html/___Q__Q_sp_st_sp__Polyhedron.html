OldPolyhedra :: QQ * Polyhedron

QQ * Polyhedron -- rescales a polyhedron by a given positive factor

Synopsis

  • Operator: *
  • Usage:
    Q = k * P
  • Inputs:
    • k, a rational number, ZZ or QQ, strictly positive
    • P, a convex polyhedron
  • Outputs:
    • Q, a convex polyhedron

Description

Rescales the Polyhedron by the strictly positive factor k. 

i1 : P = crossPolytope 3

o1 = {ambient dimension => 3           }
      dimension of lineality space => 0
      dimension of polyhedron => 3
      number of facets => 8
      number of rays => 0
      number of vertices => 6

o1 : Polyhedron
 
i2 : k = 3

o2 = 3
 
i3 : Q = k * P

o3 = {ambient dimension => 3           }
      dimension of lineality space => 0
      dimension of polyhedron => 3
      number of facets => 8
      number of rays => 0
      number of vertices => 6

o3 : Polyhedron
 
i4 : vertices Q

o4 = | -3 3 0  0 0  0 |
     | 0  0 -3 3 0  0 |
     | 0  0 0  0 -3 3 |

              3        6
o4 : Matrix QQ  <--- QQ
