OldPolyhedra :: Fan * Fan

Fan * Fan -- computes the direct product

Synopsis

  • Operator: *
  • Usage:
    F = F1 * F2
  • Inputs:
    • F1, an instance of the type Fan
    • F2, an instance of the type Fan
  • Outputs:
    • F, an instance of the type Fan

Description

Computes the direct product of two fans. This is the fan given by C=C1 x C2 for all cones C1 in F1 and C2 in F2, in the direct product of the ambient spaces.

See also directProduct(Fan,Fan). 

i1 : F1 = normalFan hypercube 1

o1 = {ambient dimension => 1         }
      number of generating cones => 2
      number of rays => 2
      top dimension of the cones => 1

o1 : Fan
 
i2 : F2 = normalFan hypercube 2

o2 = {ambient dimension => 2         }
      number of generating cones => 4
      number of rays => 4
      top dimension of the cones => 2

o2 : Fan
 
i3 : F = F1 * F2

o3 = {ambient dimension => 3         }
      number of generating cones => 8
      number of rays => 6
      top dimension of the cones => 3

o3 : Fan
 
i4 : F == normalFan hypercube 3

o4 = true
