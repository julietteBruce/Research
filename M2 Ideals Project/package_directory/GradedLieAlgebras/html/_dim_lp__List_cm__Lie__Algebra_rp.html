GradedLieAlgebras :: dim(List,LieAlgebra)

dim(List,LieAlgebra) -- compute the dimension

Synopsis

  • Function: dim
  • Usage:
    d=dim(s,L)
  • Inputs:
    • s, a list, the weight
    • L, an instance of the type LieAlgebra,
  • Outputs:
    • d, an integer, the dimension of $L$ in multi-degree $s$.

Description

i1 : L = lieAlgebra({a,b,c},Weights=>{{1,0},{2,1},{3,2}},
           Signs=>{1,1,1},LastWeightHomological=>true)

o1 = L

o1 : LieAlgebra
 
i2 : D= differentialLieAlgebra({0_L,a a,a b})

o2 = D

o2 : LieAlgebra
 
i3 : J=lieIdeal({b b + 4 a c})

o3 = J

o3 : FGLieIdeal
 
i4 : Q=D/J

o4 = Q

o4 : LieAlgebra
 
i5 : dim({7,4},Q)

o5 = 1
