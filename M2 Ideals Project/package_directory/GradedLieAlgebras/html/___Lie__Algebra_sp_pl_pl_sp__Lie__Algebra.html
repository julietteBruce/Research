GradedLieAlgebras :: LieAlgebra ++ LieAlgebra

LieAlgebra ++ LieAlgebra -- direct sum of Lie algebras

Synopsis

  • Operator: ++
  • Usage:
    S = L++M
  • Inputs:
    • L, an instance of the type LieAlgebra,
    • M, an instance of the type LieAlgebra,
  • Outputs:
    • S, an instance of the type LieAlgebra, the direct sum of $L$ and $M$
