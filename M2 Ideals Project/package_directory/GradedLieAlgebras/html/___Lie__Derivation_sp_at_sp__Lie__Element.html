GradedLieAlgebras :: LieDerivation @ LieElement

LieDerivation @ LieElement -- formal application of a derivation to a Lie element

Synopsis

  • Operator: @
  • Usage:
    y = d@x
  • Inputs:
    • d, an instance of the type LieDerivation,
    • x, an instance of the type LieElement, $x$ is an element in the source of $d$
  • Outputs:
    • y, an instance of the type LieElement, $y$ is the formal application of $d$ to the Lie element $x$
