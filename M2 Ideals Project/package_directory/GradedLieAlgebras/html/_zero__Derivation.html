GradedLieAlgebras :: zeroDerivation

zeroDerivation -- make a derivation from the zero map

Synopsis

  • Usage:
    d=zeroDerivation(L)
  • Inputs:
    • L, an instance of the type LieAlgebra,
  • Outputs:
    • d, an instance of the type LieDerivation,

Description

The zero map from $L$ to $L$ is a derivation.

i1 : L = lieAlgebra{a,b}

o1 = L

o1 : LieAlgebra
 
i2 : dL=differential L

o2 = 0

o2 : LieDerivation
 
i3 : d=zeroDerivation L

o3 = 0

o3 : LieDerivation
 
i4 : dL===d

o4 = true

Ways to use zeroDerivation :

  • "zeroDerivation(LieAlgebra)"

For the programmer

The object zeroDerivation is a method function.
