GradedLieAlgebras :: Number ExtElement

Number ExtElement -- multiplication of a number and an Ext-algebra element

Synopsis

  • Operator: SPACE
  • Usage:
    u = a x
  • Inputs:
    • a, a number,
    • x, an instance of the type ExtElement, $x$ is of type $E$, where $E$ is of type ExtAlgebra
  • Outputs:
    • u, an instance of the type ExtElement, $u$ is of type $E$, the product $a*x$

Description

The symbol SPACE is used as notation for multiplication by scalars. The scalars are of type Number if the field is QQ, otherwise the scalars are of type RingElement.

i1 : L = lieAlgebra{a,b,c}/{a b,b c}

o1 = L

o1 : LieAlgebra
 
i2 : E = extAlgebra(3,L)

o2 = E

o2 : ExtAlgebra
 
i3 : (-3) (2 ext_3-ext_4)

o3 =  - 6ext_3 + 3ext_4

o3 : E
