GradedLieAlgebras :: LieAlgebraMap == LieAlgebraMap

LieAlgebraMap == LieAlgebraMap -- whether two Lie algebra homomorphisms are defined in the same way

Synopsis

  • Operator: ==
  • Usage:
    f==g
  • Inputs:
    • f, an instance of the type LieAlgebraMap,
    • g, an instance of the type LieAlgebraMap,
  • Outputs:
    • b, a Boolean value,

Description

Two objects f and g of type LieAlgebraMap satisfies f==g if source f==source g, target f==target g, and the values of the generators of source f under f considered as elements in target g are the same as the values of the generators of source g under g.
