GradedLieAlgebras :: basis(ZZ,ZZ,ExtAlgebra)

basis(ZZ,ZZ,ExtAlgebra) -- compute a basis

Synopsis

  • Function: basis
  • Usage:
    b=basis(m,n,E)
  • Inputs:
    • m, an integer,
    • n, an integer,
    • E, an instance of the type ExtAlgebra,
  • Optional inputs:
    • Degree => ..., default value null,
    • Limit => ..., default value infinity,
    • SourceRing => ..., default value null,
    • Strategy => ..., default value null,
    • Truncate => ..., default value false,
    • Variables => ..., default value null,
  • Outputs:
    • b, a list, a basis for the part of $E$ of first degree $m$ and homological degree $n$.

Description

The optional inputs given above are not relevant for Lie algebras.
