GradedLieAlgebras :: Number LieDerivation

Number LieDerivation -- multiplication of a number and a derivation

Synopsis

  • Operator: SPACE
  • Usage:
    e = a d
  • Inputs:
    • a, a number,
    • d, an instance of the type LieDerivation,
  • Outputs:
    • e, an instance of the type LieDerivation,

Description

The symbol SPACE is used as notation for multiplication by scalars. The scalars are of type Number if the field is QQ, otherwise the scalars are of type RingElement.

i1 : L = lieAlgebra{a,b}

o1 = L

o1 : LieAlgebra
 
i2 : d = lieDerivation{a a b,b b a}

o2 = d

o2 : LieDerivation
 
i3 : describe(-2 d)

o3 = a => 2 (a b a)
     b =>  - 2 (b b a)
     map => id_L
     sign => 0
     weight => {2, 0}
     source => L
     target => L
