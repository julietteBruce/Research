GradedLieAlgebras :: dim(ZZ,VectorSpace)

dim(ZZ,VectorSpace) -- compute the dimension

Synopsis

  • Function: dim
  • Usage:
    d=dim(m,V)
  • Inputs:
    • m, an integer,
    • V, an instance of the type VectorSpace, an instance of type VectorSpace
  • Outputs:
    • d, an integer, the dimension of $V$ in first degree $m$.
