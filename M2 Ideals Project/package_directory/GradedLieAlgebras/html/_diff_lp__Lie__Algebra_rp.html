GradedLieAlgebras :: diff(LieAlgebra)

diff(LieAlgebra) -- get the differential of the generators

Synopsis

  • Function: diff
  • Usage:
    x=diff(L)
  • Inputs:
    • L, an instance of the type LieAlgebra,
  • Outputs:
    • x, a list, the list of the differential of the generators

Description

i1 : L=lieAlgebra({a,b},Signs=>1,Weights=>{{1,0},{2,1}},LastWeightHomological=>true)

o1 = L

o1 : LieAlgebra
 
i2 : D=differentialLieAlgebra{0_L,a a}

o2 = D

o2 : LieAlgebra
 
i3 : diff D

o3 = {0, (a a)}

o3 : List
 
i4 : describe D

o4 = generators => {a, b}
     Weights => {{1, 0}, {2, 1}}
     Signs => {1, 1}
     ideal => {}
     ambient => L
     diff => {0, (a a)}
     Field => QQ
     computedDegree => 0
