GradedLieAlgebras :: FGLieIdeal

FGLieIdeal -- the class of all finitely generated Lie ideals

Description

This type represents finitely generated Lie ideals. It is a subtype of LieIdeal, but not of FGSubAlgebra.

i1 : L = lieAlgebra{a,b}

o1 = L

o1 : LieAlgebra
 
i2 : S=lieIdeal{a}
