GradedLieAlgebras :: LieAlgebraMap \\ List

LieAlgebraMap \\ List -- formal application of a Lie map to every element in a list

Synopsis

  • Operator: \\
  • Usage:
    y = f\\x
  • Inputs:
    • f, an instance of the type LieAlgebraMap,
    • x, a list, the elements in $x$ belong to the source of $f$
  • Outputs:
    • y, a list, the elements in $y$ are the formal application of $f$ to the elements in $x$

Description

i1 : L = lieAlgebra{a,b}

o1 = L

o1 : LieAlgebra
 
i2 : M = L/{a a b}

o2 = M

o2 : LieAlgebra
 
i3 : f = map(M,L)

o3 = f

o3 : LieAlgebraMap
 
i4 : b = basis(4,L)

o4 = {(a a b a), (b a b a), (b b b a)}

o4 : List
 
i5 : f\\b

o5 = {(a a b a), (b a b a), (b b b a)}

o5 : List
 
i6 : f\b

o6 = {0, 0, (b b b a)}

o6 : List
