GradedLieAlgebras :: listMultiply

listMultiply -- multiplication of lists

Synopsis

  • Usage:
    m=listMultiply(x,y)
  • Inputs:
    • x, a list,
    • y, a list, $x$ and $y$ are lists of Lie elements from the same Lie algebra or lists of Ext-elements from the same Ext-algebra
  • Outputs:
    • m, a list, of lists

Description

The first list in the output consists of the first element in $x$ multiplied by all elements in $y$.

i1 : K=lieAlgebra({a,b,c},Signs=>1)/{a a,b b- 2 c a}

o1 = K

o1 : LieAlgebra
