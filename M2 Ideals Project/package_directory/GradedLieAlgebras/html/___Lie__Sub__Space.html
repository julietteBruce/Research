GradedLieAlgebras :: LieSubSpace

LieSubSpace -- the class of all Lie subspaces

Description

This type represents Lie subspaces, and has the subtypes LieIdeal, FGLieIdeal, LieSubAlgebra, FGLieSubAlgebra. A Lie subspace need not be invariant under the differential.

i1 : L = lieAlgebra{a,b}

o1 = L

o1 : LieAlgebra
 
i2 : S=lieSubSpace{a b}

o2 = S

o2 : LieSubSpace
 
i3 : I=lieIdeal{a b}

o3 = I

o3 : FGLieIdeal
 
i4 : instance(S,LieSubAlgebra)

o4 = false
 
i5 : dims(1,3,S)

o5 = {0, 1, 0}

o5 : List
 
i6 : dims(1,3,I)

o6 = {0, 1, 2}

o6 : List
