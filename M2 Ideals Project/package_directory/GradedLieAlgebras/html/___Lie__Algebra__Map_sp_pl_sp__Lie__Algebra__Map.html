GradedLieAlgebras :: LieAlgebraMap + LieAlgebraMap

LieAlgebraMap + LieAlgebraMap -- addition of Lie homomorphisms

Synopsis

  • Operator: +
  • Usage:
    u = f + g
  • Inputs:
    • f, an instance of the type LieAlgebraMap,
    • g, an instance of the type LieAlgebraMap,
  • Outputs:
    • u, an instance of the type LieAlgebraMap, the sum of $f$ and $g$

Description

The symbol + is used as infix notation for the addition of Lie homomorphisms.

i1 : M = lieAlgebra{a,b}

o1 = M

o1 : LieAlgebra
 
i2 : L = lieAlgebra{a,b}

o2 = L

o2 : LieAlgebra
 
i3 : f = map(L,M,{a,b})

o3 = f

o3 : LieAlgebraMap
 
i4 : g = map(L,M,{b,a})

o4 = g

o4 : LieAlgebraMap
 
i5 : describe(f+g)

o5 = a => a + b
     b => a + b
     source => M
     target => L
