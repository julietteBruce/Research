GradedLieAlgebras :: ExtElement + ExtElement

ExtElement + ExtElement -- addition of Ext-algebra elements

Synopsis

  • Operator: +
  • Usage:
    u=x + y
  • Inputs:
    • x, an instance of the type ExtElement, $x$ is of type $E$, where $E$ is of type ExtAlgebra
    • y, an instance of the type ExtElement, y is of type E
  • Outputs:
    • u, an instance of the type ExtElement, $u$ is of type $E$, the sum of $x$ and $y$

Description

The symbol + is used as infix notation for the Ext-algebra addition. The elements $x$ and $y$ must have the same weight and sign.
