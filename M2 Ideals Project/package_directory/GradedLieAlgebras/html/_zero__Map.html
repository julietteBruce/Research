GradedLieAlgebras :: zeroMap

zeroMap -- make the zero map

Synopsis

  • Usage:
    f=zeroMap(M,L)
  • Inputs:
    • M, an instance of the type LieAlgebra,
    • L, an instance of the type LieAlgebra,
  • Outputs:
    • f, an instance of the type LieAlgebraMap,

Description

The zero map from $L$ to $M$ is a Lie homomorphism

i1 : L = lieAlgebra{a,b}

o1 = L

o1 : LieAlgebra
 
i2 : M = lieAlgebra{a,b}

o2 = M

o2 : LieAlgebra
 
i3 : f=zeroMap(M,L)

o3 = 0

o3 : LieAlgebraMap
 
i4 : describe f

o4 = a => 0
     b => 0
     source => L
     target => M

For the programmer

The object zeroMap is a method function.
