HigherCIOperators :: exteriorMultiplication

exteriorMultiplication -- "multiplication maps in the exterior algebra"

Synopsis

  • Usage:
    mu = exteriorMultiplication n
  • Inputs:
    • n, an integer, positive integer
  • Outputs:
    • mu, a hash table, $mu\{p,q\}\wedge^p k^n\otimes \wedge^q k^n \to \wedge^{p+q}k^n$

Description

The basis of each $\wedge^p k^n$ is given in lex order.

Caveat

This is not the order of the basis in the output of koszul ff; rather, use trueKoszul ff.

See also

  • trueKoszul -- "Makes Koszul complex, with bases sorted in lex"

Ways to use exteriorMultiplication :

  • "exteriorMultiplication(ZZ)"

For the programmer

The object exteriorMultiplication is a method function.
