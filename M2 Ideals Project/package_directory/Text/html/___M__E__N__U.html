MENU -- hypertext menu item

Description

This hyeprtext item is intended to be used at most once per page, at the end. When converted by info, it produces the Menu at the end of the page. 
i1 : DIV { PARA "Hi there.",
         MENU { "some topics", TO "topic a", TO "topic b", "more topics", TO "topic c" } }

o1 = Hi there.

     Menu
     ----

     some topics

       * "topic a"
       * "topic b"

     more topics

       * "topic c"

o1 : DIV
 
i2 : info oo

o2 = Hi there.

     * Menu:


     some topics
     * topic a::
     * topic b::

     more topics
     * topic c::
 
i3 : html ooo

o3 = <div>
       <p>Hi there.</p>
       <div>
         <h3>Menu</h3>
         <h4>some topics</h4>
         <ul>
           <li>
     <span><tt>topic a</tt> (missing documentation)<!--tag: topic a-->
     </span>      </li>
           <li>
     <span><tt>topic b</tt> (missing documentation)<!--tag: topic b-->
     </span>      </li>
         </ul>
         <h4>more topics</h4>
         <ul>
           <li>
     <span><tt>topic c</tt> (missing documentation)<!--tag: topic c-->
     </span>      </li>
         </ul>
       </div>
     </div>

For the programmer

The object MENU is an intermediate markup type, with ancestor classes HypertextContainer < Hypertext < BasicList < Thing.
