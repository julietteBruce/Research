LATER -- a hypertext element for lazy evaluation

Synopsis

  • Usage:
    LATER {f}
  • Inputs:
    • f, a function
  • Outputs:
    • an instance of the type LATER

Description

When the resulting hypertext element is processed later for conversion to html or for printing, the function f will be evaluated (with no arguments) and the hypertext it returns will be formatted. 
i1 : f = () -> DIV { "hi there" }

o1 = f

o1 : FunctionClosure
 
i2 : LATER {f}

o2 = hi there

o2 : LATER
 
i3 : peek oo

o3 = LATER{f}

For the programmer

The object LATER is an intermediate markup type, with ancestor classes Hypertext < BasicList < Thing.
