LI -- hypertext list item

Synopsis

  • Usage:
    LI x
  • Inputs:
    • x
  • Outputs:
    • an instance of the type LI

Description

Entries in a UL list should be of type LI, but conversion to type LI is done automatically for the user.

See also

  • hypertext -- prepare hypertext for display

For the programmer

The object LI is a markup type, with ancestor classes HypertextContainer < Hypertext < BasicList < Thing.
