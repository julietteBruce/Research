validate -- validate a hypertext object

Synopsis

  • Usage:
    validate x
  • Inputs:
    • x, a markup list, a hypertext object to validate
  • Outputs:
    • x, a markup list, the input hypertext object, possibly with errors fixed
  • Consequences:
    • The hypertext is checked for validity, to ensure that the HTML code returned by html validate x is valid.

Description

This function is somewhat provisional. In particular, it is hard to check everything, because our hypertext format includes some entities of class IntermediateMarkUpType that don't correspond directly to HTML. Either those will have to be eliminated, or a more-final type of hypertext, convertible immediately to HTML, will have to be developed.

Ways to use validate :

  • "validate(Hypertext)"

For the programmer

The object validate is a method function.
