style(Hypertext)

Synopsis

  • Function: style
  • Usage:
    style (x,opts)
  • Inputs:
    • x, a markup list, hypertext
  • Consequences:
    • Returns a new Hypertext object with its style option updated.

Description

i1 : d = DIV { "Hello" }

o1 = Hello

o1 : DIV
 
i2 : d = style(d, "font-weight" => "bold")

o2 = Hello

o2 : DIV
 
i3 : peek d

o3 = DIV{"Hello", style => font-weight:bold;}
