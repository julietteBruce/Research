ExteriorIdeals :: stronglyStableIdeal

stronglyStableIdeal -- compute the smallest strongly stable ideal in an exterior algebra containing a given monomial ideal

Synopsis

  • Usage:
    stronglyStableIdeal I
  • Inputs:
    • I, a monomial ideal of an exterior algebra
  • Outputs:
    • an ideal, the smallest strongly stable ideal containing I

Description

Example:

i1 : E=QQ[e_1..e_4,SkewCommutative=>true]

o1 = E

o1 : PolynomialRing, 4 skew commutative variables
 
i2 : stronglyStableIdeal ideal {e_2*e_3}

o2 = ideal (e e , e e , e e )
             1 2   1 3   2 3

o2 : Ideal of E

See also

  • isStronglyStableIdeal -- whether a monomial ideal in an exterior algebra is strongly stable

Ways to use stronglyStableIdeal :

  • "stronglyStableIdeal(Ideal)"

For the programmer

The object stronglyStableIdeal is a method function.
