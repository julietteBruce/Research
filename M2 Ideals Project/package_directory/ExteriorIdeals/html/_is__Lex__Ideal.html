ExteriorIdeals :: isLexIdeal

isLexIdeal -- whether an ideal is lex

Synopsis

  • Usage:
    isLexIdeal I
  • Inputs:
    • I, a monomial ideal of an exterior algebra
  • Outputs:
    • a Boolean value, true whether ideal I is lex

Description

Other acknowledgements:

This method was taken from Chris Francisco's package: LexIdeals, which is available at LexIdeals

Examples:

i1 : E=QQ[e_1..e_4,SkewCommutative=>true]

o1 = E

o1 : PolynomialRing, 4 skew commutative variables
 
i2 : isLexIdeal ideal {e_1*e_2,e_2*e_3}

o2 = false
 
i3 : isLexIdeal ideal {e_1*e_2,e_1*e_3,e_1*e_4,e_2*e_3}

o3 = true

See also

  • lexIdeal -- compute the lex ideal with a given Hilbert function in an exterior algebra

Ways to use isLexIdeal :

  • "isLexIdeal(Ideal)"

For the programmer

The object isLexIdeal is a method function.
