ExteriorIdeals :: isHilbertSequence

isHilbertSequence -- whether the given sequence is a Hilbert sequence

Synopsis

  • Usage:
    isHilbertSequence(l,E)
  • Inputs:
    • l, a list of integers
    • E, an exterior algebra
  • Outputs:
    • a Boolean value, true whether the sequence l satisfies the Kruskal-Katona theorem in the exterior algebra E

Description

Example:

i1 : E=QQ[e_1..e_4,SkewCommutative=>true]

o1 = E

o1 : PolynomialRing, 4 skew commutative variables
 
i2 : isHilbertSequence({1,4,3,1,0},E)

o2 = true
 
i3 : isHilbertSequence({1,4,3,1,1},E)

o3 = false
