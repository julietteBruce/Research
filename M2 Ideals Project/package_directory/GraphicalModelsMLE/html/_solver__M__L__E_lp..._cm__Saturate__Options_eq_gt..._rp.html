GraphicalModelsMLE :: solverMLE(...,SaturateOptions=>...)

solverMLE(...,SaturateOptions=>...) -- use options from "saturate"

Synopsis

  • Usage:
    solverMLE(G,U,SaturateOptions=>L)
  • Inputs:
    • L, a list, of options to set up saturation. Accepts any option from the function saturate

Description

Default SaturateOptions in solverMLE are the default options in saturate. All optional input in saturate is allowed.

i1 : G=graph{{1,2},{2,3},{3,4},{1,4}}

o1 = Graph{1 => {2, 4}}
           2 => {1, 3}
           3 => {2, 4}
           4 => {1, 3}

o1 : Graph
 
i2 : U=random(ZZ^4,ZZ^4)

o2 = | 8 8 8 8 |
     | 1 3 8 5 |
     | 3 3 5 2 |
     | 7 7 7 3 |

              4        4
o2 : Matrix ZZ  <--- ZZ
 
i3 : solverMLE(G,U,SaturateOptions => {DegreeLimit=>1, MinimalGenerators => false})

o3 = (-5.76639, | 8.1875 6.3125  1.5433 2.625   |, 5)
                | 6.3125 5.1875  1.25   2.11114 |
                | 1.5433 1.25    1.5    2.25    |
                | 2.625  2.11114 2.25   5.25    |

o3 : Sequence

Further information

  • Default value: new OptionTable from {Strategy => null, BasisElementLimit => infinity, DegreeLimit => {}, MinimalGenerators => true, PairLimit => infinity}
  • Function: solverMLE -- Maximum likelihood estimate of a loopless mixed graph
  • Option key: SaturateOptions -- optional input to use options "saturate"
