MonodromySolver :: getTrackTime

getTrackTime -- elapsed time taken by solver

Synopsis

  • Usage:
    t = getTrackTime G
  • Inputs:
    • G, an instance of the type HomotopyGraph,
  • Outputs:
    • t, a real number,

Description

This only works on graphs which have been modified by a solver.

See also

  • elapsedTime -- time a computation using time elapsed

Ways to use getTrackTime :

  • "getTrackTime(HomotopyGraph)"

For the programmer

The object getTrackTime is a method function.
