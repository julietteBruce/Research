SpaceCurves :: quarticSurfaceRational

quarticSurfaceRational -- creates a QuarticSurfaceRational

Description

This function takes a polynomial ring in 4 variables over a field as the coordinate ring of P3 and creates a rational quartic surface with a double line as a QuarticSurfaceRational in this ring. To do this We blow up P2 at 10 points, listed in BlowUpPoints, and map the blownup surface to P3. The map is specified as a rational map P2->P3 given in MapToP3.

Synopsis

  • Usage:
    X = quadricSurface(R)
  • Inputs:
    • R, a ring
  • Outputs:
    • X, an instance of the type QuarticSurfaceRational
