SpaceCurves :: curve

curve -- generates a random curve

Description

The method curve generates a random curve with given input.

Synopsis

  • Usage:
    C = curve(D)
  • Inputs:
    • D, an instance of the type Divisor
  • Outputs:
    • C, an instance of the type Curve
