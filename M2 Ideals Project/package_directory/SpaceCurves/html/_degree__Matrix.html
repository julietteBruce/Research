SpaceCurves :: degreeMatrix

degreeMatrix -- computes the Hilbert-Burch degree matrix from a Betti table of ACM curves

Synopsis

  • Usage:
    M = degreeMatrix B
  • Inputs:
    • B, a Betti tally, of an ACM curve
  • Outputs:
    • M, a matrix

Description

Given the Betti table of an ACM curve, this function returns the Hilbert-Burch degree matrix. 
i1 : B = generalACMBetti {-1,-1,2}

            0 1 2
o1 = total: 1 3 2
         0: 1 . .
         1: . 3 2

o1 : BettiTally
 
i2 : degreeMatrix B

o2 = | 1 1 |
     | 1 1 |
     | 1 1 |

              3        2
o2 : Matrix ZZ  <--- ZZ

Ways to use degreeMatrix :

  • "degreeMatrix(BettiTally)"

For the programmer

The object degreeMatrix is a method function.
