SpaceCurves :: dgTable

dgTable -- prints the table of (degree,genus) pairs

Description

dgTable prints the table of (degree,genus) pairs, where the horizontal axis is the degree and the vertical is the genus. The input can be a List of Divisor, Curve, PostulationChar or Ideal.

Ways to use dgTable :

  • "dgTable(List)"

For the programmer

The object dgTable is a method function.
