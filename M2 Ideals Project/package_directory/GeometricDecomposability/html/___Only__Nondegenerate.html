GeometricDecomposability > OnlyNondegenerate

OnlyNondegenerate -- restrict to nondegenerate geometric vertex decompositions

Description

Set to true to restrict the output of findOneStepGVD to return only the indeterminates for which their geometric vertex decomposition is nondegenerate. Default value false.

See also

  • findOneStepGVD -- for which indeterminates does there exist a geometric vertex decomposition
  • OnlyDegenerate -- restrict to degenerate geometric vertex decompositions

Functions with optional argument named OnlyNondegenerate :

  • "findOneStepGVD(...,OnlyNondegenerate=>...)"

For the programmer

The object OnlyNondegenerate is a symbol.
