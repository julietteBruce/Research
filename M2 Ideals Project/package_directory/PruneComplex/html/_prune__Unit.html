PruneComplex :: pruneUnit

pruneUnit -- Prunes a unit of a differential in a list of mutable matrices

Synopsis

  • Usage:
    D = pruneUnit(C, n, u, M)
  • Inputs:
    • C, a list, a list of mutable matrices defining the differentials of a complex
    • n, an integer, the index of the differential in C
    • u, a sequence, the coordinates of the unit in the differential
    • M, a list, if provided, will initialize the list of map back to the original complex
  • Optional inputs:
    • PruningMap => ..., default value true, Whether to compute a morphism of complexes
    • UnitTest => ..., default value isUnit, Limit which units are to be pruned
  • Outputs:
    • D, a list, a list of modified mutable matrices

Description

Prune the unit in location {u} of the {n}-th differential in chain complex {C}. Moves the unit to the end, clears row and column, while keeping the neighboring differentials compatible.

Caveat

Currently only works with 1x1 units.

See also

  • pruningMap
  • pruneDiff -- Prunes a single differential in a chain complex or list of mutable matrices

Ways to use pruneUnit :

  • "pruneUnit(List,ZZ,Sequence,List)"

For the programmer

The object pruneUnit is a method function with options.
