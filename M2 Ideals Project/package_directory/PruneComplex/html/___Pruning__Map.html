PruneComplex :: PruningMap

PruningMap -- Whether to compute a morphism of complexes

Description

When true, the pruning map to the original complex will be stored in C.cache.pruningMap.

Caveat

Setting PruningMap to false will improve performance, but in some cases the grading of the resulting object may be incorrect.
