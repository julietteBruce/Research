MinimalPrimes > isPrime(Ideal)

isPrime(Ideal) -- whether an ideal is prime

Synopsis

  • Function: isPrime
  • Usage:
    isPrime I
  • Inputs:
    • I, an ideal, in a polynomial ring or quotient of one
  • Optional inputs:
    • Strategy => a string, default value null, See minimalPrimes(...,Strategy=>...)
    • Verbosity => an integer, default value 0, See minimalPrimes(...,Verbosity=>...)
    • SquarefreeFactorSize
  • Outputs:
    • a Boolean value, true if I is a prime ideal, or false otherwise

Description

This function determines whether an ideal in a polynomial ring is prime.

i1 : R = QQ[a..d];
 
i2 : I = monomialCurveIdeal(R, {1, 5, 8})
