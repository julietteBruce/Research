EquivariantGB > egb

egb -- computes equivariant Gröbner bases

Synopsis

  • Usage:
    G = egb F
  • Inputs:
    • F, a list, a list of polynomials in a ring created by buildERing.
  • Optional inputs:
    • Algorithm => ..., default value Buchberger, algorithm choice for egb
    • OutFile => ..., default value null, where to send messages
    • PrincipalSyzygies (missing documentation) => ..., default value false,
    • Symmetrize (missing documentation) => ..., default value false,
  • Outputs:
    • G, a list, an equivariant Gröbner basis for the equivariant ideal generated by F.

Description

egb uses a variant of Buchberger's algorithm.

i1 : R = buildERing({symbol x}, {1}, QQ, 2);
 
i2 : egb {x_0 + x_1}

o2 = {x }
       0

o2 : List

Caveat

The output does not necessarily belong to the same ring as the input.

Ways to use egb :

  • "egb(List)"

For the programmer

The object egb is a method function with options.
