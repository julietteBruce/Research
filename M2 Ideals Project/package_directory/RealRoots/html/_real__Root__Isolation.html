RealRoots :: realRootIsolation

realRootIsolation -- a list that isolates the real roots of a rational univariate polynomial

Synopsis

  • Usage:
    realRootIsolation(f,r)
  • Inputs:
    • f, a ring element, a rational univariate polynomial
    • r, a real number, a positive rational number
  • Outputs:
    • a list, of intervals that contain all the real roots of f

Description

This method uses a Sturm sequence and a bisection method to isolate real solutions of f in intervals of length at most r.
