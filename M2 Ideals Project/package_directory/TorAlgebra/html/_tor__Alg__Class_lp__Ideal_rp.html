TorAlgebra :: torAlgClass(Ideal)

torAlgClass(Ideal) -- the class (w.r.t. multiplication in homology) of a local ring

Synopsis

  • Function: torAlgClass
  • Usage:
    torAlgClass R or torAlgClass I
  • Inputs:
    • I, an ideal, of a polynomial algebra contained in the irrelevant maximal ideal
  • Outputs:
    • a string, the (parametrized) class of the local ring obtained by localizing the quotient by I at the irrelevant maximal ideal, provided that this ring is non-zero and of codepth at most 3 or Gorenstein or Golod; otherwise "no class"

Description

See torAlgClass(QuotientRing).

i1 : Q = QQ[x,y,z];
 
i2 : torAlgClass (ideal(x^2,y^2,z^2))

o2 = C(3)
