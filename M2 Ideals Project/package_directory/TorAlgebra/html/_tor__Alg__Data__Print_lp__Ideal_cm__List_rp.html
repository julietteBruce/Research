TorAlgebra :: torAlgDataPrint(Ideal,List)

torAlgDataPrint(Ideal,List) -- print invariants of a local ring

Synopsis

  • Function: torAlgDataPrint
  • Usage:
    torAlgDataPrint (R,L) or torAlgDataPrint(I,L)
  • Inputs:
    • I, an ideal, of a polynomial algebra contained in the irrelevant maximal ideal
    • L, a list, of keys from the hash table returned by torAlgData
  • Outputs:
    • a string, of keys specified in L together with their values

Description

Extracts data from the hash table returned by torAlgData.

i1 : Q = QQ[x,y,z];
 
i2 : I = ideal(x*y,y*z,x^3,x^2*z,x*z^2-y^3,z^3);

o2 : Ideal of Q
 
i3 : torAlgDataPrint( I, {c, e, h, m, n, Class, p, q, r} )

o3 = c=3 e=3 h=0 m=6 n=2 Class=G p=0 q=1 r=3
