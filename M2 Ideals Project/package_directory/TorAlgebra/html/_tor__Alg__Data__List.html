TorAlgebra :: torAlgDataList

torAlgDataList -- list invariants of a local ring

Synopsis

  • Usage:
    torAlgDataList(R,L) or torAlgDataList(I,L)
  • Inputs:
    • R, a quotient ring, of a polynomial algebra by an ideal contained in the irrelevant maximal ideal
    • L, a list, of keys from the hash table returned by torAlgData
  • Outputs:
    • a list, of values corresponding to the keys specified in L

Description

Extracts data from the hash table returned by torAlgData. It is also possible to call the function on the defining ideal of R; see torAlgDataList(Ideal,List).

i1 : Q = QQ[x,y,z];
 
i2 : R = Q/ideal(x*y,y*z,x^3,x^2*z,x*z^2-y^3,z^3);
 
i3 : torAlgDataList( R, {m, n, Class, p, q, r, PoincareSeries, BassSeries} )

                                      2                    2    3    4
                               (1 + T)          2 + 2T - 2T  - T  + T
o3 = {6, 2, G, 0, 1, 3, ----------------------, ----------------------}
                                  2     3    4            2     3    4
                        1 - T - 5T  - 2T  + T   1 - T - 5T  - 2T  + T

o3 : List

Ways to use torAlgDataList :

  • "torAlgDataList(QuotientRing,List)"
  • torAlgDataList(Ideal,List) -- list invariants of a local ring

For the programmer

The object torAlgDataList is a method function.
