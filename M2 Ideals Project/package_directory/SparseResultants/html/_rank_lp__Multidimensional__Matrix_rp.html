SparseResultants :: rank(MultidimensionalMatrix)

rank(MultidimensionalMatrix) -- about the border rank of a multidimensional matrix

Synopsis

  • Function: rank
  • Usage:
    rank M
  • Inputs:
    • M, a multidimensional matrix
  • Outputs:
    • an integer, a lower bound for the (border) rank of M, which is calculated as the maximum rank of all the flattenings of M

Description

In some cases, we know that the returned integer r = rank M is exactly the border rank of M. For instance, this is the case if r<3 by a result of C. Raicu. In general, however, we obtain only a lower bound and it is not known how to calculate this rank exactly without resorting to elimination.
