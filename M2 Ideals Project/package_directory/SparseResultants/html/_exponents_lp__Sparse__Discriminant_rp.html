SparseResultants :: exponents(SparseDiscriminant)

exponents(SparseDiscriminant) -- get the matrix of exponents

Synopsis

  • Function: exponents
  • Usage:
    exponents D
  • Inputs:
    • D, an instance of the type SparseDiscriminant
  • Outputs:
    • the matrix of exponents from which the sparse discriminant D is created

Description

i1 : D = denseDiscriminant(2,2);

o1 : SparseDiscriminant (sparse discriminant associated to | 0 0 0 1 1 2 |)
                                                           | 0 1 2 0 1 0 |
 
i2 : M = exponents D

o2 = | 0 0 0 1 1 2 |
     | 0 1 2 0 1 0 |

              2        6
o2 : Matrix ZZ  <--- ZZ
 
i3 : assert(D === sparseDiscriminant M)
