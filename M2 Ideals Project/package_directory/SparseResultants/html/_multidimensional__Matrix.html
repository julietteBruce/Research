SparseResultants :: multidimensionalMatrix

multidimensionalMatrix -- make a multidimensional matrix

Synopsis

  • Usage:
    multidimensionalMatrix L
  • Inputs:
    • L, a list, a list of nested lists representing a hyper-rectangular array of ring elements.
  • Outputs:
    • a multidimensional matrix, the corresponding multidimensional matrix.

Description

i1 : multidimensionalMatrix {{0, 5}, {9, 3}, {7, 2}}

o1 = {{0, 5}, {9, 3}, {7, 2}}

o1 : 2-dimensional matrix of shape 3 x 2 over ZZ
 
i2 : multidimensionalMatrix {{{1, 0}, {4, 3}}, {{3, 1}, {5, 9}}}

o2 = {{{1, 0}, {4, 3}}, {{3, 1}, {5, 9}}}

o2 : 3-dimensional matrix of shape 2 x 2 x 2 over ZZ
 
i3 : multidimensionalMatrix {{{7/3, 8, 0}, {6, 8, 3}}, {{3, 8, 2}, {9, 2, 4}}, {{0, 2, 9}, {1, 9, 5}}, {{2, 8, 4}, {9, 7, 7}}}

        3
