SparseResultants :: SparseResultant Thing

SparseResultant Thing -- evaluate a sparse resultant

Synopsis

  • Operator: SPACE
  • Usage:
    R(f)
  • Inputs:
    • R, an instance of the type SparseResultant, associated to $n+1$ integral matrices $A_0,\ldots,A_n$ with $n$ rows.
    • a thing, $n+1$ Laurent polynomials $f = (f_0,\ldots,f_n)$ in $n$ variables $x=(x_1,\ldots,x_n)$, with $f_i = \sum_{\omega\in \{columns\ of\ A_i\}} a_{i,\omega} x^{\omega}$.
  • Outputs:
    • a ring element, the $(A_0,\ldots,A_n)$-resultant of $f_0,\ldots,f_n$.
