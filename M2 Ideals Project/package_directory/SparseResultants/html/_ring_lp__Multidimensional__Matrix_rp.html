ring(MultidimensionalMatrix) -- ring of a multidimensional matrix

Synopsis

  • Function: ring
  • Usage:
    ring M
  • Inputs:
    • M, a multidimensional matrix
  • Outputs:
    • a ring, the common ring of the entries of M

Description

o1 = {{{3, 8}, {3, 6}}, {{5, 1}, {4, 2}}}

o1 : 3-dimensional matrix of shape 2 x 2 x 2 over ZZ
 
i2 : ring M

o2 = ZZ

o2 : Ring

o3 : 3-dimensional matrix of shape 2 x 2 x 2 over QQ
