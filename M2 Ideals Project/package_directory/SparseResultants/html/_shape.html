SparseResultants :: shape

shape -- shape of a multidimensional matrix

Synopsis

  • Usage:
    shape M
  • Inputs:
    • M, a multidimensional matrix, an $n$-dimensional matrix of shape $k_1\times\cdots\times k_n$
  • Outputs:
    • a list, the list of integers $\{k_1, \ldots, k_n\}$

Description

i1 : M = multidimensionalMatrix {{{0, 8, 3}, {7, 3, 2}, {2, 7, 0}, {4, 8, 4}}, {{0, 8, 1}, {3, 1, 0}, {4, 7, 4}, {0, 6, 9}}}
