SparseResultants :: genericSkewMultidimensionalMatrix

genericSkewMultidimensionalMatrix -- make a generic skew symmetric multidimensional matrix of variables

Synopsis

  • Usage:
    genericSkewMultidimensionalMatrix(n,d)
  • Inputs:
    • (n,d), two positive integers.
  • Optional inputs:
    • CoefficientRing => ..., default value QQ
    • Variable => ..., default value "a"
  • Outputs:
    • the generic skew symmetric multidimensional matrix of shape $d\times\cdots\times d$ ($n$ times).

Description

An $n$-dimensional matrix $M$ is skew symmetric if for every permutation $s$ of the set $\{0,\ldots,n-1\}$ we have permute(M,s) == sign(s)*M.
