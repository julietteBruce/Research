SparseResultants :: MultidimensionalMatrix == MultidimensionalMatrix

MultidimensionalMatrix == MultidimensionalMatrix -- equality of two multidimensional matrices

Synopsis

  • Operator: ==
  • Usage:
    M == N
  • Inputs:
    • M, a multidimensional matrix
    • N, a multidimensional matrix
  • Outputs:
    • a Boolean value, true or false, depending on whether $M$ and $N$ are equal.

Description

i1 : M = multidimensionalMatrix {{7, 6}, {9, 3}, {1, 7}}

o1 = {{7, 6}, {9, 3}, {1, 7}}

o1 : 2-dimensional matrix of shape 3 x 2 over ZZ
 
i2 : N = multidimensionalMatrix {{7, 6}, {9, 4}, {1, 7}}

o2 = {{7, 6}, {9, 4}, {1, 7}}

o2 : 2-dimensional matrix of shape 3 x 2 over ZZ
 
i3 : N' = multidimensionalMatrix {{7, 6}, {9, 3}, {1, 7}}

o3 = {{7, 6}, {9, 3}, {1, 7}}

o3 : 2-dimensional matrix of shape 3 x 2 over ZZ
 
i4 : M == N

o4 = false
 
i5 : M == N'

o5 = true
 
i6 : M - N' == 0

o6 = true
