SparseResultants :: entries(MultidimensionalMatrix)

entries(MultidimensionalMatrix) -- lists the entries of a multidimensional matrix

Synopsis

  • Function: entries
  • Usage:
    entries M
  • Inputs:
    • M, a multidimensional matrix
  • Outputs:
    • a list, the hyperrectangular nested list of the entries of M

Description

i1 : M = randomMultidimensionalMatrix {3,3}

o1 = {{8, 1, 3}, {7, 8, 3}, {3, 7, 8}}

o1 : 2-dimensional matrix of shape 3 x 3 over ZZ
 
i2 : entries M

o2 = {{8, 1, 3}, {7, 8, 3}, {3, 7, 8}}

o2 : List
 
i3 : M = randomMultidimensionalMatrix {2,2,3}

o3 = {{{8, 5, 7}, {8, 5, 2}}, {{3, 6, 3}, {6, 8, 6}}}

o3 : 3-dimensional matrix of shape 2 x 2 x 3 over ZZ
 
i4 : entries M

o4 = {{{8, 5, 7}, {8, 5, 2}}, {{3, 6, 3}, {6, 8, 6}}}

o4 : List

Ways to use this method:

  • entries(MultidimensionalMatrix) -- lists the entries of a multidimensional matrix
