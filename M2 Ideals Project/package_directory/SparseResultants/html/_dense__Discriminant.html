SparseResultants :: denseDiscriminant

denseDiscriminant -- dense discriminant (classical discriminant)

Synopsis

  • Usage:
    denseDiscriminant(d,n)
    denseDiscriminant f
  • Inputs:
    • (d,n), a pair of two integers representing, respectively, the degree and the number of variables of a Laurent polynomial $f$.
    • f, a ring element, a Laurent polynomial of degree $d$ in $n$ variables.
  • Optional inputs:
    • Algorithm => ..., default value "Poisson"
    • CoefficientRing => ..., default value ZZ
  • Outputs:
    • for (d,n), this is the same as sparseDiscriminant exponentsMatrix "generic polynomial of degree d in n variables";
    • for f, this is the same as affineDiscriminant(f).
