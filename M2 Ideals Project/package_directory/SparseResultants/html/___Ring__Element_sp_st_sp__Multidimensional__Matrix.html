RingElement * MultidimensionalMatrix -- product of a scalar with a multidimensional matrix

Synopsis

  • Operator: *
  • Usage:
    e * N
  • Inputs:
    • e, a ring element
    • M, a multidimensional matrix
  • Outputs:
    • a multidimensional matrix, the product $e*M$
