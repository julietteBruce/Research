SparseResultants :: MultidimensionalMatrix _ Sequence

MultidimensionalMatrix _ Sequence -- get entry of multidimensional matrix

Synopsis

  • Operator: _
  • Usage:
    M_l
  • Inputs:
    • M, a multidimensional matrix, an $n$-dimensional matrix.
    • l, a sequence, a sequence of $n$ nonnegative integers.
  • Outputs:
    • a ring element, the $l$-th entry of the matrix $M$ (note that indexes start from 0).
