SparseResultants :: degreeDeterminant

degreeDeterminant -- degree of the hyperdeterminant of a generic multidimensional matrix

Synopsis

  • Usage:
    degreeDeterminant n
  • Inputs:
    • n, a list, a list of positive integers $n=\{n_1,n_2,\ldots\}$
  • Outputs:
    • an integer, the degree of the hyperdeterminant of a generic multidimensional matrix of shape $n_1\times n_2\times\ldots$

Description

i1 : n = {2,3,2}

o1 = {2, 3, 2}

o1 : List
 
i2 : time degreeDeterminant n
     -- used 0.000096435 seconds

o2 = 6
 
i3 : M = genericMultidimensionalMatrix n;

o3 : 3-dimensional matrix of shape 2 x 3 x 2 over ZZ[a     ..a     ]
                                                      0,0,0   1,2,1
