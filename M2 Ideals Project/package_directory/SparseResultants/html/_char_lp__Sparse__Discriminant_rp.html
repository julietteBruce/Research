SparseResultants :: char(SparseDiscriminant)

char(SparseDiscriminant) -- get the characteristic

Synopsis

  • Function: char
  • Usage:
    char D
  • Inputs:
    • D, an instance of the type SparseDiscriminant
  • Outputs:
    • an integer, the characteristic of the ring of the polynomial representing D

Description

i1 : D = denseDiscriminant(2,2,CoefficientRing=>ZZ/331);

o1 : SparseDiscriminant (sparse discriminant associated to | 0 0 0 1 1 2 | over ZZ/331)
                                                           | 0 1 2 0 1 0 |
 
i2 : char D

o2 = 331
 
i3 : char denseDiscriminant(2,2)

o3 = 0
