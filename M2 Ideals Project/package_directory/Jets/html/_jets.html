Jets > jets

jets -- compute the jets of an object

Synopsis

  • Optional inputs:
    • Projective => ..., default value false, Option for jets

Ways to use jets :

  • jets(ZZ,AffineVariety) -- the jets of an affine variety
  • jets(ZZ,Graph) -- the jets of a graph
  • "jets(ZZ,HyperGraph)" -- see jets(ZZ,Graph) -- the jets of a graph
  • jets(ZZ,Ideal) -- compute jets of a an ideal in a polynomial ring
  • jets(ZZ,PolynomialRing) -- compute jets of a polynomial ring
  • jets(ZZ,QuotientRing) -- the jets of an affine algebra
  • jets(ZZ,RingMap) -- the jets of a homomorphism of rings

For the programmer

The object jets is a method function with options.
