zeroCode -- zero code

Synopsis

  • Usage:
    zeroCode(F,n)
  • Inputs:
    • F, a Galois field,
    • n, an integer,
  • Outputs:
    • an instance of the type LinearCode,

Description

Constructs the linear code of length n over F whose only codeword is the zero codeword.
