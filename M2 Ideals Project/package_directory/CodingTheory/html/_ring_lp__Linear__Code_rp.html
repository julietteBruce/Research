ring(LinearCode) -- the ring of a code

Synopsis

  • Function: ring
  • Usage:
    ring LinearCode
  • Inputs:
    • C, an instance of the type LinearCode,
  • Outputs:
    • a ring,

Description

Given a code C, returns the ring that contains the entries of the generator matrix of C.

i1 : C = hammingCode(2, 3)
