PHCpack :: realSlice1D

realSlice1D -- computes a real slice for a one dimensional witness set

Synopsis

  • Usage:
    slc = realSlice1D(w)
  • Inputs:
    • w, a witness set, a witness set for a solution set
  • Optional inputs:
    • searchDelta => ..., default value .1, option of realSlice1D
    • searchNpoints => ..., default value 20, option of realSlice1D
    • searchTolerance => ..., default value .0001, option of realSlice1D
  • Outputs:
    • slc, a list, list of linear equations with the largest number of real solutions

Description

A real slice is a set of linear equations with the largest number of real solutions of the equations for a given witness set.

i1 : R = CC[x,y,z];

See also

  • intersectSlice -- intersects a witness set by a slice

Ways to use realSlice1D :

  • "realSlice1D(WitnessSet)"

For the programmer

The object realSlice1D is a method function with options.
