PHCpack :: solveSystem(...,randomSeed=>...)

solveSystem(...,randomSeed=>...) -- option to set the seed of the random number generators

Synopsis

  • Usage:
    solveSystem(...,randomSeed=>ZZ)

Description

Use randomSeed=>12345 to set the seed to 12345.

Further information

  • Default value: -1
  • Function: solveSystem -- a numerical blackbox solver
  • Option key: randomSeed -- seed for the random number generators

Functions with optional argument named randomSeed :

  • solveSystem(...,randomSeed=>...) -- option to set the seed of the random number generators
