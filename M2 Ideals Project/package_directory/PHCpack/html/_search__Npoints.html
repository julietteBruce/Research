PHCpack :: searchNpoints

searchNpoints -- option of realSlice1D

Description

Before the line search, a discretization of the range of the slices is computed. The value of searchNpoints sets the number of equidistant points in this range of slices.

Functions with optional argument named searchNpoints :

  • realSlice1D(...,searchNpoints=>...) -- option of realSlice1D
  • realSlice2D(...,searchNpoints=>...) -- option of realSlice2D

For the programmer

The object searchNpoints is a symbol.
