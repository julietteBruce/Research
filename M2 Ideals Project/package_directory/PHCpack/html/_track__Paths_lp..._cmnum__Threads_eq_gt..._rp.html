PHCpack :: trackPaths(...,numThreads=>...)

trackPaths(...,numThreads=>...) -- Option to define the number of threads in the path tracker.

Synopsis

  • Usage:
    trackPaths(...,numThreads=>ZZ)

Further information

  • Default value: 0
  • Function: trackPaths -- tracks paths defined by a typical homotopy between start and end systems
  • Option key: numThreads -- the number of threads in the path tracker

Functions with optional argument named numThreads :

  • mixedVolume(...,numThreads=>...) -- option to set the number of threads when solving a start system
  • solveSystem(...,numThreads=>...) -- option to set the number of threads
  • trackPaths(...,numThreads=>...) -- Option to define the number of threads in the path tracker.
