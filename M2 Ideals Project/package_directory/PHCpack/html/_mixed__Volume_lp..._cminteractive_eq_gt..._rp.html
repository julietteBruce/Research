PHCpack :: mixedVolume(...,interactive=>...)

mixedVolume(...,interactive=>...) -- option to switch to the interactive mode of phc -m

Synopsis

  • Usage:
    solveSystem(...,interactive=>true)

Further information

  • Default value: false
  • Function: mixedVolume -- computes mixed volume of a polynomial system
  • Option key: interactive -- flag to run phc -p or phc -m in interactive mode
