PHCpack :: StableMixedVolume

StableMixedVolume -- optional input for computation of the stable mixed volume

Description

Put StableMixedVolume=>true as an argument in the function mixedVolume to count solutions in affine space.

Functions with optional argument named StableMixedVolume :

  • mixedVolume(...,StableMixedVolume=>...) -- optional input for computation of the stable mixed volume

For the programmer

The object StableMixedVolume is a symbol.
