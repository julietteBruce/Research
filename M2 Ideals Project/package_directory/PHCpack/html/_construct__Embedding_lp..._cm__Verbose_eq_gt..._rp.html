PHCpack :: constructEmbedding(...,Verbose=>...)

constructEmbedding(...,Verbose=>...) -- option to specify whether additional output is wanted

Synopsis

  • Usage:
    constructEmbedding(...,Verbose=>Boolean)

Description

Use Verbose=>true for additional output which includes the input and solution file names used by phc -c.

Further information

  • Default value: false
  • Function: constructEmbedding -- constructs an embedding of a polynomial system
  • Option key: Verbose -- request verbose feedback
