scalarProduct(RootSystem,ZZ,ZZ) -- the scalar product of two fundamental weights

Synopsis

  • Function: scalarProduct

  • Inputs:
    • R, an instance of the type RootSystem,
    • i, an integer, the number of the first fundamental weight
    • j, an integer, the number of the second fundamental weight
  • Outputs:
