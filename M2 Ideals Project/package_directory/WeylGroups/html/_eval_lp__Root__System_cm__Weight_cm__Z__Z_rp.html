WeylGroups :: eval(RootSystem,Weight,ZZ)

eval(RootSystem,Weight,ZZ) -- evaluate the dual of a simple root at a Weight

Synopsis

  • Function: eval
  • Usage:
    eval(R,v,i)
  • Inputs:
    • R, an instance of the type RootSystem,
    • v, an instance of the type Weight,
    • i, an integer, the number of the simple root
  • Outputs:
    • an integer, the dual of the i-th simple root evaluated at the weight v

Description

i1 : R=rootSystemA(2)

o1 = RootSystem{...8...}

o1 : RootSystem
 
i2 : v=weight(R,{1,2})

o2 = | 1 |
     | 2 |

o2 : Weight
 
i3 : eval(R,v,1)

o3 = 1
