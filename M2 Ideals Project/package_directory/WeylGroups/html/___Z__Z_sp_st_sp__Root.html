WeylGroups > Root > ZZ * Root

ZZ * Root -- multiplication of a root by an integer

Synopsis

  • Operator: *
  • Usage:
    n*r
  • Inputs:
    • n, an integer,
    • r, an instance of the type Root,
  • Outputs:
    • an instance of the type Weight,

Description

i1 : R=rootSystemA(3)

o1 = RootSystem{...8...}

o1 : RootSystem
 
i2 : r=simpleRoot(R,1)

o2 = |  2 |
     | -1 |
     |  0 |

o2 : Root
 
i3 : 2*r

o3 = |  4 |
     | -2 |
     |  0 |

o3 : Weight

Note that if the integer is +1 or -1, we obtain a weight that is a root.

i4 : -1*r

o4 = | -2 |
     |  1 |
     |  0 |

o4 : Root
