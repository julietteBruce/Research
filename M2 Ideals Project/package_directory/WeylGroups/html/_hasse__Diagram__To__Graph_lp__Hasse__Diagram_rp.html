WeylGroups :: hasseDiagramToGraph(HasseDiagram)

hasseDiagramToGraph(HasseDiagram) -- turning a hasse diagram into a graph (intended for graphic representation)

Synopsis

  • Function: hasseDiagramToGraph
  • Usage:
    hasseDiagramToGraph(h)
  • Inputs:
    • h, an instance of the type HasseDiagram,
  • Optional inputs:
    • labels
  • Outputs:
    • an instance of the type HasseGraph,

Description

By default, the HasseGraph will have vertices with empty labels.

i1 : R=rootSystemA(3)

o1 = RootSystem{...8...}
