WeylGroups :: whoseReflection(WeylGroupElement)

whoseReflection(WeylGroupElement) -- the positive root whose reflection is a given element of a Weyl group

Synopsis

  • Function: whoseReflection
  • Usage:
    whoseReflection(w)
  • Inputs:
    • w, an instance of the type WeylGroupElement,
  • Outputs:
    • an instance of the type Root, the root whose corresponding reflection is w

Description

i1 : R=rootSystemD(5)

o1 = RootSystem{...8...}

o1 : RootSystem
 
i2 : w=reduce(R,{3})

o2 = WeylGroupElement{RootSystem{...8...}, |  1 |}
                                           |  2 |
                                           | -1 |
                                           |  2 |
                                           |  2 |

o2 : WeylGroupElement
 
i3 : if isReflection(w) then whoseReflection(w)

o3 = |  0 |
     | -1 |
     |  2 |
     | -1 |
     | -1 |

o3 : Root

Caveat

If w is not a reflection, an error occurs.
