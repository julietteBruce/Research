WeylGroups :: dynkinExponents(DynkinType)

dynkinExponents(DynkinType) -- the exponents of the Dynkin type

Synopsis

  • Function: dynkinExponents
  • Usage:
    dynkinExponents(D)
  • Inputs:
    • D, an instance of the type DynkinType,
  • Outputs:
    • a list, of lists of integers, the exponents listed in the tables in Bourbaki, Lie groups and Lie algebras, following chapter 6.

Description

i1 : R=rootSystemE(6)++rootSystemA(4)++rootSystemB(3)++rootSystemA(4);
 
i2 : dynkinExponents(dynkinType(R))

o2 = {{1, 2, 3, 4}, {1, 2, 3, 4}, {1, 3, 5}, {1, 4, 5, 7, 8, 11}}

o2 : List
