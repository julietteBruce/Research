addRoots(RootSystem,Root,Root) -- the sum of two roots

Synopsis

  • Function: addRoots
  • Usage:
    r1 + r2
  • Inputs:
    • R, an instance of the type RootSystem,
    • r1, an instance of the type Root,
    • r2, an instance of the type Root,
  • Outputs:
    • an instance of the type Weight, the sum of r1 and r2. Actually, it is returned as an object of class Root if the resulting weight is a root.

Description

i1 : R=rootSystemA(2)
