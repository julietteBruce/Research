WeylGroupElement == WeylGroupElement -- equality of elements of Weyl groups

Synopsis

  • Operator: ==
  • Usage:
    w1 == w2
  • Inputs:
    • w1, an instance of the type WeylGroupElement,
    • w2, an instance of the type WeylGroupElement,
  • Outputs:
    • a Boolean value, true if w1 and w2 belong to the same Weyl group and coincide, and false otherwise
