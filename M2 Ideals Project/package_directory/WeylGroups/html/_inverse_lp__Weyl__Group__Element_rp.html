WeylGroups :: inverse(WeylGroupElement)

inverse(WeylGroupElement) -- the inverse to an element of a Weyl group

Synopsis

  • Function: inverse
  • Usage:
    inverse(w)
  • Inputs:
    • w, an instance of the type WeylGroupElement,
  • Outputs:
    • an instance of the type WeylGroupElement, the inverse to w

Description

i1 : w=reduce(rootSystemE(6),{1,2,3})

o1 = WeylGroupElement{RootSystem{...8...}, | -2 |}
                                           | -1 |
                                           |  1 |
                                           |  3 |
                                           |  1 |
                                           |  1 |

o1 : WeylGroupElement
 
i2 : inverse w

o2 = WeylGroupElement{RootSystem{...8...}, |  1 |}
                                           | -1 |
                                           | -2 |
                                           |  4 |
                                           |  1 |
                                           |  1 |

o2 : WeylGroupElement
