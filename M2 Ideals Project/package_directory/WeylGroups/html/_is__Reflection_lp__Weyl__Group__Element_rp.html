WeylGroups :: isReflection(WeylGroupElement)

isReflection(WeylGroupElement) -- checks whether an element of a Weyl group is a reflection

Synopsis

  • Function: isReflection
  • Usage:
    isReflection(w)
  • Inputs:
    • w, an instance of the type WeylGroupElement,
  • Outputs:
    • a Boolean value, true if w is a reflection and false otherwise

Description

i1 : R=rootSystemD(5)

o2 : WeylGroupElement
