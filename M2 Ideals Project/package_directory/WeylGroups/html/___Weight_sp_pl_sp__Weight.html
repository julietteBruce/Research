WeylGroups :: Weight + Weight

Weight + Weight -- the sum of two weights

Synopsis

  • Operator: +
  • Usage:
    p1 + p2
  • Inputs:
    • p1, an instance of the type Weight,
    • p2, an instance of the type Weight,
  • Outputs:
    • an instance of the type Weight, the sum of p1 and p2
