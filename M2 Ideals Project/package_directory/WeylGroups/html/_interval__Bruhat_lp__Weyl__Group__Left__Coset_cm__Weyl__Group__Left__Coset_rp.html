WeylGroups :: intervalBruhat(WeylGroupLeftCoset,WeylGroupLeftCoset)

intervalBruhat(WeylGroupLeftCoset,WeylGroupLeftCoset) -- elements between two given ones for the Bruhat order on a quotient of a Weyl group

Synopsis

  • Function: intervalBruhat
  • Usage:
    intervalBruhat(u,v)
  • Inputs:
    • u, an instance of the type WeylGroupLeftCoset,
    • v, an instance of the type WeylGroupLeftCoset,
  • Outputs:
    • an instance of the type HasseDiagram, of elements of minimal length in their cosets, bigger than u and smaller than v, together with the reflections to go from an element to the next one (on the right) attached to the corresponding edge.
