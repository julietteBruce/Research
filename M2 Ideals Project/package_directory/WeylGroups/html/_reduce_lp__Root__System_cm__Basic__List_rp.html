WeylGroups :: reduce(RootSystem,BasicList)

reduce(RootSystem,BasicList) -- the product of several reflections with respect to simple roots

Synopsis

  • Function: reduce
  • Usage:
    reduce(R,L)
  • Inputs:
    • R, an instance of the type RootSystem,
    • L, a basic list,
  • Outputs:
    • an instance of the type WeylGroupElement, s_{L_0}...s_{L_{\#L-1}}

Description

i1 : reduce(rootSystemF4,{1,2,3,4})

o1 = WeylGroupElement{RootSystem{...8...}, | -4 |}
                                           |  1 |
                                           |  4 |
                                           |  1 |

o1 : WeylGroupElement
