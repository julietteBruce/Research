isRoot(RootSystem,Parabolic,Root) -- check whether a root is in the sub root system of the parabolic

Synopsis

  • Function: isRoot
  • Usage:
    isRoot(R,P,r)
  • Inputs:
    • R, an instance of the type RootSystem,
    • P, an instance of the type Parabolic,
    • r, an instance of the type Root,
  • Outputs:
    • a Boolean value, true if p is a root in the sub root system of R defined by P and false otherwise
