DGAlgebras :: isAcyclic

isAcyclic -- Determines if a DGAlgebra is acyclic.

Synopsis

  • Usage:
    isAcyc = isAcyclic(A)
  • Inputs:
    • A, an instance of the type DGAlgebra,
  • Optional inputs:
    • EndDegree => ..., default value -1, Option to specify the degree to finish checking acyclicity
  • Outputs:
    • isAcyc, a Boolean value,

Description

This function determines if the DGAlgebra is acyclic.

i1 : R = ZZ/101[a,b,c,d]/ideal{a^4+b^4+c^4+d^4}

o1 = R
