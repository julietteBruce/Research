DGAlgebras :: koszulComplexDGA(Ideal)

koszulComplexDGA(Ideal) -- Returns the Koszul complex as a DGAlgebra

Synopsis

  • Function: koszulComplexDGA
  • Usage:
    A = koszulComplexDGA(I)
  • Inputs:
    • I, an ideal, An ideal of a ring R
  • Optional inputs:
    • Variable => ..., default value "T", Returns the Koszul complex as a DGAlgebra
  • Outputs:
    • A, an instance of the type DGAlgebra,

Description

To construct the Koszul complex on the set of generators of I as a DGAlgebra one uses

i1 : R = ZZ/101[a,b,c]

o1 = R

o1 : PolynomialRing
 
i2 : I = ideal{a^3,b^3,c^3,a^2*b^2*c^2}

             3   3   3   2 2 2
o2 = ideal (a , b , c , a b c )

o2 : Ideal of R
 
i3 : A = koszulComplexDGA(I)

o3 = {Ring => R                           }
      Underlying algebra => R[T ..T ]
                               1   4
                        3   3   3   2 2 2
      Differential => {a , b , c , a b c }

o3 : DGAlgebra
