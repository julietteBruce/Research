IntegralClosure :: idealizer(...,Index=>...)

idealizer(...,Index=>...) -- Sets the starting index on the new variables used to build the endomorphism ring Hom(J,J)

Description

If the program idealizer is used independently, the user will generally want to use the default value of 0. However, when used as part of the integralClosure computation the number needs to start higher depending on the level of recursion involved.

Further information

  • Default value: 0
  • Function: idealizer -- compute Hom(I,I) as a quotient ring
  • Option key: Index -- Optional input for idealizer

Functions with optional argument named Index :

  • idealizer(...,Index=>...) -- Sets the starting index on the new variables used to build the endomorphism ring Hom(J,J)
