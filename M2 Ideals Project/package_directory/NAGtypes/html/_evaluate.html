NAGtypes :: evaluate

evaluate -- evaluate a polynomial system or matrix at a point

Synopsis

  • Usage:
    y = evaluate(f,x)
  • Inputs:
    • f, a polynomial system or a matrix
    • x, a point or a matrix
  • Outputs:
    • y, the value f(x)

Description

Evaluates a PolySystem or a matrix with polynomial entries at a point.

i1 : R = CC[x,y]; S = polySystem {x^2+y^2-6, 2*x^2-y};
 
i3 : p = point {{1.0+3*ii,2.3+ii}};
 
i4 : evaluate(S,p)

o4 = | -9.71+10.6ii |
     | -18.3+11ii   |

                2          1
o4 : Matrix CC    <--- CC
              53         53
 
i5 : evaluate(jacobian S, p)

o5 = | 2+6ii  4.6+2ii |
     | 4+12ii -1      |

                2          2
o5 : Matrix CC    <--- CC
              53         53
