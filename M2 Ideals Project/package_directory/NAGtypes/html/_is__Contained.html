NAGtypes :: isContained

isContained -- Is one space contained in the other

Synopsis

  • Usage:
    b = isContained(S, T)
    b = isContained(D, E)
  • Inputs:
    • S, an instance of the type PolySpace,
    • T, an instance of the type PolySpace,
    • D, an instance of the type DualSpace,
    • E, an instance of the type DualSpace,
  • Optional inputs:
    • Tolerance => ..., default value .000001
  • Outputs:
    • b, a Boolean value, whether S is contained in T (or D in E).

Description

Determines numerically whether the first polynomial space is contained in the second.

Ways to use isContained :

  • "isContained(DualSpace,DualSpace)"
  • "isContained(PolySpace,PolySpace)"

For the programmer

The object isContained is a method function with options.
