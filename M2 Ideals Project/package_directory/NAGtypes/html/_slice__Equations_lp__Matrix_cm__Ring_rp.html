NAGtypes :: sliceEquations(Matrix,Ring)

sliceEquations(Matrix,Ring) -- slicing linear functions

Synopsis

  • Function: sliceEquations
  • Usage:
    S = sliceEquations(M,R)
    S = projectiveSliceEquations(M,R)
  • Inputs:
    • M, a matrix, contains the coefficients of the slicing linear polynomials
    • R, a ring, where the output polynomials belong
  • Outputs:
    • S, a list, contains linear polynomials

Description

A service function used in NumericalAlgebraicGeometry.

i1 : R = CC[x,y]	

o1 = R

o1 : PolynomialRing
 
i2 : sliceEquations(matrix{{1,2,3},{4,5,6*ii}}, R)

o2 = {x + 2y + 3, 4x + 5y + 6*ii}

o2 : List
 
i3 : projectiveSliceEquations(matrix{{1,2,3},{4,5,6*ii}}, CC[x,y,z])
