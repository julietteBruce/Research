NAGtypes :: polySystem

polySystem -- construct a polynomial system

Synopsis

  • Usage:
    P = polysystem F
  • Inputs:
    • F, a list or an ideal or a matrix (column matrix) with polynomial entries or a polynomial system
  • Outputs:
    • P, a polynomial system

Description

Constructs a PolySystem from the given polynomials.

i1 : R = CC[x,y]; S := polySystem {x^2+y^2-6, 2*x^2-y}

o2 = {-2} | x2+y2-6 |
     {-2} | 2x2-y   |

o2 : PolySystem
 
i3 : S = polySystem transpose matrix {{x^2+y^2-6, 2*x^2-y}}

o3 = {-2} | x2+y2-6 |
     {-2} | 2x2-y   |

o3 : PolySystem
 
i4 : T = polySystem S

o4 = T

o4 : PolySystem
