NAGtypes :: random(PolySpace)

random(PolySpace) -- random element of a subspace

Synopsis

  • Function: random
  • Usage:
    f = random S
    f = random(d,S)
  • Inputs:
    • S, an instance of the type PolySpace, an instance of the type PolySpace or an instance of the type DualSpace,
  • Optional inputs:
    • Density => ..., default value 1,
    • Height => ..., default value 10,
    • MaximalRank => ..., default value false,
    • UpperTriangular => ..., default value false,
  • Outputs:
    • f, a ring element,

Description

Provides a random element of a (finite dimensional) PolySpace or DualSpace (of the given degree/order d).

Ways to use this method:

  • "random(DualSpace)"
  • random(PolySpace) -- random element of a subspace
  • "random(ZZ,DualSpace)"
  • "random(ZZ,PolySpace)"
