NAGtypes :: polySpace

polySpace -- construct a PolySpace

Synopsis

  • Usage:
    S = polySpace M
  • Inputs:
    • M, a matrix, with one row of generators
  • Optional inputs:
    • Reduced => ..., default value false
  • Outputs:
    • S, an instance of the type PolySpace,

Description

Used to construct a finite dimensional vector subspace of a polynomial ring.

i1 : R = CC[x,y];
 
i2 : M = matrix{{y^2,x^2+x*y}};

             1       2
o2 : Matrix R  <--- R
 
i3 : S = polySpace M

o3 = | y2 x2+xy |

o3 : PolySpace

Ways to use polySpace :

  • "polySpace(Matrix)"
  • "polySpace(PolySpace)"

For the programmer

The object polySpace is a method function with options.
