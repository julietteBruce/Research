LexIdeals :: hilbertFunct(...,MaxDegree=>...)

hilbertFunct(...,MaxDegree=>...) -- bound degree through which Hilbert function is computed

Synopsis

  • Usage:
    li = hilbertFunct(I,MaxDegree=>d)
  • Inputs:
    • I, an ideal, a homogeneous ideal
    • d, an integer, the highest degree in which the Hilbert function should be computed
  • Outputs:
    • li, a list, the values of the Hilbert function through degree d

Description

If I is Artinian, the default is to return all values of the Hilbert function until it becomes zero. If not, the default maximum degree is 20. This option allows one to specify any maximum degree.

Further information

  • Default value: null
  • Function: hilbertFunct -- return the Hilbert function of a polynomial ring mod a homogeneous ideal as a list
  • Option key: MaxDegree -- optional argument for hilbertFunct
