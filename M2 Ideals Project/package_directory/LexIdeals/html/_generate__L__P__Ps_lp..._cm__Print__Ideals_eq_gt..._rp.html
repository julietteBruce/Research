LexIdeals :: generateLPPs(...,PrintIdeals=>...)

generateLPPs(...,PrintIdeals=>...) -- print LPP ideals nicely on the screen

Synopsis

  • Usage:
    li = generateLPPs(R,hilb,PrintIdeals=>true)
  • Inputs:
    • R, a polynomial ring, the ring in which to work
    • hilb, a list, the values of the Hilbert function as a list
  • Outputs:
    • li, a list, the LPP ideals with Hilbert function hilb

Description

The default value is false. If PrintIdeals is set to true, the power sequences and ideals are printed nicely on the screen to make them more readable than they are in list form.

Further information

  • Default value: false
  • Function: generateLPPs -- return all LPP ideals corresponding to a given Hilbert function
  • Option key: PrintIdeals -- optional argument for generateLPPs
