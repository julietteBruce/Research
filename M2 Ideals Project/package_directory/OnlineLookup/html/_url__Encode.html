urlEncode -- URL encoding

Description

This function provides a minimal encoding of a string in order to be used as part of a URL.

Ways to use urlEncode :

  • urlEncode(Nothing) (missing documentation)
  • urlEncode(String) (missing documentation)

For the programmer

The object urlEncode is a method function.
