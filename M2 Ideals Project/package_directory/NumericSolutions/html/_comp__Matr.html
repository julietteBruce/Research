NumericSolutions :: compMatr

compMatr -- companion matrix

Synopsis

  • Usage:
    compMatr(I,f)
  • Inputs:
    • I, a zero dimensional ideal.
    • f, a polynomial in R.
  • Outputs:
    • M, the companion matrix of the ideal I associated to the polynomial f.
