NumericSolutions :: traceForm

traceForm -- trace matrix of an ideal

Synopsis

  • Usage:
    traceForm(I)
  • Inputs:
    • I, a zero dimensional ideal.
  • Outputs:
    • S, the trace form of the ideal I. The signature of such a form equals the number of real points in the variety V(I).
