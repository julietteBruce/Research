CoincidentRootLoci :: ring(CoincidentRootLocus)

ring(CoincidentRootLocus) -- get the ring of a coincident root locus

Synopsis

  • Function: ring
  • Usage:
    ring X
  • Inputs:
    • X, a coincident root locus
  • Outputs:
    • a ring, the coordinate ring of the target of the map of X

Description

i1 : X = coincidentRootLocus {3,2}

o1 = CRL(3,2)

o1 : Coincident root locus
 
i2 : ring X

o2 = QQ[t ..t ]
         0   5

o2 : PolynomialRing
 
i3 : X' = coincidentRootLocus({3,2},ZZ/3331,Variable=>x)

o3 = CRL(3,2;ZZ/3331;'x')

o3 : Coincident root locus
 
i4 : ring X'

      ZZ
o4 = ----[x ..x ]
     3331  0   5

o4 : PolynomialRing
