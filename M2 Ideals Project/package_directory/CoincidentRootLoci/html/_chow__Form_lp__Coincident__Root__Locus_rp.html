CoincidentRootLoci :: chowForm(CoincidentRootLocus)

chowForm(CoincidentRootLocus) -- Chow form of a coincident root locus

Synopsis

  • Function: chowForm
  • Usage:
    chowForm X
  • Inputs:
    • X, a coincident root locus
  • Optional inputs:
    • AffineChartGrass => ..., default value true,
    • AffineChartProj => ..., default value true,
    • Duality => ..., default value null,
    • Variable => ..., default value null,
  • Outputs:
    • a ring element, the Chow form of $X$

Description

This is equivalent to chowForm ideal X, but should be faster. See chowForm(Ideal).

i1 : X = coincidentRootLocus {3,1}

o1 = CRL(3,1)

o1 : Coincident root locus
