CoincidentRootLoci :: isSubset(CoincidentRootLocus,CoincidentRootLocus)

isSubset(CoincidentRootLocus,CoincidentRootLocus) -- whether one object is a subset of another

Synopsis

  • Function: isSubset
  • Usage:
    isSubset(X,Y)
  • Inputs:
    • X, a coincident root locus, associated with a partition $\lambda$
    • Y, a coincident root locus, associated with a partition $\mu$
  • Outputs:
    • a Boolean value, whether $X$ is contained in $Y$, i.e., whether the partition $\mu$ is a refinement of the partition $\lambda$

Description

i1 : X = coincidentRootLocus {6,4}

o1 = CRL(6,4)

o1 : Coincident root locus
 
i2 : Y = coincidentRootLocus {5,2,2,1}

o2 = CRL(5,2,2,1)

o2 : Coincident root locus
 
i3 : isSubset(X,Y)

o3 = true
 
i4 : X' = coincidentRootLocus {6,3,1}

o4 = CRL(6,3,1)

o4 : Coincident root locus
 
i5 : isSubset(X',Y)

o5 = false
