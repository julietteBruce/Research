CoincidentRootLoci :: realroots

realroots -- real roots of a binary form

Synopsis

  • Usage:
    realroots F
  • Inputs:
    • F, a ring element, a binary form over $\mathbb{Q}$
  • Optional inputs:
    • Verbose => ..., default value false, request verbose feedback
  • Outputs:
    • a list, a list of pairs of elements of RR, the real roots of $F$ (multiple roots appear multiple times)

Description

This is a simple application of the method roots.

i1 : F = randomBinaryForm 5

        5     4       3 2     2 3       4     5
o1 = 18t  + 2t t  + 9t t  + 2t t  + 4t t  + 3t
        0     0 1     0 1     0 1     0 1     1

o1 : QQ[t ..t ]
         0   1
 
i2 : realroots F

o2 = {[-.495276, 1]}

o2 : List
