apolar(RingElement) -- the apolar ideal

Synopsis

  • Function: apolar
  • Usage:
    apolar F
  • Inputs:
    • F, a ring element, a binary form
  • Optional inputs:
    • Variable => ..., default value "t", specify a name for a variable
  • Outputs:
    • an ideal, the ideal generated by all binary forms that annihilate $F$
