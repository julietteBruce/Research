CoincidentRootLoci :: member(RingElement,CoincidentRootLocus)

member(RingElement,CoincidentRootLocus) -- test membership in a coincident root locus

Synopsis

  • Function: member
  • Usage:
    member(F,X)
  • Inputs:
    • F, a ring element, a binary form
    • X, a coincident root locus
  • Outputs:
    • a Boolean value, whether $F$ belongs to $X$

Description

i1 : R := QQ[x,y];
 
i2 : F = x^7-3*x^6*y+5*x^5*y^2-7*x^4*y^3+7*x^3*y^4-5*x^2*y^5+3*x*y^6-y^7

      7     6      5 2     4 3     3 4     2 5       6    7
o2 = x  - 3x y + 5x y  - 7x y  + 7x y  - 5x y  + 3x*y  - y

o2 : QQ[x..y]
 
i3 : member(F,coincidentRootLocus {3,2,2})

o3 = true
 
i4 : member(F,coincidentRootLocus {5,2})

o4 = false
