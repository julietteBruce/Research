CoincidentRootLoci :: subsets(CoincidentRootLocus)

subsets(CoincidentRootLocus) -- produce all the subloci

Synopsis

  • Function: subsets
  • Usage:
    subsets X
  • Inputs:
    • X, a coincident root locus
  • Outputs:
    • a list, the list of the coincident root loci that are contained in X

Description

i1 : X = coincidentRootLocus {5,2,2,1}

o1 = CRL(5,2,2,1)

o1 : Coincident root locus
 
i2 : subsets X

o2 = {CRL(10), CRL(7,3), CRL(8,2), CRL(9,1), CRL(6,4), CRL(5,5), CRL(5,3,2),
     ------------------------------------------------------------------------
     CRL(5,4,1), CRL(6,2,2), CRL(7,2,1), CRL(5,2,2,1)}

o2 : List
