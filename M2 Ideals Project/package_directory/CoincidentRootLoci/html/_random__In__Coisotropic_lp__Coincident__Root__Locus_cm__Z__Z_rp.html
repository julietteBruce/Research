CoincidentRootLoci :: randomInCoisotropic(CoincidentRootLocus,ZZ)

randomInCoisotropic(CoincidentRootLocus,ZZ) -- get a random element

Synopsis

  • Function: randomInCoisotropic
  • Usage:
    randomInCoisotropic(X,i)
  • Inputs:
    • X, a coincident root locus, of dimension $d$ and codimension $n-d$
    • i, an integer
  • Outputs:
    • an ideal, generated by $n-d+i$ linearly independent binary forms of the same degree $n$, which corresponds to a random point of the $i$-th coisotropic hypersurface $CH_i(X)\subset Grass(n-d-1+i,\mathbb{P}(Sym^n(K^2)))$
