CoincidentRootLoci :: ideal(CoincidentRootLocus)

ideal(CoincidentRootLocus) -- the defining ideal of a coincident root locus

Synopsis

  • Function: ideal
  • Usage:
    ideal X
  • Inputs:
    • X, a coincident root locus
  • Outputs:
    • an ideal, the defining homogeneous ideal of X

Description

This is the same as image map X.

i1 : X = coincidentRootLocus {3,2}

o1 = CRL(3,2)
