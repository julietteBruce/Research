CoincidentRootLoci :: randomBinaryForm

randomBinaryForm -- random homogeneous polynomial in two variables

Synopsis

  • Usage:
    randomBinaryForm d
    randomBinaryForm(d,r,c)
    randomBinaryForm(d,r,)
    randomBinaryForm(d,,c)
  • Inputs:
    • d, an integer
    • r, an integer
    • c, an integer
  • Optional inputs:
    • Variable => ..., default value "t", specify a name for a variable
  • Outputs:
    • a ring element, a random binary form of degree d, real rank r and complex rank c
