RandomCanonicalCurves :: RandomCanonicalCurves

RandomCanonicalCurves -- Construction of canonical curves of genus less or equal to 14

Description

This package bundles the constructions for random points in the moduli spaces of curves $M_g$ for $g \leq 14$ based on the proofs of unirationality of $M_g$ by Severi, Sernesi, Chang-Ran and Verra.
