LatticePolytopes :: iskCayleykEdges

iskCayleykEdges -- Checks if a polytope is Cayley of type [P_0*P_1]^k and has every edge of length k

Synopsis

  • Usage:
    iskCayleykEdges(P)
  • Inputs:
    • P, a convex polyhedron,
  • Outputs:
    • L, a sequence, (P_0,P_1,k) such that conditions are satisfied. Or declare that conditions are not satisfied.

Description

iskCayleykEdges is a function. It checks if at given polytope is of type [P_0*P_1]^k with every edge of length k
