LatticePolytopes :: adjointPolytope

adjointPolytope -- computes the adjoint of a polytope

Synopsis

  • Usage:
    adjointPolytope(X,k)
  • Inputs:
    • X, a convex polyhedron, or a a matrix where the columns are the vertice points
    • k, an integer, an integer.
  • Outputs:
    • a convex polyhedron, The adjoint Polytope of X corresponding to the line bundle kK+L, see the article Polyhedral adjunction theory by Sandra Di Rocco, Christian Haase, Benjamin Nill and Andreas Paffenholz.

Description

adjointPolytope is a function that compute the k:th adjoint of a polytope 

i1 : adjointPolytope(hypercube(2),1)
