EdgeIdeals :: Gins

Gins -- optional argument for isSCM

Description

Directs isSCM to use generic initial ideals to determine whether the Alexander dual of the edge ideal of a hypergraph is componentwise linear. See full discussion at isSCM.

See also

  • isSCM -- determines if a (hyper)graph is sequentially Cohen-Macaulay

Functions with optional argument named Gins :

  • isSCM(...,Gins=>...) -- use gins inside isSCM

For the programmer
