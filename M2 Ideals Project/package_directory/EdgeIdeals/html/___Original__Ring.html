EdgeIdeals :: OriginalRing

OriginalRing -- optional argument for inducedHyperGraph

Description

This is an option to tell inducedGraph and inducedHyperGraph whether to return a (hyper)graph with the original (larger) ring attached or the subring only involving the variables of L. The smaller ring is the default.

See also

  • inducedGraph -- returns the induced subgraph of a graph
  • inducedHyperGraph -- returns the induced subgraph of a (hyper)graph

Functions with optional argument named OriginalRing :

  • inducedGraph(...,OriginalRing=>...) -- use OriginalRing inside inducedGraph or inducedHyperGraph
  • "inducedHyperGraph(...,OriginalRing=>...)" -- see inducedGraph(...,OriginalRing=>...) -- use OriginalRing inside inducedGraph or inducedHyperGraph

For the programmer

The object OriginalRing is a symbol.
