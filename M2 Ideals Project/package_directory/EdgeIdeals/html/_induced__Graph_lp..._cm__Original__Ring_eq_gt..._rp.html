EdgeIdeals :: inducedGraph(...,OriginalRing=>...)

inducedGraph(...,OriginalRing=>...) -- use OriginalRing inside inducedGraph or inducedHyperGraph

Synopsis

  • Usage:
    J = inducedGraph(G,L,OriginalRing=>true)
    K = inducedHyperGraph(H,L,OrignalRing=>true)
  • Inputs:
    • G, a graph, the graph being considered
    • H, a hypergraph, the hypergraph being considered
    • L, a list, a list of vertices
  • Outputs:
    • G, a graph, the induced Graph of G on the vertices of L
    • K, a hypergraph, the induced HyperGraph of H on the vertices of L

Description

When OriginalRing is set to true, inducedGraph or inducedHyperGraph returns a (hyper)graph with the original (larger) ring attached rather than the subring only involving the variables of L.

Further information

  • Default value: false
  • Function: inducedGraph -- returns the induced subgraph of a graph
  • Option key: OriginalRing -- optional argument for inducedHyperGraph
