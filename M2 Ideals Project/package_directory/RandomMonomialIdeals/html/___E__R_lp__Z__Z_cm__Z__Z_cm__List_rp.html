RandomMonomialIdeals :: ER(ZZ,ZZ,List)

ER(ZZ,ZZ,List) -- Graded Erdos-Renyi type distribution on monomials over (n,D,L)

Synopsis

  • Function: ER
  • Usage:
    ER(ZZ,ZZ,List)
  • Inputs:
    • n, an integer, number of variables
    • D, an integer, maximum degree
    • L, a list, of real numbers whose i-th entry is the probability of selecting a monomial of degree i, or of integers whose i-th entry is the number of monomials of degree i in each set
  • Outputs:
    • an instance of the type Model, Erdos-Renyi type model

Description

Creates a graded ER-type model for sampling monomials in $n$ variables of degree at most $D$.
