RandomMonomialIdeals :: ER(ZZ,ZZ,RR)

ER(ZZ,ZZ,RR) -- Erdos-Renyi type distribution on monomials over (n,D,p)

Synopsis

  • Function: ER
  • Usage:
    ER(ZZ,ZZ,RR)
  • Inputs:
    • n, an integer, number of variables
    • D, an integer, maximum degree
    • p, a real number, the probability of selecting a monomial
  • Outputs:
    • an instance of the type Model, Erdos-Renyi type model

Description

Creates an ER-type model for sampling monomials in $n$ variables of degree at most $D$ independently with probability $p$.

i1 : n=3; D=4; p=0.1;
 
i4 : myModel = ER(n,D,p)
