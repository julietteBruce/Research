ReflexivePolytopesDB :: toString(KSEntry)

toString(KSEntry) -- the underlying string of a KSEntry

Synopsis

  • Function: toString
  • Usage:
    toString e
  • Inputs:
    • e, an entry from the Kreuzer-Skarke database,
  • Outputs:
    • a string,

Description

Returns the underlying string of the given entry from the Kreuzer-Skarke database.

i1 : e = KSEntry "4 5  M:53 5 N:9 5 H:3,43 [-80] id:0
       1   0   2   4 -10
       0   1   3   5  -9
       0   0   4   0  -4
       0   0   0   8  -8
       ";
 
i2 : toString e

o2 = 4 5  M:53 5 N:9 5 H:3,43 [-80] id:0
       1   0   2   4 -10
       0   1   3   5  -9
       0   0   4   0  -4
       0   0   0   8  -8
