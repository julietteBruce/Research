QuaternaryQuartics :: random(List,Ideal)

random(List,Ideal) -- a random ring element of a given degree

Synopsis

  • Function: random
  • Usage:
    random(d, I)
  • Inputs:
    • d, a list, or an integer, if the ring of $I$ is singly graded
    • I, an ideal, homogeneous
  • Optional inputs:
    • Density => ..., default value 1,
    • Height => ..., default value 10,
    • MaximalRank => ..., default value false,
    • UpperTriangular => ..., default value false,
  • Outputs:
    • a ring element, a random element in the ideal of the given degree

Description

This function should probably be in the Core of Macaulay2.
