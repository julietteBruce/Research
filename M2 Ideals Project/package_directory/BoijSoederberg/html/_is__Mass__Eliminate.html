BoijSoederberg :: isMassEliminate

isMassEliminate -- determines whether the Boij-Soederberg decomposition algorithm eliminates multiple Betti numbers at the same time

Synopsis

  • Usage:
    isMassEliminate(B)
  • Inputs:
    • B
    • a Betti tally, a Betti diagram
  • Outputs:
    • a Boolean value, True or false if...
