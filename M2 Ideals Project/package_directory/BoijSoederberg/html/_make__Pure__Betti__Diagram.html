BoijSoederberg :: makePureBettiDiagram

makePureBettiDiagram -- makes a pure Betti diagram given a list of degrees

Synopsis

  • Usage:
    makePureBettiDiagram L
  • Inputs:
    • L, of strictly increasing integers
  • Optional inputs:
    • TableEntries => a string, default value LeastIntegerEntries, proscribes the scaled versions of the entries of the pure diagram; options are LeastIntegerEntries, HerzogKuhl, and RealizationModules.
  • Outputs:
    • a Betti tally, containing the Betti numbers which satisfy the Herzog-Kuhl equations according to given options. Defaults to the minimal integral Betti numbers which satisfy the Herzog-Kuhl equations.

Description

See pureBetti for a description of the Herzog-Kuhl equations.
