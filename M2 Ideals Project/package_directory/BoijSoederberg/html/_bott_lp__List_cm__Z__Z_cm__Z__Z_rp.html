BoijSoederberg :: bott(List,ZZ,ZZ)

bott(List,ZZ,ZZ) -- cohomology table of Schur functor of tautological bundle on P^n

Synopsis

  • Function: bott
  • Usage:
    bott(L,lowDegree,highDegree)
  • Inputs:
    • L, a list, a non-increasing sequence of integers
    • lowDegree, an integer
    • highDegree, an integer
  • Outputs:
    • a Cohomology tally

Description

Produces a CohomologyTally of the vector bundle S_L(Q), between the column whose index is lowDegree and the column whose index is highDegree. See bott(List,ZZ) for the definition of Q.
