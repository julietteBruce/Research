BoijSoederberg :: facetEquation(List,ZZ,ZZ,ZZ)

facetEquation(List,ZZ,ZZ,ZZ) -- The upper facet equation corresponding to (L,i)

Synopsis

  • Function: facetEquation
  • Usage:
    facetEquation(L,i,lodeg,hideg)
  • Inputs:
    • L, a list, the degree sequence, a strictly ascending list of integers
    • i, an integer, an index in range 0..#L-2, such that L_(i+1) == L_i + 2
    • lodeg, an integer, the leftmost degree of the table
    • hideg, an integer, the rightmost degree of the table
  • Outputs:
    • A (hideg-lodeg+1) by #L matrix over ZZ whose rows correspond to slanted degrees lodeg .. hideg, such that the dot product of this matrix with any betti diagram of any finite length module is >= 0.

Description

The (entry by entry) dot product of this matrix will be >= 0 for every minimal free resolution. Of course, the converse does not hold!

The following example is from Eisenbud and Schreyer, math.AC/0712.1843v2, example 2.4. Notice that the notation here differs slightly from theirs. In both cases i refers to the index, but in Macaulay2, the first element of a list has index 0. hence in this example, i is 3 and not 4, as in the example in the paper.
