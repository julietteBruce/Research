BoijSoederberg :: eliminateBetti

eliminateBetti -- elimination table for a Betti diagram

Synopsis

  • Usage:
    eliminateBetti(B)
    eliminateBetti(I)
  • Inputs:
    • B
    • a Betti tally, a Betti diagram
    • I
    • an ideal, an Ideal
  • Optional inputs:
    • EliminationSequence (missing documentation) => ..., default value false,
  • Outputs:
    • an instance of the type BettiEliminationTally, The elimination table of B or of the Betti table of R/I
