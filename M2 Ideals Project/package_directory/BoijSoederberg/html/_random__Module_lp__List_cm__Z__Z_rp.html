BoijSoederberg :: randomModule(List,ZZ)

randomModule(List,ZZ) -- module with random relations in prescribed degrees

Synopsis

  • Function: randomModule
  • Usage:
    randomModule(L,m)
  • Inputs:
    • L, a list, a strictly increasing degree sequence of integers
    • m, an integer
  • Optional inputs:
    • CoefficientRing (missing documentation) => ..., default value ZZ/101, The base field for the resulting module
  • Outputs:
    • a module, randomly generated having m b_0 generators in degree L_0 and m b_1 relations in degree L_1, where b = pureBetti L
