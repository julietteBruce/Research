BoijSoederberg :: pureWeyman

pureWeyman -- first betti number of specific exact complex

Synopsis

  • Usage:
    pureWeyman L
  • Inputs:
    • a list, a strictly increasing sequence of degrees
  • Outputs:
    • an integer, The zero-th betti number of the corresponding pure resolution construction

Description

pureWeyman corresponds to the construction in math.AC/0709.1529v3 "The Existence of Pure Free Resolutions", Section 4.
