BoijSoederberg :: dotProduct

dotProduct -- entry by entry dot product of two Betti diagrams

Synopsis

  • Usage:
    dotProduct(M,lowestDeg,B)
    dotProduct(M,B)
    dotProduct(B,C)
    dotProduct(M,N)
  • Inputs:
    • M, a matrix
    • N, a matrix
    • lowestDeg, an integer
    • B, a Betti tally
    • C, a Betti tally
  • Outputs:
    • an integer, the entry by entry dot product

Description

In the first version (M, lowestDeg) refers to mat2betti(M, lowestDeg), and in the second version (M,B) refers to (M,0,B).
