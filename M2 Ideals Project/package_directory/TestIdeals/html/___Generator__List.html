TestIdeals :: GeneratorList

GeneratorList -- an option to specify that a certain list of elements is used to describe a Cartier action

Description

GeneratorList is an option used in various functions to specify a Cartier action, particularly on the canonical module. In particular, if it is not specified, typically the Cartier action on the canonical module will need to be recomputed.

Functions with optional argument named GeneratorList :

  • "FPureModule(...,GeneratorList=>...)" -- see FPureModule -- compute the submodule of the canonical module stable under the image of the trace of Frobenius
  • "testModule(...,GeneratorList=>...)" -- see testModule -- find the parameter test module of a reduced ring

For the programmer

The object GeneratorList is a symbol.
