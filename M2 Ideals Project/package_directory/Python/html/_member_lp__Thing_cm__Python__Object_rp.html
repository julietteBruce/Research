Python :: member(Thing,PythonObject)

member(Thing,PythonObject) -- test membership in a python object

Synopsis

  • Function: member
  • Usage:
    member(e, x)
  • Inputs:
    • e, a thing,
    • x, an instance of the type PythonObject,
  • Outputs:
    • a Boolean value, whether e is in x;

Description

This calls Python's __contains__ method, which is equivalent to using the Python in keyword.

i1 : member(toPython 3, toPython {1, 2, 3})

o1 = true
 
i2 : member(toPython 4, toPython {1, 2, 3})

o2 = false

Note that testing a non-Python object for membership will always return false.

i3 : member(3, toPython {1, 2, 3})

o3 = false

Ways to use this method:

  • "member(PythonObject,PythonObject)"
  • member(Thing,PythonObject) -- test membership in a python object
