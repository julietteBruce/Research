quotientRemainder(PythonObject,PythonObject) -- quotient and remainder of python objects

Synopsis

  • Function: quotientRemainder
  • Usage:
    quotientRemainder(x, y)
  • Inputs:
    • x, an instance of the type PythonObject,
    • y, an instance of the type PythonObject,
  • Outputs:
    • a sequence, a pair of two python objects;

Description

The quotient and remainder when x is divided by y. This calls Python's built-in divmod function.
