Python :: abs(PythonObject)

abs(PythonObject) -- absolute value of a python object

Synopsis

  • Function: abs
  • Usage:
    abs x
  • Inputs:
    • x, an instance of the type PythonObject,
  • Outputs:
    • an instance of the type PythonObject, the absolute value of x;

Description

This is equivalent to the Python abs function.

i1 : abs toPython(-12)

o1 = 12

o1 : PythonObject of class int

Ways to use this method:

  • abs(PythonObject) -- absolute value of a python object
