NumericalSchubertCalculus :: solveSchubertProblem(...,LinearAlgebra=>...)

solveSchubertProblem(...,LinearAlgebra=>...) -- switch between Linear Algebra and Parameter Homotopy

Synopsis

  • Usage:
    solveSchubertProblem(...,LinearAlgebra=>T)
  • Inputs:
    • T, a Boolean value, true for Linear Algebra, false for Parameter Homotopy

Description

Chooses the method for glueing solutions from the top of one node (tournament game) to the leaf of the previous node. When true (default), uses Linear Algebra, otherwise uses Parameter Homotopies.

Further information

  • Default value: true
  • Function: solveSchubertProblem -- uses Littlewood-Richardson homotopy algorithm to solve a Schubert problem
  • Option key: LinearAlgebra -- compute a Gröbner basis

Caveat

Parameter Homotopies are not necessary and usually take longer than simple Linear Algebra.

Functions with optional argument named LinearAlgebra :

  • solveSchubertProblem(...,LinearAlgebra=>...) -- switch between Linear Algebra and Parameter Homotopy
