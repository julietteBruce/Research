NumericalSchubertCalculus :: randomSchubertProblemInstance

randomSchubertProblemInstance -- returns a random instance of a given Schubert problem

Synopsis

  • Usage:
    randomSchubertProblemInstance(conditions,k,n)
  • Inputs:
    • conditions, a list, which is a list of Schubert conditions that are either all partitions or all brackets (see bracket2partition for details)
    • k, an integer,
    • n, an integer, $k$ and $n$ define the Grassmannian $Gr(k,n)$ of $k$-planes in $n$-space
  • Optional inputs:
    • Strategy => ..., default value "unit circle", strategy for creating a random matrix representing a flag
  • Outputs:
    • a list, random instance of the Schubert problem, which is a list of pairs of the form (condition,flag)

Description

This first verifies that the conditions are either all partitions or all brackets, and that they form a Schubert problem on $Gr(k,n)$.

Then it creates a list of random square invertible matrices that represent flags for the Schubert problem.

For instance, consider the problem of four lines, which is given by 4 partitions {1}$^4$ in $Gr(2,4)$

Caveat

The output consists of random numerical matrices that are assumed invertible. The code does not check invertibility.
