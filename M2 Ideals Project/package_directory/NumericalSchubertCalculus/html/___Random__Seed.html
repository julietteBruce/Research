NumericalSchubertCalculus :: RandomSeed

RandomSeed -- seed for the random number generators.

Description

To avoid singularities during complex path following, the homotopy methods use a random constant. The flags generated in LRtriple also depend on the values generated by the random number generator. Fixing the value of RandomSeed leads to reproducible runs.

Functions with optional argument named RandomSeed :

  • LRtriple(...,RandomSeed=>...) -- option to set the seed of the random number generators.

For the programmer

The object RandomSeed is a symbol.
