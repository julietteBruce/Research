Bertini :: makeB'InputFile

makeB'InputFile -- write a Bertini input file in a directory

Synopsis

  • Usage:
    makeB'InputFile(s)
  • Inputs:
    • s, a string, a directory where the input file will be written
  • Optional inputs:
    • AffVariableGroup => ..., default value {}, an option to group variables and use multihomogeneous homotopies
    • B'Constants => ..., default value {}, an option to designate the constants for a Bertini Input file
    • B'Functions (missing documentation) => ..., default value {},
    • B'Polynomials (missing documentation) => ..., default value {},
    • BertiniInputConfiguration (missing documentation) => ..., default value {},
    • HomVariableGroup => ..., default value {}, an option to group variables and use multihomogeneous homotopies
    • NameB'InputFile (missing documentation) => ..., default value "input",
    • NamePolynomials (missing documentation) => ..., default value {},
    • ParameterGroup (missing documentation) => ..., default value {},
    • PathVariable (missing documentation) => ..., default value {},
    • RandomComplex => ..., default value {}, an option which designates symbols/strings/variables that will be set to be a random real number or random complex number
    • RandomReal => ..., default value {}, an option which designates symbols/strings/variables that will be set to be a random real number or random complex number
    • SetParameterGroup (missing documentation) => ..., default value {},
    • StorageFolder (missing documentation) => ..., default value null,
    • VariableList (missing documentation) => ..., default value {},
    • Verbose => ..., default value false, Option to silence additional output

Description

This function writes a Bertini input file. The user can specify CONFIGS for the file using the BertiniInputConfiguration option. The user should specify variable groups with the AffVariableGroup (affine variable group) option or HomVariableGroup (homogeneous variable group) option. The user should specify the polynomial system they want to solve with the B'Polynomials option or B'Functions option. If B'Polynomials is not used then the user should use the NamePolynomials option.

i1 : R=QQ[x1,x2,y]

o1 = R

o1 : PolynomialRing
 
i2 : theDir = temporaryFileName();
 
i3 : makeDirectory theDir
 
i4 : makeB'InputFile(theDir,
             BertiniInputConfiguration=>{MPType=>2},
         AffVariableGroup=>{{x1,x2},{y}},
             B'Polynomials=>{y*(x1+x2+1)^2+1,x1-x2+1,y-2});
 
i5 : R=QQ[x1,x2,y,X]

o5 = R

o5 : PolynomialRing
 
i6 : makeB'InputFile(theDir,
             BertiniInputConfiguration=>{MPType=>2},
         AffVariableGroup=>{{x1,x2},{y}},
             NamePolynomials=>{f1,f2,f3},
             B'Functions=>{
            {X,x1+x2+1},
            {f1,y*X^2+1},
            {f2,x1-x2+1},
            {f3,y-2}});
 
i7 : R=QQ[x1,x2,y,X]

o7 = R

o7 : PolynomialRing
 
i8 : makeB'InputFile(theDir,
              BertiniInputConfiguration=>{MPype=>2},
          AffVariableGroup=>{{x1,x2},{y}},
               B'Polynomials=>{y*X^2+1,x1-x2+1,y-2},
                B'Functions=>{
                   {X,x1+x2+1}});

Variables must begin with a letter (lowercase or capital) and can only contain letters, numbers, underscores, and square brackets. "jade" should not be used in any expression. "I" can only be used to represent the complex number sqrt(-1).

Ways to use makeB'InputFile :

  • "makeB'InputFile(String)"

For the programmer

The object makeB'InputFile is a method function with options.
