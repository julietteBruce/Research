Bertini :: NumberToB'String

NumberToB'String -- Translates a number to a string that Bertini can read.

Synopsis

  • Usage:
    NumberToB'String(n)
  • Inputs:
    • n, a thing, n is a number.
  • Optional inputs:
    • M2Precision (missing documentation) => ..., default value 53,

Description

This function takes a number as an input then outputs a string to represent this number to Bertini. The numbers are converted to floating point to precision determined by the option M2Precision.

i1 : NumberToB'String(2+5*ii)

o1 = .2e1 .5e1
 
i2 : NumberToB'String(1/3,M2Precision=>16)
Warning: rational numbers will be converted to floating point.

o2 = .33333333333333331e0 .0e0
 
i3 : NumberToB'String(1/3,M2Precision=>128)
Warning: rational numbers will be converted to floating point.

o3 = .3333333333333333333333333333333333333338e0 .0e0

Ways to use NumberToB'String :

  • "NumberToB'String(Thing)"

For the programmer

The object NumberToB'String is a method function with options.
