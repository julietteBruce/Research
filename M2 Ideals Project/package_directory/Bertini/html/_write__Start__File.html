Bertini :: writeStartFile

writeStartFile -- Writes the list of list of coordinates to a file that Bertini can read.

Synopsis

  • Usage:
    writeStartFile(s,v)
  • Inputs:
    • s, a string, The directory where the Bertini file will be written.
    • v, a list, A list of list numbers that will be written to the file.
  • Optional inputs:
    • M2Precision (missing documentation) => ..., default value 53,
    • NameStartFile (missing documentation) => ..., default value "start",
    • StorageFolder (missing documentation) => ..., default value null,

Description

This function can be used to write "start" files and any other solution file using the option NameStartFile=>"AnyNameYouWant".

i1 : coordinatesOfTwoPnts={{1,0},{3,4}}

o1 = {{1, 0}, {3, 4}}

o1 : List
 
i2 : writeStartFile(storeBM2Files,coordinatesOfTwoPnts);

Ways to use writeStartFile :

  • "writeStartFile(String,List)"

For the programmer

The object writeStartFile is a method function with options.
