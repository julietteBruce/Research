Bertini :: CopyB'File

CopyB'File -- an optional argument to specify whether make a copy of the file.

Description

When set to true, a file is copy of the file is made rather than just moved. The default in moveB'File is set to false.

Functions with optional argument named CopyB'File :

  • "moveB'File(...,CopyB'File=>...)"

For the programmer

The object CopyB'File is a symbol.
