Bertini :: B'Section

B'Section -- a mutable hash table that gives information about a hyperplane used to slice a numerical variety.

Description

B'Section is a type of mutable hash table. It can be created using makeB'Section.

For the programmer

The object B'Section is a type, with ancestor classes MutableHashTable < HashTable < Thing.
