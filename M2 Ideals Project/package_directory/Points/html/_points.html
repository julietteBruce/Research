Points :: points

points -- make the ideal of a set of points

Synopsis

  • Usage:
    i = points pointsMat
  • Inputs:
    • pointsMat, a matrix, matrix whose columns are the homogeneous coordinates of the points
  • Outputs:
    • i, an ideal,
