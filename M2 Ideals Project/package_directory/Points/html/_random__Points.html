Points :: randomPoints

randomPoints -- ideal of a random set of points

Synopsis

  • Usage:
    i = randomPoints(r,n)
  • Inputs:
    • r, an integer, ambient dimension
    • n, an integer, number of points
  • Outputs:
    • i, an ideal, ideal of the random points

Description

The script defines a ring R with r+1 variables, and calls points(R,randomPointsMat(R, n))
