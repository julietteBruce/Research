InvolutiveBases :: Involutive

Involutive -- compute a (usually non-minimal) resolution using involutive bases

Description

The symbol Involutive is allowed as value for the optional argument Strategy for resolution. If provided, the resolution is constructed using janetResolution.

For the programmer

The object Involutive is a symbol.
