InvolutiveBases :: janetMultVar

janetMultVar -- return table of multiplicative variables for given module elements as determined by Janet division

Synopsis

  • Usage:
    janetMultVar M
  • Inputs:
    • M, a matrix or a list
  • Outputs:
    • list of sets of variables of the polynomial ring; the i-th set consists of the multiplicative variables for the i-th generator in J

Description

i1 : R = QQ[x1,x2,x3];
 
i2 : M = matrix {{ x1*x2*x3, x2^2*x3, x1*x2*x3^2 }};

             1       3
o2 : Matrix R  <--- R
 
i3 : janetMultVar M

o3 = {set {x1, x2}, set {x2, x3}, set {x1, x2, x3}}

o3 : List

See also

  • multVar -- extract the sets of multiplicative variables for each generator (in several contexts)
  • isPommaretBasis -- check whether or not a given Janet basis is also a Pommaret basis
