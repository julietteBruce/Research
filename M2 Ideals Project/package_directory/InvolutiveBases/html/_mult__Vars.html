InvolutiveBases :: multVars

multVars -- key in the cache table of a differential in a Janet resolution

Description

The symbol multVars is used as a key in the cache table of a differential in a resolution if it is constructed using janetResolution. In that case it stores the sets of multiplicative variables for the Janet basis given by that differential.

For the programmer

The object multVars is a symbol.
