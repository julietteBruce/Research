DiffAlg :: degree(DiffAlgElement)

degree(DiffAlgElement) -- degree of a differential form or a vector field

Synopsis

  • Function: degree
  • Usage:
    degree e
  • Inputs:
    • e, an instance of the type DiffAlgElement, a differential form or vector field
  • Outputs:
    • a list, If e is a vector field it returns {n,d}; if e is a differential form it returns {n,r,d}.

Description

This function returns the degree of a homogeneous differential form or vector field.

  • n, an integer, is the number of variables minus one
  • r, an integer, is the degree of the differential form or empty if e is a vector field
  • d, an integer, is the degree of the polynomial coefficients

Caveat

If the DiffAlgElement is non-homogeneous the function returns the highest degrees {n,r,d} of each homogeneous component in the given expression. For example, if the degree of w is {2,1,3}, then degree(w + (diff w)) returns, {2,2,3}
