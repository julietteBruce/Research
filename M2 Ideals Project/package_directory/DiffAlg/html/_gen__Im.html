DiffAlg :: genIm

genIm -- a basis of the image of a linear expression

Synopsis

  • Usage:
    genIm(expr,var)
  • Inputs:
    • expr, an instance of the type DiffAlgElement, an expression linear in the variable var
    • var, an instance of the type DiffAlgElement, this is the variable of the linear expression, it must have free and linear scalar coefficients
  • Outputs:
    • a list, a basis of the image of the linear expression

Description

This routine returns a basis of the image of expr, an homogeneous expression linear in var.


In this example we compute a basis of the image of the derivative of a projective differential 1-forms.

i5 : w1=random newForm(2,1,2,"a");
 
i6 : w2=random newForm(2,1,2,"a");
 
i7 : w3=w1+w2;
 
i8 : u=linearComb({w1,w2,w3},"a");
 
i9 : genIm(u,u)
