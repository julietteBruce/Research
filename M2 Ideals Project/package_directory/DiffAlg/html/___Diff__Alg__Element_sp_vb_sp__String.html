DiffAlg :: DiffAlgElement | String

DiffAlgElement | String -- concatenate a string with a differential form or vector field

Synopsis

  • Operator: |
  • Usage:
    e | text
  • Inputs:
    • e, an instance of the type DiffAlgElement, a differential form or a vector field
    • text, a string, a string to be concatenated
  • Outputs:
    • a string, concatenation of text with the string representation of e
