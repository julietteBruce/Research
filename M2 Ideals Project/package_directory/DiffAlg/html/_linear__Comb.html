DiffAlg :: linearComb

linearComb -- generic linear combination of elements

Synopsis

  • Usage:
    linearComb(L,varName)
  • Inputs:
    • L, a list, list of vector fields or differential forms, see DiffAlgField or DiffAlgForm
    • varName, a string, name of the generic coefficient
  • Outputs:
    • an instance of the type DiffAlgElement, generic linear combination of the list L

Description

This routine produce a generic linear combination of the elements in L. It can be used together with genKer or genIm to solve a system of homogeneous linear equations.
