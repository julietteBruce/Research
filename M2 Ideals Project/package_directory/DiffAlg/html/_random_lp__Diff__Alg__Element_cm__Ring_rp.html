DiffAlg :: random(DiffAlgElement,Ring)

random(DiffAlgElement,Ring) -- replaces the variables of the coefficient ring of a differential form or a vector field with random values

Synopsis

  • Function: random
  • Usage:
    random(e,R)
  • Inputs:
    • e, an instance of the type DiffAlgElement, a differential form or vector field with generic coefficients
    • R, a ring, the ring where the random values are taken from. Default value is an integer
  • Optional inputs:
    • Density => ..., default value 1,
    • Height => ..., default value 10,
    • MaximalRank => ..., default value false,
    • UpperTriangular => ..., default value false,
  • Outputs:
    • an instance of the type DiffAlgElement, the differential form or vector field e whose variables of the coefficient ring were evaluated at random values

Caveat

This routine depends on the Macaulay2 method random which is not implemented in some rings. The option random(...,Density=>...) applied to a non-generic form (such us 2a logarithmic form) may return a form equal to 0

Ways to use this method:
