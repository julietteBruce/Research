DiffAlg :: moduliIdeal

moduliIdeal -- ideal generated by the coefficients of a differential form or vector field

Synopsis

  • Usage:
    moduliIdeal(e)
  • Inputs:
    • e, an instance of the type DiffAlgElement, a differential form or vector field
  • Outputs:
    • an ideal, the ideal generated by the scalar coefficients of e

Description

Given a differential form or vector field, this routine returns the ideal generated by the scalar coefficients of such element.


In this example we compute the equations that the scalar coefficients of a closed differential 1-form must satisfy.
