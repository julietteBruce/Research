diff(DiffAlgForm) -- exterior differential

Synopsis

  • Function: diff
  • Usage:
    diff w
  • Inputs:
    • w, an instance of the type DiffAlgForm, a form
  • Outputs:
    • an instance of the type DiffAlgForm, the exterior differential of w
