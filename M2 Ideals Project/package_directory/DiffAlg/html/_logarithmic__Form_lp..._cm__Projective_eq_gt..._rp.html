logarithmicForm(...,Projective=>...) -- a boolean option to produce a projective logarithmic form

Synopsis

  • Usage:
    logarithmicForm(..., Projective => b)
  • Inputs:
    • b, a Boolean value, if true, it return a generic projective logarithmic form
