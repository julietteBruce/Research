DiffAlg :: isHomogeneous(DiffAlgElement)

isHomogeneous(DiffAlgElement) -- tests if a form (or field) is homogeneous

Synopsis

  • Function: isHomogeneous
  • Usage:
    isHomogeneous(e)
  • Inputs:
    • e, an instance of the type DiffAlgElement, representing a form or a field
  • Outputs:
    • a Boolean value, true if the form (or field) is homogeneous

Description

In this example we test if a vector field is homogeneous,

i1 : X = newField("3*ax_0+x_1*ax_1")

o1 = 3ax  + x ax
        0    1  1

o1 : DiffAlgField
 
i2 : isHomogeneous X

o2 = false
