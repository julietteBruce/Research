SimplicialPosets :: getFVector

getFVector -- Returns the f-vector {f_{-1},f_0,...,f_{d-1}} of the given poset.

Synopsis

  • Usage:
    L = getFVector(P)
  • Inputs:
    • P, an instance of the type Poset, A poset.
  • Outputs:
    • L, a list, The f-vector {f_-1,f_0,...,f_{d-1}} of the poset P.

Description

This is a wrapper around the rankGeneratingFunction method from the Posets package.

i1 : P = fromFVector({1,6,5,1});
 
i2 : L = getFVector(P)

o2 = {1, 6, 5, 1}

o2 : List

Ways to use getFVector :

  • "getFVector(Poset)"

For the programmer

The object getFVector is a method function.
