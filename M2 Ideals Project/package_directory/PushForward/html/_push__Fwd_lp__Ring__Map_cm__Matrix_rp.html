pushFwd(RingMap,Matrix) -- push forward of a module map via a finite ring map

Synopsis

  • Function: pushFwd
  • Usage:
    gA = pushFwd(f, g)
    gA = pushFwd g
  • Inputs:
    • f, a ring map, from a ring $A$ to a ring $B$ or (if not specified) the natural map from A = coefficientRing ring g
    • g, a matrix, (a matrix), $g : M_1 \to M_2$ of modules over $B$
  • Optional inputs:
    • NoPrune => ..., default value false, NoPrune option for pushFwd
  • Outputs:
    • gA, a matrix,

Description

If $M_1$ and $M_2$ are both finite generated as $A$-modules, via $f$, this returns the induced map on $A$-modules.
