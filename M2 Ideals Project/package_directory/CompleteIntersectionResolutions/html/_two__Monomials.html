CompleteIntersectionResolutions :: twoMonomials

twoMonomials -- tally the sequences of BRanks for certain examples

Synopsis

  • Usage:
    T = TwoMonomials(c,d)
  • Inputs:
    • c, an integer, codimension in which to work
    • d, an integer, degree of the monomials to take
  • Optional inputs:
    • Optimism => ..., default value 0
  • Outputs:
    • T, a tally,

Description

tallies the sequences of B-ranks that occur for ideals generated by pairs of monomials in R = S/(d-th powers of the variables), with full complexity (=c); that is, for an appropriate syzygy M of M0 = R/(m1, m2) where m1 and m2 are monomials of the same degree.

i1 : setRandomSeed 0

o1 = 0
 
i2 : twoMonomials(2,3)
     -- used 0.991526 seconds
2
Tally{{{1, 1}} => 2        }
      {{2, 2}, {1, 2}} => 4

     -- used 0.51784 seconds
3
Tally{{{2, 2}, {1, 2}} => 2}
      {{3, 3}, {2, 3}} => 1

     -- used 0.192088 seconds
4
Tally{{{2, 2}, {1, 2}} => 1}
