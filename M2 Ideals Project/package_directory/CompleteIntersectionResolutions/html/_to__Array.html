CompleteIntersectionResolutions :: toArray

toArray -- makes an array from a List or from a single integer

Synopsis

  • Usage:
    arr = toArray L
    arr = toArray n
  • Inputs:
    • L, a list,
    • n, an integer,
  • Outputs:
    • arr, an array,

Ways to use toArray :

  • "toArray(List)"
  • "toArray(ZZ)"

For the programmer

The object toArray is a method function.
