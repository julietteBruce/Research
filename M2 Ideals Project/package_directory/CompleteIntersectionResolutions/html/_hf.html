CompleteIntersectionResolutions :: hf

hf -- Computes the hilbert function in a range of degrees

Synopsis

  • Usage:
    H = hf(s,P)
  • Inputs:
    • s, a sequence, or List
    • P, a module, graded module
  • Outputs:
    • H, a list,

Ways to use hf :

  • "hf(List,Module)"
  • "hf(Sequence,Module)"

For the programmer

The object hf is a method function.
