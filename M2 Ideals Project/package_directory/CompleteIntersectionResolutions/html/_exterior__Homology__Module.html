CompleteIntersectionResolutions :: exteriorHomologyModule

exteriorHomologyModule -- Make the homology of a complex into a module over an exterior algebra

Synopsis

  • Usage:
    M = exteriorHomologyModule(ff, C)
  • Inputs:
    • ff, a matrix, Matrix of elements that are homotopic to 0 on C
    • C, a chain complex,
  • Outputs:
    • M, a module,

Description

Assuming that the elements of the 1xc matrix ff are null-homotopic on C, the script returns the direct sum of the homology of C as a module over a new ring, consisting of ring C with c exterior variables adjoined. The script is the main component of exteriorTorModule

See also

  • exteriorTorModule -- Tor as a module over an exterior algebra or bigraded algebra
  • makeHomotopiesOnHomology -- Homology of a complex as exterior module

Ways to use exteriorHomologyModule :

  • "exteriorHomologyModule(Matrix,ChainComplex)"

For the programmer

The object exteriorHomologyModule is a method function.
