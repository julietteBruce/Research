CompleteIntersectionResolutions :: isLinear

isLinear -- check whether matrix entries have degree 1

Synopsis

  • Usage:
    b = isLinear M
  • Inputs:
    • M, a matrix,
  • Outputs:
    • b, a Boolean value,

Description

Note that a linear matrix, in this sense, can still have different target degrees (in which case the cokernel decomposes into a direct sum by generator degree.)

Ways to use isLinear :

  • "isLinear(Matrix)"

For the programmer

The object isLinear is a method function.
