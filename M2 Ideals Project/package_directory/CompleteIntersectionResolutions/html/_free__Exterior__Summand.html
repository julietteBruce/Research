CompleteIntersectionResolutions :: freeExteriorSummand

freeExteriorSummand -- find the free summands of a module over an exterior algebra

Synopsis

  • Usage:
    F = freeExteriorSummand M
  • Inputs:
    • M, a module, over an exterior algebra
  • Outputs:
    • F, a matrix, Map from a free module to M. Image is the largest free summand
