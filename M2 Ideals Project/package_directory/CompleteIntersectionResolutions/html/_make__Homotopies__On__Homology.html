CompleteIntersectionResolutions :: makeHomotopiesOnHomology

makeHomotopiesOnHomology -- Homology of a complex as exterior module

Synopsis

  • Usage:
    (H,h) = makeHomotopiesOnHomology(ff, C)
  • Inputs:
    • ff, a matrix, matrix of elements homotopic to 0 on C
    • C, a chain complex,
  • Outputs:
    • H, a hash table, Homology of C, indexed by places in the C
    • h, a hash table, homotopies for elements of f on the homology of C

Description

The script calls makeHomotopies1 to produce homotopies for the ff_i on C, and then computes their action on the Homology of C.

See also

  • exteriorTorModule -- Tor as a module over an exterior algebra or bigraded algebra
  • exteriorExtModule -- Ext(M,k) or Ext(M,N) as a module over an exterior algebra

Ways to use makeHomotopiesOnHomology :

  • "makeHomotopiesOnHomology(Matrix,ChainComplex)"
