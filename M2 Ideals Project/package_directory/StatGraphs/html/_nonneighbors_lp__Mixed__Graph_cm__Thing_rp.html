StatGraphs :: nonneighbors(MixedGraph,Thing)

nonneighbors(MixedGraph,Thing) -- return the neighbors of a vertex of a mixed graph

Synopsis

  • Function: nonneighbors
  • Usage:
    nonneighbors (G,v)
    nonneighbors (G,v)
  • Inputs:
    • G, an instance of the type MixedGraph,
    • v, a thing, a vertex of G
  • Outputs:
    • a set,

Description

The nonneighbors of v are the all the vertices u such that u,v is not an undirected edge in G.

i1 : G = mixedGraph(graph{{3,1}},digraph {{1,2},{2,3}},bigraph {{3,4},{2,4}})

o1 = MixedGraph{Bigraph => Bigraph{2 => {4}   }}
                                   3 => {4}
                                   4 => {3, 2}
                Digraph => Digraph{1 => {2}}
                                   2 => {3}
                                   3 => {}
                Graph => Graph{1 => {3}}
                               3 => {1}

o1 : MixedGraph
 
i2 : nonneighbors (G,1)

o2 = set {2, 4}

o2 : Set
 
i3 : nonneighbors (G,2)

o3 = set {1, 3, 4}

o3 : Set
