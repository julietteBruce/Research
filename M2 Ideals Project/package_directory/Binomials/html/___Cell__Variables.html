Binomials :: CellVariables

CellVariables -- cellular variables

Description

The cellular variables of a binomial ideal are the variables which are non-zerodivisors modulo that ideal. With this option these variables, if known in advance, can be handed over to specialized functions for cellular ideals.

See also

  • cellularBinomialPrimaryDecomposition -- Primary decomposition of a cellular binomial ideal
  • cellularBinomialAssociatedPrimes -- Associated primes of a cellular binomial ideal

Functions with optional argument named CellVariables :

  • "binomialIsPrime(...,CellVariables=>...)"
  • "cellularBinomialAssociatedPrimes(...,CellVariables=>...)"
  • "cellularBinomialIsPrimary(...,CellVariables=>...)"
  • "cellularBinomialPrimaryDecomposition(...,CellVariables=>...)"
  • "cellularBinomialRadical(...,CellVariables=>...)"
  • "cellularBinomialUnmixedDecomposition(...,CellVariables=>...)"
  • "partialCharacter(...,CellVariables=>...)"
