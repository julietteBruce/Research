Binomials :: binomialIsPrimary

binomialIsPrimary -- test for primary binomial ideals

Synopsis

  • Usage:
    binomialIsPrime I
  • Inputs:
    • I, a binomial ideal
  • Outputs:
    • true or false, depending on whether I is a primary binomial ideal

Description

A binomial ideal is prime only if it is cellular. If the cellular variables are known, the function cellularBinomialIsPrimary should be used. 
i1 : R = QQ[x,y,z]

o1 = R

o1 : PolynomialRing
 
i2 : I = ideal(x-y,z^3)

                    3
o2 = ideal (x - y, z )

o2 : Ideal of R
 
i3 : binomialIsPrimary I

o3 = true
