Binomials :: isCellular

isCellular -- testing for cellular binomial ideals

Synopsis

  • Usage:
    isCellular I
  • Inputs:
    • I, a binomial ideal
  • Optional inputs:
    • ReturnCellVars => ..., default value false, return the cellular variables
  • Outputs:
    • true, or the list of cell variables if I is cellular, false otherwise.

Description

This function is the standard way to compute the cellular variables. 
i1 : R = QQ[x,y,z]

o1 = R

o1 : PolynomialRing
 
i2 : I = ideal (x-z^2,y^4)

               2       4
o2 = ideal (- z  + x, y )

o2 : Ideal of R
 
i3 : isCellular I

o3 = true
 
i4 : isCellular (I, ReturnCellVars=>true)

o4 = {x, z}

o4 : List
