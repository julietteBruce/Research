SubalgebraBases :: generators(Subring)

generators(Subring) -- A generating set of a subring

Synopsis

  • Function: generators
  • Usage:
    result = gens S
  • Inputs:
    • S, an instance of the type Subring,
  • Optional inputs:
    • CoefficientRing => ..., default value null,
  • Outputs:
    • result, a matrix,

Description

Returns a generating set for the Subring as a one-row matrix.

i1 : R = QQ[x,y,z];
 
i2 : S = subring({x^2, y^2, z^2});
 
i3 : gens S

o3 = | x2 y2 z2 |

             1       3
o3 : Matrix R  <--- R
