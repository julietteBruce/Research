SubalgebraBases :: sagbiStatus

sagbiStatus -- returns if the sagbi computation is done

Synopsis

  • Usage:
    result = sagbiStatus SB
  • Inputs:
    • SB, an instance of the type SAGBIBasis,
  • Outputs:
    • result, a Boolean value,

Description

Returns true if the sagbi computation is completed otherwise returns false.
