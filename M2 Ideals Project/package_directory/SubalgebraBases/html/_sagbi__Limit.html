SubalgebraBases :: sagbiLimit

sagbiLimit -- The current limit of the sagbi computation

Synopsis

  • Usage:
    result = sagbiLimit SB
  • Inputs:
    • SB, an instance of the type SAGBIBasis,
  • Outputs:
    • result, an integer,

Description

Returns the current limit of the sagbi computation. When the computation is resumed then this limit will be set to the new specified limit.
