SubalgebraBases :: subring

subring -- Constructs a subring of a polynomial ring.

Synopsis

  • Usage:
    A = subring M
    A = subring L
    A = subring S
  • Inputs:
    • M, a matrix, a one-row matrix whose entries are the generators for the constructed Subring.
    • L, a list, a list of generators for the constructed Subring.
    • S, an instance of the type SAGBIBasis,
  • Optional inputs:
    • GeneratorSymbol => a symbol, default value null, a symbol to be used for the variables of the subductionQuotientRing.
  • Outputs:
    • A, an instance of the type Subring,

Description

This function serves as the canonical constructor for the Subring type.

i1 : R = QQ[x];
 
i2 : S = subring {x^4+x^3, x^2+x}

o2 = QQ[p_0..p_1], subring of R

o2 : Subring
 
i3 : SB = sagbi S;
 
i4 : gens SB

o4 = | x2+x x3-x |

             1       2
o4 : Matrix R  <--- R
 
i5 : (x^3+x^2)%S

o5 = 0

o5 : R
