SubalgebraBases :: net(SAGBIBasis)

net(SAGBIBasis) -- Short summary of a sagbi basis computation object

Synopsis

  • Function: net
  • Usage:
    net S
  • Inputs:
    • S, an instance of the type SAGBIBasis,
  • Outputs:
    • a net

Description

Displays the number of sagbi generators of a S along with the degree limit.
