SubalgebraBases :: ring(SAGBIBasis)

ring(SAGBIBasis) -- The lifted ring of a SAGBIBasis computation object

Synopsis

  • Function: ring
  • Usage:
    result = ring S
  • Inputs:
    • S, an instance of the type SAGBIBasis,
  • Outputs:
    • result, a ring,

Description

Returns the lifted ring of S. This is always a PolynomialRing.

i1 : R = QQ[x,y,z];
 
i2 : S = sagbi {x^2, y^2, z^2}

o2 = SAGBIBasis Computation Object with 3 generators, Limit = 100.

o2 : SAGBIBasis
 
i3 : ring S

o3 = R

o3 : PolynomialRing
 
i4 : Q = R / ideal(x^3 - y);
 
i5 : S' = sagbi {x^3, y^2}

o5 = SAGBIBasis Computation Object with 1 generators, Limit = 100.

o5 : SAGBIBasis
 
i6 : ring S'

o6 = R

o6 : PolynomialRing
