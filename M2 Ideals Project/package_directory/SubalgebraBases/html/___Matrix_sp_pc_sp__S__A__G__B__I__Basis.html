SubalgebraBases :: Matrix % SAGBIBasis

Matrix % SAGBIBasis -- Remainder modulo a subring

Synopsis

  • Operator: %
  • Usage:
    R = M % SB
  • Inputs:
    • M, a matrix,
    • SB, an instance of the type SAGBIBasis,
  • Outputs:
    • R, a matrix, the normal form of entries of M modulo the subring

Description

Subducts M entry-wise against the sagbi generators of SB.
