net(Subring) -- Short summary of a subring

Synopsis

  • Function: net
  • Usage:
    net S
  • Inputs:
    • S, an instance of the type Subring,
  • Outputs:
    • a net

Description

Displays the ambient ring and number of generators of a Subring.
