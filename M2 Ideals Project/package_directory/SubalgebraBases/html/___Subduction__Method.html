SubalgebraBases :: SubductionMethod

SubductionMethod -- Subduction method for the Sagbi algorithm

Description

To subduct the S-pairs in sagbi, one can either use \"Top\" or \"Engine\". Engine level subduction is generally faster. To view each subduction taking place inside the Sagbi algorithm, use a high value for PrintLevel together with top-level subduction.
