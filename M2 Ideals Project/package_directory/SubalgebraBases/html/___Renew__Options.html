SubalgebraBases :: RenewOptions

RenewOptions -- Flag for reselecting the options for a sagbi computation

Description

When a sagbi computation is initialized (for the first time), the options of that computation are stored in the computation object. Whenever the computation is resumed, the same options will apply to the computation, with the exception of PrintLevel, Limit, Recompute, and RenewOptions. To change the other options in the computation, set this flag to true. The new options will be used and stored in the computation object.
