RingElement // Subring -- subductionQuotient with respect to a subring

Synopsis

  • Operator: //
  • Usage:
    r = f // S
  • Inputs:
    • f, a ring element,
    • S, an instance of the type Subring,
  • Outputs:
    • r, a ring element, a subductionQuotient of f with respect to the subring

Description

The result is a ring element that lies in the subductionQuotientRing that has one variable for generator of the subring $S$. A subduction quotient can be thought of as an expression of $f - (f\%S) \in S$ in terms of the generators of $S$. This function is equivalent to groebnerSubductionQuotient.
