SubalgebraBases :: sagbiBasis

sagbiBasis -- Constructs a computation object from a subring.

Synopsis

  • Usage:
    B = sagbiBasis S
  • Inputs:
    • S, an instance of the type Subring,
  • Optional inputs:
    • AutoSubduce => ..., default value true
    • AutoSubduceOnPartialCompletion => ..., default value false
    • Limit => ..., default value 100
    • PrintLevel => ..., default value 0
    • Recompute => ..., default value false
    • ReduceNewGenerators => ..., default value true
    • RenewOptions => ..., default value false
    • StorePending => ..., default value true
    • Strategy => ..., default value "Master"
    • SubductionMethod => ..., default value "Top"
  • Outputs:
    • B, an instance of the type SAGBIBasis,

Description

This function serves as the canonical constructor for the SAGBIBasis type. It is also the output of the sagbi function. A SAGBIBasis may be cast to a Subring using subring.
