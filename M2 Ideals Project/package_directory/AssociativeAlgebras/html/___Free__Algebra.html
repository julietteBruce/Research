AssociativeAlgebras :: FreeAlgebra

FreeAlgebra -- Type of a free algebra

Synopsis

  • Usage:
    A = QQ<|x,y|>

Description

This is the type of a free algebra over a commutative ring R (i.e. a tensor algebra over R).

i1 : A = QQ<|x,y|>

o1 = A
