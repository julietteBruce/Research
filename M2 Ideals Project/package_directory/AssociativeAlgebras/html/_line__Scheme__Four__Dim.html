AssociativeAlgebras :: lineSchemeFourDim

lineSchemeFourDim -- Compute the line scheme of a four-dimensional AS regular algebra

Synopsis

  • Usage:
    L = lineSchemeFourDim B
  • Inputs:
    • B, a quotient of a free algebra,
  • Outputs:
    • L, an ideal,

Description

This method computes the scheme that parametrizes the set of line modules over an AS-regular algebra B due to Shelton and Vancliff. More precisely, it computes the image of this scheme under the Pl√ºcker embedding.

As a first example, we see that the line scheme of the commutative polynomial ring is just the image of the Grassmannian Gr(4,2) in $\mathbb{P}^5$:

i1 : S = skewPolynomialRing(QQ,1_QQ,{x_1,x_2,x_3,x_4})
Using numthreads = 0

o1 = S

o1 : FreeAlgebraQuotient
 
i2 : L = lineSchemeFourDim(S,M);
Using numthreads = 0
Using numthreads = 0

o2 : Ideal of QQ[M   ..M   , M   ..M   , M   ]
                  1,2   1,4   2,3   2,4   3,4
 
i3 : netList minimalPrimes L

     +-------------------------------------+
o3 = |ideal(M   M    - M   M    + M   M   )|
     |       1,4 2,3    1,3 2,4    1,2 3,4 |
     +-------------------------------------+
