AssociativeAlgebras :: ncBasis

ncBasis -- Returns a basis of an noncommutative ring in specified degrees.

Synopsis

  • Usage:
    bas = ncBasis(d,e,B)
  • Inputs:
    • d, an integer, or List or InfiniteNumber
    • e, an integer, or List or InfiniteNumber
    • B, a ring,
  • Optional inputs:
    • Limit => ..., default value infinity
  • Outputs:
    • bas, a matrix,

Description

This command returns a basis (or minimal generating set, if the ground ring is not a field), of a graded noncommutative ring.
