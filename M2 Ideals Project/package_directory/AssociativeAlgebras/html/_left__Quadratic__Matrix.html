AssociativeAlgebras :: leftQuadraticMatrix

leftQuadraticMatrix -- Factors the quadratic ideal on the left or on the right.

Synopsis

  • Usage:
    M = leftQuadraticMatrix I
  • Inputs:
    • I, an ideal, or List
  • Outputs:
    • M, a matrix,

Description

This function expresses the generators of the quadratic ideal I as a product of a row or column vector of the variables times a matrix with linear entries.
