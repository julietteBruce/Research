AssociativeAlgebras :: isNormal(RingElement)

isNormal(RingElement) -- Determines if an element of a noncommutative ring is normal

Synopsis

  • Function: isNormal
  • Usage:
    isNormal x
  • Inputs:
    • x, a ring element,
  • Outputs:
    • a Boolean value,

Description

Given an element x in a noncommutative ring R, this method returns true if Rx=xR.
