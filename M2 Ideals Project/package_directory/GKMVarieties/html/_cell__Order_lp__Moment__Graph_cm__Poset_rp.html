GKMVarieties :: cellOrder(MomentGraph,Poset)

cellOrder(MomentGraph,Poset) -- define a cell order on a moment graph

Synopsis

  • Function: cellOrder
  • Usage:
    cellOrder(G,P)
  • Inputs:
    • G, a moment graph,
    • P, an instance of the type Poset, whose ground set is the vertices of G

Description

Defines a Poset $P$ to be a cell order on the MomentGraph $G$. Overwrites if there was one already defined on $G$.
