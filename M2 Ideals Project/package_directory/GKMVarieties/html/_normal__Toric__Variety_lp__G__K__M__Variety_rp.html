GKMVarieties :: normalToricVariety(GKMVariety)

normalToricVariety(GKMVariety) -- converts a GKM variety back into a toric variety

Synopsis

  • Function: normalToricVariety
  • Usage:
    Y = normalToricVariety X
  • Inputs:
    • X, a GKM variety,
  • Optional inputs:
    • CoefficientRing => ..., default value QQ,
    • MinimalGenerators => ..., default value false,
    • Variable => ..., default value x,
    • WeilToClass => ..., default value null,
  • Outputs:
    • Y, a normal toric variety,

Description

If $X$ is a GKM variety that was originally constructed using normalToricVariety, then this method reverts $X$ to a NormalToricVariety.

i1 : X = toricProjectiveSpace 2;
 
i2 : Y = makeGKMVariety X

o2 = a "GKM variety" with an action of a 2-dimensional torus

o2 : GKMVariety
 
i3 : assert(normalToricVariety Y === X)

  • normalToricVariety -- make a normal toric variety
