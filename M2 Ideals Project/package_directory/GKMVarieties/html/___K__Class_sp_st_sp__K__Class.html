GKMVarieties :: KClass * KClass

KClass * KClass -- computes the product of two equivariant K-classes

Synopsis

  • Operator: *
  • Usage:
    C1 * C2
  • Inputs:
    • C1, an equivariant K-class,
    • C2, an equivariant K-class,
  • Outputs:
    • an equivariant K-class, the product of C1 and C2

Description

This method computes the product of two equivariant $K$-classes.

i1 : Gr24 = generalizedFlagVariety("A",3,{2}); --the Grassmannian of projective lines in projective 3-space
 
i2 : O1 = ampleKClass Gr24 -- the O(1) bundle on Gr24 as an equivariant K-class
