Topcom :: topcomIsTriangulation(Matrix,List)

topcomIsTriangulation(Matrix,List) -- determine if a set of subsets is a triangulation of a point set

Synopsis

  • Function: topcomIsTriangulation
  • Usage:
    topcomIsTriangulation(A, tri)
  • Inputs:
    • A, a matrix,
    • tri, a list,
  • Optional inputs:
    • Homogenize => a Boolean value, default value true
  • Outputs:
    • a Boolean value,

Description

Caveat

topcom doesn't check that the subsets given are simplices, so we do that directly.

See also

  • topcomIsRegularTriangulation(Matrix,List) -- determine if a given triangulation is a regular triangulation
