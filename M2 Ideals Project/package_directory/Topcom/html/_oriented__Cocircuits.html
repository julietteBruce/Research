Topcom :: orientedCocircuits

orientedCocircuits -- compute the oriented cocircuits of an oriented matroid

Synopsis

  • Usage:
    orientedCocircuits om
    orientedCocircuits A
  • Inputs:
    • om, a string, representing a chirotope
    • A, a matrix,
  • Optional inputs:
    • Homogenize => a Boolean value, default value true, Only valid in the second form.
  • Outputs:
    • a list,

Description

See also

  • chirotopeString -- compute the chirotope string of a point or vector configuration
  • orientedCircuits -- compute the oriented circuits of an oriented matroid or point or vector configuration

Ways to use orientedCocircuits :

  • "orientedCocircuits(Matrix)"
  • "orientedCocircuits(String)"

For the programmer

The object orientedCocircuits is a method function with options.
