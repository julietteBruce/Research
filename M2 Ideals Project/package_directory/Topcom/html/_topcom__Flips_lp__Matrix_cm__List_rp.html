Topcom :: topcomFlips(Matrix,List)

topcomFlips(Matrix,List) -- find the neighboring triangulations (bistellar flips) of a triangulation

Synopsis

  • Function: topcomFlips
  • Usage:
    topcomFlips(A, tri)
  • Inputs:
    • A, a matrix,
    • tri, a list,
  • Optional inputs:
    • Homogenize => a Boolean value, default value true
    • RegularOnly => a Boolean value, default value true
  • Outputs:
    • a list,
