NumericalAlgebraicGeometry :: setDefault

setDefault -- set/get the default parameters for continuation algorithms

Synopsis

  • Usage:
    setDefault(p1=>v1, p2=>v2, ...), v = getDefault p
  • Inputs:
    • p, p1, p2, Symbol(s), the name(s) of parameter(s)
  • Optional inputs:
    • Attempts => ..., default value null, (meaning Attempts = 5). The maximal number of attempts (e.g., to make a random regular homotopy).
    • SingularConditionNumber => ..., default value null, (meaning SingularConditionNumber = 100000). Matrix is considered to be singular if its condition number is greater than this value.
    • Precision => ..., default value null, (meaning bits of precision)
    • AffinePatches => ..., default value null, reserved for developers
    • Bits => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • CorrectorTolerance => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • EndZoneFactor => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • ErrorTolerance => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • gamma => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • InfinityThreshold => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • Iterations => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • maxCorrSteps => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • MultistepDegree => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • NoOutput => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • Normalize => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • numberSuccessesBeforeIncrease => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • Predictor => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • Projectivize => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • ResidualTolerance => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • SLP => ..., default value null, reserved for developers
    • SLPcorrector => ..., default value null, reserved for developers
    • SLPpredictor => ..., default value null, reserved for developers
    • Software => ..., default value null, specify internal or external software
    • stepIncreaseFactor => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • tDegree => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • Tolerance => ..., default value null, specifies the tolerance of a numerical computation
    • tStep => ..., default value null, options for core functions of Numerical Algebraic Geometry
    • tStepMin => ..., default value null, options for core functions of Numerical Algebraic Geometry
  • Outputs:
    • setDefault returns null.
    • getDefault returns v, the value of the specified parameter p.

Description

To see a detailed description of an option click on its name.

These functions set/get values of optional parameters used in the functions track, solveSystem, refine as well as higher-level functions (that are under construction).

i1 : getDefault Predictor

o1 = RungeKutta4

o1 : Symbol
 
i2 : setDefault(Predictor=>Euler, CorrectorTolerance=>1e-10)
 
i3 : getDefault Predictor  

o3 = Euler

o3 : Symbol

See also

  • track -- track a linear segment homotopy given start and target system
  • solveSystem -- solve a system of polynomial equations
  • refine -- refine numerical solutions to a system of polynomial equations
  • areEqual -- determine if solutions are equal
