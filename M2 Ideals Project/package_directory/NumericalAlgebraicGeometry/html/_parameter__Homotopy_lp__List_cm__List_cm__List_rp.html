NumericalAlgebraicGeometry :: parameterHomotopy(List,List,List)

parameterHomotopy(List,List,List) -- solve a parametric system of equations

Synopsis

  • Function: parameterHomotopy
  • Usage:
    sols = parameterHomotopy(F,varsP,valuesP)
  • Inputs:
    • F, a list, contains the polynomials in the system
    • varsP, a list, names of the parameters
    • valuesP, a list, contains (possibly several sets of) values of the parameters
  • Optional inputs:
    • Software => ..., default value null, specify internal or external software
  • Outputs:
    • sols, a list, lists of lists of solutions for each set of the parameters

Description

Solves a parameteric polynomial system for several values of parameters.
