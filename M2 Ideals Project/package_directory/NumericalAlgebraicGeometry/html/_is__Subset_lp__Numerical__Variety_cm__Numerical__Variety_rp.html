NumericalAlgebraicGeometry :: isSubset(NumericalVariety,NumericalVariety)

isSubset(NumericalVariety,NumericalVariety) -- check containment

Synopsis

  • Function: isSubset
  • Usage:
    B = isSubset(V,W)
  • Inputs:
    • V, a numerical variety, or a witness set
    • W, a numerical variety, or a witness set
  • Outputs:
    • B, a Boolean value

Description

Checks containment of one variety represented numerically in the other.

Caveat

Does not work for singular components.

See also

  • WitnessSet -- a witness set
  • isOn -- determines if a point belongs to a variety

Ways to use this method:

  • isSubset(NumericalVariety,NumericalVariety) -- check containment
  • "isSubset(WitnessSet,WitnessSet)"
