NumericalAlgebraicGeometry :: vars(GateSystem)

vars(GateSystem) -- the variable gates in the evaluation circuit used for the system

Synopsis

  • Function: vars

Description

This method returns the 1-row GateMatrix that contains InputGates that are considered variables for the evaluation circuit.

Ways to use this method:
