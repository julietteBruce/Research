NumericalAlgebraicGeometry :: squareUp

squareUp -- square up a polynomial system

Synopsis

  • Usage:
    G = squareUp F\nG = squareUp(F,M)\nsquareUp(F,n)
  • Inputs:
    • F, a polynomial system
    • M, a matrix, the matrix used to square up the system (by default a random matrix is picked)
    • n, an integer, the number of polynomials to be formed (by default, this equals the number of variables)
  • Outputs:
    • G, a polynomial system

Description

Squares up an overdetermined polynomial system. Attaches keys SquareUpMatrix and SquaredUpSystem to F. 
i1 : CC[x,y]; F = polySystem {x^2+y^2,x^3+y^3,x^4+y^4}

o2 = F

o2 : PolySystem
 
i3 : G := squareUp F
