Graphs :: html(Digraph)

html(Digraph) -- Create an .svg representation of a graph or digraph

Synopsis

  • Function: html
  • Usage:
    html G
  • Inputs:
    • G, a digraph,
  • Outputs:
    • a string

Description

Uses graphviz to create an .svg representation of G, which is returned as a string.

i2 : html completeGraph 2
