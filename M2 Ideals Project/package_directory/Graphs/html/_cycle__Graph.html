Graphs :: cycleGraph

cycleGraph -- Constructs a cycle graph

Synopsis

  • Usage:
    C = cycleGraph n
  • Inputs:
    • n, an integer,
  • Outputs:
    • C, a graph, The cycle graph with n vertices

Description

A cycle graph is a graph on n vertices in which all the vertices are in a closed chain of edges.

i1 : C = cycleGraph 5

o1 = Graph{0 => {1, 4}}
           1 => {0, 2}
           2 => {1, 3}
           3 => {2, 4}
           4 => {0, 3}

o1 : Graph
