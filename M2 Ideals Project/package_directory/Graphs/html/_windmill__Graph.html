Graphs :: windmillGraph

windmillGraph -- Constructs a windmill graph

Synopsis

  • Usage:
    G = windmillGraph(k,d)
  • Inputs:
    • k, an integer,
    • d, an integer,
  • Outputs:
    • G, a graph, The windmill graph constructed by joining d copies of the complete graph K_k at a shared vertex.

Description

A whidmill joins d amount of copies of a complete graph on k vertices at exactly one shared vertex.

i1 : G = windmillGraph(4,5)
