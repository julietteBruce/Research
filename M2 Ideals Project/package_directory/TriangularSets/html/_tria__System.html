TriangularSets :: triaSystem

triaSystem -- a triangular system

Synopsis

  • Usage:
    triaSystem(R,F)
    triaSystem(R,F,H)
  • Inputs:
    • R, a ring,
    • F, a list, a list of polynomials
    • H, a list, a list of polynomials (optional)
  • Outputs:
    • an instance of the type TriaSystem, given by the pair (F,H)

Description

By default $H$ is chosen to be the set of initials of $F$.

i1 : R = QQ[a,b,c,d,e,f,g,h, MonomialOrder=>Lex];
 
i2 : F = {a*d - b*c, c*f - d*e, e*h - f*g};
 
i3 : H = {d, f, h};
 
i4 : triaSystem(R,F,H)

o4 = {a*d - b*c, c*f - d*e, e*h - f*g} / {d, f, h}

o4 : TriaSystem
 
i5 : triaSystem(R,F)

o5 = {a*d - b*c, c*f - d*e, e*h - f*g} / {d, f, h}

o5 : TriaSystem

Caveat

It is not checked that $(F,H)$ is indeed a triangular system.
