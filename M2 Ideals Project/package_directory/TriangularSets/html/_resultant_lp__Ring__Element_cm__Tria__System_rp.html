TriangularSets :: resultant(RingElement,TriaSystem)

resultant(RingElement,TriaSystem) -- iterated resultant by a triangular set

Synopsis

  • Function: resultant
  • Usage:
    resultant(f,T)
  • Inputs:
    • f, a ring element,
    • T, an instance of the type TriaSystem,
  • Outputs:
    • a ring element, iterated resultant of f by T

Description

Returns the iterated resultant of $f$ by a triangular set $T$. Let $T = (t_1,t_2,\cdots,t_k)$ where $mvar(t_1)>\cdots>mvar(t_k)$. The resultant of $f$ by $T$ is $$resultant(f,T) = resultant(\cdots(resultant(resultant(f,t_1),t_2)\cdots,t_k)$$
