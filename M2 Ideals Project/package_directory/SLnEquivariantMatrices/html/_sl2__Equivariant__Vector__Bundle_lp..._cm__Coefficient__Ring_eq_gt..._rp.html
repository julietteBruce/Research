SLnEquivariantMatrices :: sl2EquivariantVectorBundle(...,CoefficientRing=>...)

sl2EquivariantVectorBundle(...,CoefficientRing=>...) -- name for optional argument

Synopsis

  • Usage:
    W = sl2EquivariantVectorBundle(m,d,CoefficientRing=>C)

Description

This is an option to tell slEquivariantVectorBundle to define the vector bundle over a projective space whose coordinate ring has coefficients in the ring C.

Further information

  • Default value: QQ
  • Function: sl2EquivariantVectorBundle -- computes a SL(2)-equivariant vector bundle over some projective space
  • Option key: CoefficientRing -- an optional argument

Functions with optional argument named CoefficientRing :

  • flattenRing(...,CoefficientRing=>...) -- optionally specify the desired coefficient ring of the flattened ring
  • "generators(...,CoefficientRing=>...)" -- see generators(Ring) -- the list of generators of a ring
  • "Grassmannian(...,CoefficientRing=>...)" -- see Grassmannian(ZZ,ZZ) -- the Grassmannian of linear subspaces of a vector space
  • "Schubert(...,CoefficientRing=>...)" -- see Schubert(ZZ,ZZ,VisibleList) -- find the Plücker ideal of a Schubert variety
