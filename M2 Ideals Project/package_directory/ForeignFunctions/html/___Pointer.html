ForeignFunctions :: Pointer

Pointer -- pointer to memory address

Description

Pointer objects are pointers to memory addresses. These make up each ForeignObject.

i1 : x = int 20

o1 = 20

o1 : ForeignObject of type int32
 
i2 : peek x

o2 = int32{Address => 0x10c348070}

These pointers can be accessed using address.

i3 : ptr = address x

o3 = 0x10c348070

o3 : Pointer

Simple arithmetic can be performed on pointers.

i4 : ptr + 5

o4 = 0x10c348075

o4 : Pointer
 
i5 : ptr - 3

o5 = 0x10c34806d

o5 : Pointer

Functions and methods returning a pointer :

  • address -- pointer to type or object

Methods that use a pointer :

  • "* Pointer = Thing" -- see * voidstar = Thing -- assign value to object at address
  • "foreignFunction(Pointer,String,ForeignType,VisibleList)" -- see foreignFunction -- construct a foreign function
  • "foreignObject(Pointer)" -- see foreignObject -- construct a foreign object
  • ForeignPointerType Pointer -- cast a Macaulay2 pointer to a foreign pointer
  • ForeignType Pointer -- dereference a pointer
  • "Pointer + ZZ"
  • "Pointer - ZZ"
  • "ZZ + Pointer"

Fixed objects of class Pointer :

  • nullPointer -- the null pointer

For the programmer

The object Pointer is a type, with ancestor class Thing.
