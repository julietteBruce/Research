HighestWeights :: LeadingTermTest

LeadingTermTest -- check the columns of the input matrix for repeated leading terms

Description

The method propagateWeights tests whether the input map is represented by a matrix whose columns have all different leading terms. Set this argument to false to bypass this test.

See also

  • propagateWeights -- propagate (Lie theoretic) weights along equivariant maps
