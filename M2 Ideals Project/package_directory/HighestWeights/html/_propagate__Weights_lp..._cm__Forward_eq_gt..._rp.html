HighestWeights :: propagateWeights(...,Forward=>...)

propagateWeights(...,Forward=>...) -- propagate weights from domain to codomain

Description

Set this argument to true to ensure weights are propagated forward along a map of graded free modules, i.e., from domain to the codomain of the map. The default setting is false meaning that weights are propagated backwards, i.e., from codomain to domain.

Further information

  • Default value: false
  • Function: propagateWeights -- propagate (Lie theoretic) weights along equivariant maps
  • Option key: Forward -- propagate weights from domain to codomain
