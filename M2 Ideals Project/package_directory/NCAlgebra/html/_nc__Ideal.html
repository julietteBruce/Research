NCAlgebra :: ncIdeal

ncIdeal -- Define a two-sided ideal in a noncommutative ring

Synopsis

  • Usage:
    I = ncIdeal fs
  • Inputs:
    • fs, an instance of the type NCRingElement, or a List of NCRingElements
  • Outputs:
    • I, an instance of the type NCIdeal,

Description

This defines a two-sided ideal in a noncommutative ring.

i1 : A = QQ{x,y,z}

o1 = A

o1 : NCPolynomialRing
 
i2 : p = y*z + z*y - x^2

            2
o2 = zy+yz-x

o2 : A
 
i3 : q = x*z + z*x - y^2

         2
o3 = zx-y +xz

o3 : A
 
i4 : r = z^2 - x*y - y*x

      2
o4 = z -yx-xy

o4 : A
 
i5 : I = ncIdeal{p,q,r}

                             2      2      2
o5 = Two-sided ideal {zy+yz-x , zx-y +xz, z -yx-xy}

o5 : NCIdeal

Ways to use ncIdeal :

  • "ncIdeal(List)"
  • "ncIdeal(NCRingElement)"

For the programmer

The object ncIdeal is a method function.
