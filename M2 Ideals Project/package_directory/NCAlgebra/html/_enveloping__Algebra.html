NCAlgebra :: envelopingAlgebra

envelopingAlgebra -- Create the enveloping algebra

Synopsis

  • Usage:
    Ae = envelopingAlgebra(A,x)
  • Inputs:
    • A, an instance of the type NCRing,
    • x, a symbol,
  • Outputs:
    • Ae, an instance of the type NCRing,

Description

This function returns the enveloping algebra A ** Aop. The symbol argument is used to define the variables in Aop.
