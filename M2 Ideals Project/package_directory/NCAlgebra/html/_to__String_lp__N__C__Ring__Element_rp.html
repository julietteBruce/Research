NCAlgebra :: toString(NCRingElement)

toString(NCRingElement) -- Converts an NCRingElement to a string

Synopsis

  • Function: toString
  • Usage:
    str = toString f
  • Inputs:
    • f, an instance of the type NCRingElement,
  • Outputs:
    • str, a string,

Description

Converts an NCRingElement to a string. This should be readable by both Macaulay2 as well as Bergman.

i1 : A = QQ{x,y,z,w}

o1 = A

o1 : NCPolynomialRing
 
i2 : f = x^2+y^2+z^3

      3  2  2
o2 = z +y +x

o2 : A
 
i3 : toString f

o3 = z^3+y^2+x^2
