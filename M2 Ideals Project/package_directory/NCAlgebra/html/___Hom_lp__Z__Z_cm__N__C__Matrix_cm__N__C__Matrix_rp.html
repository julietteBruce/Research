NCAlgebra :: Hom(ZZ,NCMatrix,NCMatrix)

Hom(ZZ,NCMatrix,NCMatrix) -- Compute a graded component of Hom(M,N)

Synopsis

  • Function: Hom
  • Usage:
    homs = Hom(n,M,N)
  • Inputs:
    • n, an integer,
    • M, an instance of the type NCMatrix,
    • N, an instance of the type NCMatrix,
  • Outputs:
    • homs, a list,

Description

This function computes a generating set of the degree n component of Hom(M,N) as a module over the coefficient ring of A = ring M.

Ways to use this method:

  • Hom(ZZ,NCMatrix,NCMatrix) -- Compute a graded component of Hom(M,N)
