transpose(NCMatrix) -- Transposes an NCMatrix

Synopsis

  • Function: transpose
  • Usage:
    L = transpose M
  • Inputs:
    • M, an instance of the type NCMatrix,
  • Outputs:
    • L, an instance of the type NCMatrix,

Description

This command transposes an NCMatrix
