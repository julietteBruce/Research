NCAlgebra :: basis(ZZ,NCLeftIdeal)

basis(ZZ,NCLeftIdeal) -- Returns a basis of an NCLeftIdeal in a particular degree.

Synopsis

  • Function: basis
  • Usage:
    bas = basis(n,I)
  • Inputs:
    • n, an integer,
    • I, an instance of the type NCLeftIdeal,
  • Optional inputs:
    • Degree => ..., default value null,
    • Limit => ..., default value infinity,
    • SourceRing => ..., default value null,
    • Strategy => ..., default value null,
    • Truncate => ..., default value false,
    • Variables => ..., default value null,
  • Outputs:
    • bas, an instance of the type NCMatrix,

Description

This command returns a basis (or minimal generating set, if the ground ring is not a field), of a homogeneous left ideal in a noncommutative ring.
