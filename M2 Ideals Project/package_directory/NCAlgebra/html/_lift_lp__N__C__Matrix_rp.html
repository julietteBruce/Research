NCAlgebra :: lift(NCMatrix)

lift(NCMatrix) -- Lifts an NCMatrix

Synopsis

  • Function: lift
  • Usage:
    L = lift M
  • Inputs:
    • M, an instance of the type NCMatrix,
  • Optional inputs:
    • Verify => ..., default value true,
  • Outputs:
    • L, an instance of the type NCMatrix,

Description

This command lifts an NCMatrix to a matrix over its ambient NCRing.

i1 : A = QQ{x,y,z,w}

o1 = A

o1 : NCPolynomialRing
