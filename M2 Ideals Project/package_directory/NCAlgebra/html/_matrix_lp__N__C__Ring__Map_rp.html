NCAlgebra :: matrix(NCRingMap)

matrix(NCRingMap) -- An NCMatrix associated to an NCRingMap.

Synopsis

  • Function: matrix
  • Usage:
    matrix f
  • Inputs:
    • f, an instance of the type NCRingMap,
  • Optional inputs:
    • Degree => ..., default value null,
  • Outputs:
    • an instance of the type NCMatrix,

Description

This function returns a matrix whose entries are the images of the source generators in the target ring of f. The type of matrix depends on the type of the target ring.
