NCAlgebra :: leadCoefficient(NCRingElement)

leadCoefficient(NCRingElement) -- Returns the lead monomial of an NCRingElement

Synopsis

  • Function: leadCoefficient
  • Usage:
    coeff = leadCoefficient f
  • Inputs:
    • f, an instance of the type NCRingElement,
  • Outputs:
    • coeff, a ring element,

Description

Returns the lead coefficient of an NCRingElement (as an element of the base).
