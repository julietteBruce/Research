NCAlgebra :: ncGroebnerBasis

ncGroebnerBasis -- Compute a noncommutative Groebner basis.

Synopsis

  • Usage:
    Igb = ncGroebnerBasis I
  • Inputs:
    • I, an instance of the type NCIdeal, or a List of NCRingElements
  • Optional inputs:
    • DegreeLimit (missing documentation) => an integer, default value 100, the maximum degree for the calculation of a Groebner basis
    • InstallGB => a Boolean value, default value false, set this to true to install a NCGroebnerBasis without verifying that it is one
  • Outputs:
    • Igb, an instance of the type NCGroebnerBasis,

If the InstallGB flag is not set, then a call to Bergman is made, if the base ring is QQ or a finite field of characteristic p. Otherwise, an error is raised.

Calls to Bergman are usually faster, except for when the polynomial is small. See the documentation for NCRingElement % NCGroebnerBasis for details on controlling when Bergman is called.
