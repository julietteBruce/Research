NCAlgebra :: ring(NCMatrix)

ring(NCMatrix) -- Gives the ring of the NCMatrix

Synopsis

  • Function: ring
  • Usage:
    L = ring M
  • Inputs:
    • M, an instance of the type NCMatrix,
  • Outputs:
    • L, an instance of the type NCRing,

Description

This command returns the ring over which the NCMatrix is defined.

i1 : A = QQ{a,b,c,d}

o1 = A

o1 : NCPolynomialRing
 
i2 : M = ncMatrix {{a,b,c,d},{b,c,d,a}}

o2 = | a b c d |
     | b c d a |

o2 : NCMatrix
 
i3 : ring M

o3 = A

o3 : NCPolynomialRing

Ways to use this method:

  • ring(NCMatrix) -- Gives the ring of the NCMatrix
