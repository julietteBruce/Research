NCAlgebra :: isWellDefined(NCRingMap)

isWellDefined(NCRingMap) -- Determines if an NCRingMap is well-defined.

Synopsis

  • Function: isWellDefined
  • Usage:
    isWellDefined f
  • Inputs:
    • f, an instance of the type NCRingMap,
  • Outputs:
    • a Boolean value,

Description

Returns true if the given NCRingMap evaluates as 0 on the defining relations of the source.
