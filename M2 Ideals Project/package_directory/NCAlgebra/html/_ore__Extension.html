NCAlgebra :: oreExtension

oreExtension -- Creates an Ore extension of a noncommutative ring

Synopsis

  • Usage:
    oreExtension(A,sigma,delta,x) or oreExtension(A,sigma,x)
  • Inputs:
    • A, an instance of the type NCRing,
    • sigma, an instance of the type NCRingMap,
    • delta, an instance of the type NCRingMap,
    • x, an instance of the type NCRingElement, or a Symbol
  • Optional inputs:
    • Degree (missing documentation) => ..., default value 1,
  • Outputs:
    • an instance of the type NCQuotientRing,

Description

This method calls oreIdeal and returns the associated Ore extension as an NCQuotientRing.

i1 : B = skewPolynomialRing(QQ,(-1)_QQ,{x,y,z,w})
--Calling Bergman for NCGB calculation.
Complete!

o1 = B

o1 : NCQuotientRing
 
i2 : sigma = ncMap(B,B,{y,z,w,x})

o2 = NCRingMap B <--- B

o2 : NCRingMap
 
i3 : C = oreExtension(B,sigma,a)
--Calling Bergman for NCGB calculation.
Complete!

o3 = C

o3 : NCQuotientRing
