NCAlgebra :: NCMatrix * ZZ

NCMatrix * ZZ -- Product of NCMatrices

Synopsis

  • Operator: *
  • Usage:
    L = M*a
  • Inputs:
    • M, an instance of the type NCMatrix,
    • a, an integer,
  • Outputs:
    • L, an instance of the type NCMatrix,

Description

This command allows for the scaling of an NCMatrix by an element in ZZ.

i1 : B = threeDimSklyanin(QQ,{1,1,-1},{x,y,z})
--Calling Bergman for NCGB calculation.

o1 = B

o1 : NCQuotientRing
 
i2 : M = ncMatrix {{x, y, z}}

o2 = | x y z |

i3 : sigma = ncMap(B,B,{y,z,x})

o3 = NCRingMap B <--- B

o3 : NCRingMap
 
i4 : N = ncMatrix {{M},{sigma M}, {sigma sigma M}}

o4 = | x y z |
     | y z x |

i5 : N*3
