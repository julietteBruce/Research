NCMatrix % NCGroebnerBasis -- Reduces the entries of an NCMatrix with respect to an NCGroebnerBasis

Synopsis

  • Operator: %
  • Usage:
    L = M % Igb
  • Inputs:
    • M, an instance of the type NCMatrix,
    • Igb, an instance of the type NCGroebnerBasis,
  • Outputs:
    • L, an instance of the type NCMatrix,

Description
