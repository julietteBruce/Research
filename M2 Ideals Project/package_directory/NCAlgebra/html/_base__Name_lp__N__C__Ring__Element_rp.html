NCAlgebra :: baseName(NCRingElement)

baseName(NCRingElement) -- Returns the base name of a generator of an NCRing

Synopsis

  • Function: baseName
  • Usage:
    name = baseName f
  • Inputs:
    • f, an instance of the type NCRingElement,

Description

Returns the base name of a generator of an NCRing. This could be something of type IndexedVariable or a Symbol.

i1 : A = QQ{x,y,z,w}

o1 = A

o1 : NCPolynomialRing
 
i2 : baseName x

o2 = x

o2 : Symbol
 
i3 : B = QQ{p_1..p_6}

o3 = B

o3 : NCPolynomialRing
 
i4 : baseName p_1

o4 = p
      1

o4 : IndexedVariable
