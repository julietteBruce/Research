NCAlgebra :: NCRingMap NCIdeal

NCRingMap NCIdeal -- Apply a ring map to the generators of an ideal

Synopsis

  • Operator: SPACE
  • Usage:
    J = f I
  • Inputs:
    • f, an instance of the type NCRingMap,
    • I, an instance of the type NCIdeal, or NCGroebnerBasis
  • Outputs:
    • J, an instance of the type NCIdeal, or NCGroebnerBasis

Description

This function applies the ring map f to the generators of the ideal or noncommutative Groebner basis that is passed in.

i1 : A = QQ{x,y}

o1 = A

o1 : NCPolynomialRing
 
i2 : g = ncMap(A,A,{y,x})

o2 = NCRingMap A <--- A

o2 : NCRingMap
 
i3 : I = ncIdeal {x^2*y+y*x^2}

                        2  2
o3 = Two-sided ideal {yx +x y}

o3 : NCIdeal
 
i4 : g I

                       2    2
o4 = Two-sided ideal {y x+xy }

o4 : NCIdeal
