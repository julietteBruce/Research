NCAlgebra :: normalFormBergman

normalFormBergman -- Calls Bergman for a normal form calculation

Synopsis

  • Usage:
    nfList = normalFormBergman(L,Igb)
  • Inputs:
    • L, an instance of the type NCRingElement, or a List of NCRingElements
    • Igb, an instance of the type NCGroebnerBasis,
  • Outputs:
    • nfList, a list,

Description

This method takes a list of NCRingElements and calls Bergman to reduce each element to normal form relative to the given NCGroebnerBasis for the NCIdeal defining the NCRing to which the list elements belong.
