NCAlgebra :: isHomogeneous(NCRingMap)

isHomogeneous(NCRingMap) -- Determines if an NCRingMap preserves the natural grading

Synopsis

  • Function: isHomogeneous
  • Usage:
    isHomogeneous f
  • Inputs:
    • f, an instance of the type NCRingMap,
  • Outputs:
    • a Boolean value,

Description

This method returns true if the map f is degree 0; that is, if the generators of the source of f are mapped to elements of the same degree in the target.
