NCAlgebra :: kernelComponent

kernelComponent -- Computes a basis of the kernel of a ring map in a specified degree.

Synopsis

  • Usage:
    kern = kernelComponent(n,phi)
  • Inputs:
    • n, an integer,
    • phi, an instance of the type NCRingMap,
  • Outputs:
    • kern, an instance of the type NCMatrix,

Description
