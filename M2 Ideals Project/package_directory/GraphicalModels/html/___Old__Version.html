GraphicalModels :: OldVersion

OldVersion -- optional argument in gaussianVanishingIdeal to use old method for gaussianRings coming from directed graphs

Description

Alternative computation of the vanishing ideal of a Gaussian directed graphical model using (2) from Seth Sullivant, Kelli Talaska, and Jan Draisma, Trek separation for Gaussian graphical models, The Annals of Statistics 38.3 (2010): 1665-1685.
