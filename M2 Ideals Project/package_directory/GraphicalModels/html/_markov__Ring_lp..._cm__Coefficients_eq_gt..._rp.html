GraphicalModels :: markovRing(...,Coefficients=>...)

markovRing(...,Coefficients=>...) -- optional input to choose the base field in markovRing or gaussianRing

Synopsis

  • Usage:
    gaussianRing(n,Coefficients=>Ring)
    gaussianRing(G,Coefficients=>Ring)
    markovRing(d,Coefficients=>Ring)
  • Inputs:
    • d, a sequence, with positive integer entries $(d_1,\dots ,d_r)$
    • n, an integer, number of random variables
    • G, a graph, a graph, or a digraph, or an instance of the type MixedGraph with directed and bidirected edges
  • Outputs:
    • a ring,

Description

In both markovRing and gaussianRing, the default coefficient ring is QQ. Putting Coefficients => r for a choice of ring(field) r as an argument in the function markovRing or gaussianRing creates a ring with the desired coefficient ring.

i1 : R2 = markovRing ((2,2),Coefficients=>CC);
 
i2 : coefficientRing R2

o2 = CC
       53

o2 : ComplexField

Further information

  • Default value: QQ
  • Function: markovRing -- ring of joint probability distributions on several discrete random variables
  • Option key: Coefficients -- optional input to choose the base field

See also

  • markovRing -- ring of joint probability distributions on several discrete random variables
  • gaussianRing -- ring of Gaussian correlations on n random variables or a graphical model
