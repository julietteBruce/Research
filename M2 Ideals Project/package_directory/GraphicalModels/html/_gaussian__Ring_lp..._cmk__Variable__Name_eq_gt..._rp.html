GraphicalModels :: gaussianRing(...,kVariableName=>...)

gaussianRing(...,kVariableName=>...) -- symbol used for indeterminates in a ring of Gaussian joint probability distributions

Synopsis

  • Usage:
    gaussianRing(G,kVariableName=>m)
  • Inputs:
    • m, a Symbol or a String

Description

The option gaussianRing(G,kVariableName=>m) changes the symbol used for intedeterminates in the concentration matrix in a polynomial ring created with gaussianRing.

i1 : R = gaussianRing graph({{a,b},{b,c},{c,d},{a,d}})

Further information

  • Default value: k
  • Function: gaussianRing -- ring of Gaussian correlations on n random variables or a graphical model
  • Option key: kVariableName -- optional input to choose the variable names for concentration matrix in gaussianRing
