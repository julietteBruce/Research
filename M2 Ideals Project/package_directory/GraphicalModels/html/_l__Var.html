GraphicalModels :: lVar

lVar -- key in hash table gaussianRingData: labels of l variables

Description

This key is present in every gaussianRingData that comes from a graph. It is equal to the value of the optional input gaussianRing(...,lVariableName=>...).

i1 : U = graph {{1,2},{2,3}}

o1 = Graph{1 => {2}   }
           2 => {1, 3}
           3 => {2}

o1 : Graph
 
i2 : B = bigraph{{4,5}}

o2 = Bigraph{4 => {5}}
             5 => {4}

o2 : Bigraph
 
i3 : D = digraph {{1,4}}

o3 = Digraph{1 => {4}}
             4 => {}

o3 : Digraph
