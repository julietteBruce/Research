Markov :: gaussRing

gaussRing -- ring of gaussian correlations on n random variables

Synopsis

  • Usage:
    gaussRing n
  • Inputs:
    • n, an integer, the number of random variables
  • Optional inputs:
    • CoefficientRing (missing documentation) => ..., default value QQ, a coefficient field or ring
    • Variable (missing documentation) => ..., default value "s", a symbol, the variables in the ring will be s_(1,1),...
  • Outputs:
    • a ring with indeterminates s_(i,j), 1 <= i <= j <= n, a ring

Description

The routines gaussMinors (missing documentation) , gaussIdeal, gaussTrekIdeal (missing documentation) all require that the ring be created by this function. 

i1 : R = gaussRing 5;
 
i2 : gens R

o2 = {s   , s   , s   , s   , s   , s   , s   , s   , s   , s   , s   , s   ,

     ------------------------------------------------------------------------
     s   , s   , s   }
      4,4   4,5   5,5
