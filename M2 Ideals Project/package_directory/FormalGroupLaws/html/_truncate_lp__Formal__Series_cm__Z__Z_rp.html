truncate(FormalSeries,ZZ) -- truncate formal series

Synopsis

  • Function: truncate
  • Usage:
    truncate(s,n)
  • Inputs:
    • s, an instance of the type FormalSeries,
    • n, an integer,
  • Outputs:
    • an instance of the type FormalSeries,

Description

This function truncates the formal series s at the precision n.
