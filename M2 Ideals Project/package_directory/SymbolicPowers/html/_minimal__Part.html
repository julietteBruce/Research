SymbolicPowers :: minimalPart

minimalPart -- intersection of the minimal components

Synopsis

  • Usage:
    minimalPart(I)
  • Inputs:
    • I, an ideal,
  • Outputs:
    • an ideal, the intersection of the components of I corresponding to minimal primes

Description

Eliminates embedded components of a given ideal

i1 : R = QQ[x,y,z]

o1 = R

o1 : PolynomialRing
 
i2 : J = intersect(ideal(x^2,y,z^3),ideal(x,z))

                       2   3
o2 = ideal (y*z, x*y, x , z )

o2 : Ideal of R
 
i3 : minimalPart(J)

o3 = ideal (z, x)

o3 : Ideal of R
