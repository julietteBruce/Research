SymbolicPowers :: squarefreeGens

squarefreeGens -- returns all square-free monomials in a minimal generating set of the given ideal.

Synopsis

  • Usage:
    squarefreeGens(I)
  • Inputs:
    • I, an ideal,
  • Outputs:
    • a list,

Description

Given a monomial ideal $I$, returns all square-free monomials in a minimal generating set of $I$.

i1 : R = QQ[x,y,z];
 
i2 : I = ideal(x*y,y*z,x^2);

o2 : Ideal of R
 
i3 : squarefreeGens(I)

o3 = {y*z, x*y}

o3 : List

See also

  • squarefreeInCodim -- finds square-fee monomials in ideal raised to the power of the codimension.

Ways to use squarefreeGens :

  • "squarefreeGens(Ideal)"
