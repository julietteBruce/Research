SymbolicPowers :: isPacked

isPacked -- determines if a given square-free ideal is packed.

Synopsis

  • Usage:
    isPacked(I)
  • Inputs:
    • I, an ideal,
  • Outputs:
    • a Boolean value,

Description

Given a square-free monomial ideal $I$, determines if the ideal is Konig. A square-free monomial ideal $I$ of codimension $c$ is packed if every ideal obtained from it by replacing any number of variables by 1 or 0 is Konig.

i1 : R = QQ[x,y,z];
 
i2 : I = ideal(x*y,y*z,x*z);

o2 : Ideal of R
 
i3 : isPacked(I)

o3 = false

See also

  • squarefreeGens -- returns all square-free monomials in a minimal generating set of the given ideal.

Ways to use isPacked :

  • "isPacked(Ideal)"

For the programmer

The object isPacked is a method function.
