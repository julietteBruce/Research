SymbolicPowers :: CIPrimes

CIPrimes -- compute the symbolic power by taking the intersection of the powers of the primary components

Description

The default value is false. When defined to be true, the symbolicPower function tests whether the primary components are complete intersections having the same height. If each component is, then the function takes the intersection of the powers of the components: $$I^{(n)}=\cap_{p\in Ass(R/I)} p^n.$$
