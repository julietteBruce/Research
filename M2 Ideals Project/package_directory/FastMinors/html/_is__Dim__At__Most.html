FastMinors :: isDimAtMost

isDimAtMost -- returns true if we can quickly see whether the dim is at most a given number

Synopsis

  • Usage:
    isDimAtMost(n, I)
  • Inputs:
    • n, an integer, an integer
    • I, an ideal, an ideal in a polynomial ring over a field, or a quotient ring of such
  • Optional inputs:
    • PairLimit => ..., default value 100
    • SPairsFunction => ..., default value -*Function*-
    • Verbose => ..., default value false
  • Outputs:
    • true if the dimension of I is at most n or null if the function cannot tell whether the dimension is at most n

Description

This simply calls isCodimAtLeast, passing options as described there.

See also

  • isCodimAtLeast -- returns true if we can quickly see whether the codim is at least a given number

Ways to use isDimAtMost :

  • "isDimAtMost(ZZ,Ideal)"

For the programmer

The object isDimAtMost is a method function with options.
