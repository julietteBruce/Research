FastMinors :: Threads

Threads -- an option for various functions

Description

Increasing this function may tell various functions to multithread their operations. You may also want to increase allowableThreads.

See also

  • isRankAtLeast -- determines if the matrix has rank at least a number
  • getSubmatrixOfRank -- tries to find a submatrix of the given rank
  • recursiveMinors -- uses a recursive cofactor algorithm to compute the ideal of minors of a matrix

Functions with optional argument named Threads :

  • "getSubmatrixOfRank(...,Threads=>...)" -- see getSubmatrixOfRank -- tries to find a submatrix of the given rank
  • "isRankAtLeast(...,Threads=>...)" -- see isRankAtLeast -- determines if the matrix has rank at least a number
  • "recursiveMinors(...,Threads=>...)" -- see recursiveMinors -- uses a recursive cofactor algorithm to compute the ideal of minors of a matrix

For the programmer

The object Threads is a symbol.
