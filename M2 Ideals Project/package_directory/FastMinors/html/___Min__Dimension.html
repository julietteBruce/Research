FastMinors :: MinDimension

MinDimension -- an option for projDim

Description

This option is used to tell the function projDim not to look for projective dimension below the option value.

See also

  • projDim -- finds an upper bound for the projective dimension of a module

Functions with optional argument named MinDimension :

  • "projDim(...,MinDimension=>...)" -- see projDim -- finds an upper bound for the projective dimension of a module

For the programmer

The object MinDimension is a symbol.
