NoetherianOperators :: evaluate(DiffOp,AbstractPoint)

evaluate(DiffOp,AbstractPoint) -- evaluate coefficients of a differential operator

Synopsis

  • Function: evaluate
  • Usage:
    evaluate(D, p)
  • Inputs:
    • D, a differential operator,
    • p, a point, or a one-row matrix
  • Outputs:
    • a differential operator,

Description

Evaluates the polynomial coefficients of a differential operator at a point. This can be used to obtain a set of specialized Noetherian operators.

i1 : R = QQ[x,y];
 
i2 : S = diffOpRing R;
 
i3 : D = diffOp(x*dx+y*dy^3)

o3 = | ydy^3+xdx |

                1
o3 : DiffOp in S
 
i4 : evaluate(D, point{{1,2}})

o4 = | 2dy^3+dx |

                1
o4 : DiffOp in S

Ways to use this method:

  • evaluate(DiffOp,AbstractPoint) -- evaluate coefficients of a differential operator
  • "evaluate(DiffOp,Matrix)"
