NoetherianOperators :: Sampler

Sampler -- optional sampler function

Description

There are currently two methods that accept a user-specified sampler function

  • Strategy => "Hybrid" used with noetherianOperators
  • numericalNoetherianOperators

See the respective documentation nodes for details and syntax. If the option Sampler is unset, the default sampler will use Bertini.

See also

  • Strategy => "Hybrid" -- strategy for computing Noetherian operators
  • numericalNoetherianOperators -- Noetherian operators via numerical interpolation

For the programmer

The object Sampler is a symbol.
