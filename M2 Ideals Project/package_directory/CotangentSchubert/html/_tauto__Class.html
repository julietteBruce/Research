CotangentSchubert :: tautoClass

tautoClass -- Compute the class of a tautological bundle

Synopsis

  • Usage:
    tautoClass (i,j)
    tautoClass (i,j,A)
  • Inputs:
    • i, an integer,
    • j, an integer,
    • A, a ring,

Description

This function computes the i-th Chern class of the j-th tautological bundle of the flag variety whose K-theory (or cohomology) ring is given by A. If A is not specified, then the ring that was defined last is used.

For the programmer

The object tautoClass is a method function with options.
