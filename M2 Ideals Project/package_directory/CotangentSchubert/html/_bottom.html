CotangentSchubert :: bottom

bottom -- The bottom string of a puzzle

Synopsis

  • Usage:
    bottom P
  • Inputs:
    • P, an instance of the type Puzzle,

Description

Reads off the bottom boundary string of a puzzle.

For the programmer

The object bottom is a function closure.
