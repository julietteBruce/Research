CotangentSchubert :: restrict

restrict -- Restriction to fixed points

Description

Given a (K-theory or cohomology) class in a ring defined by setupCotangent with the options Presentation => Borel and Equivariance => true, this function computes its restriction to fixed points (effectively, giving the class as would be produced by the option Presentation => EquivLoc).

For the programmer

The object restrict is a method function.
