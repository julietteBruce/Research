CotangentSchubert :: canonicalClass

canonicalClass -- Class of the canonical bundle of a flag variety

Description

This function returns the class of the canonical bundle of the flag variety in the cohomology ring given as argument (or the last ring defined with setupCotangent if no argument is given).

For the programmer

The object canonicalClass is a method function with options.
