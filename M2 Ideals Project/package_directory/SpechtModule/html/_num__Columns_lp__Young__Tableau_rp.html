SpechtModule :: numColumns(YoungTableau)

numColumns(YoungTableau) -- returns the number of columns of a tableau

Synopsis

  • Function: numColumns
  • Usage:
    numcols y
  • Inputs:
    • y, an instance of the type YoungTableau,
  • Outputs:
    • an integer, the number of columns of the tableau

Description

i1 : y = youngTableau(new Partition from {2,1,1,1},{2,0,1,4,3})

o1 = | 2 0 |
     | 1 |
     | 4 |
     | 3 |

o1 : YoungTableau
 
i2 : numcols y

o2 = 2
