SpechtModule :: spechtPolynomials(Partition,PolynomialRing)

spechtPolynomials(Partition,PolynomialRing) -- the set of all Specht polynomial indexed by standard tableaux of shape p

Synopsis

  • Function: spechtPolynomials
  • Usage:
    spechtPolynomials(p,R)
  • Inputs:
    • p, an instance of the type Partition,
    • R, a polynomial ring,
  • Optional inputs:
    • AsExpression => ..., default value false, an optional argument that returns polynomials as expressions
  • Outputs:
    • a hash table, a hash table with the polynomials index by the filling of their respective tableaux

Description

The set of all the Specht polynomials for standard tableaux of a given shape p forms a basis for a module which is isomorphic to the Specht module indexed by p.
