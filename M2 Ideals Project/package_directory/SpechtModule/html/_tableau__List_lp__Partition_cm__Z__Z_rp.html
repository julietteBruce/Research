SpechtModule :: tableauList(Partition,ZZ)

tableauList(Partition,ZZ) -- the constructor for the type TableauList

Synopsis

  • Function: tableauList
  • Usage:
    tableauList(p,n)
    tableauList(p)
  • Inputs:
    • p, an instance of the type Partition, the shape for the tableaux
    • n, an integer, the number of tableaux in the list, if is not provided then the default value multinomial p is used. See multinomial.
  • Outputs:
    • an instance of the type TableauList, an empty TableauList with space for n tableaux

Description

i1 : p = new Partition from {2,1}

o1 = Partition{2, 1}

o1 : Partition
 
i2 : y = youngTableau(p,{0,1,2})

o2 = | 0 1 |
     | 2 |

o2 : YoungTableau
 
i3 : t = tableauList p

o3 = {}

o3 : TableauList
 
i4 : addTableau(t,y)

      | 2 |

o4 : TableauList
