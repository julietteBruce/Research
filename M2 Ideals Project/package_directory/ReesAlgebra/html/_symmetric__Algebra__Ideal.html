ReesAlgebra :: symmetricAlgebraIdeal

symmetricAlgebraIdeal -- Ideal of the symmetric algebra of an ideal or module

Synopsis

  • Usage:
    I = symmetricAlgebraIdeal J
  • Inputs:
    • I, an ideal,
    • I, a module,
  • Optional inputs:
    • VariableBaseName => ..., default value "w"
  • Outputs:
    • J, an ideal,

Description

Uses the built-in function symmetricAlgebra. The function returns J an ideal in a new ring, with generators corresponding to those of th eideal or module I. The name of the new generators may be set, for example to T, with the form

symmetricAlgebraIdeal(J, VariableBaseName =>"T")

See also

  • reesIdeal -- Compute the defining ideal of the Rees Algebra

Ways to use symmetricAlgebraIdeal :

  • "symmetricAlgebraIdeal(Ideal)"
  • "symmetricAlgebraIdeal(Module)"

For the programmer

The object symmetricAlgebraIdeal is a method function with options.
