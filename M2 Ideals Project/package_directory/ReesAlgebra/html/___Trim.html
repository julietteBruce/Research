ReesAlgebra :: Trim

Trim -- Choose whether to trim (or find minimal generators) for the ideal or module.

Synopsis

  • Usage:
    reesIdeal(..., Trim => true)

Description

Note that when Trim=>true, the generators used will be the ones (and in the order) M2 likes, possibly not the original ones.
