localResolution -- find a resolution over a local ring

Synopsis

  • Usage:
    localResolution M
  • Inputs:
    • M, a module
    • or
    • an ideal
    • Strategy, see resolution(...,Strategy=>...)
    • SyzygyLimit, see resolution(...,SyzygyLimit=>...)
    • HardDegreeLimit, see resolution(...,HardDegreeLimit=>...)
    • LengthLimit, see resolution(...,LengthLimit=>...)
    • StopBeforeComputation, see resolution(...,StopBeforeComputation=>...)
    • DegreeLimit, see resolution(...,DegreeLimit=>...)
    • FastNonminimal, see resolution(...,FastNonminimal=>...)
    • SortStrategy, see resolution(...,SortStrategy=>...)
    • PairLimit, see resolution(...,PairLimit=>...)
  • Optional inputs:
    • DegreeLimit => ..., default value null
    • FastNonminimal => ..., default value false
    • HardDegreeLimit => ..., default value {}
    • LengthLimit => ..., default value infinity
    • PairLimit => ..., default value infinity
    • SortStrategy => ..., default value 0
    • StopBeforeComputation => ..., default value false
    • Strategy => ..., default value null
    • SyzygyLimit => ..., default value infinity
  • Outputs:
    • a chain complex

Description

This method has option inputs that it inherits from

resolution.

This function iterates

localsyz to obtain a resolution over the local ring.
