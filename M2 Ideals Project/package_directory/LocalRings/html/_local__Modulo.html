localModulo -- find the pre-image (pullback) of image of a map over a local ring

Synopsis

  • Usage:
    localModulo(m,n)
  • Inputs:
    • m, a matrix
    • n, a matrix
  • Outputs:
    • a matrix, whose image is the pre-image (pullback) of the image of n under m

Description

The maps m and n must have the same target, and their sources and targets must be free. If m is null, then it is taken to be the identity. If n is null, it is taken to be zero.
