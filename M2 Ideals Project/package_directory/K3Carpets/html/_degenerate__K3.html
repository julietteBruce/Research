K3Carpets :: degenerateK3

degenerateK3 -- Ideal of a degenerate K3 surface X_e(a,b)

Synopsis

  • Usage:
    I = degenerateK3(a,b,e)
    I = degenerateK3(a,b,t)
  • Inputs:
    • a, an integer,
    • b, an integer, a and b should be positive
    • e, a sequence, of two integers (e_1,e_2)
    • t, a list, of two integers {t_1,t_2}
  • Optional inputs:
    • Characteristic => an integer, default value 32003, the characteristic of the ground field
  • Outputs:
    • I, an ideal,

Description

The routine computes the homogeneous ideal of the degenerate K3 surface in $\mathbb P^{a+b+1}$ associated as in HREF{y}{x} to a polynomial $$p(z)=z^2-e_1z+e_2=(z-t_1)(z-t_2)$$ In case $p(z)=(z-1)^2$ it coincides with carpet(a,b).
