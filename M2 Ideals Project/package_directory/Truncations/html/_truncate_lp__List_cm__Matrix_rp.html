Truncations > truncate(List,Matrix)

truncate(List,Matrix) -- truncation of a map of free modules

Synopsis

  • Function: truncate
  • Usage:
    truncate(degs, f)
  • Inputs:
    • degs, a list, an integer or a list, a single degree, a multidegree, or a list of degrees or multidegrees
    • f, a matrix, a graded map between graded modules (not necessarily free modules)
  • Optional inputs:
    • MinimalGenerators => a Boolean value, default value true, indicates whether the source and target of the result should be trimmed
  • Outputs:
    • a matrix, a graded map between the truncations of the source and target of $f$

Description

This function truncates the source and target of a module map, and returns the induced map between them.
