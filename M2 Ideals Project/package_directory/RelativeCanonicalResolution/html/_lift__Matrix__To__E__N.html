RelativeCanonicalResolution :: liftMatrixToEN

liftMatrixToEN -- Lifts a matrix between bundles on the scroll to the associated Eagon-Northcott type complexes

Synopsis

  • Usage:
    A=liftMatrixToEN(Psi,e)
  • Inputs:
    • Psi, a matrix, the matrix that defines map between vector bundles on the scroll.
    • e, a list, the type of the scroll $P(E)$
  • Outputs:
    • A, a matrix, the induced map

Description

Given a map between vector bundles F and G on a normalized scroll of type e, this function computes the induced map between the first modules in the Eagon-Northcott type resolution of F and G.
