RelativeCanonicalResolution :: eagonNorthcottType

eagonNorthcottType -- Computes the Eagon-Northcott type resolution

Synopsis

  • Usage:
    resX=resCurveOnScroll(Phi,b)
  • Inputs:
    • Phi, a matrix, the matrix whose maximal minors define the scroll
    • b, an integer, the twist
  • Outputs:
    • ENresolution, a chain complex, the Eagon-Northcott type resolution

Description

This function computes the Eagon-Northcott type resolution associated to the matrix Phi defining the scroll and an integer b defining the twist $O_{P(E)}(bR)$. The way the function works is similar to the way the "Eagon-Northcott" function by Greg Smith works.
