futureParser -- forward reference to a parser not defined yet

Synopsis

  • Usage:
    futureParser p
  • Inputs:
    • p, a symbol
  • Outputs:
    • a parser, a parser that will pass its input to the value of the symbol p at that time

Description

This function makes recursive parsers possible.

i1 : p = futureParser q

o1 = p

o1 : Parser
 
i2 : m = p : charAnalyzer

o2 = m

o2 : FunctionClosure
 
i3 : q = constParser "abc"

o3 = q

o3 : Parser
 
i4 : m "abc"

o4 = abc
