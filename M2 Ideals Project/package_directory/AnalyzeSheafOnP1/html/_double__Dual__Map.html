AnalyzeSheafOnP1 :: doubleDualMap

doubleDualMap -- map from a module to its double dual

Synopsis

  • Usage:
    e = doubleDualMap M
  • Inputs:
    • M, a module,
  • Outputs:
    • e, a matrix, map from M to double dual

Description

provide the natural map M --> Hom(Hom(M,S),S), where S = ring M.

Ways to use doubleDualMap :

  • "doubleDualMap(Module)"

For the programmer

The object doubleDualMap is a method function.
