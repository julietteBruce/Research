AnalyzeSheafOnP1 :: showSheafOnP1

showSheafOnP1 -- Prints the analysis of a sheaf on P1

Synopsis

  • Usage:
    showSheafOnP1 M
  • Inputs:
    • M, a module,
    • M, a coherent sheaf,

Description

prints out the twists of the line bundle summands and the annihilators of the (cyclic) torsion components.

See also

  • AnalyzeSheafOnP1 -- Describe a graded module over k[x,y] without 0-dimensional torsion

Ways to use showSheafOnP1 :

  • "showSheafOnP1(CoherentSheaf)"
  • "showSheafOnP1(Module)"

For the programmer

The object showSheafOnP1 is a method function.
