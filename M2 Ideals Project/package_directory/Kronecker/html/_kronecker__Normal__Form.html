Kronecker :: kroneckerNormalForm

kroneckerNormalForm -- Kronecker normal form of a matrix of linear forms

Synopsis

  • Usage:
    (B,P,Q) = kroneckerNormalForm A
    (B,P) = kroneckerNormalForm(A,ChangeMatrix=>{true,false})
    (B,Q) = kroneckerNormalForm(A,ChangeMatrix=>{false,true})
    B = kroneckerNormalForm(A,ChangeMatrix=>{false,false})
    (B,P,Q) = kroneckerNormalForm(A,x,y)
    (B,P) = kroneckerNormalForm(A,x,y,ChangeMatrix=>{true,false})
    (B,Q) = kroneckerNormalForm(A,x,y,ChangeMatrix=>{false,true})
    B = kroneckerNormalForm(A,x,y,ChangeMatrix=>{false,false})
  • Inputs:
    • A, a matrix, a matrix of linear forms in two variables
    • x, a ring element
    • y, a ring element
  • Optional inputs:
    • ChangeMatrix => ..., default value {true, true}
  • Outputs:
    • B, a matrix, the Kronecker normal form of A
    • P, a matrix, invertible (left) change of basis matrix
    • Q, a matrix, invertible (right) change of basis matrix

Description

This function produces a matrix of linear forms B, and invertible matrices P and Q such that B = P*A*Q and B is in Kronecker normal form. The optional parameters x and y, if given, should be complementary linear forms. They are used to fill in nonzero entries of all blocks in the Kronecker normal form for which there is a choice of which linear forms to use. If they are not given, they are assumed to be the first two variables of the ring.
