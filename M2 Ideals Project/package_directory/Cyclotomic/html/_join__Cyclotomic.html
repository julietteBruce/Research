Cyclotomic :: joinCyclotomic

joinCyclotomic -- Join ideals in polynomial rings over different cyclotomic fields

Synopsis

  • Usage:
    joinCyclotomic l
  • Inputs:
    • li, a list of ideals in polynomial rings over cyclotomic fields
  • Outputs:
    • li2, The list of ideals in a common ring.
