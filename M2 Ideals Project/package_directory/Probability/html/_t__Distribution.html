Probability :: tDistribution

tDistribution -- Student's t-distribution

Synopsis

  • Usage:
    tDistribution df
  • Inputs:
    • df, a number, the degrees of freedom (positive);
  • Outputs:
    • a continuous probability distribution,

Description

The Student's t-distribution.

i1 : X = tDistribution 5

o1 = t(5)

o1 : ContinuousProbabilityDistribution
 
i2 : density_X 2

o2 = .0650903103262165

o2 : RR (of precision 53)
 
i3 : probability_X 3

o3 = .984950376051269

o3 : RR (of precision 53)
 
i4 : quantile_X 0.4

o4 = -.267180865704146

o4 : RR (of precision 53)
 
i5 : random X

o5 = 1.42096698549053

o5 : RR (of precision 53)
