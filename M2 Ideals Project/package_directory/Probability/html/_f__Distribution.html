Probability :: fDistribution

fDistribution -- F-distribution

Synopsis

  • Usage:
    fDistribution(d1, d2)
  • Inputs:
    • d1, a number, the numerator degrees of freedom (positive);
    • d2, a number, the denominator degrees of freedom (positive);
  • Outputs:
    • a continuous probability distribution,

Description

The F-distribution, widely used in ANOVA.
