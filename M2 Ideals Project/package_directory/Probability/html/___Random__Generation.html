Probability :: RandomGeneration

RandomGeneration -- random generation function

Description

A key in ProbabilityDistribution objects and an option for discreteProbabilityDistribution and continuousProbabilityDistribution for setting the random generation function to be used by random(ProbabilityDistribution).

Functions with optional argument named RandomGeneration :

  • "continuousProbabilityDistribution(...,RandomGeneration=>...)" -- see continuousProbabilityDistribution -- construct a continuous probability distribution
  • "discreteProbabilityDistribution(...,RandomGeneration=>...)" -- see discreteProbabilityDistribution -- construct a discrete probability distribution

For the programmer

The object RandomGeneration is a symbol.
