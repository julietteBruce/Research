Probability :: normalDistribution

normalDistribution -- normal distribution

Synopsis

  • Usage:
    normalDistribution(mu, sigma)
  • Inputs:
    • mu, a number, the mean;
    • sigma, a number, the standard deviation (positive);
  • Outputs:
    • a continuous probability distribution,

Description

The normal distribution with mean mu and standard deviation sigma.

i1 : X = normalDistribution(30, 5)

o1 = N(30,5)

o1 : ContinuousProbabilityDistribution
 
i2 : density_X 25

o2 = .0483941449038287

o2 : RR (of precision 53)
 
i3 : probability_X 35

o3 = .841344746068543

o3 : RR (of precision 53)
 
i4 : quantile_X 0.75

o4 = 33.3724487509804

o4 : RR (of precision 53)
 
i5 : random X

o5 = 28.8971809640044

o5 : RR (of precision 53)

With no arguments, the standard normal distribution with mean 0 and standard deviation 1 is returned.

i6 : Z = normalDistribution()

o6 = N(0,1)

o6 : ContinuousProbabilityDistribution
 
i7 : quantile_Z 0.975

o7 = 1.95996398454006

o7 : RR (of precision 53)
