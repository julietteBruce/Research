Probability :: poissonDistribution

poissonDistribution -- Poisson distribution

Synopsis

  • Usage:
    poissonDistribution lambda
  • Inputs:
    • lambda, a number, the rate parameter;
  • Outputs:
    • a discrete probability distribution,

Description

The Poisson distribution, the distribution of the number of events to occur during some interval of time when the expected number of events is lambda.
