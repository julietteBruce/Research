Macaulay2Doc :: Inhomogeneous

Inhomogeneous -- a Strategy option value

Description

Inhomogeneous -- a strategy used with the keyword Strategy.

This is the default Gröbner basis algorithm used if the submodule is inhomogeneous, and the ring is a (quotient of) a polynomial ring over a field.

See also

  • gb(...,Strategy=>...) -- compute a Gröbner basis

For the programmer

The object Inhomogeneous is a symbol.
