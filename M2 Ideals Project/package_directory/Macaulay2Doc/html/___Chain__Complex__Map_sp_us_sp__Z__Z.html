ChainComplexMap _ ZZ -- component map

Synopsis

  • Operator: _
  • Usage:
    p_i
  • Inputs:
    • p, a chain complex map, a map D <- C of chain complexes, of degree d, say
    • i, an integer
  • Outputs:
    • a matrix, the component D_(i+d) <- C_i
