Macaulay2Doc :: debuggingMode

debuggingMode -- whether to enter the debugger when an error occurs

Synopsis

  • Usage:
    debuggingMode = true
  • Consequences:
    • the debugger will be entered when an error occurs

For the programmer

The object debuggingMode is a Boolean value.
