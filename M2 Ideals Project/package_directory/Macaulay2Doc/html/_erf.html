Macaulay2Doc :: erf

erf -- error function

Synopsis

  • Usage:
    erf x
  • Inputs:
    • x, a real number,
  • Outputs:
    • a real number, the error function of x;

Description

The error function \(\operatorname{erf} x = \frac{2}{\sqrt\pi}\int_0^x e^{-t^2}\,dt\).

i1 : erf 2

o1 = .995322265018953

o1 : RR (of precision 53)

See also

  • erfc -- complementary error function
  • inverseErf -- inverse error function

Ways to use erf :

  • erf(RR) (missing documentation)

For the programmer

The object erf is a method function.
