Macaulay2Doc :: uniquePermutations

uniquePermutations -- produce all unique permutations of a list

Synopsis

  • Usage:
    uniquePermutations x
  • Inputs:
    • x, a visible list or an integer
  • Outputs:
    • a list of all distinct permutations of the visible list x, or, if x is an integer, of the list of integers from 0 through n-1

Description

i1 : uniquePermutations {a,b,a,b}

o1 = {{a, b, a, b}, {a, b, b, a}, {a, a, b, b}, {b, a, a, b}, {b, a, b, a},
     ------------------------------------------------------------------------
     {b, b, a, a}}

o1 : List

Ways to use uniquePermutations :

  • "uniquePermutations(VisibleList)"
  • "uniquePermutations(ZZ)"

For the programmer

The object uniquePermutations is a method function.
