Macaulay2Doc :: factor

factor -- factor a ring element or a ZZ-module

Description

(Disambiguation: for division of matrices, which can be thought of as factoring one homomorphism through another, instead see Matrix // Matrix. For lifting a map between modules to a map between their free resolutions, see extend.)

Ways to use factor :

  • factor(Module) -- factor a ZZ-module
  • "factor(QQ)" -- see factor(RingElement) -- factor a ring element
  • factor(RingElement) -- factor a ring element
  • "factor(ZZ)" -- see factor(RingElement) -- factor a ring element

For the programmer

The object factor is a method function with options.
