wedgeProduct(ZZ,ZZ,Module) -- the exterior multiplication map

Synopsis

  • Function: wedgeProduct
  • Usage:
    wedgeProduct(p,q,F)
  • Inputs:
    • p, an integer
    • q, an integer
    • F, a module, a free module
  • Outputs:
    • a matrix, representing the multiplication map from exteriorPower(p,M) ** exteriorPower(q,M) to exteriorPower(p+q,M)
