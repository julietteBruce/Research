>> -- a binary operator, uses include bit shifting, or attaching optional inputs to functions

Ways to use symbol >> :

  • "Boolean >> Function" -- see OptionTable >> Function -- attaching options to a function
  • "List >> Function" -- see OptionTable >> Function -- attaching options to a function
  • OptionTable >> Function -- attaching options to a function
  • "CC >> ZZ" -- see right shift
  • "RR >> ZZ" -- see right shift
  • "RRi >> ZZ" -- see right shift
  • "ZZ >> ZZ" -- see right shift

For the programmer

The object >> is a keyword.

This operator may be used as a binary operator in an expression like x>>y. The user may install binary methods for handling such expressions with code such as

         X >> Y := (x,y) -> ...

where X is the class of x and Y is the class of y.
