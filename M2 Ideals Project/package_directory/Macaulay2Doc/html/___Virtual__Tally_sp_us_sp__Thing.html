Macaulay2Doc :: VirtualTally _ Thing

VirtualTally _ Thing -- get a count from a tally

Synopsis

  • Operator: _
  • Usage:
    t_x
  • Inputs:
    • t, a virtual tally
    • x, a thing
  • Outputs:
    • an integer, the number of times x is counted by t

Description

i1 : t = tally apply(1..10000, i -> # factor i)

o1 = Tally{0 => 1   }
           1 => 1280
           2 => 4097
           3 => 3695
           4 => 894
           5 => 33

o1 : Tally
 
i2 : t_5

o2 = 33
 
i3 : t_6

o3 = 0
