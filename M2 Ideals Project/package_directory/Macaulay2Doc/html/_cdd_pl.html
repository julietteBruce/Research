cdd+

The program cdd+, written by Komei Fukuda, is a C++ implementation of the double description method of Motzkin, et al., for generating all vertices and extreme rays of a general convex polyhedron in $\RR^d$ given by a system of linear inequalities. It is available at http://www.ifor.math.ethz.ch/~fukuda/cdd_home/.
