Macaulay2Doc :: isQuotientRing

isQuotientRing -- whether something is a quotient ring

Synopsis

  • Usage:
    isQuotientRing R
  • Inputs:
    • R, a ring
  • Outputs:
    • a Boolean value, true if R is a quotient ring and false otherwise

Description

i1 : S = ZZ/3[x,y,z];
 
i2 : isQuotientRing S

o2 = false
 
i3 : R = S/(x^2-y*z);
 
i4 : isQuotientRing R

o4 = true
 
i5 : ambient R

o5 = S

o5 : PolynomialRing
 
i6 : symAlg = symmetricAlgebra R^2;
 
i7 : isQuotientRing symAlg

o7 = false
 
i8 : sing = singularLocus R;
 
i9 : isQuotientRing sing

o9 = true

See also

  • ambient -- ambient free module of a subquotient, or ambient ring

Ways to use isQuotientRing :

  • "isQuotientRing(QuotientRing)"
  • "isQuotientRing(Ring)"

For the programmer

The object isQuotientRing is a method function.
