Macaulay2Doc :: Complement

Complement -- a Strategy option value

Description

The standard (default) strategy for mingens or trim is Complement, at least in the case where the module (or ideal) is homogeneous, and the ring is (quotient ring of) a polynomial ring over a field. The other strategy, Inhomogeneous, is experimental.

For the programmer

The object Complement is a symbol.
