Macaulay2Doc :: BesselJ

BesselJ -- Bessel function of the first kind

Synopsis

  • Usage:
    BesselJ(n,x)
    BesselJ_n x
  • Inputs:
    • n, an integer, the order
    • x, an integer, a rational number, or a real number
  • Outputs:
    • the Bessel function of the first kind of order n at x

Description

i1 : BesselJ_0 .5

o1 = .938469807240813

o1 : RR (of precision 53)
 
i2 : BesselJ_2 3p200

o2 = .486091260585891076907831094114984034801662265643298727377034

o2 : RR (of precision 200)

See Bessel function.

See also

  • BesselY -- Bessel function of the second kind

Ways to use BesselJ :

  • "BesselJ(ZZ,Constant)"
  • "BesselJ(ZZ,Number)"

For the programmer

The object BesselJ is a method function.
