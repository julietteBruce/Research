Macaulay2Doc :: toCC

toCC -- convert to high-precision complex number

Description

Synopsis

  • Usage:
    toCC(prec,x,y)
    toCC(prec,x)
  • Inputs:
    • prec, an integer, the number of bits of precision desired
    • x, an integer, a rational number, or a real number
    • y, an integer, a rational number, or a real number
  • Outputs:
    • a complex number, the complex number with real part x and complex part y. If y is omitted, the imaginary part is zero. 
i1 : toCC(200,7)

o1 = 7

o1 : CC (of precision 200)
 
i2 : toCC(100,7,3.)

o2 = 7+3*ii

o2 : CC (of precision 100)

Synopsis

  • Usage:
    toCC(x,y)
    toCC x
  • Inputs:
    • x, a real number
    • y, a real number
  • Outputs:
    • a complex number, the complex number with real part x and complex part y. If y is omitted, the imaginary part is zero. The precision of the result is the minimum precision of the arguments. 
i3 : toCC(3.,4.)

o3 = 3+4*ii

o3 : CC (of precision 53)
 
i4 : toCC(3.p100,4.p200)

o4 = 3+4*ii

o4 : CC (of precision 100)
