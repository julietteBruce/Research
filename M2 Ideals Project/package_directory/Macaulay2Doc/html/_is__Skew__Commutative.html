Macaulay2Doc :: isSkewCommutative

isSkewCommutative -- whether a ring has skew commuting variables

Synopsis

  • Usage:
    isSkewCommutative R
  • Inputs:
    • R, a ring
  • Outputs:
    • a Boolean value, true if R has skew commuting variables and false otherwise

Description

i1 : A = ZZ/3[a,b,c];
 
i2 : isSkewCommutative A

o2 = false
 
i3 : B = QQ[a..d,SkewCommutative=>{a,b}]

o3 = B

o3 : PolynomialRing, 2 skew commutative variables
 
i4 : isSkewCommutative B

o4 = true
 
i5 : C = B[x,y]

o5 = C

o5 : PolynomialRing
 
i6 : isSkewCommutative C

o6 = true
 
i7 : b_C * a_C

o7 = -a*b

o7 : C
 
i8 : D = B/(a*d-b*c)

o8 = D

o8 : QuotientRing
 
i9 : isSkewCommutative D

o9 = true
