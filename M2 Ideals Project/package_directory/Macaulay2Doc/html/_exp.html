Macaulay2Doc :: exp

exp -- exponential function

Synopsis

  • Usage:
    exp x
    exp I
  • Inputs:
    • x, a real number
    • I, a real interval
  • Outputs:
    • the exponential of x
    • a real interval, an interval containing the exponentials of points of I

Description

i1 : exp 1p300

o1 = 2.7182818284590452353602874713526624977572470936999595749669676277240766
     3035354759457138218

o1 : RR (of precision 300)
 
i2 : exp(pi*ii)

o2 = -1+1.22464679914735e-16*ii

o2 : CC (of precision 53)

Ways to use exp :

  • "exp(CC)"
  • "exp(RR)"
  • "exp(RRi)"
  • exp(RingElement)

For the programmer

The object exp is a method function.
