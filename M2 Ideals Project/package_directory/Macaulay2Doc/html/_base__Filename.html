Macaulay2Doc :: baseFilename

baseFilename -- the base part of a filename or path

Synopsis

  • Usage:
    baseFilename fn
  • Inputs:
    • fn, a string, a filename or path
  • Outputs:
    • the last component of the path

Description

i1 : baseFilename "/a/b/dir/"

o1 = dir
 
i2 : baseFilename "/a/b/file"

o2 = file
