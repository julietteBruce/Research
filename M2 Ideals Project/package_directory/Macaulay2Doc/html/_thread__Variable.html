threadVariable -- create a symbol whose value in one thread is not shared with others

Synopsis

  • Usage:
    threadVariable foo
  • Outputs:
    • a new symbol, whose name is "foo", for example, whose values in each thread will be independent of each other, with initial value null

Description

i1 : threadVariable x

o1 = x

o1 : Symbol
 
i2 : x = 1

o2 = 1
 
i3 : t = schedule ( () -> ( x = 2 ; x ) )

o3 = <<task, result available, task done>>

o3 : Task
 
i4 : while not isReady t do null
 
i5 : taskResult t

o5 = 2
 
i6 : x

o6 = 1

For the programmer

The object threadVariable is a keyword.
