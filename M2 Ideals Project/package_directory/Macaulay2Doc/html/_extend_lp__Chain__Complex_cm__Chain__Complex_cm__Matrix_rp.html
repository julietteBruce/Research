extend(ChainComplex,ChainComplex,Matrix) -- extend a module map to a chain map, if possible

Synopsis

  • Function: extend
  • Usage:
    extend(D,C,f0)
  • Inputs:
    • D, a chain complex
    • C, a chain complex
    • f0, a matrix, a map from C_0 to D_0
  • Optional inputs:
    • Verify => a Boolean value, default value true, whether to check that the map extends
  • Outputs:
    • a chain complex map, a chain complex map f: D <--- C of degree 0 extending f0 in the sense that f_0==f0
