Macaulay2Doc :: hilbertSeries(...,Reduce=>...)

hilbertSeries(...,Reduce=>...) -- reduce the Hilbert series

Synopsis

  • Usage:
    hilbertSeries(..., Reduce => true)
  • Consequences:
    • the resulting rational function is reduced by cancelling factors of the numerator that occur explicitly as factors of the denominator. See also reduceHilbert.

Description

i1 : R = QQ[x,y,z];
 
i2 : hilbertSeries ideal (x,y)

               2
     1 - 2T + T
o2 = -----------
              3
       (1 - T)

o2 : Expression of class Divide
 
i3 : hilbertSeries(ideal (x,y), Reduce => true)

        1
o3 = -------
     (1 - T)

o3 : Expression of class Divide

Further information

  • Default value: false
  • Function: hilbertSeries -- compute the Hilbert series
  • Option key: Reduce -- an optional argument

Functions with optional argument named Reduce :

  • hilbertSeries(...,Reduce=>...) -- reduce the Hilbert series
