Macaulay2Doc :: ambient

ambient -- ambient free module of a subquotient, or ambient ring

Description

Each module M in Macaulay2 is a quotient, submodule or subquotient of a free module, called the ambient free module of M.

The ambient ring of a quotient ring R/I is the polynomial ring of that it is a quotient. The ambient ring of a Galois field is the quotient ring that it was constructed from.
