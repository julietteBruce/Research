Macaulay2Doc :: rsort(...,DegreeOrder=>...)

rsort(...,DegreeOrder=>...) -- specify Ascending, Descending, or null

Description

The meaning is the same as from sort(...,MonomialOrder=>...).

Further information

  • Default value: null
  • Function: rsort -- sort a list or matrix in reverse order
  • Option key: DegreeOrder -- sort primarily by degree

Functions with optional argument named DegreeOrder :

  • rsort(...,DegreeOrder=>...) -- specify Ascending, Descending, or null
  • sort(...,DegreeOrder=>...) -- specify Ascending, Descending, or null
  • sortColumns(...,DegreeOrder=>...) -- specify Ascending, Descending, or null
