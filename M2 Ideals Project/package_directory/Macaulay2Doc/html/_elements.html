Macaulay2Doc :: elements

elements -- list of elements

Synopsis

  • Usage:
    elements S
  • Inputs:
    • S, a set, or a tally
  • Outputs:
    • a list

Description

i1 : set{1,2,3,1,2,4}

o1 = set {1, 2, 3, 4}

o1 : Set
 
i2 : elements oo

o2 = {1, 2, 3, 4}

o2 : List
 
i3 : tally apply(10, i -> random 5)

o3 = Tally{0 => 2}
           1 => 1
           2 => 1
           3 => 5
           4 => 1

o3 : Tally
 
i4 : elements oo

o4 = {0, 0, 1, 2, 3, 3, 3, 3, 3, 4}

o4 : List

See also

  • tally -- tally the elements of a list, sequence, array, or string
  • set -- make a set

Ways to use elements :

  • "elements(Set)"
  • "elements(Tally)"

For the programmer

The object elements is a method function.
