Macaulay2Doc :: isReal

isReal -- whether a number is real

Synopsis

  • Usage:
    isReal x
  • Inputs:
    • x, a number
  • Outputs:
    • a Boolean value, whether x is real

Description

i1 : isReal 2.

o1 = true
 
i2 : isReal sqrt(-1)

o2 = false

Ways to use isReal :

  • "isReal(CC)"
  • "isReal(Constant)"
  • "isReal(InfiniteNumber)"
  • "isReal(QQ)"
  • "isReal(RR)"
  • "isReal(RRi)"
  • "isReal(ZZ)"

For the programmer

The object isReal is a method function.
