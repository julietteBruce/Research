setIOExclusive -- exclusive I/O for the current thread

Synopsis

  • Usage:
    setIOExclusive()
  • Consequences:
    • the current thread becomes the only one permitted to use the files stdio and stderr

See also

  • parallel programming with threads and tasks
  • setIOUnSynchronized -- unsynchronized I/O for threads
  • setIOSynchronized -- synchronized I/O for threads

For the programmer

The object setIOExclusive is a compiled function.
