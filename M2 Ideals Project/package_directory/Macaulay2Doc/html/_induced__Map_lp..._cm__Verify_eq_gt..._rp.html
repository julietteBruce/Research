Macaulay2Doc :: inducedMap(...,Verify=>...)

inducedMap(...,Verify=>...) -- verify that a map is well-defined

Description

Verify => true -- an option for inducedMap which requests verification that the induced map produced is well defined.

Further information

  • Default value: true
  • Function: inducedMap -- compute an induced map
  • Option key: Verify -- an optional argument

Functions with optional argument named Verify :

  • "extend(...,Verify=>...)" -- see extend(ChainComplex,ChainComplex,Matrix) -- extend a module map to a chain map, if possible
  • inducedMap(...,Verify=>...) -- verify that a map is well-defined
  • "lift(...,Verify=>...)" -- see lift -- lift to another ring
