Macaulay2Doc :: printingLeadLimit

printingLeadLimit -- maximum number of leading zeroes to use when printing real numbers

Synopsis

  • Usage:
    printingLeadLimit = n
  • Inputs:
    • n, an integer
  • Consequences:
    • Real numbers are printed with at most n leading zeroes.

Description

i1 : 1/30000000000.

o1 = 3.33333333333333e-11

o1 : RR (of precision 53)
 
i2 : printingLeadLimit

o2 = 5
 
i3 : printingLeadLimit = 20

o3 = 20
 
i4 : 1/30000000000.

o4 = .0000000000333333333333333

o4 : RR (of precision 53)

  • printingAccuracy -- current accuracy for printing numbers
  • printingTrailLimit -- maximum number of additional trailing digits to use when printing real numbers
  • printingSeparator -- string used to separate mantissa from exponent when printing real numbers
  • format -- format a string or a real number

For the programmer

The object printingLeadLimit is an integer.
