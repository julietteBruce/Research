Macaulay2Doc :: isSubset(Set,Set)

isSubset(Set,Set) -- whether one object is a subset of another

Synopsis

  • Function: isSubset
  • Usage:
    isSubset(x,y)
  • Inputs:
    • x, a set, a set or a list
    • y, a set, a set or a list
  • Outputs:
    • a Boolean value, whether every element of x is in y

Description

i1 : isSubset(set{a},set{a,b,c})

o1 = true
 
i2 : isSubset({a},set{a,b,c})

o2 = true
 
i3 : isSubset({a,a},{a,b,c})

o3 = true

See also

  • Set -- the class of all sets

Ways to use this method:

  • "isSubset(Set,VisibleList)"
  • "isSubset(VisibleList,Set)"
  • "isSubset(VisibleList,VisibleList)"
