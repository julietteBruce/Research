using Macaulay2 with emacs

Emacs is a free text editor written by Richard Stallman, which can be used as an ascii-oriented interface to Macaulay2. The best way to edit Macaulay2 code or to run Macaulay2 is with emacs. Emacs is available as part of many operating systems, as well as from http://www.gnu.org/software/emacs/.
