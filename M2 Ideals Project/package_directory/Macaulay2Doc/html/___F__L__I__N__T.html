FLINT

The library FLINT (Fast Library for Number Theory) is a library for computations in number theory, consisting mainly of routines for integer and polynomial arithmetic and linear algebra. It was written by William Hart, Mike Hansen, Sebastian Pancratz, Fredrik Johansson, and others, and is available at http://flintlib.org/.
