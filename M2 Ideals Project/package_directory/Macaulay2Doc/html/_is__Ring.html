Macaulay2Doc :: isRing

isRing -- whether something is a ring

Synopsis

  • Usage:
    isRing R
  • Inputs:
    • R, a thing
  • Outputs:
    • a Boolean value, true if R is a ring and false otherwise

Description

i1 : isRing QQ

o1 = true
 
i2 : R = QQ[x,y];
 
i3 : isRing(R)

o3 = true
 
i4 : isRing(R^1)

o4 = false
 
i5 : isRing(GF(2,3)[x,dx, WeylAlgebra => {x => dx}])

o5 = true

Ways to use isRing :

  • "isRing(Ring)"
  • "isRing(Thing)"

For the programmer

The object isRing is a method function.
