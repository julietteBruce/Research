Macaulay2Doc :: isPolynomialRing

isPolynomialRing -- whether something is a polynomial ring

Synopsis

  • Usage:
    isPolynomialRing R
  • Inputs:
    • R, a thing
  • Outputs:
    • a Boolean value, true if R is a polynomial ring and false otherwise

Description

i1 : isPolynomialRing CC

o1 = false
 
i2 : isPolynomialRing(ZZ/7[x,y])

o2 = true
 
i3 : isPolynomialRing(QQ[x,dx, WeylAlgebra => {x => dx}])

o3 = true
 
i4 : isPolynomialRing(GF(2)[x,y, SkewCommutative => true])

o4 = true
 
i5 : isPolynomialRing(ZZ/101[x,y,z]/(x^2-y*z))

o5 = false

Ways to use isPolynomialRing :

  • "isPolynomialRing(PolynomialRing)"
  • "isPolynomialRing(Thing)"

For the programmer

The object isPolynomialRing is a method function.
