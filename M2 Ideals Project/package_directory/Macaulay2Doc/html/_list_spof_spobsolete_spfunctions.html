list of obsolete functions

  • obsolete classes
    • Thread has been replaced by Task
  • obsolete functions
    • 'mutableZero' has been replaced by mutableMatrix
    • 'unlist' has been replaced by toSequence
    • 'evaluate' has been replaced by value
    • 'seq x' has been replaced by 'singleton x', which has been replaced by '1:x'
    • 'verticalJoin' has been replaced by stack
    • 'netRows' has been replaced by unstack
    • 'name' has been replaced by toString
    • 'quote' has been replaced by symbol
    • 'Numeric' has been replaced by numeric
    • 'submodule' has been removed
    • 'monomialCurve' has been replaced by monomialCurveIdeal
    • 'assign' has been replaced by <-
    • 'minprimes' has been replaced by independentSets
    • 'elapsedTime' has been renamed to cpuTime
    • 'pushForward1(f,M)' has been replaced by 'relations coimage map(M,f)'
    • adjoint1 has been replaced by adjoint', whose interface is different
  • obsolete methods
    • Wikipedia(String) has been replaced: use wikipedia instead
    • 'map(Ideal)' has been removed: use 'map(module I,module I, 1)' instead
    • 'map(Ideal,Ideal)' has been removed: use 'map(module I,module J)' instead
    • 'map(Module,Matrix)' has been replaced: use 'map(M,,f)' instead
    • 'map(Module,RingElement)' has been removed: use 'map(M,M,r)' instead
    • 'RingElement _ ZZ' has been replaced: use 'part(n,f)' instead
    • 'RingElement _ List' has been replaced: use 'part(d,f)' instead
    • 'diff(RingElement)' has been removed: use 'diff(vars ring f, f)' instead
    • 'diff(Matrix)' has been removed: use 'diff(vars ring f, f)' instead
    • 'map(Module,Module)' has been removed: use 'inducedMap' instead
    • 'monomialIdeal(R)' has been removed: use 'monomialIdeal(0_R)' instead
