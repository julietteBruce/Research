Macaulay2Doc :: codim(Ideal,Generic=>...)

codim(Ideal,Generic=>...)

Synopsis

  • Usage:
    codim(...,Generic=>true)
  • Consequences:
    • Allows the computation of the codimension to proceed without an error message, even if the ring is defined over the integers. The computation proceeds by effectively tensoring first with the rational numbers.

Further information

  • Default value: false
  • Method: codim(Ideal) -- compute the codimension
  • Option key: Generic -- an optional argument
