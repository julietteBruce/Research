Macaulay2Doc :: String ^ Sequence

String ^ Sequence -- vertically stacked copies of a string

Synopsis

  • Operator: ^
  • Usage:
    s^(height,depth)
  • Inputs:
    • s, a string
    • (height,depth), a pair of integers
  • Outputs:
    • a net, height + depth copies of s, stacked vertically. There are depth lines below the base line

Description

i1 : s = "|"

o1 = |
 
i2 : s^(4,3)

     |
     |
     |
o2 = |
     |
     |
     |
 
i3 : n = net(x_0)

o3 = x
      0
 
i4 : n0 = s^(height n, depth n)

o4 = |
     |
 
i5 : n0|n|n0

o5 = |x |
     | 0|
