Macaulay2Doc :: Symbol _ Ring

Symbol _ Ring -- get a ring variable by name

Synopsis

  • Operator: _
  • Usage:
    x_R
  • Inputs:
    • x, a symbol, or a ring element
    • R, a ring
  • Outputs:
    • a ring element, the variable of R whose name is x

Description

i1 : R = ZZ[x,y,z];
 
i2 : use R;
 
i3 : S = ZZ[x,t];
 
i4 : x

o4 = x

o4 : S
The symbol x now refers to the variable x in the ring S. There are various ways of now referring to x in the ring R. 
i5 : R_0

o5 = x

o5 : R
 
i6 : R_"x"

o6 = x

o6 : R
 
i7 : use R;
 
i8 : x

o8 = x

o8 : R
