mutableIdentity(Ring,ZZ) -- make a mutable identity matrix

Synopsis

  • Function: mutableIdentity
  • Usage:
    mutableIdentity(R,nrows)
  • Inputs:
    • R, a ring
    • nrows, an integer
  • Optional inputs:
    • Dense => ..., default value true, whether the encoding of the matrix should be dense or not: see MutableMatrix
  • Outputs:
    • a mutable matrix, an nrows by nrows mutable identity matrix filled with elements of the ring R
