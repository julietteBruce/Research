Macaulay2Doc :: setGroupID

setGroupID -- set the process group identifier

Synopsis

  • Usage:
    setGroupID(pid,pgid)
  • Inputs:
    • pid, an integer
    • pgid, an integer
  • Consequences:
    • The process group id of the process with process id pid is set to pgid. If pid is 0, the current process is affected. If pgid is 0, the new process group id is equal to the process id.

See also

  • processID -- the process identifier
  • groupID -- the process group identifier

For the programmer

The object setGroupID is a compiled function.
