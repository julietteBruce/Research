Macaulay2Doc :: OptionTable ++ OptionTable

OptionTable ++ OptionTable -- a binary operator, usually used for direct sum

Synopsis

  • Operator: ++
  • Usage:
    x ++ y
  • Inputs:
    • x, an option table
    • y, an option table
  • Outputs:
    • an option table, a new OptionTable obtained by merging x and y, preferring the default values provided by y

Description

Alternatively, y can be a list of options.

i1 : options res ++ { Alpha => Omega }

o1 = OptionTable{Alpha => Omega                }
                 DegreeLimit => null
                 FastNonminimal => false
                 HardDegreeLimit => {}
                 LengthLimit => infinity
                 PairLimit => infinity
                 SortStrategy => 0
                 StopBeforeComputation => false
                 Strategy => null
                 SyzygyLimit => infinity

o1 : OptionTable
