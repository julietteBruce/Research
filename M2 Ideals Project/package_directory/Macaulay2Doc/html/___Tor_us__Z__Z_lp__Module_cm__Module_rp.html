Macaulay2Doc :: Tor_ZZ(Module,Module)

Tor_ZZ(Module,Module) -- compute a Tor module

Synopsis

  • Scripted functor: Tor
  • Usage:
    Tor_i(M,N)
  • Inputs:
    • i, an integer
    • M, a module
    • N, a module
  • Outputs:
    • a module, the i-th Tor module of M and N

Description

If N is a ring (instead of a module), then the free module of rank 1 over it is used instead. If M or N is an ideal of ring, then the underlying module is used instead.

Ways to use this method:

  • "Tor_ZZ(Ideal,Ideal)"
  • "Tor_ZZ(Ideal,Module)"
  • "Tor_ZZ(Ideal,Ring)"
  • "Tor_ZZ(Module,Ideal)"
  • Tor_ZZ(Module,Module) -- compute a Tor module
  • "Tor_ZZ(Module,Ring)"
