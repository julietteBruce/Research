Macaulay2Doc :: rowRankProfile

rowRankProfile -- find the row rank profile of a mutable matrix

Synopsis

  • Usage:
    rowRankProfile m
  • Inputs:
    • m, a mutable matrix, over RR or CC
  • Outputs:
    • a list, the lexicographically smallest list of indices of linearly independent rows generating the row space of m

Description

i1 : rowRankProfile mutableMatrix {{1,2,3}, {0,0,0.}, {3,4,5} }

o1 = {0, 2}

o1 : List

See also

  • columnRankProfile -- find the column rank profile of a mutable matrix
