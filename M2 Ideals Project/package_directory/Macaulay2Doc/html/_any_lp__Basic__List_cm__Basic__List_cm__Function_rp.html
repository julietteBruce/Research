Macaulay2Doc :: any(BasicList,BasicList,Function)

any(BasicList,BasicList,Function) -- whether any corresponding elements of a pair of lists satisfy a condition

Synopsis

  • Function: any
  • Usage:
    any(v,w,f)
  • Inputs:
    • v, a basic list
    • w, a basic list
    • f, a function, a function of two variables that returns true or false
  • Outputs:
    • a Boolean value, true if f returns true when applied to some pair (x,y) of corresponding elements of v and w, and false otherwise

Description

i1 : any((1,2,3,4),(2,3,4,5), (i,j) -> i>j)

o1 = false
 
i2 : any((1,2,5,4),(2,3,4,5), (i,j) -> i>j)

o2 = true
 
i3 : any((1,2,5,4),(2,3,4,5), x -> (print x; false))
(1, 2)
(2, 3)
(5, 4)
(4, 5)

o3 = false
 
i4 : any((1,2,5,4),(2,3,4,5), x -> (print x; true))
(1, 2)

o4 = true
