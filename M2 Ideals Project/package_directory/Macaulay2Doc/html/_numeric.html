Macaulay2Doc :: numeric

numeric -- convert to floating point

Synopsis

  • Usage:
    numeric x
    numeric(prec,x)
    numeric_prec x
  • Inputs:
    • x
    • prec, an integer, the number of bits of precision to use in the computation
  • Outputs:
    • the expression obtained from x by converting the integers, rational numbers, and symbolic constants within it to double floating point numbers.

Description

i1 : x = {1,2/3,pi}

         2
o1 = {1, -, pi}
         3

o1 : List
 
i2 : numeric oo

o2 = {1, .666667, 3.14159}

o2 : List
 
i3 : numeric_200 pi

o3 = 3.14159265358979323846264338327950288419716939937510582097494

o3 : RR (of precision 200)
 
i4 : numeric_100 oo

o4 = 3.14159265358979323846264338328

o4 : RR (of precision 100)
