Macaulay2Doc :: topLevelMode

topLevelMode -- the current top level mode

Synopsis

  • Usage:
    topLevelMode = x
  • Inputs:
    • x, a symbol, TeXmacs, or Standard
  • Consequences:
    • the interpreter will produce input and output prompts appropriate for the mode, and will format output values appropriately

Description

This variable is intended for internal use only.

For the programmer

The object topLevelMode is a symbol.
