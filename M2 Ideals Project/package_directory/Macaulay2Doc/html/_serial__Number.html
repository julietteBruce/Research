Macaulay2Doc :: serialNumber

serialNumber -- serial number of a dictionary, task, symbol, mutable hash table, or mutable list,

Synopsis

  • Usage:
    serialNumber x
  • Inputs:
    • x
  • Outputs:
    • an integer, the serial number of x

Description

i1 : serialNumber asdf

o1 = 6131654
 
i2 : serialNumber foo

o2 = 6131662
 
i3 : serialNumber ZZ

o3 = 1000050

For the programmer

The object serialNumber is a compiled function.
