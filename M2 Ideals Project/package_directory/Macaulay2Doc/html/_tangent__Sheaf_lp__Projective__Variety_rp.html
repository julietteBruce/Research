Macaulay2Doc :: tangentSheaf(ProjectiveVariety)

tangentSheaf(ProjectiveVariety) -- tangent sheaf of a projective variety

Synopsis

  • Function: tangentSheaf
  • Usage:
    tangentSheaf X
  • Inputs:
    • X, a projective variety
  • Optional inputs:
    • Minimize => a Boolean value, default value true, whether to apply minimalPresentation to the result before returning it
  • Outputs:
    • a coherent sheaf

Description

Computes the tangent sheaf of the projective variety X

Tangent sheaf of the projective plane: 

i1 : P = Proj(QQ[a,b,c])

o1 = P

o1 : ProjectiveVariety
 
i2 : TP = tangentSheaf(P)

o2 = image {-2} | a  b 0 |
           {-2} | -c 0 b |
           {-2} | 0  c a |

                                         3
o2 : coherent sheaf on P, subsheaf of OO  (2)
                                        P
 
i3 : HH^0(TP(-1))

       3
o3 = QQ

o3 : QQ-module, free
 
i4 : HH^1(TP(-3))

       1
o4 = QQ

o4 : QQ-module, free
