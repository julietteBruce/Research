use(Package) -- activate a previously dismissed package

Synopsis

  • Function: use
  • Usage:
    use PACKAGENAME
  • Inputs:
    • PACKAGENAME, a package, which was previously loaded, and then dismissed.
  • Consequences:
    • activates the package, making all of its exported symbols visible
    • runs the function stored under the key
    • symbol use
    • in the package, if there is one

Description

This function is only necessary after dismissing a package.
