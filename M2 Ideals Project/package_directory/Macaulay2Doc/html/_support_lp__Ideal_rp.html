Macaulay2Doc :: support(Ideal)

support(Ideal) -- list of variables occurring in the generators of an ideal

Synopsis

  • Function: support
  • Usage:
    support I
  • Inputs:
    • I, an ideal, an ideal in a polynomial ring
  • Outputs:
    • a list, of ring elements, the variables occurring in the generators of the ideal

Description

i1 : R = QQ[a..g]

o1 = R

o1 : PolynomialRing
 
i2 : I = ideal(b,c,e+f)

o2 = ideal (b, c, e + f)

o2 : Ideal of R
 
i3 : support I

o3 = {b, c, e, f}

o3 : List
