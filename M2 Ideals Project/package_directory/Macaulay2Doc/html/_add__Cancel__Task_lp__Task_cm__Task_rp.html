addCancelTask(Task,Task) -- specify that the completion of one task triggers the cancellation of another

Synopsis

  • Function: addCancelTask
  • Usage:
    addCancelTask(t,u)
  • Inputs:
    • t, a task,
    • u, a task,
  • Consequences:
    • when the task t is finished, then the task u will be cancelled

Ways to use this method:

  • addCancelTask(Task,Task) -- specify that the completion of one task triggers the cancellation of another
