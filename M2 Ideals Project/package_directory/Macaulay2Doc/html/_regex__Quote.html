regexQuote -- escape special characters in regular expressions

Synopsis

  • Usage:
    regexQuote s
  • Inputs:
    • s, a string,
  • Outputs:
    • a string, obtained from s by escaping all of the characters that have special meanings in regular expressions (\, ^, $, ., |, ?, *, +, (, ), [, ], {, }) with a \.

Description

i1 : match("2+2", "2+2")

o1 = false
 
i2 : regexQuote "2+2"

o2 = 2\+2
 
i3 : match(oo, "2+2")

o3 = true
