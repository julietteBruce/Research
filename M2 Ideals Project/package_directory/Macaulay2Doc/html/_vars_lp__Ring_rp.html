Macaulay2Doc :: vars(Ring)

vars(Ring) -- row matrix of the variables

Synopsis

  • Function: vars
  • Usage:
    vars R
  • Inputs:
    • R, a ring
  • Outputs:
    • a matrix, a matrix with one row whose entries are the variables of the ring R

Description

i1 : S = QQ[w,x,y,z];
 
i2 : vars S

o2 = | w x y z |

             1       4
o2 : Matrix S  <--- S
 
i3 : ideal vars S

o3 = ideal (w, x, y, z)

o3 : Ideal of S

                            1
