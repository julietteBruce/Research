Macaulay2Doc :: xor

xor -- exclusive disjunction

Synopsis

  • Usage:
    t xor u
  • Inputs:
    • t, a Boolean value,
    • u, a Boolean value,
  • Outputs:
    • a Boolean value, equivalent to t and not u or not t and u

See also

  • and -- conjunction
  • or -- disjunction

  • "Boolean xor Boolean"

For the programmer

The object xor is a keyword.

This operator may be used as a binary operator in an expression like x xor y. The user may install binary methods for handling such expressions with code such as

         X  xor  Y := (x,y) -> ...

where X is the class of x and Y is the class of y.
