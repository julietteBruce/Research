Macaulay2Doc :: height(File)

height(File) -- get window height

Synopsis

  • Function: height
  • Usage:
    height f
  • Inputs:
    • f, a file
  • Outputs:
    • an integer, the height of the window or terminal attached to the file f, if any, else 0

Ways to use this method:

  • height(File) -- get window height
