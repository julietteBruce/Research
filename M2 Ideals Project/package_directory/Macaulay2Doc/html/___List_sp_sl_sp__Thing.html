List / Thing -- vector division

Synopsis

  • Operator: /
  • Usage:
    v/c
  • Inputs:
    • v, a list, to be treated as a vector
    • c, a thing, a number or scalar ring element
  • Outputs:
    • a list, the quotient vector; every element of v is divided by c

Description

i1 : {1,2,3,4} / 3

      1  2     4
o1 = {-, -, 1, -}
      3  3     3

o1 : List
