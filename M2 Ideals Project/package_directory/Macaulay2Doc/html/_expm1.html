Macaulay2Doc :: expm1

expm1 -- exponential minus 1

Synopsis

  • Usage:
    expm1 x
    expm1 I
  • Inputs:
    • x
    • I, a real interval
  • Outputs:
    • a real number, the quantity exp(x)-1
    • a real interval, an interval containing the exponential of the points of I minus one

Description

i1 : expm1 1p100e-10

o1 = 1.00000000005000000000166666667e-10

o1 : RR (of precision 100)
 
i2 : exp(1p100e-10)-1

o2 = 1.00000000005000000000937586788e-10

o2 : RR (of precision 100)

Ways to use expm1 :

  • "expm1(RR)"
  • "expm1(RRi)"
  • expm1(RingElement) (missing documentation)

For the programmer

The object expm1 is a method function.
