time -- time a computation

Synopsis

  • Usage:
    time e

Description

time e evaluates e, prints the amount of cpu time used, and returns the value of e. 
i1 : time 3^30
     -- used 4.918e-6 seconds

o1 = 205891132094649

See also

  • timing -- time a computation
  • cpuTime -- seconds of cpu time used since Macaulay2 began
  • elapsedTiming -- time a computation using time elapsed
  • elapsedTime -- time a computation using time elapsed

For the programmer

The object time is a keyword.
