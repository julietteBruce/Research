GradedModule ** Module -- a binary operator, usually used for tensor product or Cartesian product

Synopsis

  • Operator: **
  • Usage:
    C ** M
  • Inputs:
    • C, a graded module
    • M, a module
  • Outputs:
    • a graded module, the tensor product of C with M
