Macaulay2Doc :: exponents

exponents -- the exponents of a polynomial

Synopsis

  • Usage:
    exponents m
  • Inputs:
    • m, a ring element or a monoid element, a monomial
  • Outputs:
    • a list, for a monomial, provides the list of exponents; for a polynomial, provides a list whose elements are the lists of exponents of the terms of f

Description

i1 : R = ZZ/101[x,y,z];
 
i2 : exponents (f = x^2 - 7 + x*y*z^11 + y)

o2 = {{1, 1, 11}, {2, 0, 0}, {0, 1, 0}, {0, 0, 0}}

o2 : List
 
i3 : leadMonomial f

          11
o3 = x*y*z

o3 : R
 
i4 : exponents leadMonomial f

o4 = {{1, 1, 11}}

o4 : List
