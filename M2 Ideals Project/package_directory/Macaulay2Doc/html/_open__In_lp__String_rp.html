Macaulay2Doc :: openIn(String)

openIn(String) -- open an input file

Synopsis

  • Function: openIn
  • Usage:
    openIn fn
  • Inputs:
    • fn, a string
  • Outputs:
    • a file, an open input file whose filename is fn. Filenames starting with ! or with $ are treated specially, see openInOut.

Description

i1 : "test-file" << "hi there" << close;
 
i2 : g = openIn "test-file"

o2 = test-file

o2 : File
 
i3 : fileLength g

o3 = 8
 
i4 : atEndOfFile g

o4 = false
 
i5 : read g

o5 = hi there
 
i6 : atEndOfFile g

o6 = true
 
i7 : close g

o7 = test-file

o7 : File
 
i8 : removeFile "test-file"

A filename starting with ~/ will have the tilde replaced by the user's home directory.
