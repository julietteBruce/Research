Macaulay2Doc :: truncate

truncate

Description

This function is a method function, defined in the core so multiple packages can add methods to it.

  • truncate(List,Ring)
  • "truncate(Constant)" -- see truncate(Number) -- round a number toward zero

  • truncate(ZZ,Ideal)
  • truncate(ZZ,Matrix)
  • truncate(ZZ,Module)
