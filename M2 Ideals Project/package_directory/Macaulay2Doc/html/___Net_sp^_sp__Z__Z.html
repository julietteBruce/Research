Macaulay2Doc :: Net ^ ZZ

Net ^ ZZ -- raise a net or string

Synopsis

  • Operator: ^
  • Usage:
    n^i
  • Inputs:
    • n, a net, or a a string
    • i, an integer
  • Outputs:
    • a net, obtained by elevating n by raising its characters by i rows

Description

If n is a string, then n^0 is an easy way to convert it to a net. 

i1 : s = "x"

o1 = x
 
i2 : class(s^0)

o2 = Net

o2 : Type
 
i3 : n = s^1|"ij"

     x
o3 =  ij
 
i4 : n^-3

o4 = 

     x
      ij
