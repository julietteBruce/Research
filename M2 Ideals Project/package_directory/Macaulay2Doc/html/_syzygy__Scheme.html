syzygyScheme -- construct a syzygy scheme

Description

syzygyScheme(C,i,v) -- produce the syzygy scheme from a map v : R^j ---> C_i which selects some syzygies from a resolution C.

For the programmer

The object syzygyScheme is a function closure.
