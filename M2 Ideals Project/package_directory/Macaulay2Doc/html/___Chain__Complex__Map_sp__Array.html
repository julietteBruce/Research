ChainComplexMap Array -- degree shift

Synopsis

  • Operator: SPACE
  • Usage:
    g = f[i]
  • Inputs:
    • f, a chain complex map,
    • i, an array, an array [i] containing an integer i
  • Outputs:
    • g, a chain complex map, a new chain complex map g in which g_j is f_(i+j).
