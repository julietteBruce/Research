kernel(ChainComplexMap) -- kernel of a chain complex map

Synopsis

  • Function: kernel
  • Usage:
    kernel f
  • Inputs:
    • f, a chain complex map, a map of chain complexes C --> D
  • Optional inputs:
    • SubringLimit => ..., default value infinity,
  • Outputs:
    • a chain complex, the kernel of f

Description

If f is a graded module map, then the result will be a graded module.
