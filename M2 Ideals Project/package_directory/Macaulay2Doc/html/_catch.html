catch -- catch a thrown exception

Synopsis

  • Usage:
    catch c
  • Outputs:
    • the value obtained by evaluating the code c, or, if a throw was executed during the evaluation of c, the argument given to throw.

Description

i1 : catch scan(0..10, i -> if i == 5 then throw 18 else print i)
0
1
2
3
4

o1 = 18
