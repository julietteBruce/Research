Macaulay2Doc :: cpuTime

cpuTime -- seconds of cpu time used since Macaulay2 began

Synopsis

  • Usage:
    cpuTime()
  • Outputs:
    • a real number, the number of seconds of cpu time used since the program was started

Description

i1 : t1 = cpuTime()

o1 = 153.893689894

o1 : RR (of precision 53)
 
i2 : for i from 0 to 1000000 do 223131321321*324234324324;
 
i3 : t2 = cpuTime()

o3 = 157.406106395

o3 : RR (of precision 53)
 
i4 : t2-t1

o4 = 3.51241650099999

o4 : RR (of precision 53)

Caveat

Under Linux, the cpu time reported is that consumed by the thread, but under MAC OS X, it is that consumed by the process.

For the programmer

The object cpuTime is a compiled function.
