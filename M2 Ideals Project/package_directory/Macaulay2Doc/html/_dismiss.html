dismiss -- dismiss a package

Synopsis

  • Usage:
    dismiss pkg
  • Inputs:
    • pkg, a package,
  • Consequences:
    • the dictionary of the package pkg is removed from dictionaryPath, so the symbols exported by pkg are no longer available for use.

Description

The package itself is still accessible under its name in the PackageDictionary, and its exported symbols can be made available again with use.

i1 : newPackage "PKG"

o1 = PKG

o1 : Package
 
i2 : export "x1"

o2 = {x1}

o2 : List
 
i3 : x1=3

o3 = 3
