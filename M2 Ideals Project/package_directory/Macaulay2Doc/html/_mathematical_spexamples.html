mathematical examples

In this section we present some tutorials that aim to introduce the user to some mathematical ways of using Macaulay2. The tutorials are relatively independent of each other, and each one introduces the use of some features of Macaulay2 in a slow and leisurely way, assuming the reader is already familiar with the mathematical concepts involved. David Eisenbud joins us as a co-author of these tutorials.
