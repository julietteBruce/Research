Macaulay2Doc :: Matrix ^ ZZ

Matrix ^ ZZ -- power

Synopsis

  • Operator: ^
  • Usage:
    f^n
  • Inputs:
    • f, a matrix
    • n, an integer
  • Outputs:
    • a matrix, f^n

Description

i1 : R = ZZ/7[x]/(x^6-3*x-4)

o1 = R

o1 : QuotientRing
 
i2 : f = matrix{{x,x+1},{x-1,2*x}}

o2 = | x   x+1 |
     | x-1 2x  |

             2       2
o2 : Matrix R  <--- R
 
i3 : f^2

o3 = | 2x2-1  3x2+3x |
     | 3x2-3x -2x2-1 |

             2       2
o3 : Matrix R  <--- R
 
i4 : f^1000

o4 = | 3x5-2x4-2x2+2x+3 -2x5+x3-x2+x+1  |
     | x5+x4-2x2-3x+1   -x5+2x4-3x3+x-3 |

             2       2
o4 : Matrix R  <--- R

If the matrix is invertible, then f^-1 is the inverse.
