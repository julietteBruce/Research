GradedModuleMap | GradedModuleMap -- a binary operator, often used for horizontal concatenation

Synopsis

  • Operator: |
  • Usage:
    f|g
  • Inputs:
    • f, a graded module map
    • g, a graded module map
  • Outputs:
    • the map of graded modules whose component in degree i is f_i|g_i see Matrix | Matrix
