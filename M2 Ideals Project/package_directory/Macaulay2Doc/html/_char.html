Macaulay2Doc :: char

char -- computes the characteristic of the ring or field

Synopsis

  • Usage:
    char F
  • Inputs:
    • F, a ring, or an affine variety or a projective variety
  • Outputs:
    • the characteristic of the ring. If F is an affine or projective variety, then the characteristic of the corresponding ring is returned

Description

i1 : R = ZZ/10007[x,y];
 
i2 : char R

o2 = 10007
 
i3 : R = ZZ[x]/823671827384723894723894723892

o3 = R

o3 : QuotientRing
 
i4 : char R

o4 = 823671827384723894723894723892

Ways to use char :

  • "char(AffineVariety)"
  • "char(InexactField)"
  • "char(ProjectiveVariety)"
  • "char(Ring)"

For the programmer

The object char is a method function.
