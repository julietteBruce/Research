Macaulay2Doc :: super

super -- get the ambient module

Description

super M -- yields the module that the module M is a submodule of.
super f -- if f is a map whose target is a submodule of M, yields the composite of f with the inclusion into M.

See also

  • cover -- get the covering free module
  • ambient -- ambient free module of a subquotient, or ambient ring

Ways to use super :

  • "super(CoherentSheaf)"
  • "super(GradedModule)"
  • "super(Matrix)"
  • "super(Module)"
  • "super(Vector)"

For the programmer

The object super is a method function.
