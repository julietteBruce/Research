Macaulay2Doc :: stdio

stdio -- the standard input output file

Synopsis

  • Usage:
    stdio
  • Outputs:
    • the standard input output file

Description

Use this file to get input from the terminal, or to display information on the user's screen. This is the file always used by print and << if it is not explicitly given a file.

i1 : << "hi" << endl
hi

o1 = stdio

o1 : File
 
i2 : stdio << "hi" << endl
hi

o2 = stdio

o2 : File

See also

  • stderr -- the standard error output file
  • File << Thing -- print to a file
