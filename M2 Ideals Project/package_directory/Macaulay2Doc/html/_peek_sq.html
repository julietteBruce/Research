Macaulay2Doc :: peek'

peek' -- examine contents of an object

Synopsis

  • Usage:
    peek'(n, o)
  • Inputs:
    • n, an integer,
    • o, a thing, any Macaulay2 object
  • Outputs:
    • a net, laying out the contents of o, bypassing installed formatting and printing methods to depth n

Description

i1 : s = factor 112

      4
o1 = 2 7

o1 : Expression of class Product
 
i2 : peek s

              4
o2 = Product{2 , 7}
 
i3 : peek'_2 s

o3 = Product{Power{2, 4}, Power{7, 1}}

See also

  • peek -- examine contents of an object

Ways to use peek' :

  • "peek'(ZZ,Thing)"

For the programmer

The object peek' is a method function.
