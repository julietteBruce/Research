M2SingularBook -- Macaulay2 examples for the Singular book

M2SingularBook consists of Macaulay2 translations of the examples in the book A Singular introduction to commutative algebra

Each example is numbered as it is in Greuel and Pfister's book. Links to Macaulay2 routines and concepts are also provided.
