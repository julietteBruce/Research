Macaulay2Doc :: gradedModule

gradedModule -- make a graded module

Synopsis

  • Usage:
    gradedModule v
  • Inputs:
    • v, a list, a module, or a list or sequence of modules
  • Outputs:
    • the graded module with the i-th element of v installed as its i-th component

Description

i1 : gradedModule ZZ^2

           2
o1 = 0 : ZZ

o1 : GradedModule
 
i2 : gradedModule(ZZ^2,ZZ^3,ZZ^400)

           2
o2 = 0 : ZZ   

           3
     1 : ZZ   

           400
     2 : ZZ

o2 : GradedModule
If v is a chain complex then the return value is the graded module underlying it.
