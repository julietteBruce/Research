Macaulay2Doc :: asinh

asinh -- inverse hyperbolic sine

Synopsis

  • Usage:
    asinh x
  • Inputs:
    • x
  • Outputs:
    • a number, the inverse hyperbolic sine of x

Description

i1 : asinh .2

o1 = .198690110349241

o1 : RR (of precision 53)

See Hyperbolic function.

Ways to use asinh :

  • "asinh(Number)"
  • asinh(Constant) (missing documentation)

For the programmer

The object asinh is a method function.
