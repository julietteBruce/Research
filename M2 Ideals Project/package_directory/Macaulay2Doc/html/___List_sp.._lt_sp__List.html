Macaulay2Doc :: List ..< List

List ..< List -- rectangular sequences of consecutive lists and sequences

Synopsis

  • Operator: ..<
  • Usage:
    s ..< t
  • Inputs:
    • s, a list
    • t, a list
  • Outputs:
    • a sequence, the rectangular sequence of consecutive lists of the same length, from s to t, in lexicographic order, not including the endpoint, obtained by applying the operator ..< recursively

Description

i1 : {a,1} ..< {d,4}

o1 = ({a, 1}, {a, 2}, {a, 3}, {b, 1}, {b, 2}, {b, 3}, {c, 1}, {c, 2}, {c, 3})

o1 : Sequence
 
i2 : {1} ..< {5}

o2 = ({1}, {2}, {3}, {4})

o2 : Sequence

Ways to use this method:

  • List ..< List -- rectangular sequences of consecutive lists and sequences
  • "Sequence ..< Sequence"
