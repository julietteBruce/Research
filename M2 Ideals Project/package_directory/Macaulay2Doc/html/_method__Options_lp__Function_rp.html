methodOptions(Function) -- recover the options used when a method function was created

Synopsis

  • Function: methodOptions
  • Usage:
    methodOptions f
  • Inputs:
    • f, a function
  • Outputs:
    • an option table, the options used when f was created by method

Description

                 TypicalValue => Thing
