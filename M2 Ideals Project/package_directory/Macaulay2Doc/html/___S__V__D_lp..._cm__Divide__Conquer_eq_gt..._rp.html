Macaulay2Doc :: SVD(...,DivideConquer=>...)

SVD(...,DivideConquer=>...) -- Use the lapack divide and conquer SVD algorithm

Synopsis

  • Usage:
    SVD(M, DivideConquer=>true)

Description

For large matrices, this algorithm is often much faster. 
i1 : M = random(RR^200, RR^200);

                200          200
o1 : Matrix RR      <--- RR
              53           53
 
i2 : time SVD(M);
     -- used 0.0276256 seconds
 
i3 : time SVD(M, DivideConquer=>true);
     -- used 0.0227895 seconds

Further information

  • Default value: true
  • Function: SVD -- singular value decomposition of a matrix
  • Option key: DivideConquer -- an optional argument

Functions with optional argument named DivideConquer :

  • SVD(...,DivideConquer=>...) -- Use the lapack divide and conquer SVD algorithm
