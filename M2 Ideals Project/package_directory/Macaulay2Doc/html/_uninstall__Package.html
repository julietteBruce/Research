uninstallPackage -- uninstall a package and remove its documentation

Synopsis

  • Usage:
    uninstallPackage PackageName
  • Inputs:
    • PackageName, a package, or String, the package to uninstall
  • Optional inputs:
    • InstallPrefix => a string, default value "/home/m2user/Library/Application Support/Macaulay2/local/", see installPackage(...,InstallPrefix=>...) -- load and install a package and its documentation
  • Consequences:
    • the links to the files and the files of the specified package created by installPackage, in case encapsulation is enabled, are removed, for every version of the package.
