Macaulay2Doc :: hilbertPolynomial(CoherentSheaf)

hilbertPolynomial(CoherentSheaf) -- compute the Hilbert polynomial of the coherent sheaf

Synopsis

  • Function: hilbertPolynomial
  • Usage:
    hilbertPolynomial S
  • Inputs:
    • S, a coherent sheaf
  • Optional inputs:
    • Projective => ..., default value true, choose how to display the Hilbert polynomial
  • Outputs:
    • a projective Hilbert polynomial, unless the option Projective is false

Description

We compute the Hilbert polynomial of a coherent sheaf.
