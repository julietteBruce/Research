Macaulay2Doc :: Resolution

Resolution -- the class of all resolution computations

Description

These resolutions are internal engine objects not meant to be examined by the user.

The symbol Resolution is also used in a ChainComplex to store the resolution it comes from.

Methods that use a resolution :

  • new ChainComplex from Resolution (missing documentation)
  • "ring(Resolution)" -- see ring -- get the associated ring of an object
  • "status(Resolution)" -- see status -- status of a resolution computation

For the programmer

The object Resolution is a type, with ancestor classes MutableHashTable < HashTable < Thing.
