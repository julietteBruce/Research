Macaulay2Doc :: intersection(RRi,RRi)

intersection(RRi,RRi) -- computes intersection of input intervals

Synopsis

  • Function: intersection
  • Usage:
    K = intersection(I,J)
    K = intersection(I,J,Precision => prec)
  • Inputs:
    • I, a real interval,
    • J, a real interval,
  • Optional inputs:
    • Precision => an integer, default value -1, specifies the desired precision of the output, a value of -1 uses the minimum precision of the inputs.
  • Outputs:
    • K, a real interval,

Description

Returns the intersection of the two input intervals or an empty interval if they do not intersect.
