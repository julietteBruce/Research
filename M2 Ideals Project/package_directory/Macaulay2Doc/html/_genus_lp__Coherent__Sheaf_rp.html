genus(CoherentSheaf) -- arithmetic genus

Synopsis

  • Function: genus
  • Usage:
    genus F
  • Inputs:
    • F, a coherent sheaf, a coherent sheaf, a module, or an ideal
  • Outputs:
    • an integer

Description

Computes the arithmetic genus of the coherent sheaf F that is (-1)^dim-support * (chi(F) - 1)). If F is a module over a ring, then the genus of F~ is computed. If I is an ideal in a ring R then the genus of (R/I)~ is computed.

o1 = V
