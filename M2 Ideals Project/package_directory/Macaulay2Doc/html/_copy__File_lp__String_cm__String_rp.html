copyFile(String,String)

Synopsis

  • Function: copyFile
  • Usage:
    copyFile(src,dst)
  • Inputs:
    • src, a string, the filename or path to an existing regular file
    • dst, a string, the filename or path to the copy to be made
  • Optional inputs:
    • UpdateOnly => a Boolean value, default value false, whether to copy file only if the destination file does not exist or is older than the source file
    • Verbose => a Boolean value, default value false, whether to report individual file operations
  • Consequences:
    • the file may be copied

Description

i1 : src = temporaryFileName()

o1 = /private/tmp/M2-56793-0/0
 
i2 : dst = temporaryFileName()

o2 = /private/tmp/M2-56793-0/1
 
i3 : src << "hi there" << close

o3 = /private/tmp/M2-56793-0/0

o3 : File
 
i4 : copyFile(src,dst,Verbose=>true)
 -- copying: /private/tmp/M2-56793-0/0 -> /private/tmp/M2-56793-0/1
 
i5 : get dst

o5 = hi there
 
i6 : copyFile(src,dst,Verbose=>true,UpdateOnly => true)
 -- skipping: /private/tmp/M2-56793-0/0 not newer than /private/tmp/M2-56793-0/1
 
i7 : src << "ho there" << close

o7 = /private/tmp/M2-56793-0/0

o7 : File
 
i8 : copyFile(src,dst,Verbose=>true,UpdateOnly => true)
 -- skipping: /private/tmp/M2-56793-0/0 not newer than /private/tmp/M2-56793-0/1
 
i9 : get dst
