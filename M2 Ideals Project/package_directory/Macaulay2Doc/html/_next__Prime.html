Macaulay2Doc :: nextPrime

nextPrime -- compute the smallest prime greater than or equal to a given number

Synopsis

  • Usage:
    nextPrime n
  • Inputs:
    • n, a number,
  • Outputs:
    • an integer, the smallest prime $\ge n$

Description

i1 : nextPrime 10000

o1 = 10007
 
i2 : nextPrime 3.5678

o2 = 5
 
i3 : nextPrime (3/7)

o3 = 2

See also

  • isPrime -- whether a integer or polynomial is prime
  • getPrimeWithRootOfUnity -- find a prime p with a primitive n-th root of unity r in ZZ/p

Ways to use nextPrime :

  • "nextPrime(Number)"

For the programmer

The object nextPrime is a method function.
