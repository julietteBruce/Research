Macaulay2Doc :: map(Module)

map(Module) -- identity map

Synopsis

  • Function: map
  • Usage:
    map M
  • Inputs:
    • M, a module
  • Optional inputs:
    • Degree => ..., default value null, specify the degree of a map
    • DegreeLift => ..., default value null, make a ring map
    • DegreeMap => ..., default value null, make a ring map
  • Outputs:
    • a matrix, the identity map on M

Description

i1 : f = map ZZ^3

o1 = | 1 0 0 |
     | 0 1 0 |
     | 0 0 1 |

              3        3
o1 : Matrix ZZ  <--- ZZ
 
i2 : g = id_(ZZ^3)

o2 = | 1 0 0 |
     | 0 1 0 |
     | 0 0 1 |

              3        3
o2 : Matrix ZZ  <--- ZZ
 
i3 : f === g

o3 = true

Caveat

may be removed
