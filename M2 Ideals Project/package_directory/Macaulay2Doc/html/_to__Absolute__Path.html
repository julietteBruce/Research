Macaulay2Doc :: toAbsolutePath

toAbsolutePath -- the absolute path version of a file name

Synopsis

  • Usage:
    toAbsolutePath filename
  • Inputs:
    • filename, a string
  • Outputs:
    • a string, the absolute (real) path name of filename

Description

i1 : toAbsolutePath "a/b.m2"

o1 = /private/tmp/macaulay2-20230205-25510-hsiem6/M2-release-1.21/M2/BUILD/
     build-brew/Macaulay2/packages/a/b.m2

Paths of the form foo/x/../bar, are shortened to foo/bar without checking the file system to see whether x is a symbolic link. For the other behavior, see realpath.
