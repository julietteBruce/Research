RevLex -- reverse lexicographic ordering

Description

The reverse lexicographic order is defined by: $x^A > x^B$ if the FIRST non-zero entry of the vector of integers A-B is NEGATIVE. This is a local order, not a global order. Therefore Gröbner bases over this ring only give generators over the local ring whose fractions are all elements not in the ideal generated by the variables. 
i1 : R = QQ[a..d,MonomialOrder => RevLex, Global => false];
