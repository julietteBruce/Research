Macaulay2Doc :: notify

notify -- whether to notify the user when a file is loaded

Synopsis

  • Usage:
    notify = true
    notify = false
  • Consequences:
    • If notify is set to true, then each time a file or a package is loaded, a message will be displayed.

Description

i1 : notify = true

o1 = true
