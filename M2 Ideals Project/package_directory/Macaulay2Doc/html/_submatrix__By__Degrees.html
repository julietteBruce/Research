Macaulay2Doc :: submatrixByDegrees

submatrixByDegrees -- submatrix consisting of rows and columns in an interval or box of degrees

Synopsis

  • Usage:
    submatrixByDegrees(m, targetBox, sourceBox)
  • Inputs:
    • m, a matrix, A matrix between free modules
    • targetBox, a sequence, A sequence of 2 degree vectors, (lodegree, hidegree). All rows whose multi-degree is greater than or equal (in each component) to lodegree, and less than or equal to hidegree, are selected
    • sourceBox, a sequence, Same as targetBox, except this governs which columns are selected
  • Outputs:
    • a matrix, The submatrix of m consisting of those rows (respectively columns) whose (multi-)degree lie in the box targetBox, (respectively columnBox)

Description

If only one degree (as integer, or list of integers) is given for targetBox or sourceBox, then only rows or columns that match that exact degree are used.

For multidegrees, the interval is a box.

i8 : S = QQ[a..d, Degrees=>{2:{1,0},2:{0,1}}];

Caveat

The degrees are taken from the target and source free modules, not from the matrix entries themselves.
