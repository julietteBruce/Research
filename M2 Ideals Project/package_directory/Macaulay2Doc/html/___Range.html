Macaulay2Doc :: Range

Range -- can be assigned a integral interval

Synopsis

  • Usage:
    getPrimeWithRootOfUnity(p,r,Range=>(a,b))
  • Inputs:
    • a, an integer,
    • b, an integer,

Description

Specifies an integral interval in which we search for a prime number with desired properties. If $b< a+2*log a$ an error message will be returned. The default value is $(10^4,3*10^4)$

See also

  • nextPrime -- compute the smallest prime greater than or equal to a given number
  • getPrimeWithRootOfUnity -- find a prime p with a primitive n-th root of unity r in ZZ/p

Functions with optional argument named Range :

  • "getPrimeWithRootOfUnity(...,Range=>...)" -- see getPrimeWithRootOfUnity -- find a prime p with a primitive n-th root of unity r in ZZ/p

For the programmer

The object Range is a symbol.
