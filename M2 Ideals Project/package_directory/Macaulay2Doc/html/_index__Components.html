Macaulay2Doc :: indexComponents

indexComponents -- specify keys for components of a direct sum

Description

indexComponents -- a symbol used as a key in a direct sum under which to store a hash table in which to register preferred keys used to index the components of the direct sum.

See also

  • directSum -- direct sum of modules or maps
  • components -- list the components of a direct sum
  • indices -- indices of a polynomial; also components for a direct sum

For the programmer

The object indexComponents is a symbol.
