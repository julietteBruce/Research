Macaulay2Doc :: midpoint

midpoint -- midpoint of an interval

Synopsis

  • Usage:
    x = midpoint I
  • Inputs:
    • I, a real interval,
  • Outputs:
    • x, a real number,

Description

Returns the midpoint (the average of the endpoints) of the input interval.

See also

  • left -- left endpoint of an interval
  • right -- right endpoint of an interval
  • diameter -- diameter of an interval

For the programmer

The object midpoint is a compiled function.
