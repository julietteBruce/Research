package -- get containing package

Synopsis

  • Usage:
    package x
  • Inputs:
    • x, a thing,
  • Outputs:
    • a package, the package in which the documentation key x was defined

Description

i1 : package sin

o1 = Core

o1 : Package
 
i2 : package poly

o2 = Classic

o2 : Package
