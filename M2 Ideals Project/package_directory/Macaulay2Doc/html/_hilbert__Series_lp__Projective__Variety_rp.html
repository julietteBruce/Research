Macaulay2Doc :: hilbertSeries(ProjectiveVariety)

hilbertSeries(ProjectiveVariety) -- compute the Hilbert series of a projective variety

Synopsis

  • Function: hilbertSeries
  • Usage:
    hilbertSeries V
  • Inputs:
    • V, a projective variety
  • Optional inputs:
    • Order => ..., default value infinity, display the truncated power series expansion
    • Reduce => ..., default value false, reduce the Hilbert series
  • Outputs:
    • a divide expression, the Hilbert series

Description

We compute the Hilbert series of a projective variety, that is, the Hilbert series of the homogeneous coordinate ring of V. The saturation of the ideal may need to be computed.

This method is not implemented yet.

Ways to use this method:
