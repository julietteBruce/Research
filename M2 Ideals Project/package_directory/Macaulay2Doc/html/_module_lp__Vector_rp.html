Macaulay2Doc :: module(Vector)

module(Vector) -- the module of a vector

Synopsis

  • Function: module
  • Usage:
    module v
  • Inputs:
    • v, a vector
  • Outputs:
    • the module that contains the vector v

Description

The class of v is also equal to the module of v. 
i1 : F = ZZ^4

       4
o1 = ZZ

o1 : ZZ-module, free
 
i2 : v = F_2

o2 = | 0 |
     | 0 |
     | 1 |
     | 0 |

       4
o2 : ZZ
 
i3 : module v

       4
o3 = ZZ

o3 : ZZ-module, free
 
i4 : class v

       4
o4 = ZZ

o4 : ZZ-module, free
