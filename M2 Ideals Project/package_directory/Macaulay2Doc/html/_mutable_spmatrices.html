Macaulay2Doc :: mutable matrices

mutable matrices

To and from matrices: 
i1 : m = matrix{{1,2,3},{4,5,6}}

o1 = | 1 2 3 |
     | 4 5 6 |

              2        3
o1 : Matrix ZZ  <--- ZZ
 
i2 : n = mutableMatrix m

o2 = | 1 2 3 |
     | 4 5 6 |

o2 : MutableMatrix
 
i3 : m2 = matrix n

o3 = | 1 2 3 |
     | 4 5 6 |

              2        3
o3 : Matrix ZZ  <--- ZZ
 
i4 : m2 - m == 0

o4 = true
Modifying and accessing entries. Remember that the upper left entry is (0,0), not (1,1)! 
i5 : n_(0,0) = 212314323

o5 = 212314323
 
i6 : n_(0,0)

o6 = 212314323
 
i7 : n

     | 4         5 6 |
