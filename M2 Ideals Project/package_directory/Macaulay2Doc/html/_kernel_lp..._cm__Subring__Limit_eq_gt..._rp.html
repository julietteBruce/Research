Macaulay2Doc :: kernel(...,SubringLimit=>...)

kernel(...,SubringLimit=>...)

Description

SubringLimit => n -- an option for kernel which causes the computation of the kernel of a ring map to stop after n elements have been discovered.

Further information

  • Default value: infinity
  • Function: kernel -- kernel of a ringmap, matrix, or chain complex
  • Option key: SubringLimit -- stop after finding enough elements of a subring

Functions with optional argument named SubringLimit :

  • "gb(...,SubringLimit=>...)" -- see gb -- compute a Gröbner basis
  • kernel(...,SubringLimit=>...)
