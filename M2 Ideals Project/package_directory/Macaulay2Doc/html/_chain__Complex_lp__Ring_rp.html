Macaulay2Doc :: chainComplex(Ring)

chainComplex(Ring) -- make an empty chain complex over a ring

Synopsis

  • Function: chainComplex
  • Usage:
    chainComplex R
  • Inputs:
    • R, a ring
  • Outputs:
    • a chain complex, an empty chain complex over R

Description

The resulting chain complex can be further populated with modules and maps. 
i1 : C = chainComplex ZZ

o1 = 0

o1 : ChainComplex
 
i2 : C.dd_101 = random(ZZ^2,ZZ^3)

o2 = | 8 3 8 |
     | 1 7 3 |

              2        3
o2 : Matrix ZZ  <--- ZZ
 
i3 : C.dd_105 = id_(ZZ^2)

o3 = | 1 0 |
     | 0 1 |

              2        2
o3 : Matrix ZZ  <--- ZZ
 
i4 : C

       2       3                       2       2
o4 = ZZ  <-- ZZ  <-- 0   <-- 0   <-- ZZ  <-- ZZ
                                              
     100     101     102     103     104     105

o4 : ChainComplex
 
i5 : C.dd

             2                  3
o5 = 100 : ZZ  <------------- ZZ  : 101
                  | 8 3 8 |
                  | 1 7 3 |

             2                2
     104 : ZZ  <----------- ZZ  : 105
                  | 1 0 |
                  | 0 1 |

o5 : ChainComplexMap
