Macaulay2Doc :: asin

asin -- arcsine

Synopsis

  • Usage:
    asin x
    asin I
  • Inputs:
    • x
    • I, a real interval
  • Outputs:
    • a number, the arcsine (in radians) of x
    • a real interval, an interval containing the arcsines of the points of I

Description

i1 : asin 1

o1 = 1.5707963267949

o1 : RR (of precision 53)

See Trigonometric function.

Ways to use asin :

  • "asin(CC)"
  • "asin(RR)"
  • "asin(RRi)"
  • asin(RingElement) (missing documentation)

For the programmer

The object asin is a method function.
