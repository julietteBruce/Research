random(ZZ,Ring) -- get a random homogeneous element from a graded ring

Synopsis

  • Function: random
  • Usage:
    random(d, R)
  • Inputs:
    • d, an integer, an integer or a list, the degree or multi-degree to use;
    • R, a ring,
  • Optional inputs:
    • Density => ..., default value 1, make a random module map
    • Height => ..., default value 10
    • MaximalRank => ..., default value false, make a random module map
    • UpperTriangular => ..., default value false, make a random module map
  • Outputs:
    • a ring element, a random homogeneous element of the ring R of degree d
