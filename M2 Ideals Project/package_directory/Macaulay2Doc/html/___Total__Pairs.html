Macaulay2Doc :: TotalPairs

TotalPairs -- whether to display the total number of S-pairs

Description

TotalPairs -- an option for status which specifies whether to display the total number of S-pairs.

Functions with optional argument named TotalPairs :

  • "status(...,TotalPairs=>...)" -- see status -- status of a resolution computation

For the programmer

The object TotalPairs is a symbol.
