Macaulay2Doc :: fileMode(ZZ,File)

fileMode(ZZ,File) -- set file mode

Synopsis

  • Function: fileMode
  • Usage:
    fileMode(mo,f)
  • Inputs:
    • mo, an integer
    • f, a file
  • Consequences:
    • the mode of the open file f is set to mo

Description

i1 : fn = temporaryFileName()

o1 = /private/tmp/M2-55783-0/0
 
i2 : f = fn << "hi there"

o2 = /private/tmp/M2-55783-0/0

o2 : File
 
i3 : m = 7 + 7*8 + 7*64

o3 = 511
 
i4 : fileMode(m,f)
 
i5 : fileMode f

o5 = 511
 
i6 : close f

o6 = /private/tmp/M2-55783-0/0

o6 : File
 
i7 : fileMode fn

o7 = 511
 
i8 : removeFile fn
