maxAllowableThreads -- the maximum possible number of simultaneously running tasks

Synopsis

  • Usage:
    maxAllowableThreads
  • Outputs:
    • an integer, the maximum number to which allowableThreads can be set

Description

i1 : maxAllowableThreads

o1 = 4
