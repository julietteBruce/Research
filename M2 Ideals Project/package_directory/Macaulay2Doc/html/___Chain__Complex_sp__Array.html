ChainComplex Array -- degree shift

Synopsis

  • Operator: SPACE
  • Usage:
    D = C[i]
  • Inputs:
    • C, a chain complex,
    • i, an array, an array [i] containing an integer i
  • Outputs:
    • D, a chain complex, a new chain complex D in which D_j is C_(i+j). The signs of the differentials are reversed if i is odd.
