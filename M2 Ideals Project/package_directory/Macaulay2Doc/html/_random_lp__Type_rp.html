random(Type) -- get a random object of a type

Synopsis

  • Function: random
  • Usage:
    random T
  • Inputs:
    • T, a type,
  • Optional inputs:
    • Height => an integer, default value 10, get a random homogeneous element from a graded ring
    • Density => ..., default value 1, make a random module map
    • MaximalRank => ..., default value false, make a random module map
    • UpperTriangular => ..., default value false, make a random module map
  • Outputs:
    • a thing, a random instance of the type T

Description

If the Height option specifies a number h and T is ZZ then the integers returned are in the range [0, h); for QQ the numerator and denominator are in the range [1, h].
