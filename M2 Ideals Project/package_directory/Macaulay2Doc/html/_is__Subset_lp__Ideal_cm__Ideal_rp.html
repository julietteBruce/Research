isSubset(Ideal,Ideal) -- whether one object is a subset of another

Synopsis

  • Function: isSubset
  • Usage:
    isSubset(I,J)
  • Inputs:
    • I, an ideal
    • J, an ideal
  • Outputs:
    • a Boolean value, whether I is contained in J
