Macaulay2Doc :: truncateOutput

truncateOutput

Synopsis

  • Usage:
    truncateOutput w
  • Inputs:
    • w, an integer
  • Consequences:
    • The maximum output line width is set to w, which should be an integer or infinity. This function works by assigning a value to Thing#{Standard,BeforePrint}, which may conflict with other BeforePrint methods installed by the user, or those installed by the system that do line wrapping.

Ways to use truncateOutput :

  • "truncateOutput(InfiniteNumber)"
  • "truncateOutput(ZZ)"

For the programmer

The object truncateOutput is a method function.
