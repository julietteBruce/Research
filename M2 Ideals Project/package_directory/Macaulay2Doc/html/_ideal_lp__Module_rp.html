ideal(Module) -- converts a module to an ideal

Synopsis

  • Function: ideal
  • Usage:
    ideal M
  • Inputs:
    • M, a module, which is a submodule of a free module of rank 1
  • Outputs:
    • an ideal, given by the generators of M

Description
