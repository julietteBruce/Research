List << List -- component-wise comparison of lists

Synopsis

  • Operator: <<
  • Usage:
    L1 << L2
  • Inputs:
    • L1, a list
    • L2, a list
  • Outputs:
    • a Boolean value, whether the first list is less than or equal to the second list in each component

Code

../../../../Macaulay2/m2/lists.m2:178:24-178:58: --source code:
List << List := (A, B) -> all(min(#A, #B), i -> A#i <= B#i)
