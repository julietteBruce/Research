Macaulay2Doc :: all(HashTable,Function)

all(HashTable,Function) -- whether all key/value pairs in a hash table satisfy a specified condition

Synopsis

  • Function: all
  • Usage:
    all(H,f)
  • Inputs:
    • H, a hash table
    • f, a function, , which returns true or false
  • Outputs:
    • a Boolean value, true if freturns true when applied to all key/value pairs of H and false otherwise

Description

i1 : all(hashTable{1=>3, 2=>2, 3=>1}, (a,b) -> a == b)

o1 = false
 
i2 : all(hashTable{1=>1, 2=>2, 3=>3}, (a,b) -> a == b)

o2 = true
