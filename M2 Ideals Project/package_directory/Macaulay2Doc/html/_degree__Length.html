Macaulay2Doc :: degreeLength

degreeLength -- the length of the degree vector

Synopsis

  • Usage:
    degreeLength A
  • Inputs:
    • A, a ring or a monoid,
  • Outputs:
    • an integer, the length of a multidegree vector used as a degree of elements of A.

Description

Example degreeLength ZZ degreeLength(ZZ[]) degreeLength(ZZ[x]) degreeLength(ZZ[x, Degrees => {{1,2,3}}]) degreeLength(ZZ[x,y,z, DegreeRank => 3]) degreeLength(GF 9)

See also

  • degree
  • degreesRing -- the ring or monoid of degrees
  • degreesMonoid -- the ring or monoid of degrees
  • monoid(...,DegreeRank=>...) -- specify the degrees of the variables

Ways to use degreeLength :

  • "degreeLength(Monoid)"
  • "degreeLength(Ring)"

For the programmer

The object degreeLength is a method function.
