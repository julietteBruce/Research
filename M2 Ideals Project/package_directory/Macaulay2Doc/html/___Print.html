Macaulay2Doc :: Print

Print -- top level method for printing results

Synopsis

  • Usage:
    X#{Standard,Print} = f
  • Inputs:
    • X, a type
    • f, a function, that can print something of type X
  • Consequences:
    • at top level, whenever it is time to print an output value of type X, the function f will be called

Description

The function f is responsible for printing the output prompt and for applying the BeforePrint and AfterPrint methods, if desired.
