Macaulay2Doc :: size2

size2 -- number of binary digits to the left of the point

Synopsis

  • Usage:
    size2 x
  • Inputs:
    • x, a number
  • Outputs:
    • the smallest number n such that 2^n is strictly greater than the absolute value of x

Description

This function is not implemented for rational numbers. 
i1 : size2 4

o1 = 3
 
i2 : size2 3

o2 = 2
 
i3 : size2 4.

o3 = 3
 
i4 : size2 3.99999999

o4 = 2
 
i5 : size2 0

o5 = -1073741824
 
i6 : size2 0.

o6 = -1073741824
 
i7 : size2 (1/0.)

o7 = 1073741823
 
i8 : size2 (1/0.-1/0.)

o8 = 1073741823

Ways to use size2 :

  • "size2(CC)"
  • "size2(RR)"
  • "size2(RRi)"
  • "size2(ZZ)"

For the programmer

The object size2 is a compiled function.
