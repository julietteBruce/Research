Macaulay2Doc :: isOutputFile(File)

isOutputFile(File) -- whether a file is open for output

Synopsis

  • Function: isOutputFile
  • Usage:
    isOutputFile f
  • Inputs:
    • f, a file
  • Outputs:
    • a Boolean value, whether f is an open output file

Description

i1 : f = "test-file" << "hi there"

o1 = test-file

o1 : File
 
i2 : isOutputFile f

o2 = true
 
i3 : close f

o3 = test-file

o3 : File
 
i4 : isOutputFile f

o4 = false
 
i5 : get "test-file"

o5 = hi there
 
i6 : removeFile "test-file"

See also

  • openOut -- open an output file
  • openInOut -- open an input output file
  • close -- close a file
  • get -- get the contents of a file
  • removeFile -- remove a file
