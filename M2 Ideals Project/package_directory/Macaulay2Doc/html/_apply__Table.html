Macaulay2Doc :: applyTable

applyTable -- apply a function to each element of a table

Synopsis

  • Usage:
    applyTable(T, f)
  • Inputs:
    • T, a list, a table (list of lists of the same length)
    • f, a function,
  • Outputs:
    • A, a list, a table of the same shape as $T$, where the function $f$ has been applied elementwise

Description

i1 : t = table({1,3,5,7}, {0,1,2,4}, (i,j) -> i^j);
 
i2 : netList t

     +-+-+--+----+
o2 = |1|1|1 |1   |

     +-+-+--+----+

     +-+-+--+----+
