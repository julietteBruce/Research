Macaulay2Doc :: Unique

Unique -- do not return repeated polynomial roots

Description

A booleanBoolean, to select whether to return repeated roots or not.

Functions with optional argument named Unique :

  • "roots(RingElement,Unique=>...)" -- see roots(RingElement) -- compute the roots of a polynomial

For the programmer

The object Unique is a symbol.
