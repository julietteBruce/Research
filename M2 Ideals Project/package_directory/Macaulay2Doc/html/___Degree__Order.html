Macaulay2Doc :: DegreeOrder

DegreeOrder -- sort primarily by degree

Description

DegreeOrder -- an optional argument for use with certain functions, used to specify sort order.

Functions with optional argument named DegreeOrder :

  • rsort(...,DegreeOrder=>...) -- specify Ascending, Descending, or null
  • sort(...,DegreeOrder=>...) -- specify Ascending, Descending, or null
  • sortColumns(...,DegreeOrder=>...) -- specify Ascending, Descending, or null

For the programmer

The object DegreeOrder is a symbol.
