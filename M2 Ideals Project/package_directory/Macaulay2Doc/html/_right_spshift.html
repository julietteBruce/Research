Macaulay2Doc :: right shift

right shift

Synopsis

  • Usage:
    x >> j
    x >> I
  • Inputs:
    • x
    • j
    • I, a real interval
  • Outputs:
    • the integer obtained from x by shifting its binary representation rightward j places
    • a real interval, an interval containing all shifts of the binary representations elements of I by j places rightward 
i1 : 256 >> 5

o1 = 8
 
i2 : 256. >> 555

o2 = 2.17066284129402e-165

o2 : RR (of precision 53)
