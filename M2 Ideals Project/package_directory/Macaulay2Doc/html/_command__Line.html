commandLine -- the command line arguments

Synopsis

  • Usage:
    commandLine

Description

A constant whose value is the list of arguments passed to the interpreter, including argument 0, the name of the program.

See also

  • scriptCommandLine -- the command line arguments to be used when running a script

For the programmer

The object commandLine is a list.
