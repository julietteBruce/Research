Macaulay2Doc :: syz(Matrix)

syz(Matrix) -- compute the syzygy matrix

Synopsis

  • Function: syz
  • Usage:
    syz h
  • Inputs:
    • h, a matrix, a matrix
  • Optional inputs:
    • Algorithm => ..., default value Inhomogeneous, see gb(...,Algorithm=>...)
    • BasisElementLimit => ..., default value infinity, see gb(...,BasisElementLimit=>...)
    • DegreeLimit => ..., default value {}, see gb(...,DegreeLimit=>...)
    • GBDegrees => ..., default value null, see gb(...,GBDegrees=>...)
    • HardDegreeLimit => ..., default value null, see gb(...,HardDegreeLimit=>...)
    • MaxReductionCount => an integer, default value 10, the maximum number of reductions of an S-pair done before requeueing it, if the Inhomogeneous algorithm is in use
    • PairLimit => ..., default value infinity, see gb(...,PairLimit=>...)
    • StopBeforeComputation => ..., default value false, see gb(...,StopBeforeComputation=>...)
    • Strategy => ..., default value {}, see gb(...,Strategy=>...)
    • SyzygyLimit => ..., default value infinity, see gb(...,SyzygyLimit=>...)
    • SyzygyRows => ..., default value infinity, see gb(...,SyzygyRows=>...)
  • Outputs:
    • a matrix, the matrix of minimal or trimmed generators for the syzygies among the columns of h

i1 : R = QQ[a..g];
