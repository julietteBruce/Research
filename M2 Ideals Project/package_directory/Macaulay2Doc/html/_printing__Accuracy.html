Macaulay2Doc :: printingAccuracy

printingAccuracy -- current accuracy for printing numbers

Synopsis

  • Usage:
    printingAccuracy = n
  • Inputs:
    • n, an integer
  • Consequences:
    • Henceforth, inexact numbers are printed with at most n digits to the right of the decimal point displayed. The special case where n=-1 is interpreted as meaning n=infinity, and this case is used when a number appears alone on an output line.
