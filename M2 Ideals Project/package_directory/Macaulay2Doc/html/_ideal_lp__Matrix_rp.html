Macaulay2Doc :: ideal(Matrix)

ideal(Matrix) -- make an ideal

Synopsis

  • Function: ideal
  • Usage:
    ideal M
  • Inputs:
    • M, a matrix, whose entries are ring elements
  • Outputs:
    • an ideal, which is generated by all the entries in M

Description

i1 : R = ZZ/7[w,x,y,z];
 
i2 : f = vars R

o2 = | w x y z |

             1       4
o2 : Matrix R  <--- R
 
i3 : ideal f

o3 = ideal (w, x, y, z)

o3 : Ideal of R
 
i4 : g = matrix{{x^2-w*y, x*y-w*z, x*z-y^2},{y^2-x*z,x^2*y-z^2,x^3-y*z}}

o4 = | x2-wy xy-wz  -y2+xz |
     | y2-xz x2y-z2 x3-yz  |

             2       3
o4 : Matrix R  <--- R
 
i5 : ideal g

             2         2                    2     2     2         3
o5 = ideal (x  - w*y, y  - x*z, x*y - w*z, x y - z , - y  + x*z, x  - y*z)

o5 : Ideal of R

The functions minors, pfaffians and fitting ideal are typically more useful methods of creating an ideal from a matrix.
