Macaulay2Doc :: log

log -- logarithm function

Synopsis

  • Usage:
    log x
    log(b,x)
    log_b x
    log I
    log(b,I)
    log_b I
    log(J,x)
    log_J x
    log(J,I)
    log_J I
  • Inputs:
    • x, a real number
    • b, a real number, the base for the logarithm
    • I, a real interval
    • J, a real interval, an interval of bases for the logarithm
  • Outputs:
    • the logarithm of x
    • a real interval, an interval containing the logarithms of points of I
    • a real interval, an interval containing the logarithms for bases in J

Description

i1 : log 10

o1 = 2.30258509299405

o1 : RR (of precision 53)
 
i2 : log_2 10

o2 = 3.32192809488736

o2 : RR (of precision 53)
 
i3 : log_10 2p100

o3 = .301029995663981195213738894725

o3 : RR (of precision 100)

Ways to use log :

  • "log(CC)"
  • "log(RR)"
  • "log(RR,RR)"
  • "log(RRi)"
  • "log(RRi,RRi)"
  • log(RR,CC) (missing documentation)
  • log(RR,RRi) (missing documentation)
  • log(RRi,RR) (missing documentation)

For the programmer

The object log is a method function.
