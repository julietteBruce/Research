Macaulay2Doc :: undocumented

undocumented -- declare that something need not be documented

Synopsis

  • Usage:
    undocumented key
  • Inputs:
    • key, a thing, a documentation key, or a list of keys
  • Consequences:
    • the documentation key(s) are designated as keys not needing documentation, thus avoiding warning messages when a package is installed.
