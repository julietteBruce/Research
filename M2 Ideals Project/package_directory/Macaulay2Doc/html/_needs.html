needs -- read Macaulay2 commands if necessary

Synopsis

  • Usage:
    needs "f"
  • Consequences:
    • The file named f is loaded with load if it hasn't been loaded yet; if it has changed since the last time it was loaded, it will be loaded again, from the same location as the time before, without searching along the path.

Description

If the variable notify is set to true, then an informational message is displayed after the file is loaded.

For the programmer

The object needs is a function closure.
