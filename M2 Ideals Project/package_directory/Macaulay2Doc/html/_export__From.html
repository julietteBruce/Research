exportFrom -- export symbols from a package's private dictionary

Synopsis

  • Usage:
    exportFrom(pkg, {"symbol1", "symbol2"})
  • Inputs:
    • pkg, a package, the package containing the symbols
    • a list, of strings, corresponding to the symbols to export
  • Consequences:
    • package symbols provided in the list will be added to the dictionary of the current package, and will be exported

Description

This function can be used to export specific symbols from the private dictionary of another package.

i1 : exportFrom_Core {"HTML"}

o1 = {HTML}

o1 : List

See also

  • export -- export symbols from a package
  • importFrom -- import symbols to the current private dictionary
