Macaulay2Doc :: toSequence

toSequence -- convert to sequence

Synopsis

  • Usage:
    toSequence x
  • Inputs:
    • x, a basic list or a string, or an instance of a class with the iterator method installed
  • Outputs:
    • a sequence,

Description

toSequence x yields the elements of a x as a sequence.

i1 : toSequence {1, 2, 3}

o1 = (1, 2, 3)

o1 : Sequence
 
i2 : toSequence "foo"

o2 = (f, o, o)

o2 : Sequence
 
i3 : toSequence iterator {1, 2, 3}

o3 = (1, 2, 3)

o3 : Sequence

If x is already a sequence, then x is returned.

i4 : toSequence (1, 2, 3)

o4 = (1, 2, 3)

o4 : Sequence

Ways to use toSequence :

  • "toSequence(BasicList)"
  • "toSequence(String)"
  • "toSequence(Thing)"

For the programmer

The object toSequence is a compiled function.
