Macaulay2Doc :: comodule

comodule -- submodule to quotient module

Synopsis

  • Usage:
    comodule M
    quotient M
  • Inputs:
    • M, a module, or an ideal
  • Outputs:
    • a module, the quotient module N/M, if M is given as a submodule $M \subset N$, where N is either a free module or a quotient of a free module.

Description

i1 : R = QQ[a..d];
 
i2 : I = ideal(a,b,c,d^3);

o2 : Ideal of R
 
i3 : comodule I

o3 = cokernel | a b c d3 |
