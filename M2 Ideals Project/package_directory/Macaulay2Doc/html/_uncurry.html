uncurry -- uncurry a function

Description

uncurry(f, (a,b)) -- computes ((f a) b).
uncurry(f, (a,b,c)) -- computes (((f a) b) c).
uncurry(f, (a,b,c,d)) -- computes ((((f a) b) c) d).
... and so on.
