Macaulay2Doc :: commonRing

commonRing -- find smallest containing ring

Synopsis

  • Usage:
    commonRing v
  • Inputs:
    • v, a list of numbers, ring elements, matrices, or things with rings, see
    • ring
  • Outputs:
    • a ring, the smallest ring to which all members of the list can be promoted
