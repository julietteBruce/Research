<< -- a binary operator (file output, ...)

Ways to use symbol << :

  • "<< Thing" -- see File << Thing -- print to a file
  • "File << Manipulator" -- see File << Thing -- print to a file
  • "File << Net" -- see File << Thing -- print to a file
  • "File << String" -- see File << Thing -- print to a file
  • "File << Symbol" -- see File << Thing -- print to a file
  • File << Thing -- print to a file
  • "Nothing << Manipulator" -- see File << Thing -- print to a file
  • "Nothing << Thing" -- see File << Thing -- print to a file
  • "String << Thing" -- see File << Thing -- print to a file
  • "CC << ZZ" -- see left shift
  • "RR << ZZ" -- see left shift
  • "RRi << ZZ" -- see left shift
  • "ZZ << ZZ" -- see left shift
  • List << List -- component-wise comparison of lists
  • "NetFile << Manipulator" -- see NetFile -- the class of all net files
  • "NetFile << Net" -- see NetFile -- the class of all net files
  • "NetFile << String" -- see NetFile -- the class of all net files

For the programmer

The object << is a keyword.

This operator may be used as a binary operator in an expression like x<<y. The user may install binary methods for handling such expressions with code such as

         X << Y := (x,y) -> ...

where X is the class of x and Y is the class of y.

This operator may be used as a prefix unary operator in an expression like <<y. The user may install a method for handling such expressions with code such as

           << Y := (y) -> ...

where Y is the class of y.
