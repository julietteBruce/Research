Macaulay2Doc :: String ..< String

String ..< String -- a sequence of consecutive strings, not including the endpoint

Synopsis

  • Operator: ..<
  • Usage:
    s ..< t
  • Inputs:
    • s, a string
    • t, a string
  • Outputs:
    • a sequence, the sequence of consecutive strings of the same length, from s to t, in lexicographic order, inclusive

Description

i1 : "a" ..< "z"

o1 = (a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x,
     ------------------------------------------------------------------------
     y)

o1 : Sequence
 
i2 : "aa" ..< "ce"

o2 = (aa, ab, ac, ad, ba, bb, bc, bd)

o2 : Sequence
 
i3 : "aaa" ..< "bcd"

o3 = (aaa, aab, aac, aba, abb, abc)

o3 : Sequence
 
i4 : "佖" ..< "佥"

o4 = (佖, 佗, 佘, 余, 佚, 佛, 作, 佝, 佞, 佟, 你, 佡, 佢, 佣, 佤)

o4 : Sequence
