flip(Module,Module) -- matrix of commutativity of tensor product

Synopsis

  • Function: flip
  • Usage:
    flip(F,G)
  • Inputs:
    • F, a module
    • G, a module
  • Outputs:
    • a matrix, the matrix representing the natural isomorphism G ** F <-- F ** G
