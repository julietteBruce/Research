floor(Number) -- floor function

Synopsis

  • Function: floor
  • Usage:
    floor x
  • Inputs:
    • x, a number
  • Outputs:
    • the greatest integer less than or equal to the number x. The imaginary part of a complex number is ignored.

Description

i1 : floor (7.234232131231*10^6)

o1 = 7234232
