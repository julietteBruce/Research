Macaulay2Doc :: resolution(...,StopBeforeComputation=>...)

resolution(...,StopBeforeComputation=>...) -- whether to stop the computation immediately

Description

StopBeforeComputation -- keyword for an optional argument used with resolution.

Tells whether to start the computation, with the default value being true. This can be useful when you want to obtain the partially computed resolution contained in an interrupted computation.

Further information

  • Default value: false
  • Function: resolution -- projective resolution
  • Option key: StopBeforeComputation -- an optional argument

Functions with optional argument named StopBeforeComputation :

  • "gb(...,StopBeforeComputation=>...)" -- see gb -- compute a Gröbner basis
  • "pushForward(...,StopBeforeComputation=>...)" -- see pushForward(RingMap,Module)
  • resolution(...,StopBeforeComputation=>...) -- whether to stop the computation immediately
  • "syz(...,StopBeforeComputation=>...)" -- see syz(Matrix) -- compute the syzygy matrix
