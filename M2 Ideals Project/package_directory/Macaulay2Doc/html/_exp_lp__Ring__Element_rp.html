exp(RingElement) -- exponential function

Synopsis

  • Function: exp
  • Usage:
    exp x
  • Inputs:
    • x, a ring element
  • Outputs:
    • the exponential of x, provided x is nilpotent, and the denominators required have reciprocals in the ring of x.
