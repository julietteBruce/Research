Macaulay2Doc :: Module _ List

Module _ List -- map from free module to some generators

Synopsis

  • Operator: _
  • Usage:
    M_p
  • Inputs:
    • M, a module, or an ideal
    • p, a list, of integers
  • Outputs:
    • f, a matrix, a map from a free module to the module M which sends the basis vectors to the generators of M whose index numbers are listed.

Description

If M is an ideal, then the map maps to module M. 
i1 : R = QQ[x,y,z]

o1 = R

o1 : PolynomialRing
 
i2 : I = ideal vars R

o2 = ideal (x, y, z)

o2 : Ideal of R
 
i3 : f = I_{0,2}

o3 = {1} | 1 0 |
     {1} | 0 0 |
     {1} | 0 1 |

i4 : image f
