Macaulay2Doc :: peek

peek -- examine contents of an object

Synopsis

  • Usage:
    peek o
  • Inputs:
    • o, a thing, any Macaulay2 object
  • Outputs:
    • a net, laying out the internal contents of o to depth 1, bypassing installed methods for displaying the object

Description

This function is used during debugging Macaulay2 programs to examine the internal structure of objects.

i1 : set {1, 2, 3}

o1 = set {1, 2, 3}

o1 : Set
 
i2 : peek oo

o2 = Set{1 => 1}
         2 => 1
         3 => 1
 
i3 : new MutableHashTable from { a => 3, b => 44}

o3 = MutableHashTable{...2...}

o3 : MutableHashTable
 
i4 : peek oo

o4 = MutableHashTable{a => 3 }
                      b => 44

See also

  • peek' -- examine contents of an object

For the programmer

The object peek is a function closure.
