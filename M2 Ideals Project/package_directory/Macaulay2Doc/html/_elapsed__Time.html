elapsedTime -- time a computation using time elapsed

Synopsis

  • Usage:
    elapsedTime e

Description

elapsedTime e evaluates e, prints the amount of time elapsed, and returns the value of e. 
i1 : elapsedTime sleep 1
 -- 1.02258 seconds elapsed

o1 = 0

See also

  • elapsedTiming -- time a computation using time elapsed
  • cpuTime -- seconds of cpu time used since Macaulay2 began

For the programmer

The object elapsedTime is a keyword.
