Macaulay2Doc :: width(File)

width(File) -- get window width

Synopsis

  • Function: width
  • Usage:
    width f
  • Inputs:
    • f, a file
  • Outputs:
    • an integer, the width of the window or terminal attached to the file f, if any, else 0
