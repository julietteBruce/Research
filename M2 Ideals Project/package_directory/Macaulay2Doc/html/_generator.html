Macaulay2Doc :: generator

generator -- provide a single generator

Synopsis

  • Usage:
    generator I
  • Inputs:
    • I, an ideal or a module
  • Outputs:
    • the single generator of I, if it has just one

Description

If the number of apparent generators is greater than 1, then trim will be called.

i1 : I = ideal (4,6)

o1 = ideal (4, 6)

o1 : Ideal of ZZ
 
i2 : generator I

o2 = 2
 
i3 : M = image matrix {{4,6},{0,0}}

o3 = image | 4 6 |
           | 0 0 |

                               2
o3 : ZZ-module, submodule of ZZ
 
i4 : generator M

o4 = | 2 |
     | 0 |

o4 : image | 2 |
           | 0 |

See also

  • generators -- provide matrix or list of generators

Ways to use generator :

  • "generator(Ideal)"
  • "generator(Module)"

For the programmer

The object generator is a method function.
