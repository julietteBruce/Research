Macaulay2Doc :: sec

sec -- secant

Synopsis

  • Usage:
    sec x
    sec I
  • Inputs:
    • x
    • I, a real interval
  • Outputs:
    • a real number, the secant of x
    • a real interval, an interval containing the secants of the points of I

Description

i1 : sec(pi/3)

o1 = 2

o1 : RR (of precision 53)

See Trigonometric function.

Ways to use sec :

  • "sec(CC)"
  • "sec(RR)"
  • "sec(RRi)"
  • sec(RingElement) (missing documentation)

For the programmer

The object sec is a method function.
