Macaulay2Doc :: Set - Set

Set - Set -- set difference

Synopsis

  • Operator: -
  • Usage:
    x - y
  • Inputs:
    • x, a set, or a list
    • y, a set, or a list
  • Outputs:
    • a set, or a list, consisting of those elements of x not in y

Description

At least one of x, y must be a set, and the other may be a list. If x is a list, then so is the result. 
i1 : set{a,b,c} - set{a}

o1 = set {b, c}

o1 : Set
 
i2 : set{a,b,c} - {a}

o2 = set {b, c}

o2 : Set
 
i3 : {a,b,c} - set{a}

o3 = {b, c}

o3 : List
