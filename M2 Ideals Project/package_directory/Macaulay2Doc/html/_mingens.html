Macaulay2Doc :: mingens

mingens -- minimal generator matrix

Synopsis

  • Optional inputs:
    • Strategy => ..., default value null, a Strategy option value

Description

Every ideal and module in Macaulay2 comes equipped with a generating set, which is sometimes a larger generating set than is required. This routine returns a matrix whose columns generate the given ideal or module.

See also

  • trim -- minimize generators and relations
  • minimalPresentation -- compute a minimal presentation

Ways to use mingens :

  • mingens(GroebnerBasis) -- (partially constructed) minimal generator matrix
  • "mingens(Ideal)" -- see mingens(Module) -- minimal generator matrix
  • mingens(Module) -- minimal generator matrix

For the programmer

The object mingens is a method function with options.
