Macaulay2Doc :: _*

_* -- a unary postfix operator, used for indicating pushforward maps

Ways to use symbol _* :

  • Ideal _* -- get the list of generators of an ideal

  • Monoid _* (missing documentation)

  • RingFamily _* -- a unary postfix operator, used for indicating pushforward maps

For the programmer

The object _* is a keyword.

This operator may be used as a postfix unary operator in an expression like x_*. The user may install a method for handling such expressions with code such as

         X _*   := (x) -> ...

where X is the class of x.
