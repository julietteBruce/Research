Macaulay2Doc :: numgens

numgens -- the number of generators

Ways to use numgens :

  • numgens(CoherentSheaf) -- the number of generators of the underlying module
  • numgens(Ideal) -- number of generators of an ideal
  • numgens(Module) -- number of generators of a module
  • numgens(Monoid) -- number of generators of a monoid
  • "numgens(EngineRing)" -- see numgens(Ring) -- number of generators of a polynomial ring
  • "numgens(FractionField)" -- see numgens(Ring) -- number of generators of a polynomial ring
  • "numgens(MonomialIdeal)" -- see numgens(Ring) -- number of generators of a polynomial ring
  • "numgens(PolynomialRing)" -- see numgens(Ring) -- number of generators of a polynomial ring
  • "numgens(QuotientRing)" -- see numgens(Ring) -- number of generators of a polynomial ring
  • numgens(Ring) -- number of generators of a polynomial ring

For the programmer

The object numgens is a method function.
