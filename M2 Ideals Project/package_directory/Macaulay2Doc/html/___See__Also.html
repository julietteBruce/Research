SeeAlso -- cross-references to other documentation nodes

Synopsis

  • Usage:
    document { ... , SeeAlso => { ... }, ... }

Description

This option inserts into a documentation page a sentence instructing the reader to see some other topics.

The entries may have the special forms used with TO. As an example, here is the code for the SeeAlso part of a documentation node referencing this node.


SeeAlso => {[document, SeeAlso]}

See also

  • TOH -- hypertext documentation link followed by a headline
  • Subnodes -- a menu of documentation nodes

Functions with optional argument named SeeAlso :

  • "document(...,SeeAlso=>...)"

For the programmer

The object SeeAlso is a symbol.
