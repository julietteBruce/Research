Vector ** Vector -- tensor product

Synopsis

  • Operator: **
  • Usage:
    v ** w
  • Inputs:
    • v, a vector
    • w, a vector
  • Outputs:
    • a vector, the tensor product of v and w

Description

If v is in the module M, and w is in the module N, then v**w is in the module M**N.
