Macaulay2Doc :: List | List

List | List -- join lists, sequences or arrays

Synopsis

  • Operator: |
  • Usage:
    v|w
  • Inputs:
    • v, a list, an array, or a sequence
    • w, of the same type as v
  • Outputs:
    • The join of the lists, sequences or arrays.

Description

i1 : {1,2,3}|{4,5,6}

o1 = {1, 2, 3, 4, 5, 6}

o1 : List
 
i2 : (a,b,c)|(1,2)

o2 = (a, b, c, 1, 2)

o2 : Sequence

See also

  • join -- join lists, sequences, and iterable objects
