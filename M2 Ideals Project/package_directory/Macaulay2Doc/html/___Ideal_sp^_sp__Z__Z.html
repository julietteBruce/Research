Ideal ^ ZZ -- power of an ideal

Synopsis

  • Operator: ^
  • Usage:
    I^n
  • Inputs:
    • I, an ideal
    • n, an integer, at least zero
  • Outputs:
    • an ideal, the ideal I^n

Description

i1 : R = QQ[a..d];
 
i2 : I = ideal(a^2, b^2-c*d);
