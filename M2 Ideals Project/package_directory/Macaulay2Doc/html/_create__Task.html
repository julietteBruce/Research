createTask -- create a task

Synopsis

  • Usage:
    createTask(f,x)
    createTask f
  • Inputs:
    • f, a function,
    • x, a thing,
  • Outputs:
    • a task, a task, which when scheduled, will apply the function f to the argument x. In the second form, where x is not specified, it is take to be ().

Description

i1 : f = x -> 2^x

o1 = f

o1 : FunctionClosure
 
i2 : t = createTask(f,3)

o2 = <<task, created>>

o2 : Task
 
i3 : schedule t

o3 = <<task, running>>

o3 : Task
 
i4 : while not isReady t do sleep 1
 
i5 : taskResult t

o5 = 8

Ways to use createTask :

  • "createTask(Function)"
  • "createTask(Function,Thing)"

For the programmer

The object createTask is a compiled function.
