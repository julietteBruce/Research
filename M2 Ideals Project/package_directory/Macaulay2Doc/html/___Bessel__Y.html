Macaulay2Doc :: BesselY

BesselY -- Bessel function of the second kind

Synopsis

  • Usage:
    BesselY(n,x)
    BesselY_n x
  • Inputs:
    • n, an integer, the order
    • x, an integer, a rational number, or a real number
  • Outputs:
    • the Bessel function of the second kind of order n at x

Description

i1 : BesselY_0 .5

o1 = -.444518733506707

o1 : RR (of precision 53)
 
i2 : BesselY_2 3p200

o2 = -.160400393484923729675768299537980918104012040174378220981908

o2 : RR (of precision 200)

See Bessel function.

See also

  • BesselJ -- Bessel function of the first kind

Ways to use BesselY :

  • "BesselY(ZZ,Constant)"
  • "BesselY(ZZ,Number)"

For the programmer

The object BesselY is a method function.
