Macaulay2Doc :: current

current -- the pseudocode that produced an error

Synopsis

  • Usage:
    current
  • Outputs:
    • an instance of the type Pseudocode, the pseudocode that produced an error, or null, if none

Description

Use value to evaluate the code again, for debugging purposes.
