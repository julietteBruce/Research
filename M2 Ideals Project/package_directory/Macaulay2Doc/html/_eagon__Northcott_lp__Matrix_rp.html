eagonNorthcott(Matrix) -- Eagon-Northcott complex of a matrix of linear forms

Synopsis

  • Function: eagonNorthcott
  • Usage:
    eagonNorthcott f
  • Inputs:
    • f, a matrix, a matrix of linear forms
  • Outputs:
    • C, a chain complex, the Eagon-Northcott complex of f

Description

The Eagon-Northcott complex is an explicit chain complex that gives a minimal projective resolution of the cokernel of the matrix maximal minors of a generic matrix of linear forms.
