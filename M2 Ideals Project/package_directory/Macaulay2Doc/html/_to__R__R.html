Macaulay2Doc :: toRR

toRR -- convert to high-precision real number

Synopsis

  • Usage:
    toRR(prec,x)
  • Inputs:
    • prec, an integer, the number of bits of precision desired
    • x, a real number, an integer, or a rational number
  • Outputs:
    • a real number, the result of converting x to a high-precision real number

Description

i1 : toRR(200,1/7)

o1 = .142857142857142857142857142857142857142857142857142857142857

o1 : RR (of precision 200)
 
i2 : precision oo

o2 = 200

For the programmer

The object toRR is a compiled function.
