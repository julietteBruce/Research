Macaulay2Doc :: localDictionaries

localDictionaries -- get local dictionaries

Synopsis

  • Usage:
    localDictionaries f
  • Inputs:
    • f, () or a function, a symbol, an instance of the type Pseudocode, or a dictionary
  • Outputs:
    • a list, a list of the local dictionaries associated with the lexical scopes containing f

Description

i1 : f := x -> y -> z -> 11;
 
i2 : d := localDictionaries ((f 22) 33)

o2 = {LocalDictionary{...1...}, LocalDictionary{...1...},
     ------------------------------------------------------------------------
     LocalDictionary{...2...}}

o2 : List
 
i3 : peek d

o3 = {LocalDictionary{"y" => y}, LocalDictionary{"x" => x},
                                                           
     ------------------------------------------------------------------------
     LocalDictionary{"d" => d}}
                     "f" => f
 
i4 : d#0#"y"

o4 = y

o4 : Symbol
 
i5 : value d#0#"y"

o5 = 33
 
i6 : peek localDictionaries()

o6 = {LocalDictionary{"d" => d}}
                      "f" => f

Ways to use localDictionaries :

  • "localDictionaries(Dictionary)"
  • "localDictionaries(Function)"
  • "localDictionaries(Pseudocode)"
  • "localDictionaries(Symbol)"

For the programmer

The object localDictionaries is a compiled function.
