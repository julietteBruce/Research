Macaulay2Doc :: clearOutput

clearOutput -- forget output values

Synopsis

  • Usage:
    clearOutput

Description

clearOutput is a command that attempts to release memory by clearing the values retained by the output line symbols.

See also

  • clearAll -- forget everything

For the programmer

The object clearOutput is a command.
