isSubset(RRi,RRi) -- subset test for intervals

Synopsis

  • Function: isSubset
  • Usage:
    x = isSubset(I,J)
  • Inputs:
    • I, a real interval,
    • J, a real interval,
  • Outputs:
    • x, a Boolean value,

Description

Returns true if interval I is a subset of interval J.
