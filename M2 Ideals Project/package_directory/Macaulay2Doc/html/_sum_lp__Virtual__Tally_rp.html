Macaulay2Doc :: sum(VirtualTally)

sum(VirtualTally) -- sum of elements

Synopsis

  • Function: sum

Description

sum v yields the sum of the elements in the tally v. 

i1 : a = tally{1,1,1,1,1,10,10,10,100,100}

o1 = Tally{1 => 5  }
           10 => 3
           100 => 2

o1 : Tally
 
i2 : sum a

o2 = 235
