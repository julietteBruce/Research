Macaulay2Doc :: showUserStructure

showUserStructure -- show parent structure for those types defined by user

Synopsis

  • Usage:
    showUserStructure
  • Outputs:
    • a display of the parent structure of the types defined by the user and assigned to global variables

Description

Each type is displayed to the right of its parent.

A type is an instance of the class Type.

i1 : R = QQ[x,y]

o1 = R

o1 : PolynomialRing
 
i2 : X = new Type of List

o2 = X

o2 : Type
 
i3 : Y = new Type of X

o3 = Y

o3 : Type
 
i4 : Z = new Type of X

o4 = Z

o4 : Type
 
i5 : showUserStructure

o5 = Thing : BasicList : RingElement : R
                         VisibleList : List : X : Y
                                                  Z

o5 : Descent

For the programmer

The object showUserStructure is a command.
