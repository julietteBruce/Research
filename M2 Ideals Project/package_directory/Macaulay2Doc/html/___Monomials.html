Macaulay2Doc :: Monomials

Monomials -- whether to display the number of monomial

Description

Monomials -- an option for status which specifies whether to display the number of monomials.

Functions with optional argument named Monomials :

  • "coefficients(...,Monomials=>...)" -- see coefficients -- monomials and their coefficients
  • "status(...,Monomials=>...)" -- see status -- status of a resolution computation

For the programmer

The object Monomials is a symbol.
