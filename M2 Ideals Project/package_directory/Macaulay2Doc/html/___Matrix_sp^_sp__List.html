Macaulay2Doc :: Matrix ^ List

Matrix ^ List -- select rows

Synopsis

  • Operator: ^
  • Usage:
    f_rows
  • Inputs:
    • f, a matrix, a matrix between free modules
    • rows, a list, a list of integers denoting the choice of rows
  • Outputs:
    • a matrix, the submatrix of f corresponding to the rows in the list rows.

Description

This is the same as calling submatrix(f,rows,). 
i1 : R = QQ[a..f];
 
i2 : p = matrix {{a,b,c},{d,e,f}}

o2 = | a b c |
     | d e f |

             2       3
o2 : Matrix R  <--- R
 
i3 : p^{1}

o3 = | d e f |

             1       3
o3 : Matrix R  <--- R
 
i4 : p^{1,0}

o4 = | d e f |
     | a b c |

             2       3
o4 : Matrix R  <--- R
