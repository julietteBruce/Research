entries(Vector) -- lists the entries of a vector

Synopsis

  • Function: entries
  • Usage:
    entries v
  • Inputs:
    • v, a vector
  • Outputs:
    • a List of the coordinates of v

Description

i1 : R = (ZZ[x,y])^3;
 
i2 : v = vector {1,x,x*y};
 
i3 : entries v

o3 = {1, x, x*y}
