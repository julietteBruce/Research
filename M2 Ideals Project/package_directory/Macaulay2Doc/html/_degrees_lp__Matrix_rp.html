Macaulay2Doc :: degrees(Matrix)

degrees(Matrix) -- degrees of target and source

Synopsis

  • Function: degrees
  • Usage:
    degrees f
  • Inputs:
    • f, a matrix
  • Outputs:
    • a list, a list {x,y} where x is the list of degrees of the target of f and y is the list of degrees of the source of f.

Description

i1 : S = ZZ/101[x,y,z,Degrees => {{2,3},{1,2},{2,0}}];
 
i2 : degrees vars S

o2 = {{{0, 0}}, {{2, 3}, {1, 2}, {2, 0}}}

o2 : List

See also

  • degreeLength -- the length of the degree vector
  • degreesRing -- the ring or monoid of degrees
