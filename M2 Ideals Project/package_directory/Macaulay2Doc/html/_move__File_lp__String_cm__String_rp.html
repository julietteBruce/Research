moveFile(String,String)

Synopsis

  • Function: moveFile
  • Usage:
    moveFile(src,dst)
  • Inputs:
    • src, a string, the filename or path to an existing file
    • dst, a string, the new filename or path to a location (on the same file system). Omit this argument and an appropriately numbered backup filename will be invented.
  • Optional inputs:
    • Verbose => a Boolean value, default value false, whether to report individual file operations
  • Outputs:
    • the name of the backup file if one was created, or null
  • Consequences:
    • the file will be moved by creating a new link to the file and removing the old one

Description

i1 : src = temporaryFileName()

o1 = /private/tmp/M2-53666-0/0
 
i2 : dst = temporaryFileName()

o2 = /private/tmp/M2-53666-0/1
 
i3 : src << "hi there" << close

o3 = /private/tmp/M2-53666-0/0

o3 : File
 
i4 : moveFile(src,dst,Verbose=>true)
--moving: /private/tmp/M2-53666-0/0 -> /private/tmp/M2-53666-0/1
 
i5 : get dst

o5 = hi there
 
i6 : bak = moveFile(dst,Verbose=>true)
--backup file created: /private/tmp/M2-53666-0/1.bak

o6 = /private/tmp/M2-53666-0/1.bak
 
i7 : removeFile bak
