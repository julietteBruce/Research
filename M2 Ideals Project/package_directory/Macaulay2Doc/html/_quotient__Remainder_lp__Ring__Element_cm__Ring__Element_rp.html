quotientRemainder(RingElement,RingElement) -- quotient and remainder

Synopsis

  • Function: quotientRemainder
  • Usage:
    (q,r) = quotientRemainder(f,g)
  • Inputs:
    • f, a ring element
    • g, a ring element
  • Outputs:
    • q, a ring element, the quotient for the division of f by g
    • r, a ring element, the remainder for the division of f by g
