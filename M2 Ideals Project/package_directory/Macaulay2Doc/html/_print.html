Macaulay2Doc :: print

print -- print something

Synopsis

  • Usage:
    print x

Description

print x prints x on the standard output followed by a new line. 
i1 : print "Hello world!"
Hello world!
The return value is null.

For the programmer

The object print is a function closure.
