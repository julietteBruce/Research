PencilsOfQuadrics :: randomExtension

randomExtension -- a random extension of a vector bundle on E by another vector bundle

Synopsis

  • Usage:
    V=randomExtension(V1,V2)
    V=randomExtension(Y1,Y2)
  • Inputs:
    • V1, an instance of the type VectorBundleOnE,
    • V2, an instance of the type VectorBundleOnE,
    • Y1, a matrix, a matrix which is the yAction of some VectorBundleOnE
    • Y2, a matrix, a matrix which is the yAction of some VectorBundleOnE
  • Outputs:
    • V, an instance of the type VectorBundleOnE,

Description

Chooses a random extension of V2 by V1, where V1, V2 are vector bundles on E represented by the type VectorBundleOnE.
