PencilsOfQuadrics :: yAction

yAction -- defines a vector bundle on E

Synopsis

  • Usage:
    M = V.yAction
  • Outputs:
    • M, a matrix, a square matrix on PP^1, representing the action of y

Description

A matrix representing the action of y for the hyperelliptic curve E with equation y^2 - (-1)^g * f.

i2 : R = kk[s,t];
 
i3 : f = (s+2*t)*(s+t)*(s-t)*(s-2*t);
 
i4 : L0 = randomLineBundle(0,f)

o4 = VectorBundleOnE{...1...}

o4 : VectorBundleOnE
