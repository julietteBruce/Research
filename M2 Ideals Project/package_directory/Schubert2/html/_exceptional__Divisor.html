Schubert2 :: exceptionalDivisor

exceptionalDivisor -- get the exceptional divisor of a blowup

Synopsis

  • Usage:
    exceptionalDivisor X
  • Inputs:
    • X, an abstract variety, a blowup, built using blowup
  • Outputs:
    • an abstract sheaf, the exceptional divisor on X, expressed as a line bundle

Description

i1 : T = abstractProjectiveSpace 2

o1 = T

o1 : a flag bundle with subquotient ranks {1..2}
 
i2 : S = abstractProjectiveSpace 0

o2 = S

o2 : a flag bundle with subquotient ranks {1, 0}
 
i3 : i = map(T,S,OO_S) -- inclusion of a point in P^2

o3 = i

o3 : a map to T from S
 
i4 : X = first blowup(i)

o4 = X

o4 : an abstract variety of dimension 2
