Schubert2 :: incidenceCorrespondence(List,AbstractSheaf)

incidenceCorrespondence(List,AbstractSheaf) -- build containment correspondence between two Grassmannians

Synopsis

  • Function: incidenceCorrespondence
  • Usage:
    incidenceCorrespondence(L,E)
  • Inputs:
    • L, a list, of two integers, $L={a,b}$, such that $0 \leq a$ and $0\leq b$.
    • E, an abstract sheaf, a vector bundle of rank $n$, such that $a\leq n$ and $b\leq n$.
  • Outputs:
    • an incidence correspondence, from $G1 = G(a,E)$ to $G2 = G(b,E)$, same as incidenceCorrespondence(G1,G2)

Description

i1 : B = flagBundle({3,1})

o1 = B

o1 : a flag bundle with subquotient ranks {3, 1}
 
i2 : E = first bundles B

o2 = E

o2 : an abstract sheaf of rank 3 on B
 
i3 : incidenceCorrespondence({1,2},E)

o3 = an incidence correspondence

o3 : an incidence correspondence from a flag bundle to a flag bundle
