kernelBundle(ZZ,AbstractSheaf,AbstractSheaf)

Synopsis

  • Function: kernelBundle
  • Usage:
    kernelBundle(k,B,A)
  • Inputs:
    • k, an integer,
    • B, an abstract sheaf,
    • A, an abstract sheaf,
  • Outputs:
    • an abstract variety, the kernel bundle on the locus where the rank of a generic map from A to B is at most k, assuming that the rank doesn't drop further
