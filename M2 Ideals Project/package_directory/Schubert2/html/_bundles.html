Schubert2 :: bundles

bundles -- get the list of tautological line bundles on a flag bundle

Synopsis

  • Usage:
    bundles X
  • Inputs:
    • X, an abstract flag bundle,
  • Outputs:
    • a list, of bundles, the tautological bundles on the flagBundle X

Description

i1 : X = flagBundle({2,3,4})

o1 = X

o1 : a flag bundle with subquotient ranks {2..4}
 
i2 : L = bundles X

o2 = (a sheaf, a sheaf, a sheaf)

o2 : Sequence
 
i3 : rank \ L

o3 = (2, 3, 4)

o3 : Sequence

Ways to use bundles :

  • "bundles(FlagBundle)"

The object bundles is a method function.
