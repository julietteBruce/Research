Schubert2 :: use(AbstractVariety)

use(AbstractVariety) -- assign values to variables associated with a variety

Synopsis

  • Function: use
  • Usage:
    use X
  • Inputs:
    • X, an abstract variety,
  • Outputs:
    • an abstract variety, X
  • Consequences:
    • The generators of the intersection ring of X are assigned to their variables. See use(Ring).
    • The named bundles on X are assigned to their variables.
