Schubert2 :: Correspondence _*

Correspondence _* -- pushforward along a correspondence

Synopsis

  • Operator: _*
  • Usage:
    I_*
  • Inputs:
    • I, an instance of the type Correspondence, from $X$ to $Y$
  • Outputs:
    • a function, the pushforward operator associated to $I$, which can be applied to abstract sheaves or cycles on $X$

Description

See incidenceCorrespondence for an example of the usage.
