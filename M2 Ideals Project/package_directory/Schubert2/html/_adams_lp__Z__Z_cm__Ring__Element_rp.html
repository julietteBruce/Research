adams(ZZ,RingElement) -- compute Adams operations

Synopsis

  • Function: adams
  • Usage:
    adams(i,f)
    adams_i f
  • Inputs:
    • i, an integer,
    • f, a ring element,
  • Outputs:
    • a ring element, the Chern character of the effect of $i$-th Adams operation on the abstract sheaf whose Chern character is $f$

Description

i1 : X = abstractVariety(3, QQ[c,d,e,Degrees=>{1,2,3}])

o1 = X

o1 : an abstract variety of dimension 3
 
i2 : adams(3, 1 + c + d + e)
