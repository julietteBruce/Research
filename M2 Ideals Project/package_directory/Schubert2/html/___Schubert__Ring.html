Schubert2 :: SchubertRing

SchubertRing -- a symbol used internally as a key

Description

If $G$ is a Grassmannian (i.e. a FlagBundle with rank sequence of length 2), and if the Schubert ring of $G$ has already been built, then it is stored in G.cache.SchubertRing. This should not be used as an accessor; see schubertRing.

See also

  • schubertRing -- get the Schubert ring of a Grassmannian

For the programmer

The object SchubertRing is a symbol.
