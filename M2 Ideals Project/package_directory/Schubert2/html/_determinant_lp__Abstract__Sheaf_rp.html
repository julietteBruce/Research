determinant(AbstractSheaf) -- determinant of an abstract sheaf

Synopsis

  • Function: determinant
  • Usage:
    det F
  • Inputs:
    • F, an abstract sheaf,
  • Optional inputs:
    • Strategy => ..., default value null,
  • Outputs:
    • an abstract sheaf, the determinant of F

Description

The determinant of F is the $n$-th exterior power of F, where $n$ is the rank of F.
