Schubert2 :: AbstractSheaf - AbstractSheaf

AbstractSheaf - AbstractSheaf -- difference of two abstract sheaves

Synopsis

  • Operator: -
  • Usage:
    F-G
  • Inputs:
    • F, an abstract sheaf,
    • G, an abstract sheaf,
  • Outputs:
    • an abstract sheaf, the difference of $F$ and $G$. If one of the arguments is an integer, it represents the trivial sheaf of that rank.

Description

i1 : X = abstractProjectiveSpace 1

o1 = X

o1 : a flag bundle with subquotient ranks {2:1}
 
i2 : OO_X(1) - OO_X(2)

o2 = a sheaf

o2 : an abstract sheaf of rank 0 on X
 
i3 : chi oo

o3 = -1

The sheaves can be on different varieties if one of the varieties is over the other.
