Schubert2 :: sectionClass(AbstractVarietyMap)

sectionClass(AbstractVarietyMap) -- get the class of the image of a section of a map of abstract varieties

Synopsis

  • Function: sectionClass
  • Usage:
    sectionClass f
  • Inputs:
    • f, an abstract variety map,
  • Outputs:
    • a ring element, the class of a section of $f$. (This means, more or less, any cycle class of relative dimension zero whose integral is 1.)

Description

i1 : X = abstractProjectiveSpace 4

o1 = X

o1 : a flag bundle with subquotient ranks {1, 4}
 
i2 : f = X.StructureMap

o2 = f

o2 : a map to point from X
