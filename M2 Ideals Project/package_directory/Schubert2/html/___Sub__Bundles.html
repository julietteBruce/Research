Schubert2 :: SubBundles

SubBundles -- get the subbundles of the tautological filtration on a flag bundle

Synopsis

  • Usage:
    F.SubBundles
  • Inputs:
    • F, an abstract flag bundle,
  • Outputs:
    • a list whose elements are the subbundles $0 = E_0 \subseteq{} E_1 \subseteq{} ... \subseteq{} E_n = f^* B$ in the tautological filtration of $f^* B$, where $F$ is a flag bundle parametrizing filtrations of an abstract sheaf $B$ on a variety $X$, and $f : F \rightarrow X$ is the structure map of $F$

o1 = F
