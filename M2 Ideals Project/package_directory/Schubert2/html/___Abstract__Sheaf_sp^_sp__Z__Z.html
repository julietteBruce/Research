AbstractSheaf ^ ZZ -- power of an abstract sheaf

Synopsis

  • Operator: ^
  • Usage:
    F^n
  • Inputs:
    • F, an abstract sheaf,
    • n, an integer,
  • Outputs:
    • an abstract sheaf, the n-th Cartesian power of F, which amounts to multiplication by n in the Grothendieck group

Description

Maybe we should remove this operation, in favor of ZZ * AbstractSheaf.
