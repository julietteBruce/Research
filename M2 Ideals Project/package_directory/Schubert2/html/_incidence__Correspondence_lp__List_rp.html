Schubert2 :: incidenceCorrespondence(List)

incidenceCorrespondence(List) -- build containment correspondence between two Grassmannians

Synopsis

  • Function: incidenceCorrespondence
  • Usage:
    incidenceCorrespondence L
  • Inputs:
    • L, a list, of three integers, $L={a,b,n}$, such that $0 \leq a \leq n$ and $0\leq b \leq n$.
  • Outputs:
    • an incidence correspondence, from $G1 = G(a,n)$ to $G2 = G(b,n)$, same as incidenceCorrespondence(G1,G2)

Description

i1 : I = incidenceCorrespondence({1,2,3})

o1 = I

o1 : an incidence correspondence from a flag bundle to a flag bundle
 
i2 : source I

o2 = a flag bundle

o2 : a flag bundle with subquotient ranks {1..2}
 
i3 : target I

o3 = a flag bundle

o3 : a flag bundle with subquotient ranks {2, 1}
