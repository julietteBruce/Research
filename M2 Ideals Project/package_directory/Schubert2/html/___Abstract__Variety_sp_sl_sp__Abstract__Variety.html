AbstractVariety / AbstractVariety -- get a structure map

Synopsis

  • Operator: /
  • Usage:
    X/S
  • Inputs:
    • X, an abstract variety,
    • S, an abstract variety,
  • Outputs:
    • an abstract variety map, the structure map or composition of structure maps $X \rightarrow{} S$
