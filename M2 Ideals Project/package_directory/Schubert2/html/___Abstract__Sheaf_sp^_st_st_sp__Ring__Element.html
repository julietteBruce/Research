Schubert2 :: AbstractSheaf ^** RingElement

AbstractSheaf ^** RingElement -- tensor power of an abstract sheaf

Synopsis

  • Operator: ^**
  • Usage:
    F^**n
  • Inputs:
    • F, an abstract sheaf, of rank 1 if n is not an integer
    • n, a ring element, in (or promotable to) the intersection ring of the variety of $F$
  • Outputs:
    • an abstract sheaf, the $n$-th tensor power of $F$

Description

i1 : pt = base n

o1 = pt

o1 : an abstract variety of dimension 0
 
i2 : X = abstractProjectiveSpace'(3,pt)

o2 = X

o2 : a flag bundle with subquotient ranks {3, 1}
 
i3 : A = intersectionRing X

o3 = A

o3 : QuotientRing
 
i4 : ch ((OO_X(1)) ^** 2)

                2   4 3
o4 = 1 + 2h + 2h  + -h
                    3

o4 : A
 
i5 : ch ((OO_X(1)) ^** 3)

              9 2   9 3
o5 = 1 + 3h + -h  + -h
              2     2

o5 : A
 
i6 : ch ((OO_X(1)) ^** n)

               1 2 2   1 3 3
o6 = 1 + n*h + -n h  + -n h
               2       6

o6 : A
