Schubert2 :: todd

todd -- compute the Todd class of an abstract sheaf, variety, map

Synopsis

  • Usage:
    todd F
    todd X
    todd f
  • Inputs:
    • F, an abstract sheaf,
    • X, an abstract variety,
    • f, an abstract variety map,
  • Outputs:
    • a ring element, the Todd class of $F$, of the (absolute) tangent bundle of $X$, of the (relative) tangent bundle of $f$
