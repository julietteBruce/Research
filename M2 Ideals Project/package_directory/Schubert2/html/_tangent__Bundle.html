Schubert2 :: tangentBundle

tangentBundle -- get the tangent bundle

Synopsis

  • Usage:
    tangentBundle X
    tangentBundle f
  • Inputs:
    • X, an abstract variety,
    • f, an abstract variety map,
  • Outputs:
    • an abstract sheaf, the (absolute) tangent bundle of $X$ or the (relative) tangent bundle of $f$
