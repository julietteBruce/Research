Schubert2 :: QuotientBundles

QuotientBundles -- get the quotient bundles of the tautological filtration on a flag bundle

Synopsis

  • Usage:
    F.QuotientBundles
  • Inputs:
    • F, an abstract flag bundle,
  • Outputs:
    • a list $\{E_n/E_n, E_n/E_{n-1}, ..., E_n/E_1, E_n/E_0\}$ whose elements are the quotient bundles in the tautological filtration $0 = E_0 \subseteq{} E_1 \subseteq{} ... \subseteq{} E_n = f^* B$, where $F$ is a flag bundle parametrizing filtrations of an abstract sheaf $B$ on a variety $X$, and $f : F \rightarrow X$ is the structure map of $F$
