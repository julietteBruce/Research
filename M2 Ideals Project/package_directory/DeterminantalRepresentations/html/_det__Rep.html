DeterminantalRepresentations :: detRep

detRep -- compute determinantal representations

Synopsis

  • Usage:
    detRep f
    detRep(f, Tolerance => 1e-6)
  • Inputs:
    • f, a ring element, a polynomial with real coefficients
  • Optional inputs:
    • HyperbolicPt => ..., default value null,
    • Software => ..., default value M2engine,
    • Tolerance => ..., default value .000001
  • Outputs:
    • a list, of matrices, each giving a determinantal representation of $f$

Description

This method is a wrapper function for the various methods implemented in this package. Currently it accepts quadrics (in any number of variables), bivariate polynomials or homogeneous trivariate polynomials, and cubic polynomials in 4 variables. By default, all polynomials are assumed to have real coefficients. For any polynomial falling in one of these categories, the user may call this method, and the correct algorithm will be automatically applied. For details on each case, see the pages below.

  • Determinantal representations of hyperbolic plane cubics
