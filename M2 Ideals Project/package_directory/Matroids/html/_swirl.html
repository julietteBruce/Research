Matroids :: swirl

swirl -- swirl matroid

Synopsis

  • Usage:
    M = swirl r
  • Inputs:
    • r, an integer,
  • Outputs:
    • a matroid, a free swirl

Description

The family of swirls appears in Oxley, p. 664. The rank-r free swirl has 2r elements and rank r.

i1 : areIsomorphic(swirl 3, uniformMatroid_3 6)

o1 = true
 
i2 : M = swirl 4

o2 = a "matroid" of rank 4 on 8 elements

o2 : Matroid
 
i3 : betti ideal M
