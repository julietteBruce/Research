XML :: getChildren

getChildren -- get the list of children of an XML node

Synopsis

  • Usage:
    getChildren n
  • Inputs:
    • n, an instance of the type LibxmlNode or an instance of the type LibxmlAttribute
  • Outputs:
    • the list of children of n
