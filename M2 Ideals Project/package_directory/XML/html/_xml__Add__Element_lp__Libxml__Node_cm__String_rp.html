XML :: xmlAddElement(LibxmlNode,String)

xmlAddElement(LibxmlNode,String) -- add an element to an XML node

Synopsis

  • Function: xmlAddElement
  • Usage:
    m = xmlAddElement(n,s)
  • Inputs:
    • n, an instance of the type LibxmlNode,
    • s, a string,
  • Outputs:
    • m, an instance of the type LibxmlNode, a new node named s
  • Consequences:
    • The new node m is attached as the last child of n.
