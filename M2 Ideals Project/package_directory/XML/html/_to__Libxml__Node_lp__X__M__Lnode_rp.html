XML :: toLibxmlNode(XMLnode)

toLibxmlNode(XMLnode) -- convert a hashtable of type XMLnode to an object of type LibxmlNode

Synopsis

  • Function: toLibxmlNode
  • Usage:
    toLibxmlNode x
  • Inputs:
    • x, an instance of the type XMLnode,
  • Outputs:
    • an instance of the type LibxmlNode,

Description

i1 : new XMLnode from {
          tag => "foo",
          children => {
               new XMLnode from {
                    tag => "bar",
                    children => { " chicken coop " } },
               " hi there ",
               new XMLnode from { tag => "bar" } } }

o1 = <foo
       <bar " chicken coop "
       " hi there "
       <bar

o1 : XMLnode
 
i2 : toLibxmlNode oo

o2 = <foo><bar> chicken coop </bar> hi there <bar/></foo>

o2 : LibxmlNode (xmlTypeDescription)
