XML :: xmlNewRoot(String)

xmlNewRoot(String) -- create the root node of an XML document

Synopsis

  • Function: xmlNewRoot
  • Usage:
    xmlNewRoot v
  • Inputs:
    • v, a string,
  • Outputs:
    • an instance of the type LibxmlNode, the new root node of a new XML document, with name given by v

Description

i1 : xmlNewRoot "foo"

o1 = <foo/>

o1 : LibxmlNode (xmlTypeDescription)
 
i2 : xmlDocDump oo

o2 = <?xml version="1.0" encoding="UTF-8"?>
     <foo/>
