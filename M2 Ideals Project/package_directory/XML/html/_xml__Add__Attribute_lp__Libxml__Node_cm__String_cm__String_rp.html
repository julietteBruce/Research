XML :: xmlAddAttribute(LibxmlNode,String,String)

xmlAddAttribute(LibxmlNode,String,String) -- add an attribute to an XML node

Synopsis

  • Function: xmlAddAttribute
  • Usage:
    p = xmlAddAttribute(n,a,x)
  • Inputs:
    • n, an instance of the type LibxmlNode,
    • a, a string,
    • x, a string,
  • Outputs:
    • p, an instance of the type LibxmlAttribute, a new attribute with name given by a and content given by x
  • Consequences:
    • p is added to the list of attributes of n

Description

i1 : n = xmlParse ///<bar>asdf</bar>///

o1 = <bar>asdf</bar>

o1 : LibxmlNode (xmlTypeDescription)
 
i2 : a = xmlAddAttribute(n,"foo","the value")

o2 = foo = "the value"

o2 : LibxmlAttribute
 
i3 : xmlGetName a

o3 = foo
 
i4 : n

o4 = <bar foo="the value">asdf</bar>

o4 : LibxmlNode (xmlTypeDescription)
