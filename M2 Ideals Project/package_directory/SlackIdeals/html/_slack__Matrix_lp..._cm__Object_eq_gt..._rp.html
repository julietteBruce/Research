SlackIdeals :: slackMatrix(...,Object=>...)

slackMatrix(...,Object=>...) -- specify combinatorial object

Synopsis

  • Usage:
    slackMatrix(..., Object => "polytope")
    slackMatrix(..., Object => "matroid")

Description

Indicates whether an input list should be treated as the vertices of a polytope or the ground set of a matroid.

Further information

  • Default value: polytope
  • Function: slackMatrix -- computes the slack matrix of a given realization
  • Option key: Object -- select the combinatorial object which the input should be interpreted as
