SlackIdeals :: universalIdeal

universalIdeal -- computes the universal realization ideal of a matroid

Synopsis

  • Usage:
    U = universalIdeal M
    U = universalIdeal V
  • Inputs:
    • M, a matroid, a matroid
    • V, a list, a list of points of a representation of a matroid
  • Optional inputs:
    • CoefficientRing => ..., default value QQ, specifies the coefficient ring of the underlying ring of the ideal
    • Vars => ..., default value {}, specifies the variables to use to create the underlying ring of the ideal
  • Outputs:
    • U, an ideal, the universal realization ideal in Plücker and slack variables

Description

The universal realization ideal defines a variety containing points (p, S) which are pairs of a Plücker vector and a slack matrix coming from the same realization of the matroid.

i1 : M = matroid({0, 1, 2, 3, 4, 5}, {{1, 2, 3}, {0, 2, 4}, {0, 3, 5}, {1, 4, 5}}, EntryMode => "nonbases");
 
i2 : U = universalIdeal M

Caveat

A list of points representing a matroid are always assumed to be in affine space and are projectivized by appending 1 to vector of coordinates.

  • "universalIdeal(Matroid)"

For the programmer

The object universalIdeal is a method function with options.
