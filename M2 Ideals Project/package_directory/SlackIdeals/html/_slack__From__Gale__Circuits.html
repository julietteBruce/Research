SlackIdeals :: slackFromGaleCircuits

slackFromGaleCircuits -- computes the slack matrix of a polytope from a Gale transform of the polytope

Synopsis

  • Usage:
    M = slackFromGaleCircuits(G, Tolerance => 12)
    M = slackFromGaleCircuits G
  • Inputs:
    • G, a matrix, matrix of the form matrix(RR, {...}) whose columns are the vectors of a Gale transform
  • Optional inputs:
    • Tolerance => ..., default value 14, specifies the tolerance to compute the slack matrix of a polytope from a Gale transform of a polytope
  • Outputs:
    • SP, a matrix, the slack matrix of the polytope whose Gale transform is G

Description

This function computes the slack matrix of a polytope starting from a matrix whose columns correspond to the vertices of a Gale transform of the polytope. The second argument Tolerance, which is optional, is a positive integer that expresses the tolerance used in the computations, i.e. if an entry of a matrix is less than 10^(-Tolerance), then it will be treated as zero. The default value for the tolerance is 14.
