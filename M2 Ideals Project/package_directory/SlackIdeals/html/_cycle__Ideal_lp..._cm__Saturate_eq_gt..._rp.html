SlackIdeals :: cycleIdeal(...,Saturate=>...)

cycleIdeal(...,Saturate=>...) -- specifies saturation strategy to be used

Synopsis

  • Usage:
    cycleIdeal(..., Saturate => "all")
    cycleIdeal(..., Saturate => "each")

Further information

  • Default value: all
  • Function: cycleIdeal -- constructs the cycle ideal of a realization
  • Option key: Saturate -- choose whether to saturate with respect to the product of all variables at the same time or variable by variable.
