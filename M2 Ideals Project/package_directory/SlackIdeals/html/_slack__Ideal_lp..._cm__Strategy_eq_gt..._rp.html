SlackIdeals :: slackIdeal(...,Strategy=>...)

slackIdeal(...,Strategy=>...) -- specifies saturation strategy to be used

Synopsis

  • Usage:
    slackIdeal(..., Strategy => Iterate)
    slackIdeal(..., Strategy => Bayer)
    slackIdeal(..., Strategy => Linear)
    slackIdeal(..., Strategy => Eliminate)

Further information

  • Default value: Eliminate
  • Function: slackIdeal -- computes the slack ideal
  • Option key: Strategy -- an optional argument
