VectorGraphics :: viewPort

viewPort -- ViewPort of view port

Description

viewPort gives the range of view port occupied by a GraphicsObject, as computed by the package. See also ViewPort.

Caveat

At the moment viewPort does not take into account the width of "stroke"s.

For the programmer

The object viewPort is a function closure.
