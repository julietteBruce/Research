VectorGraphics :: bisector

bisector -- Angle bisector of two lines

Synopsis

  • Usage:
    bisector ( p, p1, p2 )

Description

Returns the location of the intersection of the angle bisectors of (p,p1) and (p,p2) with the line (p1,p2).
