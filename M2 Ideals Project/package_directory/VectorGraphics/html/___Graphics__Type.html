VectorGraphics :: GraphicsType

GraphicsType -- VectorGraphics type

Description

A particular type of type, used by VectorGraphics, similar to SelfInitializingType.

Fixed objects of class GraphicsType :

  • Circle -- An SVG circle
  • Ellipse -- An SVG ellipse
  • GraphicsHtml -- Html content inside a VectorGraphics object
  • GraphicsList -- A list of VectorGraphics objects
  • GraphicsText -- SVG text
  • Light -- A source of light
  • Line -- An SVG line
  • Path -- An SVG path
  • Polygon -- An SVG polygon
  • Polyline -- An SVG sequence of lines

For the programmer

The object GraphicsType is a type, with ancestor classes Type < MutableHashTable < HashTable < Thing.
