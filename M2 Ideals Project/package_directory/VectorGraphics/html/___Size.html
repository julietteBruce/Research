VectorGraphics :: Size

Size -- Set the size of a picture

Description

An option to fix the size of the GraphicsObject in HTML em units (1em = font size = width of one line of text). Only has an effect if in the outermost GraphicsObject. Can be either a vector { width, height } or a number (diagonal).

For the programmer

The object Size is a symbol.
