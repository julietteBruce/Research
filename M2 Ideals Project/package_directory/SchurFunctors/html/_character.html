SchurFunctors :: character

character -- Determines the character of a composition of Schur functors applied to the representation of GL(V) on V

Synopsis

  • Usage:
    character(L,d)
  • Inputs:
    • L, a list, A nested list whose entries are partitions
    • d, an integer, An integer
  • Outputs:
    • p, a ring element, A symmetric polynomial

Description

Given a list L of partitions {L1,...,Ln} computes the character of the composition of Schur functors SL1(SL2(...(SLn(V)))) applied to the canonical representation of GL(V) where dim(V)=d
