TensorComplexes :: tensor(LabeledModule,LabeledModule)

tensor(LabeledModule,LabeledModule) -- creates the tensor product of two labeled modules, as a labeled module

Synopsis

  • Function: tensor
  • Usage:
    tensor(F,E)
  • Inputs:
    • F, a free module with labeled basis,
    • E, a free module with labeled basis,
  • Outputs:
    • a free module with labeled basis,

Description

This tensor(F,E) is the same as tensorProduct(F,E). See tensorProduct for more details.
