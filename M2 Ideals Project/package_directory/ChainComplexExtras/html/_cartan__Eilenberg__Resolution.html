ChainComplexExtras :: cartanEilenbergResolution

cartanEilenbergResolution -- Computes free resolution of a ChainComplex

Synopsis

  • Usage:
    m = cartanEilenbergResolution C
  • Inputs:
    • C, a chain complex,
  • Optional inputs:
    • LengthLimit => ..., default value infinity, How many steps to compute
  • Outputs:
    • m, a chain complex map, surjective quasi-isomorphism from a free ChainComplex

Description

Uses a different algorithm than resolutionOfChainComplex, often slower and less nearly minimal, to compute a free resolution of a chain complex. See resolutionOfChainComplex for an example.

See also

  • resolutionOfChainComplex -- free resolution of a chain complex

Ways to use cartanEilenbergResolution :

  • "cartanEilenbergResolution(ChainComplex)"

For the programmer

The object cartanEilenbergResolution is a method function with options.
