ChainComplexExtras :: taylor

taylor -- Gives the nth differential in the Taylor resolution of a monomial ideal I.

Synopsis

  • Usage:
    T = taylor(n,I)
  • Inputs:
    • n,
    • I,
  • Outputs:
    • T, the nth differential in the Taylor Resolution of I.

Description

i1 : R = ZZ/101[a,b]

o1 = R

o1 : PolynomialRing
 
i2 : I = monomialIdeal (ideal vars R)^3
