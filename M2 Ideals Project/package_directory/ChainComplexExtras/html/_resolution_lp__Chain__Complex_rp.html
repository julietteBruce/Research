ChainComplexExtras :: resolution(ChainComplex)

resolution(ChainComplex) -- Resolves a ChainComplex.

Synopsis

  • Function: resolution
  • Usage:
    resolution C
  • Inputs:
    • C, a chain complex,
  • Optional inputs:
    • DegreeLimit => ..., default value null,
    • FastNonminimal => ..., default value false,
    • HardDegreeLimit => ..., default value {},
    • LengthLimit => ..., default value infinity,
    • PairLimit => ..., default value infinity,
    • SortStrategy => ..., default value 0,
    • StopBeforeComputation => ..., default value false,
    • Strategy => ..., default value null,
    • SyzygyLimit => ..., default value infinity,
  • Outputs:
    • ChainComplex

Description

Returns a surjective ChainComplexMap that is a quasi-isomorphism from a (generally non-minimal) ChainComplex of free modules. resolution C is the same as resolutionOfChainComplex C. The quasi-isomorphism is computed by the method of iterated mapping cones. (For the computation of the Cartan-Eilenberg resolution, which is usually slower and results in a larger complex, use cartanEilenbergResolution C
