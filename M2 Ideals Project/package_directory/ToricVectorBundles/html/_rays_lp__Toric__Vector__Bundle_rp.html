ToricVectorBundles :: rays(ToricVectorBundle)

rays(ToricVectorBundle) -- the rays of the underlying fan

Synopsis

  • Function: rays
  • Usage:
    L = rays E
  • Inputs:
    • E, an instance of the type ToricVectorBundle
  • Outputs:
    • L, a list

Description

Returns the rays of the fan of the underlying toric variety as a list. Each ray is given as a one column matrix. 

i1 : E = cotangentBundle projectiveSpaceFan 2

o1 = {dimension of the variety => 2 }
      number of affine charts => 3
      number of rays => 3
      rank of the vector bundle => 2

o1 : ToricVectorBundleKlyachko
 
i2 : rays E

o2 = {| -1 |, | 0 |, | 1 |}
      | -1 |  | 1 |  | 0 |

o2 : List
