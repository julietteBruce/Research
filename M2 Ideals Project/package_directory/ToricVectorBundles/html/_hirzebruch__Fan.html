ToricVectorBundles :: hirzebruchFan

hirzebruchFan -- the fan of the n-th Hirzebruch surface

Synopsis

  • Usage:
    F = hirzebruchFan n
  • Inputs:
    • n, an integer, positive
  • Outputs:
    • F, an instance of the type Fan

Description

Generates the fan of the $n$-th Hirzebruch surface. 

i1 : F = hirzebruchFan 3

o1 = F

o1 : Fan
 
i2 : rays F

o2 = | 1 0  0 -1 |
     | 0 -1 1 3  |

              2        4
o2 : Matrix ZZ  <--- ZZ
