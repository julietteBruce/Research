FourTiTwo :: InputType

InputType

Description

Put InputType => "lattice" as an argument in the functions toricMarkov and hilbertBasis

See also

  • toricMarkov -- calculates a generating set of the toric ideal I_A, given A; invokes "markov" from 4ti2
  • hilbertBasis -- calculates the Hilbert basis of the cone; invokes "hilbert" from 4ti2

Functions with optional argument named InputType :

  • "hilbertBasis(...,InputType=>...)" -- see hilbertBasis -- calculates the Hilbert basis of the cone; invokes "hilbert" from 4ti2
  • "toricMarkov(...,InputType=>...)" -- see toricMarkov -- calculates a generating set of the toric ideal I_A, given A; invokes "markov" from 4ti2
