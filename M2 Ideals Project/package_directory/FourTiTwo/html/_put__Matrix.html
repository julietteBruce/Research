FourTiTwo :: putMatrix

putMatrix -- writes a matrix into a file formatted for 4ti2

Synopsis

  • Usage:
    putMatrix(F,A)
  • Inputs:
    • F, a file,
    • A, a matrix,

Description

Write the matrix A in file F in 4ti2 format.

i1 : A = matrix "1,1,1,1; 1,2,3,4"

o1 = | 1 1 1 1 |
     | 1 2 3 4 |

              2        4
o1 : Matrix ZZ  <--- ZZ
 
i2 : s = temporaryFileName()

o2 = /private/tmp/M2-18205-0/0
 
i3 : F = openOut(s)

o3 = /private/tmp/M2-18205-0/0

o3 : File
 
i4 : putMatrix(F,A)
 
i5 : close(F)

o5 = /private/tmp/M2-18205-0/0

o5 : File
 
i6 : getMatrix(s)

o6 = | 1 1 1 1 |
     | 1 2 3 4 |

              2        4
o6 : Matrix ZZ  <--- ZZ

See also

  • getMatrix -- reads a matrix from a 4ti2-formatted input file

Ways to use putMatrix :

  • "putMatrix(File,Matrix)"

For the programmer

The object putMatrix is a method function.
