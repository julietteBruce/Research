Resultants :: Grass(...,Variable=>...)

Grass(...,Variable=>...) -- specify a name for a variable

Synopsis

  • Usage:
    Variable => x

Description

This is an option used to specify a symbol to be used as a name for the generator of the coordinate ring of the Grassmannian Grass, or some other ring.

Further information

  • Default value: p
  • Function: Grass -- coordinate ring of a Grassmannian
  • Option key: Variable -- specify a name for a variable
