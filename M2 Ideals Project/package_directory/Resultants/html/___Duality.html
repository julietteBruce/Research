Resultants :: Duality

Duality -- whether to use dual Plücker coordinates

Synopsis

  • Usage:
    Duality => true/false

Description

This is a option typically used to specify whether to perform internal computations using the dual Plücker coordinates.

See also

  • chowForm -- Chow form of a projective variety
  • hurwitzForm -- Hurwitz form of a projective variety
  • tangentialChowForm -- higher Chow forms of a projective variety

Functions with optional argument named Duality :

  • "cayleyTrick(...,Duality=>...)"
  • "chowForm(...,Duality=>...)"
  • "hurwitzForm(...,Duality=>...)"
  • "isInCoisotropic(...,Duality=>...)"
  • "tangentialChowForm(...,Duality=>...)"
