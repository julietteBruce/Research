Resultants :: resultant

resultant -- multipolynomial resultant

Synopsis

  • Usage:
    resultant F
  • Inputs:
    • F, a matrix, a row matrix whose entries are $n+1$ homogeneous polynomials $F_0,\ldots,F_n$ in $n+1$ variables (or a list to be interpreted as such a matrix)
  • Optional inputs:
    • Algorithm => ..., default value "Poisson",
  • Outputs:
    • a ring element, the resultant of $F_0,\ldots,F_n$

Description

Let $F_0,\ldots,F_n$ be $n+1$ homogeneous polynomials in $n+1$ variables $x_0,\ldots,x_n$ over a commutative ring $K$. The resultant $R(F_0,\ldots,F_n)$ is a certain polynomial in the coefficients of $F_0,\ldots,F_n$; when $K$ is an algebraically closed field, $R(F_0,\ldots,F_n)$ vanishes if and only if $F_0,\ldots,F_n$ have a common nontrivial root. For the general theory, see one of the following: Using Algebraic Geometry, by David A. Cox, John Little, Donal O'shea; Discriminants, Resultants, and Multidimensional Determinants, by Israel M. Gelfand, Mikhail M. Kapranov and Andrei V. Zelevinsky.
