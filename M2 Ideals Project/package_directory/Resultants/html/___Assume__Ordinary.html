Resultants :: AssumeOrdinary

AssumeOrdinary -- whether the expected codimension is 1

Synopsis

  • Usage:
    AssumeOrdinary => true/false

Description

This is an option for dualVariety and tangentialChowForm that controls the option SubringLimit, used internally with kernel and gb. You can set this to true when the expected output should represent a hypersurface.

See also

  • dualVariety -- projective dual variety
  • tangentialChowForm -- higher Chow forms of a projective variety

Functions with optional argument named AssumeOrdinary :

  • "dualVariety(...,AssumeOrdinary=>...)"
  • "tangentialChowForm(...,AssumeOrdinary=>...)"

For the programmer

The object AssumeOrdinary is a symbol.
