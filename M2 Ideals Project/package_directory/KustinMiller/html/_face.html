KustinMiller :: face

face -- Generate a face.

Synopsis

  • Usage:
    face(L)
    face(L,R)
    face(m)
  • Inputs:
    • L, a list,
    • R, a polynomial ring,
    • m, a ring element, a monomial
  • Outputs:
    • an instance of the type Face,

Description

Generates a face out of a list L or a squarefree monomial. If L is not empty or a monomial the argument R is not required.

i1 : K=QQ;
 
i2 : R=K[x_0..x_4];
 
i3 : F=face {x_0,x_1}

o3 = x  x  
      0  1

o3 : face with 2 vertices in R

See also

  • faces -- computes all faces of a certain codimension of a Cone or Polyhedron
  • facets -- Giving the facet inequalities of a cone or polyhedron.
