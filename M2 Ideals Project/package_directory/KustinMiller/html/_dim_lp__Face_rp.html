dim(Face) -- The dimension of a face.

Synopsis

  • Function: dim
  • Usage:
    dim F
  • Inputs:
    • F, an instance of the type Face,
  • Outputs:
    • an integer, bigger or equal to -1

Description

Returns the dimension of a Face, i.e., the number of vertices F minus 1.
