BettiCharacters > action

action -- define finite group action

Synopsis

  • Optional inputs:
    • Sub => ..., default value true, format ring actors as one-row substitution matrices

Description

This function is provided by the package BettiCharacters.

Use this function to set up a finite group action on a minimal free resolution or graded module. See the specific use cases for more details.

Ways to use action :
