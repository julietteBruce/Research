BettiCharacters > actors > numActors

numActors -- number of acting elements

Synopsis

  • Usage:
    numActors(A)
  • Inputs:
    • A, an instance of the type Action,
  • Outputs:
    • an integer,

Description

This function is provided by the package BettiCharacters.

Returns the number of group elements passed by the user when defining the given action. This number is not necessarily the order of the acting group because in order to compute characters it is enough to work with a representative of each conjugacy class of the group.

See also

  • action -- define finite group action

Ways to use numActors :

  • "numActors(Action)"

For the programmer

The object numActors is a method function.
