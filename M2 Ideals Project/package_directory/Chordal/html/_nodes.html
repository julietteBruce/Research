Chordal :: nodes

nodes -- list of nodes of a chordal network

Synopsis

  • Usage:
    nodes N
    nodes(N,i)
  • Inputs:
    • N, an instance of the type ChordalNet,
    • i, a ring element, a variable (optional)
  • Optional inputs:
    • Reverse (missing documentation) => ..., default value false,
  • Outputs:
    • a list, consisting of chordal network nodes

Description

This method returns the list of nodes of the chordal network. If the optional argument i is given, then only the nodes of rank i are returned.

i1 : R = QQ[x_0..x_3, MonomialOrder=>Lex];
