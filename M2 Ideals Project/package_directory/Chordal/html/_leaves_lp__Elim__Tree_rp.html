Chordal :: leaves(ElimTree)

leaves(ElimTree) -- leaves of an elimination tree

Synopsis

  • Function: leaves
  • Usage:
    leaves tree
  • Inputs:
    • tree, an instance of the type ElimTree,
  • Outputs:
    • a list, of leaves of the tree

Description

i1 : G = graph(toList(0..9),{
         {0,{6,7}},{1,{4,9}},{2,{3,5}},{3,{7,8}},
         {4,{5,8}},{5,{8}},{6,{8,9}},{7,{8}},{8,{9}} });
