Chordal :: writeDotFile(String,Function,ChordalNet)

writeDotFile(String,Function,ChordalNet) -- writes a chordal network to a dot file

Synopsis

  • Function: writeDotFile
  • Usage:
    writeDotFile(fileName,label,N)
    writeDotFile(fileName,tree)
  • Inputs:
    • fileName, a string,
    • label, a function,
    • N, an instance of the type ChordalNet,

Description

Writes the code for an inputted chordal network (or elimination tree) to be constructed in Graphviz with specified file name
