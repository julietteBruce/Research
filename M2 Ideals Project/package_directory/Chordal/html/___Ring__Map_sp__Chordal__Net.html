Chordal :: RingMap ChordalNet

RingMap ChordalNet -- apply ring map to a chordal network

Synopsis

  • Operator: SPACE
  • Usage:
    f N
    f tree
  • Inputs:
    • f, a ring map,
    • N, an instance of the type ChordalNet,
  • Outputs:
    • an instance of the type ChordalNet, image of N under the ring map f
