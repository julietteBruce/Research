Chordal :: reduceNet

reduceNet -- reduces a chordal network

Synopsis

  • Usage:
    reduceNet N
  • Inputs:
    • N, an instance of the type ChordalNet,
  • Outputs:
    • no output
  • Consequences:
    • the input chordal network is modified

Description

Simplifies a chordal network by applying a sequence of reduction rules (while preserving the underlying variety).

i1 : I = adjacentMinorsIdeal(QQ,2,4);

o1 : Ideal of QQ[a..h]
 
i2 : N = chordalNet I;
 
i3 : chordalTria N;
 
i4 : codimCount N

       5     4     3
o4 = 2t  + 6t  + 3t

o4 : ZZ[t]
 
i5 : reduceNet N
 
i6 : codimCount N

       4     3
o6 = 5t  + 3t

o6 : ZZ[t]
