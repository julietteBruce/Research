Chordal :: dim(ChordalNet)

dim(ChordalNet) -- dimension of a chordal network

Synopsis

  • Function: dim
  • Usage:
    dim N
    codim N
  • Inputs:
    • N, an instance of the type ChordalNet,
  • Outputs:
    • an integer, dimension of the associated variety

Description

This method computes the dimension of the variety associated to a chordal network.

i1 : I = adjacentMinorsIdeal(QQ,2,10);

o1 : Ideal of QQ[a..t]
 
i2 : N = chordalNet I;
 
i3 : chordalTria N;
 
i4 : dim N

o4 = 11
