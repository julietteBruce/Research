Complexes :: isQuasiIsomorphism(ComplexMap)

isQuasiIsomorphism(ComplexMap) -- whether a map of complexes is a quasi-isomorphism

Synopsis

  • Function: isQuasiIsomorphism
  • Usage:
    isQuasiIsomorphism f
  • Inputs:
    • f, a map of complexes,
  • Optional inputs:
    • Concentration => a sequence, default value (-infinity,infinity), restricts attention to the induced maps indexed by elements in the given interval
  • Outputs:
    • a Boolean value, that is true when $f$ is a morphism of complexes such that the induced maps on homology are all isomorphisms

Description

The cone of a map $f \colon C \to D$ is acyclic exactly when $f$ is a quasi-isomorphism.
