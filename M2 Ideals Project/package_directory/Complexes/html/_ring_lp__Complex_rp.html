Complexes :: ring(Complex)

ring(Complex) -- access the ring of a complex or a complex map

Synopsis

  • Function: ring
  • Usage:
    ring C
  • Inputs:
    • C, a complex, or a ComplexMap
  • Outputs:
    • a ring,

Description

Every complex or complex map has a base ring. This function access that information.

i1 : S = ZZ/101[a,b,c,d];
 
i2 : C = freeResolution coker vars S

i3 : ring C
