Complexes :: target(ComplexMap)

target(ComplexMap) -- get the target of a map of chain complexes

Synopsis

  • Function: target
  • Usage:
    C = target f
  • Inputs:
    • f, a map of complexes,
  • Outputs:
    • C, a complex,

Description

Given a complex map $f : C \to D$ this method returns the chain complex $D$.
