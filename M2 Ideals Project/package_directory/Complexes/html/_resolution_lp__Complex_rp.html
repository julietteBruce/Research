Complexes :: resolution(Complex)

resolution(Complex) -- minimal free resolution of a complex

Synopsis

  • Function: resolution
  • Usage:
    resolution C
  • Inputs:
    • C, a complex,
  • Optional inputs:
    • LengthLimit => an integer, default value infinity, this is used to limit somehow the computation where resolutions might be too long or infinite
    • DegreeLimit => ..., default value null, unused
    • FastNonminimal => ..., default value false, unused
    • HardDegreeLimit => ..., default value {}, unused
    • PairLimit => ..., default value infinity, unused
    • SortStrategy => ..., default value 0, unused
    • StopBeforeComputation => ..., default value false, unused
    • Strategy => ..., default value null, unused
    • SyzygyLimit => ..., default value infinity, unused
  • Outputs:
    • a complex,

Description

Given a complex $C$, this method produces a quasi-isomorphic complex $F$ all of whose terms are free modules. The algorithm used minimizes the ranks of the free modules in $F$.

This method just accesses the cached value from the method resolutionMap(Complex), which does the actual computation.
