Cremona :: Dominant

Dominant

Description

This is an optional argument for toMap. When true or infinity is passed to this option, the kernel of the returned ring map will be zero.

Functions with optional argument named Dominant :

  • "rationalMap(...,Dominant=>...)"
  • "toMap(...,Dominant=>...)"

For the programmer

The object Dominant is a symbol.
