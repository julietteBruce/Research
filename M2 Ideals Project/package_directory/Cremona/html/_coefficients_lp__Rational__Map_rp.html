Cremona :: coefficients(RationalMap)

coefficients(RationalMap) -- coefficient matrix of a rational map

Synopsis

  • Function: coefficients
  • Usage:
    coefficients phi
  • Inputs:
    • phi, a rational map
  • Optional inputs:
    • Monomials => ..., default value null,
    • Variables => ..., default value null,
  • Outputs:
    • a matrix, the coefficient matrix of the polynomials defining phi
