RationalMap == RationalMap -- equality of rational maps

Synopsis

  • Operator: ==
  • Usage:
    phi == psi
  • Inputs:
    • phi, a rational map
    • psi, a rational map
  • Outputs:
    • a Boolean value, whether phi and psi are the same rational map
