Cremona :: parametrize(Ideal)

parametrize(Ideal) -- parametrization of linear varieties and hyperquadrics

Synopsis

  • Function: parametrize
  • Usage:
    parametrize I
  • Inputs:
    • I, an ideal, the ideal of a linear variety or of a hyperquadric
  • Outputs:
    • a rational map, a birational map phi such that I == image phi

Description

This function has been improved and extended in the package MultiprojectiveVarieties, see parametrize(MultiprojectiveVariety).
