Cremona :: degreeMap(RationalMap)

degreeMap(RationalMap) -- degree of a rational map

Synopsis

  • Function: degreeMap
  • Usage:
    degreeMap Phi
  • Inputs:
    • Phi, a rational map
  • Optional inputs:
    • BlowUpStrategy => ..., default value "Eliminate",
    • Certify => ..., default value false, whether to ensure correctness of output
    • Verbose => ..., default value true,
  • Outputs:
    • an integer, the degree of Phi. So this value is 1 if and only if the map is birational onto its image.

Description

This computation is done through the corresponding method for ring maps. See degreeMap(RingMap) for more details and examples.
