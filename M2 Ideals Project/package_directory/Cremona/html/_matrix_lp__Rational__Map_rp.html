Cremona :: matrix(RationalMap)

matrix(RationalMap) -- the matrix associated to a rational map

Synopsis

  • Function: matrix
  • Usage:
    matrix Phi
  • Inputs:
    • Phi, a rational map
  • Optional inputs:
    • Degree => ..., default value null,
  • Outputs:
    • a matrix, the matrix associated to the ring map defining the rational map Phi

Description

This is equivalent to matrix map Phi. Moreover, the command matrix Phi is equivalent to matrix(0,Phi), and more generally the command matrix(i,Phi) returns the matrix of the i-th representative of Phi.

See also

  • map(RationalMap) -- get the ring map defining a rational map
  • matrix(RingMap) -- the matrix associated to a ring map
