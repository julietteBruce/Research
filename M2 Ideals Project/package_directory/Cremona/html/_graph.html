Cremona :: graph

graph -- closure of the graph of a rational map

Synopsis

  • Usage:
    graph phi
  • Inputs:
    • phi, a rational map
  • Optional inputs:
    • BlowUpStrategy => ..., default value "Eliminate",
  • Outputs:
    • a rational map, the first projection
    • a rational map, the second projection
