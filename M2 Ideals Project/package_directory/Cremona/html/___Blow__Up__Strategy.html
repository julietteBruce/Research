Cremona :: BlowUpStrategy

BlowUpStrategy

Description

This is an optional argument for graph, and for the methods that eventually call it. Currently, the possible values are "Eliminate" and "Saturate", which indicate two different ways of computing the (closure of the) graph of a rational map. The default choice is "Eliminate" and this is generally preferable.
