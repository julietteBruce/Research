Cremona :: quadroQuadricCremonaTransformation

quadroQuadricCremonaTransformation -- quadro-quadric Cremona transformations

Synopsis

  • Usage:
    quadroQuadricCremonaTransformation(n,i)
    quadroQuadricCremonaTransformation(n,i,K)
  • Inputs:
    • n, an integer, the dimension of the projective space
    • i, an integer, the i-th case in the classification for P^n (for instance, if n=5 then 1<=i<=39)
    • K, a ring, the ground field (optional, the default value is QQ)
  • Outputs:
    • a rational map, an example of quadro-quadric Cremona transformation over K, according to the classifications given in the paper Quadro-quadric Cremona transformations in low dimensions via the JC-correspondence, by Pirio and Russo.
