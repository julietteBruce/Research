Cremona :: isDominant

isDominant -- whether a rational map is dominant

Synopsis

  • Usage:
    isDominant phi
  • Inputs:
    • phi, a rational map
  • Optional inputs:
    • Certify => ..., default value false, whether to ensure correctness of output
    • Verbose => ..., default value true,
  • Outputs:
    • a Boolean value, whether phi is dominant

Description

This method is based on the fibre dimension theorem. A more standard way would be to perform the command kernel map phi == 0.

i1 : P8 = ZZ/101[x_0..x_8];
 
i2 : phi = rationalMap ideal jacobian ideal det matrix{{x_0..x_4},{x_1..x_5},{x_2..x_6},{x_3..x_7},{x_4..x_8}};

o2 : RationalMap (rational map from PP^8 to PP^8)
 
i3 : time isDominant(phi,Certify=>true)
Certify: output certified!
