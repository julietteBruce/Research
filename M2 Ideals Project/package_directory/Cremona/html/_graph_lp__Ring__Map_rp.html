Cremona :: graph(RingMap)

graph(RingMap) -- closure of the graph of a rational map

Synopsis

  • Function: graph
  • Usage:
    graph phi
  • Inputs:
    • phi, a ring map, representing a rational map $\Phi:X \dashrightarrow Y$
  • Optional inputs:
    • BlowUpStrategy => ..., default value "Eliminate",
  • Outputs:
    • a ring map, representing the projection on the first factor $\mathbf{Graph}(\Phi) \to X$
    • a ring map, representing the projection on the second factor $\mathbf{Graph}(\Phi) \to Y$
