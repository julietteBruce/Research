Cremona :: parametrize

parametrize -- parametrization of a rational projective variety

Description

See parametrize(MultiprojectiveVariety) and parametrize(QuotientRing).

Ways to use parametrize :

  • parametrize(Ideal) -- parametrization of linear varieties and hyperquadrics
  • "parametrize(PolynomialRing)" -- see parametrize(Ideal) -- parametrization of linear varieties and hyperquadrics
  • "parametrize(QuotientRing)" -- see parametrize(Ideal) -- parametrization of linear varieties and hyperquadrics

For the programmer

The object parametrize is a method function.
