Cremona :: flatten(RationalMap)

flatten(RationalMap) -- write source and target as nondegenerate varieties

Synopsis

  • Function: flatten
  • Usage:
    flatten phi
  • Inputs:
    • phi, a rational map
  • Outputs:
    • a rational map, a rational map isomorphic to the original map, flattened in the sense that the ideals of source and target contain no linear forms
