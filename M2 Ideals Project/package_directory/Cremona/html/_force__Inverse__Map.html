Cremona :: forceInverseMap

forceInverseMap -- declare that two rational maps are one the inverse of the other

Synopsis

  • Usage:
    forceInverseMap(Phi,Psi)
  • Inputs:
    • Phi, a rational map
    • Psi, a rational map
  • Outputs:
    • null

Description

This method allows to inform the system that two maps are one the inverse of the other without performing any computation. This is useful in particular if you calculate the inverse map using your own method.

Caveat

If the declaration is false, nonsensical answers may result.

See also

  • isInverseMap(RationalMap,RationalMap) -- checks whether two rational maps are one the inverse of the other
  • inverse(RationalMap) -- inverse of a birational map
