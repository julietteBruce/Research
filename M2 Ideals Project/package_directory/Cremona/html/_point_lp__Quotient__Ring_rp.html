Cremona :: point(QuotientRing)

point(QuotientRing) -- pick a random rational point on a projective variety

Synopsis

  • Function: point
  • Usage:
    point R
  • Inputs:
    • R, a quotient ring, the homogeneous coordinate ring of a closed subscheme $X\subseteq\mathbb{P}^n$ over a finite ground field
  • Outputs:
    • an ideal, an ideal in R defining a point on $X$

Description

This function is a variant of the randomKRationalPoint function, which has been further improved and extended in the package MultiprojectiveVarieties, see point(MultiprojectiveVariety).
