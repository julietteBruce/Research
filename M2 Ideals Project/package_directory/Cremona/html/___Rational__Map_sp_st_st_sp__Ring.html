Cremona :: RationalMap ** Ring

RationalMap ** Ring -- change the coefficient ring of a rational map

Synopsis

  • Operator: **
  • Usage:
    phi ** K
  • Inputs:
    • phi, a rational map, defined over a coefficient ring F
    • K, a ring, the new coefficient ring (which must be a field)
  • Outputs:
    • a rational map, a rational map defined over K, obtained by coercing the coefficients of the forms defining phi into K

Description

It is necessary that all forms in the old coefficient ring F can be automatically coerced into the new coefficient ring K.

i1 : QQ[vars(0..5)]

o1 = QQ[a..f]

o1 : PolynomialRing
 
i2 : phi = rationalMap {e^2-d*f, c*e-b*f, c*d-b*e, c^2-a*f, b*c-a*e, b^2-a*d}
