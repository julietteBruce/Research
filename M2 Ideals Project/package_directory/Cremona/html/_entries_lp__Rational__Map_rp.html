Cremona :: entries(RationalMap)

entries(RationalMap) -- the entries of the matrix associated to a rational map

Synopsis

  • Function: entries
  • Usage:
    entries Phi
  • Inputs:
    • Phi, a rational map
  • Outputs:
    • a list, the entries of the matrix associated to the ring map defining the rational map Phi

Description

This is equivalent to flatten entries matrix Phi.
