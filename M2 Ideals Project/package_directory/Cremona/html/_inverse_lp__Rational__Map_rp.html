Cremona :: inverse(RationalMap)

inverse(RationalMap) -- inverse of a birational map

Synopsis

  • Function: inverse
  • Usage:
    inverse phi
    inverse(phi,Certify=>b)
  • Inputs:
    • phi, a rational map, which has to be birational, and b is a boolean value, that is, true or false (the default value is false)
  • Outputs:
    • a rational map, the inverse map of phi

Description

inverse(phi,Certify=>true) is the same as inverseMap(phi,Certify=>true,Verbose=>false), while inverse(phi,Certify=>false) applies a middle ground approach between inverseMap(phi,Certify=>true) and inverseMap(phi,Certify=>false). The procedure for the latter is as follows: It first computes the inverse map of phi using psi = inverseMap phi. Then it is checked that psi phi p == p and phi psi q == q, where p,q are, respectively, a random point on the source and the target of phi. Finally, if the tests pass, the command forceInverseMap(phi,psi) is invoked and psi is returned.

i1 : R = QQ[x_0..x_4]; phi = rationalMap minors(4,random(R^{4:1},R^5)) -- special Cremona transformation of P^4 of type (4,4)
