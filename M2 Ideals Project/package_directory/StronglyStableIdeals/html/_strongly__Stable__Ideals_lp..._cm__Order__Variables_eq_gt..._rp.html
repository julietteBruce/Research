StronglyStableIdeals :: stronglyStableIdeals(...,OrderVariables=>...)

stronglyStableIdeals(...,OrderVariables=>...) -- Option to set the order of indexed variables

Description

This option can be used to specify the order of the indexed variables of the polynomial ring containing the ideals, when calling stronglyStableIdeals giving as input only the number of variables of the polynomial ring. If OrderVariables is set to Up then x_i < x_j iff i < j, otherwise if OrderVariables is set to Down then x_i < x_j iff i > j.

The default is Down.

i1 : stronglyStableIdeals(3, 3, OrderVariables=>Down)

                  3           2         2
o1 = {ideal (x , x ), ideal (x , x x , x )}
              0   1           1   0 1   0

o1 : List
 
i2 : stronglyStableIdeals(3, 3, OrderVariables=>Up)

                  3           2         2
o2 = {ideal (x , x ), ideal (x , x x , x )}
              2   1           1   2 1   2

o2 : List

Further information

  • Default value: Down
  • Function: stronglyStableIdeals -- Compute the saturated strongly stable ideals in the given ambient space with given Hilbert polynomial
  • Option key: OrderVariables -- Option to set the order of indexed variables
