StronglyStableIdeals :: isRegSegment

isRegSegment -- reg-segment ideals

Synopsis

  • Usage:
    isRegSegment J
  • Inputs:
    • J, a monomial ideal, or
    • J, an ideal, a strongly stable ideal.
  • Outputs:
    • a sequence,

Description

Calls the method isGenSegment on the ideal truncate(regularity J, J).

i1 : QQ[x,y,z];
 
i2 : J = ideal(x^2,x*y^3,y^4)

             2     3   4
o2 = ideal (x , x*y , y )

o2 : Ideal of QQ[x..z]
 
i3 : isRegSegment J

o3 = (false, )

o3 : Sequence
 
i4 : isGenSegment(truncate(regularity J,J))

o4 = (false, )

o4 : Sequence
 
i5 : isGenSegment J

o5 = (true, {4, 3, 1})

o5 : Sequence

Ways to use isRegSegment :

  • "isRegSegment(Ideal)"
  • "isRegSegment(MonomialIdeal)"

For the programmer

The object isRegSegment is a method function.
