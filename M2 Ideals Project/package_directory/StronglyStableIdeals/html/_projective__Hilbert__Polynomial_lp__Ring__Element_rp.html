StronglyStableIdeals :: projectiveHilbertPolynomial(RingElement)

projectiveHilbertPolynomial(RingElement)

Synopsis

  • Function: projectiveHilbertPolynomial
  • Usage:
    projectiveHilbertPolynomial p
  • Inputs:
    • p, a ring element, a Hilbert polynomial.
  • Outputs:
    • a projective Hilbert polynomial,

Description

Convert a RingElement representing a Hilbert polynomial to a ProjectiveHilbertPolynomial.

i1 : QQ[t];
 
i2 : projectiveHilbertPolynomial (3*t+4)

o2 = P  + 3*P
      0      1

o2 : ProjectiveHilbertPolynomial
