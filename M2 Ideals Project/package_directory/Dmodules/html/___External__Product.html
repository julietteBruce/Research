Dmodules :: ExternalProduct

ExternalProduct -- external product of modules or complexes

Synopsis

  • Optional inputs:
    • TwistMap => ..., default value false, indicates whether TwistMap should be computed

Ways to use ExternalProduct :

  • "ExternalProduct(ChainComplex,ChainComplex)"
  • "ExternalProduct(Module,Module)"

For the programmer

The object ExternalProduct is a method function with options.
