Dmodules :: pInfo

pInfo -- prints tracing info

Description

Prints tracing information according to the print level set by Dtrace.

See also

  • Dtrace -- set the depth of comments made by D-module routines

Ways to use pInfo :

  • "pInfo(ZZ,List)"
  • "pInfo(ZZ,Thing)"

For the programmer

The object pInfo is a method function.
