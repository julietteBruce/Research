Dmodules :: AppellF1

AppellF1 -- Appell F1 system of PDE's

Synopsis

  • Usage:
    AppellF1 {a0,a1,a2,a3}
  • Inputs:
    • a list,
  • Optional inputs:
    • Vars => ..., default value Global,
  • Outputs:
    • an ideal, which represents Appell F1 system of PDE's associated to the parameters a0, a1, a2, and a3.

Caveat

AppellF1 always returns a different ring and will use variables x and y. Input should be a List of 4 numbers.
