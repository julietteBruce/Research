Dmodules :: reiffen

reiffen -- Reiffen's curve

Synopsis

  • Usage:
    reiffen(p,q)
  • Inputs:
    • p, an integer,
    • q, an integer,
  • Outputs:
    • a ring element, the defining polynomial of Reiffen's (p,q)-curve

Description

Reiffen's (p,q)-curve is a singular curve defined by $f_{p,q} = x_1^p+x_2^q+x_1x_2^{q-1} = 0$ for $p>=4$ and $q>=p+1$.

i1 : reiffen(4,5)

        4    5    4
o1 = x x  + x  + x
      1 2    2    1

o1 : QQ[x ..x ]
         1   2

Ways to use reiffen :

  • "reiffen(ZZ,ZZ)"

For the programmer

The object reiffen is a method function.
