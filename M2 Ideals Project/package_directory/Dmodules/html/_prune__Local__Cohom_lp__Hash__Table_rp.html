Dmodules :: pruneLocalCohom(HashTable)

pruneLocalCohom(HashTable) -- prunes local cohomology modules

Synopsis

  • Function: pruneLocalCohom
  • Usage:
    pruneLocalCohom H
  • Inputs:
    • a hash table, the output of localCohom
  • Outputs:
    • a hash table

Description

This function applies Dprune to all the keys of H.

See also

  • localCohom -- local cohomology
  • Dprune -- prunes a D-module

Ways to use this method:

  • pruneLocalCohom(HashTable) -- prunes local cohomology modules
