Dmodules :: AnnFs(List)

AnnFs(List) -- the annihilating ideal of f_1^{s_1}...f_r^{s_r}

Synopsis

  • Function: AnnFs
  • Usage:
    AnnFs F
  • Inputs:
    • F, a list, {f_1,...,f_r}, a list of polynomials in n variables (f_i has to be an element of A_n, the Weyl algebra)
  • Outputs:
    • an ideal, an ideal in A_n<t_1,..., t_r,dt_1,...,dt_r>
