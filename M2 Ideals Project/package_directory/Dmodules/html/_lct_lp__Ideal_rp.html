Dmodules :: lct(Ideal)

lct(Ideal) -- compute the log canonical threshold for an ideal

Synopsis

  • Function: lct
  • Usage:
    l = lct I
  • Inputs:
    • I, an ideal, an ideal in a polynomial ring
  • Optional inputs:
    • Strategy => ..., default value GeneralBernsteinSato, specify strategy for computing lct
  • Outputs:
    • l, a ring element, a rational number, the log canonical threshold of I

Description

i1 : QQ[x_1..x_3];
 
i2 : I = ideal (x_2^2-x_1*x_3, x_1^3-x_3^2);

o2 : Ideal of QQ[x ..x ]
                  1   3
 
i3 : lct I

     17
o3 = --
     12

o3 : QQ
