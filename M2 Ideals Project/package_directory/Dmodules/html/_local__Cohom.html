Dmodules :: localCohom

localCohom -- local cohomology

Synopsis

  • Optional inputs:
    • LocStrategy => ..., default value null, specify localization strategy for local cohomology
    • Strategy => ..., default value Walther, specify strategy for local cohomology

Description

Local cohomology of a polynomial ring:
  • localCohom(Ideal)
  • localCohom(List,Ideal)
  • localCohom(ZZ,Ideal)
Local cohomology of a holonomic module:
  • localCohom(Ideal,Module)
  • localCohom(ZZ,Ideal,Module)
  • localCohom(List,Ideal,Module)

See also

  • pruneLocalCohom -- prunes local cohomology modules

Ways to use localCohom :

  • localCohom(Ideal) -- local cohomology of a polynomial ring
  • localCohom(Ideal,Module) -- local cohomology of a D-module
  • localCohom(List,Ideal) -- local cohomology of a polynomial ring
  • localCohom(List,Ideal,Module) -- local cohomology of a D-module
  • localCohom(ZZ,Ideal) -- local cohomology of a polynomial ring
  • localCohom(ZZ,Ideal,Module) -- local cohomology of a D-module

For the programmer

The object localCohom is a method function with options.
