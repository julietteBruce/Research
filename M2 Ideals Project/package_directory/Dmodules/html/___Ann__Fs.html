Dmodules :: AnnFs

AnnFs -- the annihilating ideal of f^s

Description

Either a single or a list of polynomials can be supplied for f.

See also

  • AnnIFs -- the annihilating ideal of f^s for an arbitrary D-module
  • WeylAlgebra -- an optional argument

Ways to use AnnFs :

  • AnnFs(List) -- the annihilating ideal of f_1^{s_1}...f_r^{s_r}
  • AnnFs(RingElement) -- the annihilating ideal of f^s

For the programmer

The object AnnFs is a method function.
