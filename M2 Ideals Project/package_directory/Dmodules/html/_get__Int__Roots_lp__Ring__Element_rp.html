getIntRoots(RingElement) -- get integer roots of a b-function

Synopsis

  • Function: getIntRoots
  • Usage:
    getIntRoots b
  • Inputs:
    • b, a ring element, the output of one of the b-function routines
  • Outputs:
    • a list, the list of the integer roots of b
