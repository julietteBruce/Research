Dmodules :: bFunction

bFunction -- b-function

Synopsis

  • Optional inputs:
    • Strategy => ..., default value IntRing, specify strategy for computing b-function

Description

  • bFunction(Ideal,List) - for an ideal
  • bFunction(Module,List,List) - for a module

Ways to use bFunction :

  • bFunction(Ideal,List) -- b-function of an ideal
  • bFunction(Module,List,List) -- b-function of a holonomic D-module

For the programmer

The object bFunction is a method function with options.
