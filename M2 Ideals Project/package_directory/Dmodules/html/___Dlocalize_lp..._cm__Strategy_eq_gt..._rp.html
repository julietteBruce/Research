Dmodules :: Dlocalize(...,Strategy=>...)

Dlocalize(...,Strategy=>...) -- strategy for computing a localization of a D-module

Description

  • Oaku -- use Oaku's algorithm
  • OTW -- use Oaku-Takayama-Walther's algorithm
  • OTWcyclic -- use Oaku-Takayama-Walther's algorithm for a cyclic module

Further information

  • Default value: OTW
  • Function: Dlocalize -- localization of a D-module
  • Option key: Strategy -- an optional argument
