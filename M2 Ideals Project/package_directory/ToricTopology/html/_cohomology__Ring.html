ToricTopology :: cohomologyRing

cohomologyRing -- Compute the cohomology ring of a small cover or quasi-toric manifold

Synopsis

  • Usage:
    cohomologyRing(N)
    cohomologyRing(M)
  • Inputs:
    • N, a small cover,
    • M, a quasi-toric manifold,
  • Outputs:
    • a quotient ring,

Description

Compute the cohomology ring of a small cover (over ZZ/2) or quasi-toric manifold (over ZZ).

Ways to use cohomologyRing :

  • "cohomologyRing(QuasiToricManifold)"
  • "cohomologyRing(SmallCover)"

For the programmer

The object cohomologyRing is a method function with options.
