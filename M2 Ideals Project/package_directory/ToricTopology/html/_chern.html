ToricTopology :: chern

chern -- Compute the Chern classes of a quasi-toric manifold

Synopsis

  • Usage:
    chern(M)
  • Inputs:
    • M, a quasi-toric manifold,
  • Optional inputs:
    • CoefficientRing (missing documentation) => ..., default value ZZ,
  • Outputs:
    • a list,

Description

Compute the Chern classes of a quasi-toric manifold. The output is a list of elements in the cohomology ring of M.

See also

  • cohomologyRing -- Compute the cohomology ring of a small cover or quasi-toric manifold

Ways to use chern :

  • "chern(QuasiToricManifold)"

For the programmer

The object chern is a method function with options.
