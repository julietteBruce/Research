ToricTopology :: bettiQTM

bettiQTM -- Compute the betti numbers of a quasi-toric manifold

Synopsis

  • Usage:
    bettiQTM(k,M)
    bettiQTM(M)
  • Inputs:
    • k, an integer,
    • M, a quasi-toric manifold,
  • Outputs:
    • an integer,
    • a list,

Description

Compute the betti numbers of a quasi-toric manifold. If a dimension k is specified, then only the k-th betti number of M is computed. If no dimension is specified, all the betti numbers between 0 and the dimension of M are computed.

Ways to use bettiQTM :

  • "bettiQTM(QuasiToricManifold)"
  • "bettiQTM(ZZ,QuasiToricManifold)"

For the programmer

The object bettiQTM is a method function.
