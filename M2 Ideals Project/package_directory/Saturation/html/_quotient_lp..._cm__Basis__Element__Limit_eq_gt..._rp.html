Saturation :: quotient(...,BasisElementLimit=>...)

quotient(...,BasisElementLimit=>...)

Description

The value for this optional argument for saturate and quotient is passed through to gb.

Further information

  • Function: quotient -- quotient or division
  • Option key: BasisElementLimit -- an optional argument
