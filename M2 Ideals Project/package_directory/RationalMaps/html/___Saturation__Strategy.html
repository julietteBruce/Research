RationalMaps :: SaturationStrategy

SaturationStrategy -- a strategy for determining whether a map is birational and computing its inverse

Description

SaturationStrategy is a valid value for the Strategy Option for inverseOfMap, isBirationalMap, and isEmbedding. By choosing Strategy=>SaturationStrategy, the equations of the ideal of definition of the Rees algebra are generated by saturating the ideal of definition of the symmetric algebra into a non-zero element. Notice that in this package (and in particular, in this Strategy option) the rings are assumed to be integral domains. This Strategy appears to be slower in some examples.
