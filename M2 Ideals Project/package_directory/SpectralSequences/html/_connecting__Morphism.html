SpectralSequences :: connectingMorphism

connectingMorphism -- use spectral sequences to compute connecting morphisms

Synopsis

  • Usage:
    g = connectingMorphism(f, n)
  • Inputs:
    • f, a chain complex map,
    • n, an integer,
  • Outputs:
    • g, a matrix,

Description

Given a morphism $f: A \rightarrow B$ of chain complexes returns the connecting map $H_{n+1}(coker f) \rightarrow H_n (im f)$.

Ways to use connectingMorphism :

  • connectingMorphism(ChainComplexMap,ZZ) -- use spectral sequences to compute connecting morphisms

For the programmer

The object connectingMorphism is a method function.
