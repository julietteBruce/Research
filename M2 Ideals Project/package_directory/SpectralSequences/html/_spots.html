SpectralSequences :: spots

spots -- which spots does the given page has a module.

Synopsis

  • Usage:
    s = spots P
  • Inputs:
    • P, a Page,
  • Outputs:
    • s, a list,

Description

Returns a list of all the spots where the given page has a module.

Ways to use spots :

  • "spots(Page)"
  • spots(ChainComplex) -- which spots does the given chain complex has a module.
  • spots(FilteredComplex) -- which spots does the given filtered complex has a module.
  • spots(PageMap)

For the programmer

The object spots is a method function.
