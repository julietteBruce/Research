SpectralSequences :: SpectralSequencePage _ List

SpectralSequencePage _ List -- the module in the i,j position on the page

Synopsis

  • Operator: _
  • Usage:
    M = P_L
  • Inputs:
    • P, a spectral sequence page,
    • L, a list, A list L = \{i,j\} \ of integers
  • Outputs:
    • M, a module,

Description

Returns the module in the \{i,j\} \ position in the spectral sequence page. (Using homological or lower indexing conventions.) The relationship $E_{i,j} = E^{-i,-j}$ holds.
