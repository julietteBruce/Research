SpectralSequences :: FilteredComplex ^ ZZ

FilteredComplex ^ ZZ -- the filtered pieces

Synopsis

  • Operator: ^
  • Usage:
    C = K ^ j
  • Inputs:
    • K, a filtered chain complex,
    • j, an integer, an integer, infinity, or -infinity
  • Outputs:
    • C, a chain complex,

Description

Returns the chain complex in (cohomological) filtration degree j. The relationship $K ^ j = K _{(-j)}$ holds.

i1 : A = QQ[x,y];
 
i2 : C = koszul vars A;
 
i3 : K = filteredComplex C
