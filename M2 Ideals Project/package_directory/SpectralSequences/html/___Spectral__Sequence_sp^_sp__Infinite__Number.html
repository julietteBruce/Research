SpectralSequence ^ InfiniteNumber -- the infinity page of a spectral sequence

Synopsis

  • Operator: ^
  • Usage:
    P = E^k
  • Inputs:
    • E, a spectral sequence,
    • k, an infinite number,
  • Outputs:

Description

Returns the infinity page a spectral sequence.
