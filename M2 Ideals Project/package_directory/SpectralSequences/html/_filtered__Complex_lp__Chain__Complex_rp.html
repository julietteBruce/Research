SpectralSequences :: filteredComplex(ChainComplex)

filteredComplex(ChainComplex) -- obtain a filtered complex from a chain complex

Synopsis

  • Function: filteredComplex
  • Usage:
    K = filteredComplex C
  • Inputs:
    • C, a chain complex,
  • Optional inputs:
    • ReducedHomology => a Boolean value, default value true,
    • Shift => an integer, default value 0,
  • Outputs:
    • K, a filtered chain complex,

Description

Produces the filtered complex obtained by successively truncating the complex.
