SpectralSequences :: spectralSequence(FilteredComplex)

spectralSequence(FilteredComplex) -- construct a spectral sequence from a filtered complex

Synopsis

  • Function: spectralSequence
  • Usage:
    E = spectralSequence K
  • Inputs:
    • K, a filtered chain complex, A filtered complex
  • Optional inputs:
    • Prune => ..., default value false,
  • Outputs:
    • E, a spectral sequence,

Description

Returns the spectral sequence associated to the filtered complex.
