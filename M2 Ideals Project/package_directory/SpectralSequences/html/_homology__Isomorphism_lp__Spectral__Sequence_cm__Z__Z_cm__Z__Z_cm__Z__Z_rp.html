SpectralSequences :: homologyIsomorphism(SpectralSequence,ZZ,ZZ,ZZ)

homologyIsomorphism(SpectralSequence,ZZ,ZZ,ZZ) -- the homology isomorphism

Synopsis

  • Function: homologyIsomorphism
  • Usage:
    g = homologyIsomorphism(SpectralSequence, ZZ, ZZ, ZZ)
  • Inputs:
    • E, a spectral sequence,
    • p, an integer,
    • q, an integer,
    • r, an integer,
  • Outputs:
    • g, a matrix,

Description

Computes the isomorphism $ker d^r_{p,q} / image d^r_{p + r, q - r + 1} \rightarrow E^{r+1}_{p,q}$
