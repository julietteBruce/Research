SpectralSequences :: spots(FilteredComplex)

spots(FilteredComplex) -- which spots does the given filtered complex has a module.

Synopsis

  • Function: spots
  • Usage:
    s = spots L
  • Inputs:
    • L, a filtered chain complex,
  • Outputs:
    • s, a list,

Description

Returns a list of all the spots where the given filtered complex has a module.

Ways to use this method:

  • spots(FilteredComplex) -- which spots does the given filtered complex has a module.
