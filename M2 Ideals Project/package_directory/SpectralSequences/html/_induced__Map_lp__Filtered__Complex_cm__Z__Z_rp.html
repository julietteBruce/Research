inducedMap(FilteredComplex,ZZ) -- the i th inclusion map in a filtered complex

Synopsis

  • Function: inducedMap
  • Usage:
    f = inducedMap(K,i)
  • Inputs:
    • K, a filtered chain complex,
    • i, an integer,
  • Optional inputs:
    • Degree => ..., default value null,
    • Verify => ..., default value true,
  • Outputs:
    • f, a chain complex map,

Description

Returns the chain complex map specifying the inclusion of the i piece of the filtered complex to the ambient chain complex.
