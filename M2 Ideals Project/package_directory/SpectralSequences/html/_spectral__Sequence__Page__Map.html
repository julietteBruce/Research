SpectralSequences :: spectralSequencePageMap

spectralSequencePageMap -- compute the maps on a spectral sequence page

Synopsis

  • Usage:
    d = spectralSequencePageMap(FilteredComplex, ZZ)
  • Inputs:
    • K, a filtered chain complex,
    • n, an integer,
  • Optional inputs:
    • Prune => ..., default value false,
  • Outputs:
    • D, a spectral sequence page map,

Description

Returns the differentials of a spectral sequence page.

Ways to use spectralSequencePageMap :

  • "spectralSequencePageMap(FilteredComplex,ZZ)"

For the programmer

The object spectralSequencePageMap is a method function with options.
