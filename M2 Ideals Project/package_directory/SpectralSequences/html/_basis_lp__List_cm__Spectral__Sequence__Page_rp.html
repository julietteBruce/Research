SpectralSequences :: basis(List,SpectralSequencePage)

basis(List,SpectralSequencePage) -- generators of a particular degree

Synopsis

  • Function: basis
  • Usage:
    B = basis(L, E)
  • Inputs:
    • L, a list,
    • E, a spectral sequence page,
  • Optional inputs:
    • Degree => ..., default value null,
    • Limit => ..., default value infinity,
    • SourceRing => ..., default value null,
    • Strategy => ..., default value null,
    • Truncate => ..., default value false,
    • Variables => ..., default value null,
  • Outputs:
    • B, a matrix, Note!! The output should actually be a page!!;

Description

Returns generators for the requested (multi)degree of the spectral sequence page. It is designed to extend the function basis which can be applied to modules, for instance.

As a specific example consider the filtered complex $K$ below, obtained by multiplying the minimal free resolution of the rational quartic space curve by successive powers of the irrelevant ideal.

i1 : B = QQ[a..d];
 
i2 : J = ideal vars B;

o2 : Ideal of B
 
i3 : C = complete res monomialCurveIdeal(B,{1,3,4});
 
i4 : K = filteredComplex(J,C,4);

We compute the degree $0$ piece of the $E^3$ page below.

i5 : E = prune spectralSequence K;
