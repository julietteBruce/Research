SimplicialComplexes :: image(SimplicialMap)

image(SimplicialMap) -- construct the image of a simplicial map

Synopsis

  • Function: image
  • Usage:
    image f
  • Inputs:
    • f, a map of abstract simplicial complexes,
  • Outputs:
    • an abstract simplicial complex, the image of the source of $f$ in the target of $f$

Description

The image of $f$ is the subcomplex of the target of $f$ whose faces are of the form $f(F)$ for some face $F$ in the source of $f$.
